# Spanish (Español) — Overview

## About This Topic

This topic provides a comprehensive, progressive curriculum for learning Spanish from scratch to an advanced level. The 25 lessons are organized into 5 stages that build upon each other, moving from foundational concepts to practical real-world application.

Spanish is spoken by approximately 500 million people worldwide across 21 countries, making it one of the most widely spoken languages on Earth.

## Prerequisites

- No prior Spanish knowledge required (the course begins from zero)
- Basic understanding of grammar concepts in your native language is helpful

## Difficulty

- **Tier**: Beginner (starts from scratch, progresses to advanced)
- **Estimated scope**: A1 through B2/C1 CEFR levels

## Learning Path

```
Stage 1: Foundations (01-05)
  Build the basics: pronunciation, greetings, nouns, essential verbs, present tense
  ↓
Stage 2: Elementary (06-10)
  Expand: adjectives, pronouns, numbers, conversation patterns, daily vocabulary
  ↓
Stage 3: Intermediate Grammar (11-15)
  Master verb system: past tenses, future/conditional, subjunctive, full conjugation
  ↓
Stage 4: Advanced (16-20)
  Refine: complex grammar, discourse markers, regional variations, idioms, vocabulary
  ↓
Stage 5: Practical Application (21-25)
  Apply: reading, writing, culture, business Spanish, exam preparation
```

## How to Use This Course

1. **Follow the order**: Lessons build on each other. Complete each stage before moving to the next.
2. **Do the exercises**: Every lesson includes practice exercises with answers. Work through them.
3. **Review regularly**: Language learning requires repetition. Review previous lessons periodically.
4. **Practice actively**: Don't just read — write out conjugations, say sentences aloud, use new vocabulary.
5. **Reference lessons**: Lessons 15 (conjugation), 18 (regional), and 20 (vocabulary) serve as useful references to return to.
