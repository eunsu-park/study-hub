## 1. Review: Present Subjunctive

Quick reference for the "opposite vowel" rule:

| | -AR → -e | -ER/-IR → -a |
|---|---|---|
| yo | hable | coma / viva |
| tú | hables | comas / vivas |
| él | hable | coma / viva |
| nosotros | hablemos | comamos / vivamos |
| ellos | hablen | coman / vivan |

Key irregulars: **sea, esté, vaya, haya, sepa, dé**

## 2. Imperfect Subjunctive

### Formation

Take the **ellos preterite** form, remove **-ron**, and add endings:

| Ellos preterite | Stem | -ra form (yo) | -se form (yo) |
|----------------|------|---------------|---------------|
| habla**ron** | habla- | habla**ra** | habla**se** |
| comie**ron** | comie- | comie**ra** | comie**se** |
| vivie**ron** | vivie- | vivie**ra** | vivie**se** |
| fue**ron** | fue- | fue**ra** | fue**se** |
| tuvie**ron** | tuvie- | tuvie**ra** | tuvie**se** |
| dije**ron** | dije- | dije**ra** | dije**se** |

### Full Conjugation (-ra form, more common)

| Person | hablar | tener | ir/ser |
|--------|--------|-------|--------|
| yo | hablara | tuviera | fuera |
| tú | hablaras | tuvieras | fueras |
| él/ella | hablara | tuviera | fuera |
| nosotros | habláramos | tuviéramos | fuéramos |
| vosotros | hablarais | tuvierais | fuerais |
| ellos | hablaran | tuvieran | fueran |

> Both -ra and -se forms are correct. The -ra form is more common in speech;
> the -se form is more common in literature.

## 3. Present Perfect Subjunctive

```
HABER (present subjunctive) + past participle
```

| Person | Form |
|--------|------|
| yo | haya hablado |
| tú | hayas comido |
| él/ella | haya vivido |
| nosotros | hayamos hecho |
| ellos | hayan dicho |

**Use**: When the subjunctive trigger is in the present but the action is in the past:
- Espero que **hayas dormido** bien. (I hope you slept well.)
- No creo que **haya llegado**. (I don't think he's arrived.)
- Me alegra que **hayas aprobado**. (I'm glad you passed.)

---

## 4. Pluperfect Subjunctive

```
HABER (imperfect subjunctive) + past participle
```

| Person | Form |
|--------|------|
| yo | hubiera hablado |
| tú | hubieras comido |
| él/ella | hubiera vivido |
| nosotros | hubiéramos hecho |
| ellos | hubieran dicho |

**Use**: Contrary-to-fact past situations:
- Si **hubiera estudiado** más, habría aprobado.
  (If I had studied more, I would have passed.)
- Ojalá **hubiéramos** llegado a tiempo.
  (I wish we had arrived on time.)

---

## 5. Sequence of Tenses

```
+------------------------------------------------------------+
|  MAIN CLAUSE TENSE          →  SUBORDINATE SUBJUNCTIVE     |
+------------------------------------------------------------+
|  Present / Future / Command  →  Present Subjunctive         |
|  Quiero que vengas.            (I want you to come.)        |
|                                                            |
|  Present + past action       →  Present Perfect Subjunctive |
|  Espero que haya llegado.      (I hope he's arrived.)       |
|                                                            |
|  Past / Conditional          →  Imperfect Subjunctive       |
|  Quería que vinieras.          (I wanted you to come.)      |
|                                                            |
|  Past + earlier past         →  Pluperfect Subjunctive      |
|  Dudaba que hubiera venido.    (I doubted he had come.)     |
+------------------------------------------------------------+
```

## 6. Si-Clauses (Conditionals)

### Type 1: Real/Likely (Si + Present Indicative)

- Si **llueve**, no vamos. (If it rains, we won't go.)
- Si **tienes** hambre, come algo. (If you're hungry, eat something.)

### Type 2: Unlikely/Hypothetical (Si + Imperfect Subjunctive)

- Si **tuviera** dinero, **viajaría** por el mundo.
  (If I had money, I would travel the world.)
- Si **fueras** presidente, ¿qué **harías**?
  (If you were president, what would you do?)

### Type 3: Impossible/Past (Si + Pluperfect Subjunctive)

- Si **hubiera** estudiado más, **habría** aprobado.
  (If I had studied more, I would have passed.)
- Si no **hubiéramos** salido tarde, **habríamos** llegado a tiempo.
  (If we hadn't left late, we would have arrived on time.)

```
+----------------------------------------------------------+
|  CONDITIONAL TYPES SUMMARY                                |
+----------------------------------------------------------+
|  Type | Si clause              | Result clause            |
|  ─────┼────────────────────────┼──────────────────────── |
|   1   | present indicative     | present/future/command   |
|   2   | imperfect subjunctive  | conditional              |
|   3   | pluperfect subjunctive | conditional perfect      |
+----------------------------------------------------------+
|  NEVER use present subjunctive after SI!                  |
|  Si llueva... ✗   Si llueve... ✓                         |
+----------------------------------------------------------+
```

## 7. Common Subjunctive Triggers Expanded

### Wishes and Influence

| Trigger | Example |
|---------|---------|
| querer que | Quiero que **vengas**. |
| esperar que | Espero que **estés** bien. |
| desear que | Deseo que **tengas** éxito. |
| pedir que | Te pido que **hables** más despacio. |
| decir que (command) | Le dije que **viniera**. |
| aconsejar que | Te aconsejo que **estudies**. |
| prohibir que | Prohíben que **fumemos** aquí. |

### Emotions

| Trigger | Example |
|---------|---------|
| alegrarse de que | Me alegro de que **hayas** venido. |
| sentir que | Siento que **estés** enfermo. |
| temer que | Temo que no **llegue** a tiempo. |
| sorprender que | Me sorprende que **sepas** eso. |

### Doubt and Denial

| Trigger | Example |
|---------|---------|
| dudar que | Dudo que **sea** verdad. |
| no creer que | No creo que **pueda** venir. |
| negar que | Niega que lo **haya** hecho. |
| no es seguro que | No es seguro que **funcione**. |

## 8. Subjunctive vs. Indicative: Tricky Cases

| Affirmative → Indicative | Negative → Subjunctive |
|-------------------------|----------------------|
| Creo que **tiene** razón. | No creo que **tenga** razón. |
| Es seguro que **viene**. | No es seguro que **venga**. |
| Es verdad que **sabe**. | No es verdad que **sepa**. |

**Aunque** (although/even though):
- Aunque **llueve**, salgo. (Although it's raining [fact], I go out.) — Indicative
- Aunque **llueva**, saldré. (Even if it rains [uncertain], I'll go out.) — Subjunctive

**Tal vez / Quizás** (perhaps):
- Tal vez **viene**. (Perhaps he's coming. — speaker somewhat sure)
- Tal vez **venga**. (Perhaps he'll come. — more uncertain)

## 9. Practice Exercises

### Exercise 1: Imperfect Subjunctive

Conjugate in the imperfect subjunctive (-ra form).

1. Si yo (tener) ________ más tiempo...
2. Quería que tú (venir) ________.
3. Ojalá (ser) ________ más fácil.
4. Nos pidió que (hacer) ________ el trabajo.
5. Si ellos (poder) ________, vendrían.

<details>
<summary>Answers</summary>

1. **tuviera** 2. **vinieras** 3. **fuera** 4. **hiciéramos** 5. **pudieran**

</details>

### Exercise 2: Si-Clauses

Complete with the correct tense.

1. Si (llover) ________, no iremos al parque. [Type 1]
2. Si yo (ser) ________ tú, estudiaría más. [Type 2]
3. Si nosotros (saber) ________ antes, habríamos ayudado. [Type 3]
4. Si (tener, tú) ________ hambre, come algo. [Type 1]

<details>
<summary>Answers</summary>

1. **llueve** (present indicative)
2. **fuera** (imperfect subjunctive)
3. **hubiéramos sabido** (pluperfect subjunctive)
4. **tienes** (present indicative)

</details>

### Exercise 3: Choose the Correct Mood

Indicative or subjunctive?

1. Es cierto que Juan (habla / hable) tres idiomas.
2. No es cierto que María (habla / hable) chino.
3. Aunque (llueve / llueva) mañana, iremos.
4. Creo que (puedes / puedas) hacerlo.
5. No creo que (puedes / puedas) terminarlo hoy.

<details>
<summary>Answers</summary>

1. **habla** (indicative — es cierto = certainty)
2. **hable** (subjunctive — no es cierto = denial)
3. **llueva** (subjunctive — uncertain future: even if)
4. **puedes** (indicative — affirmative creer)
5. **puedas** (subjunctive — negative creer = doubt)

</details>
