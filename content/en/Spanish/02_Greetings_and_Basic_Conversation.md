## 1. Greetings by Time of Day

```
┌─────────────────────────────────────────────────────────────────┐
│                    Greetings by Time                             │
│                                                                 │
│   06:00 ─────── 12:00 ─────── 20:00 ─────── 06:00              │
│   │  Buenos días  │ Buenas tardes │ Buenas noches  │            │
│   │  (Good morning)│(Good afternoon)│(Good evening/  │           │
│   │               │               │  Good night)   │            │
│                                                                 │
│   ※ Buenas noches serves as both greeting AND farewell         │
│   ※ Informal shortcut: ¡Buenas! (works any time)              │
└─────────────────────────────────────────────────────────────────┘
```

| Time | Greeting | Usage |
|------|----------|-------|
| Morning (~6:00-12:00) | ¡Buenos días! | Formal & informal |
| Afternoon (~12:00-20:00) | ¡Buenas tardes! | Formal & informal |
| Evening/Night (~20:00+) | ¡Buenas noches! | Formal & informal |
| Any time | ¡Hola! | Informal |
| Any time | ¡Buenas! | Casual shortcut |

## 2. Informal Greetings

### 2.1 Common Informal Greetings

| Spanish | English | Notes |
|---------|---------|-------|
| ¡Hola! | Hi! | Universal, any time |
| ¿Qué tal? | How's it going? | Very common |
| ¿Cómo estás? | How are you? | Standard informal |
| ¿Qué hay? | What's up? | Casual |
| ¿Qué pasa? | What's happening? | Very casual |
| ¿Cómo te va? | How's it going for you? | Friendly |
| ¿Qué hay de nuevo? | What's new? | Catching up |

### 2.2 Regional Informal Greetings

| Expression | Region | Meaning |
|-----------|--------|---------|
| ¿Qué onda? | Mexico | What's up? |
| ¿Qué hubo? / ¿Quiubo? | Colombia | What's up? |
| ¿Cómo andás? | Argentina (voseo) | How are you? |
| ¿Qué tal, tío/tía? | Spain | What's up, dude? |
| ¿Cómo vas? | General | How are you going? |

## 3. Formal vs Informal: Tú vs Usted

### 3.1 The Two "You"s

```
┌─────────────────────────────────────────────────────────────────┐
│   TÚ (informal "you")              USTED (formal "you")        │
│   ├── Friends & peers               ├── Strangers               │
│   ├── Family members                 ├── Elderly people          │
│   ├── Children                       ├── Bosses / superiors      │
│   ├── Same-age classmates            ├── Officials / police      │
│   └── Informal situations            └── Formal situations       │
│                                                                 │
│   Verb: 2nd person singular          Verb: 3rd person singular  │
│   ¿Cómo estás? (How are you?)       ¿Cómo está? (How are you?) │
│   ¿Tienes tiempo? (Do you have      ¿Tiene tiempo?              │
│    time?)                                                       │
│                                                                 │
│   ※ When in doubt, use USTED — it's always respectful.        │
│   ※ In Latin America, usted is used more broadly.             │
│   ※ In Spain, tú is used more casually.                       │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 Quick Verb Form Comparison

| Context | Tú | Usted |
|---------|-----|-------|
| How are you? | ¿Cómo **estás**? | ¿Cómo **está**? |
| Where are you from? | ¿De dónde **eres**? | ¿De dónde **es**? |
| What's your name? | ¿Cómo **te** llamas? | ¿Cómo **se** llama? |
| Do you speak...? | ¿**Hablas** español? | ¿**Habla** español? |
| Can you help me? | ¿**Puedes** ayudarme? | ¿**Puede** ayudarme? |

## 4. Self-Introduction

### 4.1 Building Blocks

```
Name:      Me llamo [name].        → My name is [name].
           Mi nombre es [name].    → My name is [name].

Origin:    Soy de [country/city].  → I'm from [country/city].
           Soy [nationality].      → I'm [nationality].

Age:       Tengo [number] años.    → I'm [number] years old.

Work:      Trabajo como [job].     → I work as [job].
           Trabajo en [company].   → I work at [company].

Study:     Estudio [subject].      → I study [subject].
           Soy estudiante.         → I'm a student.

Language:  Hablo [language].       → I speak [language].
           Estoy aprendiendo español. → I'm learning Spanish.
```

### 4.2 Common Professions

| Spanish | English |
|---------|---------|
| profesor/a | teacher |
| ingeniero/a | engineer |
| médico/a | doctor |
| abogado/a | lawyer |
| estudiante | student |
| programador/a | programmer |
| diseñador/a | designer |
| empresario/a | entrepreneur |

### 4.3 Sample Introduction Dialogues

```
Dialogue 1 (Informal):
A: ¡Hola! ¿Cómo te llamas?
B: Me llamo Min-jun. ¿Y tú?
A: Soy María. ¿De dónde eres?
B: Soy de Corea del Sur. Soy estudiante.
A: ¡Qué bien! Mucho gusto, Min-jun.
B: Igualmente, María.

Dialogue 2 (Formal):
A: Buenos días. ¿Cómo se llama usted?
B: Me llamo Park Min-jun. Mucho gusto.
A: Encantada. Soy la doctora García. ¿De dónde es?
B: Soy de Seúl, Corea del Sur.
A: ¿Y qué hace aquí en Madrid?
B: Estudio español en la universidad.
```

## 5. Responding to "How Are You?"

### 5.1 Response Spectrum

```
Very positive:
  ¡Estupendo!         → Wonderful!
  ¡De maravilla!      → Marvelous!
  ¡Genial!            → Great!
  ¡Muy bien!          → Very well!

Positive:
  Bien, gracias.      → Fine, thanks.
  Todo bien.          → All good.
  No me quejo.        → Can't complain.

Neutral:
  Así así.            → So-so.
  Regular.            → Okay.
  Tirando.            → Getting by. (Spain, colloquial)
  Ahí vamos.          → Hanging in there.

Negative:
  No muy bien.        → Not very well.
  Un poco cansado/a.  → A bit tired.
  Fatal.              → Terrible.
```

### 5.2 The Complete Exchange

```
A: ¡Hola! ¿Qué tal?        → Hi! How's it going?
B: Bien, gracias. ¿Y tú?   → Fine, thanks. And you?
A: Muy bien, también.       → Very well, too.

A: ¿Cómo está usted?       → How are you? (formal)
B: Bien, gracias. ¿Y usted? → Fine, thanks. And you?
```

## 7. Polite Expressions

### 7.1 Thanking

| Expression | Meaning | Formality |
|-----------|---------|-----------|
| Gracias. | Thank you. | Universal |
| Muchas gracias. | Thank you very much. | Universal |
| Mil gracias. | A thousand thanks. | Warm |
| Te/Le agradezco. | I appreciate it. | Formal |
| Muy amable. | Very kind of you. | Formal |

**Responses to thanks:**
- De nada. — You're welcome.
- No hay de qué. — Don't mention it.
- Es un placer. — It's a pleasure.

### 7.2 Apologizing

```
┌─────────────────────────────────────────────────────────────────┐
│   Perdón / Perdona (tú) / Perdone (Ud.)                        │
│   → Sorry / Excuse me (minor things, getting attention)         │
│                                                                 │
│   Lo siento (mucho).                                            │
│   → I'm (very) sorry. (genuine apology, expressing sympathy)   │
│                                                                 │
│   Disculpa (tú) / Disculpe (Ud.)                                │
│   → Excuse me. (interrupting, passing through)                  │
│                                                                 │
│   Con permiso.                                                   │
│   → Excuse me. (asking to pass, leave a room)                   │
│                                                                 │
│   Fue mi culpa. → It was my fault.                              │
│   No fue mi intención. → It wasn't my intention.                │
└─────────────────────────────────────────────────────────────────┘
```

**Responses to apologies:**
- No pasa nada. — It's okay. / No big deal.
- No te preocupes. — Don't worry.
- Está bien. — It's all right.

### 7.3 Making Requests

| Politeness Level | Spanish | English |
|-----------------|---------|---------|
| Direct | ¿Puedes ayudarme? | Can you help me? |
| Polite | ¿Me podrías ayudar? | Could you help me? |
| Very polite | ¿Sería posible...? | Would it be possible...? |
| Formal | Me gustaría... | I would like... |
| Very formal | Quisiera... | I would like... (politer) |

## 8. Basic Conversation Patterns

### 8.1 At a Café

```
Cliente: ¡Buenos días!
Camarero: ¡Buenos días! ¿Qué desea?
Cliente: Un café con leche, por favor.
Camarero: ¿Algo más?
Cliente: No, eso es todo. Gracias.
Camarero: Son dos euros cincuenta.
Cliente: Aquí tiene. Gracias.
Camarero: Gracias a usted.
```

### 8.2 Asking for Directions

```
A: Disculpe, ¿dónde está la estación de metro?
B: Siga todo recto y gire a la derecha en la segunda calle.
A: ¿Está lejos?
B: No, está a cinco minutos a pie.
A: Muchas gracias.
B: De nada.
```

### 8.3 Meeting Someone New

```
A: ¡Hola! ¿Eres nuevo aquí?
B: Sí, me llamo Carlos. ¡Mucho gusto!
A: Encantada, Carlos. Soy Lucía. ¿De dónde eres?
B: Soy de México. ¿Y tú?
A: De aquí, de Madrid. ¡Bienvenido!
B: ¡Gracias! Todos son muy amables.
```

## 9. Useful First-Day Phrases

### 9.1 Survival Phrases

```
┌─────────────────────────────────────────────────────────────────┐
│   Essential Phrases for Day One                                  │
│                                                                 │
│   Por favor.                    → Please.                        │
│   Gracias.                      → Thank you.                     │
│   Sí. / No.                    → Yes. / No.                     │
│   No entiendo.                  → I don't understand.            │
│   ¿Puede repetir, por favor?   → Can you repeat, please?        │
│   ¿Cómo se dice ___ en español? → How do you say ___ in Spanish?│
│   ¿Qué significa ___?          → What does ___ mean?            │
│   Más despacio, por favor.      → More slowly, please.          │
│   No hablo español (muy bien).  → I don't speak Spanish (well). │
│   Hablo un poco de español.     → I speak a little Spanish.     │
│   ¿Habla inglés?               → Do you speak English?          │
│   ¿Dónde está el baño?         → Where is the bathroom?         │
│   ¿Cuánto cuesta?              → How much does it cost?         │
│   La cuenta, por favor.         → The check, please.            │
│   ¡Ayuda!                       → Help!                         │
└─────────────────────────────────────────────────────────────────┘
```

### 9.2 At the Train Station

```
Viajero: Buenos días. Un billete a Barcelona, por favor.
Taquillero: ¿De ida o de ida y vuelta?
Viajero: De ida y vuelta, por favor.
Taquillero: ¿Para cuándo?
Viajero: Para hoy. ¿A qué hora sale el próximo tren?
Taquillero: A las 14:30. Andén número 5.
Viajero: Perfecto. ¿Cuánto cuesta?
Taquillero: Sesenta y cinco euros.
Viajero: Aquí tiene. Gracias.
Taquillero: Buen viaje.
```
