- Use the present tense for habitual actions, current states, and near future

## 1. Spanish Verb Basics

### 1.1 The Infinitive Form

Every Spanish verb has an infinitive form ending in **-ar**, **-er**, or **-ir**:

```
┌─────────────────────────────────────────────────────────────────┐
│   Group 1: -AR verbs (~75% of all verbs — the largest group)   │
│   hablar (to speak), estudiar (to study), trabajar (to work)   │
│                                                                 │
│   Group 2: -ER verbs (~15%)                                     │
│   comer (to eat), beber (to drink), leer (to read)             │
│                                                                 │
│   Group 3: -IR verbs (~10%)                                     │
│   vivir (to live), escribir (to write), abrir (to open)        │
│                                                                 │
│   To conjugate: remove the ending (-ar/-er/-ir) to get the     │
│   stem, then add the appropriate person/number ending.          │
│                                                                 │
│   hablar → habl- + -o = hablo (I speak)                        │
│   comer  → com-  + -o = como (I eat)                           │
│   vivir  → viv-  + -o = vivo (I live)                          │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 Subject Pronouns

| Singular | Plural |
|----------|--------|
| yo (I) | nosotros/as (we) |
| tú (you, informal) | vosotros/as (you all, Spain) |
| él (he) | ellos (they, masc.) |
| ella (she) | ellas (they, fem.) |
| usted (you, formal) | ustedes (you all, formal / LatAm) |

> **Pro-drop**: Spanish often omits subject pronouns because verb endings already indicate the subject. *Hablo español* = "I speak Spanish" (no need for *yo*). Use pronouns for emphasis or clarity.

## 2. -AR Verb Conjugation

### 2.1 Pattern: hablar (to speak)

| Subject | Stem + Ending | Full Form | Meaning |
|---------|--------------|-----------|---------|
| yo | habl + **-o** | habl**o** | I speak |
| tú | habl + **-as** | habl**as** | you speak |
| él/ella/Ud. | habl + **-a** | habl**a** | he/she speaks |
| nosotros/as | habl + **-amos** | habl**amos** | we speak |
| vosotros/as | habl + **-áis** | habl**áis** | you all speak |
| ellos/ellas/Uds. | habl + **-an** | habl**an** | they speak |

### 2.2 Example: estudiar (to study)

```
Estudio español todos los días.   → I study Spanish every day.
¿Estudias en la universidad?      → Do you study at the university?
Ella estudia medicina.            → She studies medicine.
Estudiamos juntos.                → We study together.
Estudian mucho para el examen.    → They study a lot for the exam.
```

### 2.3 Ten Essential -AR Verbs

| Verb | Meaning | Example |
|------|---------|---------|
| hablar | to speak | Hablo dos idiomas. (I speak two languages.) |
| estudiar | to study | Estudio español. (I study Spanish.) |
| trabajar | to work | Trabajo en una oficina. (I work in an office.) |
| comprar | to buy | Compro comida. (I buy food.) |
| llamar | to call | Llamo a mi madre. (I call my mother.) |
| mirar | to look at | Miro la televisión. (I watch television.) |
| escuchar | to listen | Escucho música. (I listen to music.) |
| caminar | to walk | Camino al parque. (I walk to the park.) |
| cocinar | to cook | Cocino la cena. (I cook dinner.) |
| necesitar | to need | Necesito ayuda. (I need help.) |

## 3. -ER Verb Conjugation

### 3.1 Pattern: comer (to eat)

| Subject | Stem + Ending | Full Form | Meaning |
|---------|--------------|-----------|---------|
| yo | com + **-o** | com**o** | I eat |
| tú | com + **-es** | com**es** | you eat |
| él/ella/Ud. | com + **-e** | com**e** | he/she eats |
| nosotros/as | com + **-emos** | com**emos** | we eat |
| vosotros/as | com + **-éis** | com**éis** | you all eat |
| ellos/ellas/Uds. | com + **-en** | com**en** | they eat |

### 3.2 Example: beber (to drink)

```
Bebo café por la mañana.          → I drink coffee in the morning.
¿Bebes agua?                       → Do you drink water?
Él bebe jugo de naranja.          → He drinks orange juice.
Bebemos vino con la cena.         → We drink wine with dinner.
No beben alcohol.                  → They don't drink alcohol.
```

### 3.3 Ten Essential -ER Verbs

| Verb | Meaning | Example |
|------|---------|---------|
| comer | to eat | Como a las dos. (I eat at two.) |
| beber | to drink | Bebo mucha agua. (I drink a lot of water.) |
| leer | to read | Leo un libro. (I read a book.) |
| aprender | to learn | Aprendo español. (I learn Spanish.) |
| comprender | to understand | No comprendo. (I don't understand.) |
| correr | to run | Corro por la mañana. (I run in the morning.) |
| creer | to believe | Creo que sí. (I think/believe so.) |
| vender | to sell | Vende coches. (He sells cars.) |
| responder | to answer | Respondo al correo. (I answer the email.) |
| deber | must, should | Debo estudiar. (I must study.) |

## 4. -IR Verb Conjugation

### 4.1 Pattern: vivir (to live)

| Subject | Stem + Ending | Full Form | Meaning |
|---------|--------------|-----------|---------|
| yo | viv + **-o** | viv**o** | I live |
| tú | viv + **-es** | viv**es** | you live |
| él/ella/Ud. | viv + **-e** | viv**e** | he/she lives |
| nosotros/as | viv + **-imos** | viv**imos** | we live |
| vosotros/as | viv + **-ís** | viv**ís** | you all live |
| ellos/ellas/Uds. | viv + **-en** | viv**en** | they live |

> **Note**: -ER and -IR verbs share the same endings except for nosotros (-emos vs -imos) and vosotros (-éis vs -ís).

### 4.2 Example: escribir (to write)

```
Escribo en mi diario.             → I write in my diary.
¿Escribes correos electrónicos?   → Do you write emails?
Ella escribe poemas.              → She writes poems.
Escribimos cartas.                → We write letters.
Escriben mensajes.                → They write messages.
```

### 4.3 Ten Essential -IR Verbs

| Verb | Meaning | Example |
|------|---------|---------|
| vivir | to live | Vivo en Seúl. (I live in Seoul.) |
| escribir | to write | Escribo un correo. (I write an email.) |
| abrir | to open | Abro la puerta. (I open the door.) |
| recibir | to receive | Recibo un regalo. (I receive a gift.) |
| decidir | to decide | Decido ahora. (I decide now.) |
| compartir | to share | Comparto mi comida. (I share my food.) |
| asistir | to attend | Asisto a clase. (I attend class.) |
| subir | to go up | Subo las escaleras. (I go up the stairs.) |
| descubrir | to discover | Descubro algo nuevo. (I discover something new.) |
| existir | to exist | Existe una solución. (A solution exists.) |

## 5. Building Basic Sentences

### 5.1 Word Order: Subject-Verb-Object (SVO)

```
┌─────────────────────────────────────────────────────────────────┐
│   Basic pattern: Subject + Verb + Object/Complement             │
│                                                                 │
│   (Yo) estudio español.        → I study Spanish.               │
│   María come una manzana.      → María eats an apple.           │
│   (Nosotros) vivimos en Madrid. → We live in Madrid.            │
│                                                                 │
│   ※ Subject pronouns are usually omitted (pro-drop):           │
│   Hablo español. = I speak Spanish. (yo is implied)             │
│   Estudia medicina. = He/She studies medicine.                  │
│                                                                 │
│   ※ Spanish word order is more flexible than English:           │
│   Leo un libro. = Un libro leo. (emphasis on "a book")          │
│   Both are grammatically correct.                               │
└─────────────────────────────────────────────────────────────────┘
```

### 5.2 Adding Details

```
Time expressions:
  Estudio español por la mañana.         → I study Spanish in the morning.
  Trabajo todos los días.                → I work every day.
  Siempre como a las dos.               → I always eat at two.

Location:
  Estudio en la biblioteca.              → I study in the library.
  Trabajo en una oficina.                → I work in an office.

Manner:
  Habla español muy bien.               → He/She speaks Spanish very well.
  Caminamos lentamente.                 → We walk slowly.

Frequency:
  siempre (always), a menudo (often), a veces (sometimes),
  casi nunca (almost never), nunca (never)
```

## 6. Question Formation

### 6.1 Question Words (Interrogatives)

| Spanish | English | Example |
|---------|---------|---------|
| ¿Qué? | What? | ¿Qué estudias? (What do you study?) |
| ¿Quién? / ¿Quiénes? | Who? | ¿Quién habla? (Who speaks?) |
| ¿Cuándo? | When? | ¿Cuándo comes? (When do you eat?) |
| ¿Dónde? | Where? | ¿Dónde vives? (Where do you live?) |
| ¿Adónde? | Where to? | ¿Adónde vas? (Where are you going?) |
| ¿Por qué? | Why? | ¿Por qué estudias? (Why do you study?) |
| ¿Cómo? | How? | ¿Cómo estás? (How are you?) |
| ¿Cuánto/a? | How much? | ¿Cuánto cuesta? (How much does it cost?) |
| ¿Cuántos/as? | How many? | ¿Cuántos años tienes? (How old are you?) |
| ¿Cuál? / ¿Cuáles? | Which? | ¿Cuál prefieres? (Which do you prefer?) |

> **All question words carry accent marks.** Without the accent, they become relative pronouns: *que* (that), *donde* (where), etc.

### 6.2 Three Ways to Form Yes/No Questions

```
Method 1: Intonation only (raise pitch at end)
  ¿Hablas español?           → Do you speak Spanish?
  (Same word order as statement, just with rising intonation)

Method 2: Invert subject and verb
  ¿Habla usted español?      → Do you speak Spanish?
  ¿Estudia María medicina?   → Does María study medicine?

Method 3: Add a tag question
  Hablas español, ¿no?       → You speak Spanish, right?
  Hablas español, ¿verdad?   → You speak Spanish, right?
```

> **Note**: Spanish uses inverted question marks (¿) at the start. Always include both ¿ and ? in writing.

## 7. Negation

### 7.1 Basic Negation: Place *no* Before the Verb

```
Hablo español.       → I speak Spanish.
No hablo español.    → I don't speak Spanish.

Tiene tiempo.        → He/She has time.
No tiene tiempo.     → He/She doesn't have time.

Es mi libro.         → It's my book.
No es mi libro.      → It's not my book.
```

### 7.2 Negative Words

| Negative | Affirmative opposite | Meaning |
|----------|---------------------|---------|
| no | sí | no / not |
| nada | algo | nothing / anything |
| nadie | alguien | nobody / anybody |
| nunca / jamás | siempre | never / always |
| ninguno/a | alguno/a | none / some |
| tampoco | también | neither / also |
| ni...ni | o...o | neither...nor / either...or |

### 7.3 Double Negatives (Correct in Spanish!)

```
┌─────────────────────────────────────────────────────────────────┐
│   Unlike English, double negatives are REQUIRED in Spanish.     │
│                                                                 │
│   No tengo nada.       → I don't have anything. (lit: not nothing)│
│   No veo a nadie.      → I don't see anyone.                   │
│   No como nunca carne. → I never eat meat.                      │
│   No quiero ni café ni té. → I want neither coffee nor tea.    │
│                                                                 │
│   Alternative (negative word before verb, no "no"):             │
│   Nunca como carne.     → I never eat meat.                     │
│   Nadie habla.          → Nobody speaks.                        │
│   Nada funciona.        → Nothing works.                        │
│                                                                 │
│   Both patterns are correct:                                    │
│   No viene nunca. = Nunca viene. (He/She never comes.)         │
└─────────────────────────────────────────────────────────────────┘
```

## 8. Present Tense Uses

The Spanish present tense covers several English tenses:

```
1. Habitual actions:
   Estudio español todos los días.
   → I study Spanish every day.

2. Current states:
   Vivo en Madrid.
   → I live in Madrid.

3. General truths:
   La Tierra gira alrededor del Sol.
   → The Earth revolves around the Sun.

4. Near future (with time expression):
   Mañana trabajo temprano.
   → Tomorrow I work early. (= I will work / I'm working tomorrow)

5. Ongoing action (informal, instead of progressive):
   ¿Qué lees? (= ¿Qué estás leyendo?)
   → What are you reading?
```

## 9. Summary: Three Groups Compared

```
┌────────┬──────────┬──────────┬──────────┐
│ Person │ -AR      │ -ER      │ -IR      │
│        │ (hablar) │ (comer)  │ (vivir)  │
├────────┼──────────┼──────────┼──────────┤
│ yo     │ habl-o   │ com-o    │ viv-o    │
│ tú     │ habl-as  │ com-es   │ viv-es   │
│ él     │ habl-a   │ com-e    │ viv-e    │
│ nos.   │ habl-amos│ com-emos │ viv-imos │
│ vos.   │ habl-áis │ com-éis  │ viv-ís   │
│ ellos  │ habl-an  │ com-en   │ viv-en   │
├────────┴──────────┴──────────┴──────────┤
│ Pattern:                                 │
│ • "yo" always ends in -o                │
│ • -AR: a-themed vowels (a, as, amos)    │
│ • -ER/-IR: e-themed vowels (es, e, en)  │
│ • -ER vs -IR differ only in nos. & vos. │
└──────────────────────────────────────────┘
```
