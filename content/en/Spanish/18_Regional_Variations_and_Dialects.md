## 2. Spain: Peninsular Spanish

### Vosotros Conjugation (Spain Only)

Latin America uses **ustedes** for all plural "you." Spain distinguishes:
- **vosotros/as** → informal plural
- **ustedes** → formal plural

| Tense | -AR (hablar) | -ER (comer) | -IR (vivir) |
|-------|-------------|-------------|-------------|
| Present | habl**áis** | com**éis** | viv**ís** |
| Preterite | habl**asteis** | com**isteis** | viv**isteis** |
| Imperfect | habl**abais** | com**íais** | viv**íais** |
| Future | habl**aréis** | com**eréis** | viv**iréis** |
| Conditional | habl**aríais** | com**eríais** | viv**iríais** |
| Pres. Subjunctive | habl**éis** | com**áis** | viv**áis** |
| Imperative | habl**ad** | com**ed** | viv**id** |

Associated forms:
- Object pronoun: **os** — Os quiero. (I love you all.)
- Possessive: **vuestro/a/os/as** — Vuestra casa es bonita.

### Pronunciation Features

```
+----------------------------------------------------------+
|          PENINSULAR SPANISH PRONUNCIATION                 |
+----------------------------------------------------------+
|                                                          |
|  1. DISTINCIÓN (c/z vs s):                               |
|     cena → [θéna]  (not [séna])                         |
|     zapato → [θapáto]                                    |
|     cerveza → [θerβéθa]                                  |
|                                                          |
|  2. Strong /x/ (j, ge, gi):                              |
|     jota → [xóta]  (harsh, from back of throat)         |
|     gente → [xénte]                                      |
|                                                          |
|  3. Final -d:                                            |
|     Madrid → [maðríð] or [maðrí] (weakened/dropped)     |
|     ciudad → [θjuðáð]                                    |
|                                                          |
|  4. ll pronunciation:                                    |
|     Many regions: [ʎ] (lateral palatal)                  |
|     Yeísmo zones: [j] (same as y)                        |
+----------------------------------------------------------+
```

## 3. Mexico

### Characteristics

```
+----------------------------------------------------------+
|               MEXICAN SPANISH FEATURES                    |
+----------------------------------------------------------+
|                                                          |
|  Pronunciation:                                          |
|  • Seseo: c/z pronounced as [s] (like all of L.A.)      |
|  • Final -s retained clearly                             |
|  • Consonant clusters preserved: extra [éks.tra]         |
|                                                          |
|  Grammar:                                                |
|  • Ustedes used (no vosotros)                            |
|  • Preterite preferred: Hoy comí. (not He comido hoy.)  |
|  • Diminutives (-ito/-ita) used very frequently          |
|                                                          |
|  Influence:                                              |
|  • Náhuatl (Aztec) vocabulary: chocolate, tomate,        |
|    aguacate, chile, guajolote                            |
|  • US English influence (northern border regions)        |
+----------------------------------------------------------+
```

### Mexican Expressions

| Expression | Meaning |
|------------|---------|
| ¿Qué onda? | What's up? / How's it going? |
| ¡Órale! | Cool! / Really? / Let's go! |
| ¡Híjole! | Wow! (surprise) |
| No manches | No way! / You're kidding! |
| ¡Ándale! | Come on! / Hurry up! |
| Padre / Padrísimo | Awesome, cool |
| Chido/Chida | Cool, nice |
| Güey | Dude, buddy (very informal) |
| Neta | Really, for real |
| Fresa | Snobby, preppy person |

### Diminutive Culture

Mexico uses diminutives more than most other Spanish-speaking countries:

| Diminutive | From | Meaning |
|-----------|------|---------|
| ahorita | ahora | right now / in a moment |
| tantito | tanto | just a little bit |
| momentito | momento | just a moment |
| lueguito | luego | right away |
| cerquita | cerca | very close |
| poquito | poco | a tiny bit |

> **Ahorita** can mean "right now," "in a moment," or even "later" depending
> on context and tone — a famously flexible word.

## 4. Argentina and Uruguay: Rioplatense Spanish

### The Voseo System

Argentina and Uruguay use **vos** instead of **tú**:

| Verb | tú form | vos form |
|------|---------|----------|
| hablar | tú **hablas** | vos **hablás** |
| comer | tú **comes** | vos **comés** |
| vivir | tú **vives** | vos **vivís** |
| ser | tú **eres** | vos **sos** |
| tener | tú **tienes** | vos **tenés** |
| poder | tú **puedes** | vos **podés** |
| querer | tú **quieres** | vos **querés** |
| pensar | tú **piensas** | vos **pensás** |
| venir | tú **vienes** | vos **venís** |

```
+----------------------------------------------------------+
|              VOSEO RULES                                  |
+----------------------------------------------------------+
|                                                          |
|  1. Stress falls on the FINAL syllable                   |
|  2. NO stem changes (pensás, not piensás)                |
|  3. Imperative: hablá, comé, viví (stress on last vowel) |
|                                                          |
|  Examples:                                               |
|  ¿Vos tenés tiempo? (Do you have time?)                  |
|  ¿De dónde sos? (Where are you from?)                    |
|  Vení acá. (Come here.)                                  |
|  Decime la verdad. (Tell me the truth.)                  |
+----------------------------------------------------------+
```

> **Where is vos used?** Primarily Argentina and Uruguay. Also parts of
> Central America (Guatemala, Honduras, El Salvador, Costa Rica, Nicaragua),
> and the Paisa region of Colombia (Medellín).

### Pronunciation: Sheísmo

The most distinctive feature of Argentine/Uruguayan Spanish:

| Letter | General Spanish | Rioplatense |
|--------|----------------|-------------|
| **ll** | [j] (English "y") | [ʃ] (English "sh") or [ʒ] ("zh") |
| **y** | [j] | [ʃ] or [ʒ] |

- pollo: general [pójo] → Argentine [póʃo]
- yo: general [jo] → Argentine [ʃo]
- llamar: general [jamár] → Argentine [ʃamár]

Argentine intonation also has a distinctive melodic quality influenced by Italian immigration.

### Argentine Expressions (Lunfardo)

| Expression | Meaning |
|-----------|---------|
| Che | Hey, buddy (form of address) |
| Boludo/a | Friend / idiot (context-dependent) |
| ¡Qué quilombo! | What a mess! |
| Bárbaro | Great, awesome |
| Copado/a | Cool, nice |
| Morfar | To eat |
| Laburo | Work, job |
| Mango | Money, peso |
| Bondi | Bus |
| Pibe/Piba | Young person, kid |
| Re- | Very (re lindo = very pretty) |

> **Lunfardo** is Buenos Aires slang, heavily influenced by Italian immigrants.
> It's commonly found in tango lyrics.

## 5. Caribbean Spanish

### Cuba, Dominican Republic, Puerto Rico

```
+----------------------------------------------------------+
|           CARIBBEAN PRONUNCIATION FEATURES                |
+----------------------------------------------------------+
|                                                          |
|  1. Final/syllable-final -s → [h] or DROPPED            |
|     estos → [éhtoh] or [éto]                             |
|     más → [máh]                                          |
|                                                          |
|  2. Final -r → -l or DROPPED                             |
|     comer → [komél]                                      |
|     puerta → [pwélta]                                    |
|                                                          |
|  3. Rapid speech with syllable shortening                |
|                                                          |
|  4. African-influenced vocabulary and rhythm              |
+----------------------------------------------------------+
```

### Caribbean Expressions

| Expression | Region | Meaning |
|------------|--------|---------|
| ¿Qué lo qué? | Cuba/DR | What's up? |
| ¡Wepa! | Puerto Rico | Wow! / Cool! |
| Chévere | General Caribbean | Cool, nice |
| Asere | Cuba | Friend, buddy |
| Vaina | DR/Venezuela | Thing (generic) |

## 7. Chile

### Pronunciation

- Final -s weakening: [h] or dropped (like Caribbean)
- -ado → -ao: cansado → cansao
- Unique ch pronunciation: closer to [ʃ] than [tʃ]

### Chilean Expressions

| Expression | Meaning |
|-----------|---------|
| ¿Cachai? | You know? (Do you get it?) |
| Po (from pues) | Sentence-final emphasis: Sí po, No po |
| Fome | Boring |
| Pololo/Polola | Boyfriend/girlfriend |
| Guagua | Baby |
| Luca | 1,000 pesos |
| Bacán | Cool |
| Al tiro | Right away |
| Poh | (emphatic particle) |

## 8. Pronunciation Variation Map

### Three Key Phenomena

| Phenomenon | Description | Where |
|-----------|-------------|-------|
| **Distinción** | c/z = [θ], s = [s] | Most of Spain |
| **Seseo** | c/z/s all = [s] | All of Latin America, southern Spain |
| **Ceceo** | c/z/s all = [θ] | Parts of Andalusia |
| **Yeísmo** | ll = y = [j] | Most of Spanish-speaking world |
| **Sheísmo** | ll = y = [ʃ] or [ʒ] | Argentina, Uruguay |
| **S-aspiration** | Final -s → [h] | Caribbean, Chile, Andalusia, parts of L.A. |

### Pronunciation of Key Words by Region

| Word | Spain (Madrid) | Mexico | Argentina | Caribbean |
|------|---------------|--------|-----------|-----------|
| zapato | [θapáto] | [sapáto] | [sapáto] | [sapáto] |
| cena | [θéna] | [séna] | [séna] | [séna] |
| pollo | [pójo] | [pójo] | [póʃo] | [pójo] |
| estos | [éstos] | [éstos] | [éstos] | [éhtoh] |

## 10. Practice Exercises

### Exercise 1: Regional Identification

Which region does each sentence represent?

1. "¿Vos tenés tiempo para tomar un mate?"
2. "¿Qué onda, güey? ¿Todo bien?"
3. "Vosotros sois muy majos."
4. "¿Cachai lo que te digo?"
5. "A la orden, con mucho gusto."

<details>
<summary>Answers</summary>

1. **Argentina/Uruguay** (vos usage, mate culture)
2. **Mexico** (güey, qué onda)
3. **Spain** (vosotros, majos)
4. **Chile** (cachai)
5. **Colombia** (a la orden)

</details>

### Exercise 2: Vocabulary Conversion

Convert Spain Spanish to Latin American equivalents.

1. ordenador → ________
2. móvil → ________
3. coche → ________
4. piso → ________
5. patatas → ________
6. zumo → ________

<details>
<summary>Answers</summary>

1. **computadora** 2. **celular** 3. **carro / auto** 4. **departamento / apartamento**
5. **papas** 6. **jugo**

</details>

### Exercise 3: Vosotros vs. Ustedes vs. Vos

Choose the correct form for each context.

1. Spain (informal, plural): ¿________ (tener) hambre, chicos?
2. Latin America (plural): ¿________ (tener) hambre, chicos?
3. Argentina (singular): ¿________ (tener) tiempo?

<details>
<summary>Answers</summary>

1. ¿**Tenéis** hambre? (vosotros form)
2. ¿**Tienen** hambre? (ustedes form)
3. ¿**Tenés** tiempo? (vos form)

</details>

### Exercise 4: Pronunciation Notation

Write the pronunciation of these words in both Spain and Latin America.

1. zapato — Spain: ________ / Latin America: ________
2. cena — Spain: ________ / Latin America: ________
3. pollo — General: ________ / Argentina: ________

<details>
<summary>Answers</summary>

1. Spain: [θapáto] / Latin America: [sapáto]
2. Spain: [θéna] / Latin America: [séna]
3. General: [pójo] / Argentina: [póʃo]

</details>
