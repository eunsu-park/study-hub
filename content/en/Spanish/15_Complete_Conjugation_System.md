## 1. Verb Classification Overview

```
+----------------------------------------------------------+
|           SPANISH VERB CLASSIFICATION                     |
+----------------------------------------------------------+
|                                                          |
|  By ending:   -AR (most common)                          |
|               -ER                                        |
|               -IR                                        |
|                                                          |
|  By regularity:  Regular                                 |
|                   Stem-changing (boot verbs)              |
|                   Irregular (unique patterns)             |
|                   Reflexive (-se ending)                  |
|                                                          |
|  By mood:  Indicative (facts)                            |
|            Subjunctive (subjectivity)                     |
|            Imperative (commands)                          |
|                                                          |
+----------------------------------------------------------+
```

## 2. Stem-Changing Verbs: E→IE

The stem vowel **e** changes to **ie** in all forms EXCEPT nosotros and vosotros.

### PENSAR (to think) — Present Indicative

| Person | Conjugation |
|--------|-------------|
| yo | p**ie**nso |
| tú | p**ie**nsas |
| él/ella | p**ie**nsa |
| nosotros | p**e**nsamos |
| vosotros | p**e**nsáis |
| ellos | p**ie**nsan |

### Common E→IE Verbs

| -AR | -ER | -IR |
|-----|-----|-----|
| cerrar (to close) | entender (to understand) | preferir (to prefer) |
| empezar (to begin) | perder (to lose) | sentir (to feel) |
| pensar (to think) | querer (to want) | mentir (to lie) |
| despertar (to wake) | tener (to have)* | divertirse (to have fun) |
| sentar (to sit) | | venir (to come)* |

> *tener and venir have additional irregularities (tengo, vengo).

## 3. Stem-Changing Verbs: O→UE

### PODER (to be able) — Present Indicative

| Person | Conjugation |
|--------|-------------|
| yo | p**ue**do |
| tú | p**ue**des |
| él/ella | p**ue**de |
| nosotros | p**o**demos |
| vosotros | p**o**déis |
| ellos | p**ue**den |

### Common O→UE Verbs

| -AR | -ER | -IR |
|-----|-----|-----|
| contar (to count/tell) | poder (can) | dormir (to sleep) |
| encontrar (to find) | volver (to return) | morir (to die) |
| recordar (to remember) | mover (to move) | |
| costar (to cost) | llover (to rain) | |
| almorzar (to lunch) | | |

> Special: **jugar** (to play) changes u→ue: juego, juegas, juega...

## 4. Stem-Changing Verbs: E→I

Only -IR verbs. The change happens in present AND preterite (3rd person).

### PEDIR (to ask for) — Present Indicative

| Person | Conjugation |
|--------|-------------|
| yo | p**i**do |
| tú | p**i**des |
| él/ella | p**i**de |
| nosotros | p**e**dimos |
| vosotros | p**e**dís |
| ellos | p**i**den |

### Common E→I Verbs

| Verb | Meaning |
|------|---------|
| pedir | to ask for |
| servir | to serve |
| repetir | to repeat |
| seguir | to follow |
| vestirse | to get dressed |
| reír | to laugh |
| sonreír | to smile |
| elegir | to choose |

## 5. The Boot Pattern

Stem changes follow a "boot" shape in conjugation tables — changes occur in all
forms **except** nosotros and vosotros:

```
+------------------+
|  yo    ✓  |  nosotros ✗  |
|  tú    ✓  |  vosotros ✗  |
|  él    ✓  |  ellos    ✓  |
+------------------+
     \_________/
      The "boot"
```

> In the **preterite**, E→I and O→UE -IR verbs change in 3rd person only:
> - dormir → durmió, durmieron
> - pedir → pidió, pidieron

## 6. Reflexive Verb Review

### Reflexive Pronoun Placement

| With conjugated verb | With infinitive/gerund |
|---------------------|----------------------|
| **Me** levanto. | Voy a levant**arme**. |
| **Te** duchas. | Estoy duchándo**me**. |
| **Se** viste. | Quiere vestir**se**. |

### Common Reflexive Verbs by Category

| Daily Routine | Emotions | Other |
|--------------|----------|-------|
| levantarse (get up) | alegrarse (be happy) | irse (leave) |
| acostarse (go to bed) | enojarse (get angry) | llamarse (be called) |
| ducharse (shower) | preocuparse (worry) | sentarse (sit down) |
| vestirse (get dressed) | sentirse (feel) | quedarse (stay) |
| lavarse (wash) | aburrirse (get bored) | mudarse (move) |
| peinarse (comb hair) | divertirse (have fun) | darse cuenta (realize) |

## 7. Progressive Tenses

### Formation

```
ESTAR (conjugated) + gerund (-ando / -iendo)
```

| Verb Type | Gerund | Example |
|-----------|--------|---------|
| -AR | -**ando** | hablando (speaking) |
| -ER | -**iendo** | comiendo (eating) |
| -IR | -**iendo** | viviendo (living) |

### Irregular Gerunds

| Infinitive | Gerund | Reason |
|-----------|--------|--------|
| leer | **leyendo** | vowel + -iendo → -yendo |
| oír | **oyendo** | |
| ir | **yendo** | |
| dormir | **durmiendo** | o→u stem change |
| pedir | **pidiendo** | e→i stem change |
| decir | **diciendo** | e→i |
| seguir | **siguiendo** | e→i |

### Tense Combinations

| Tense | Example | Translation |
|-------|---------|-------------|
| Present progressive | Estoy hablando. | I am speaking. |
| Past progressive | Estaba hablando. | I was speaking. |
| Future progressive | Estaré hablando. | I will be speaking. |

## 8. Master Conjugation Table

Full conjugation of **HABLAR** across all major tenses:

| Tense | yo | tú | él/ella | nosotros | ellos |
|-------|----|----|---------|----------|-------|
| **Present** | hablo | hablas | habla | hablamos | hablan |
| **Preterite** | hablé | hablaste | habló | hablamos | hablaron |
| **Imperfect** | hablaba | hablabas | hablaba | hablábamos | hablaban |
| **Future** | hablaré | hablarás | hablará | hablaremos | hablarán |
| **Conditional** | hablaría | hablarías | hablaría | hablaríamos | hablarían |
| **Pres. Subj.** | hable | hables | hable | hablemos | hablen |
| **Imp. Subj.** | hablara | hablaras | hablara | habláramos | hablaran |

| Compound Tense | yo form |
|----------------|---------|
| Present Perfect | he hablado |
| Pluperfect | había hablado |
| Future Perfect | habré hablado |
| Conditional Perfect | habría hablado |
| Present Perfect Subj. | haya hablado |
| Pluperfect Subj. | hubiera hablado |

## 9. Practice Exercises

### Exercise 1: Stem-Changing Verbs

Conjugate in the present tense.

1. Yo ________ (querer) ir al cine.
2. Ellos ________ (dormir) ocho horas.
3. Tú ________ (pedir) la cuenta.
4. Nosotros ________ (poder) ayudarte.
5. Ella ________ (volver) a las cinco.
6. Yo ________ (seguir) estudiando.

<details>
<summary>Answers</summary>

1. **quiero** (e→ie) 2. **duermen** (o→ue) 3. **pides** (e→i)
4. **podemos** (no change — nosotros) 5. **vuelve** (o→ue) 6. **sigo** (e→i + irregular yo)

</details>

### Exercise 2: Progressive Tenses

Form the present progressive.

1. (yo / estudiar) → ________
2. (ella / dormir) → ________
3. (nosotros / leer) → ________
4. (ellos / pedir) → ________

<details>
<summary>Answers</summary>

1. Estoy **estudiando**. 2. Está **durmiendo**. 3. Estamos **leyendo**. 4. Están **pidiendo**.

</details>

### Exercise 3: Identify the Tense

What tense is each verb form?

1. habíamos comido → ________
2. habrán llegado → ________
3. comiera → ________
4. haya dicho → ________
5. estaría hablando → ________

<details>
<summary>Answers</summary>

1. Pluperfect (indicative)
2. Future perfect
3. Imperfect subjunctive
4. Present perfect subjunctive
5. Conditional progressive

</details>
