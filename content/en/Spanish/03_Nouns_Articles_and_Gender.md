## 1. What is Grammatical Gender?

Every Spanish noun has a grammatical gender: **masculine** or **feminine**. This is NOT about biological sex — even objects like "table" (la mesa, feminine) and "book" (el libro, masculine) have grammatical gender.

```
┌─────────────────────────────────────────────────────────────────┐
│   English: "the book", "the table", "the house"                │
│   (same article regardless of noun)                             │
│                                                                 │
│   Spanish: "el libro", "la mesa", "la casa"                    │
│   (article changes to match noun gender)                        │
│                                                                 │
│   Why it matters:                                               │
│   • Articles must agree: el libro / la mesa                     │
│   • Adjectives must agree: libro nuevo / mesa nueva             │
│   • Pronouns must agree: lo compré / la compré                  │
└─────────────────────────────────────────────────────────────────┘
```

> **For Korean speakers**: Korean nouns don't have grammatical gender (책, 탁자 are gender-neutral). In Spanish, you must memorize the gender of every noun along with its article. Always learn **el libro** not just "libro."

## 2. Gender Rules

### 2.1 Masculine Noun Markers

```
Typically masculine:
  -o ending:    el libro (book), el gato (cat), el año (year)
  -or ending:   el doctor, el amor (love), el color
  -aje ending:  el viaje (trip), el paisaje (landscape)
  -mente ending: el accidente, el continente
  Days:         el lunes, el martes, el miércoles...
  Months:       enero, febrero (no article usually, but masculine)
  Languages:    el español, el inglés, el coreano
```

### 2.2 Feminine Noun Markers

```
Typically feminine:
  -a ending:      la casa (house), la mesa (table), la persona (person)
  -ción ending:   la nación (nation), la canción (song)
  -sión ending:   la televisión, la decisión
  -dad ending:    la ciudad (city), la universidad (university)
  -tad ending:    la libertad (freedom), la amistad (friendship)
  -ez ending:     la vez (time), la vejez (old age)
  -eza ending:    la belleza (beauty), la naturaleza (nature)
  -tud ending:    la actitud, la juventud (youth)
  Letters:        la a, la be, la ce...
```

### 2.3 Important Exceptions

```
┌─────────────────────────────────────────────────────────────────┐
│   Masculine words ending in -a (memorize these!):               │
│   el día (day)        el mapa (map)       el problema (problem) │
│   el programa          el sistema          el clima (climate)    │
│   el tema (topic)      el idioma (language) el drama             │
│   el planeta           el poema            el diploma            │
│                                                                 │
│   ※ Many -ma words from Greek are masculine                    │
│                                                                 │
│   Feminine words ending in -o (memorize these!):                │
│   la mano (hand)      la foto (photo)     la moto (motorcycle)  │
│   la radio (radio)                                              │
│                                                                 │
│   Gender-neutral (same form, article changes):                  │
│   el/la estudiante    el/la artista       el/la turista         │
│   el/la periodista    el/la pianista      el/la dentista        │
└─────────────────────────────────────────────────────────────────┘
```

## 3. Definite Articles

### 3.1 The Four Forms of "the"

| | Masculine | Feminine |
|---|-----------|----------|
| **Singular** | **el** | **la** |
| **Plural** | **los** | **las** |

```
el libro   → the book          la casa   → the house
los libros → the books         las casas → the houses
```

### 3.2 Special Rule: el + Feminine Nouns with Stressed a-/ha-

Feminine singular nouns beginning with stressed **a-** or **ha-** take **el** (for sound reasons), but remain feminine:

```
el agua    → the water    (feminine! → las aguas, el agua fría)
el águila  → the eagle    (feminine! → las águilas)
el alma    → the soul     (feminine! → las almas)
el hambre  → the hunger   (feminine! → las hambres)
el aula    → the classroom (feminine! → las aulas)

※ Only when a-/ha- is stressed:
la amiga (friend) → NOT "el amiga" (stress on -mi-, not a-)
```

### 3.3 Contractions

Spanish has exactly two mandatory contractions with articles:

```
a + el = al       →  Voy al parque. (I go to the park.)
de + el = del     →  El libro del profesor. (The professor's book.)

※ No contraction with la/los/las:
  Voy a la escuela.  (Not "al escuela")
  El libro de la profesora.
```

### 3.4 When to Use Definite Articles (Different from English)

```
Spanish uses "the" where English doesn't:
  Me gusta el chocolate.       → I like chocolate. (general concept)
  La vida es bella.            → Life is beautiful.
  Los perros son leales.       → Dogs are loyal.
  El español es útil.          → Spanish is useful.
  Tengo clase los lunes.       → I have class on Mondays.

Spanish omits "the" where English uses it:
  Soy estudiante.              → I'm a student. (profession, no article)
  Necesito café.               → I need coffee. (some, no article)
```

## 4. Indefinite Articles

### 4.1 The Four Forms of "a/an/some"

| | Masculine | Feminine |
|---|-----------|----------|
| **Singular** | **un** | **una** |
| **Plural** | **unos** | **unas** |

```
un libro   → a book            una casa   → a house
unos libros → some books       unas casas → some houses
```

### 4.2 When to Omit Indefinite Articles

```
After ser + profession (without adjective):
  Soy profesor.          → I'm a teacher.
  Es médica.             → She's a doctor.
  BUT: Es un buen médico. → He's a good doctor. (adjective → use article)

After sin (without):
  Salgo sin paraguas.    → I go out without an umbrella.

After tener in negation:
  No tengo coche.        → I don't have a car.
```

## 5. Plural Formation

### 5.1 Three Rules

```
┌─────────────────────────────────────────────────────────────────┐
│   Rule 1: Ends in vowel → add -s                               │
│   libro → libros        casa → casas       estudiante → estudiantes │
│                                                                 │
│   Rule 2: Ends in consonant → add -es                           │
│   papel → papeles       ciudad → ciudades   profesor → profesores │
│                                                                 │
│   Rule 3: Ends in -z → change z to c, add -es                  │
│   lápiz → lápices       vez → veces         pez → peces         │
│                                                                 │
│   Special cases:                                                │
│   Words ending in -s (unstressed) → no change                   │
│   el lunes → los lunes     la crisis → las crisis               │
│   el paraguas → los paraguas                                    │
└─────────────────────────────────────────────────────────────────┘
```

## 6. Essential Nouns: Family

| Spanish | English | Gender |
|---------|---------|--------|
| la familia | family | f |
| el padre / la madre | father / mother | m / f |
| el hijo / la hija | son / daughter | m / f |
| el hermano / la hermana | brother / sister | m / f |
| el abuelo / la abuela | grandfather / grandmother | m / f |
| el tío / la tía | uncle / aunt | m / f |
| el primo / la prima | cousin | m / f |
| el esposo / la esposa | husband / wife | m / f |
| el nieto / la nieta | grandson / granddaughter | m / f |
| el sobrino / la sobrina | nephew / niece | m / f |
| el suegro / la suegra | father-in-law / mother-in-law | m / f |
| el cuñado / la cuñada | brother-in-law / sister-in-law | m / f |
| el bebé | baby | m |
| los padres | parents | m pl |
| los hijos | children (sons & daughters) | m pl |

**Example sentences:**
- Tengo dos hermanos. — I have two siblings.
- Mi madre es profesora. — My mother is a teacher.
- Mis abuelos viven en Busan. — My grandparents live in Busan.

> **Note**: Masculine plural can include both genders: *los hermanos* = brothers, or brothers and sisters.

## Quick Reference

```
┌─────────────────────────────────────────────────────────────────┐
│   ARTICLES          Masculine    Feminine                       │
│   Definite (the)    el / los     la / las                       │
│   Indefinite (a)    un / unos    una / unas                     │
│                                                                 │
│   GENDER CLUES                                                  │
│   Masculine: -o, -or, -aje, Greek -ma                           │
│   Feminine:  -a, -ción, -sión, -dad, -tad, -ez, -eza           │
│   Exceptions: el día, el mapa, la mano, la foto                │
│                                                                 │
│   PLURALS                                                       │
│   Vowel → +s    Consonant → +es    -z → -ces                   │
│                                                                 │
│   CONTRACTIONS                                                  │
│   a + el = al          de + el = del                            │
└─────────────────────────────────────────────────────────────────┘
```
