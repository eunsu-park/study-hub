## 1. Question Words (Interrogatives)

All Spanish question words carry an **accent mark** when used in questions.

| Question Word | English | Example |
|---------------|---------|---------|
| **¿Qué?** | What? | ¿Qué quieres? (What do you want?) |
| **¿Quién? / ¿Quiénes?** | Who? | ¿Quién es? (Who is it?) |
| **¿Cuándo?** | When? | ¿Cuándo vienes? (When are you coming?) |
| **¿Dónde?** | Where? | ¿Dónde vives? (Where do you live?) |
| **¿Adónde?** | Where to? | ¿Adónde vas? (Where are you going?) |
| **¿Por qué?** | Why? | ¿Por qué estudias? (Why do you study?) |
| **¿Cómo?** | How? | ¿Cómo estás? (How are you?) |
| **¿Cuánto/a?** | How much? | ¿Cuánto cuesta? (How much does it cost?) |
| **¿Cuántos/as?** | How many? | ¿Cuántos años tienes? (How old are you?) |
| **¿Cuál? / ¿Cuáles?** | Which? | ¿Cuál prefieres? (Which do you prefer?) |

### ¿Qué? vs. ¿Cuál?

```
+----------------------------------------------------------+
|  ¿QUÉ? = asks for a DEFINITION or IDENTITY               |
|  ¿Qué es esto?  (What is this? — asking what it IS)      |
|  ¿Qué hora es?  (What time is it?)                       |
|                                                          |
|  ¿CUÁL? = asks for a CHOICE or SELECTION                  |
|  ¿Cuál es tu nombre?  (What is your name? — selection)   |
|  ¿Cuál prefieres?     (Which do you prefer?)             |
|                                                          |
|  Before a NOUN → always use ¿Qué?                        |
|  ¿Qué libro quieres?  (NOT *¿Cuál libro?)               |
+----------------------------------------------------------+
```

## 2. Question Formation Methods

### Method 1: Inversion (Subject-Verb → Verb-Subject)

- Juan habla español. → **¿Habla** Juan español?
- Ella trabaja aquí. → **¿Trabaja** ella aquí?

### Method 2: Intonation Only (Rising Pitch)

- **¿**Tú hablas español**?** (same word order, rising tone)

### Method 3: Tag Questions

- Hablas español, **¿verdad?** (You speak Spanish, right?)
- Hablas español, **¿no?** (You speak Spanish, don't you?)
- Vienes mañana, **¿cierto?** (You're coming tomorrow, right?)

> Spanish uses **inverted question marks** ¿...? at the beginning and end.
> The same applies to exclamation marks: ¡...!

## 3. Negation Basics

### Simple Negation: NO before the Verb

- **No** hablo francés. (I don't speak French.)
- **No** tengo dinero. (I don't have money.)
- **No** quiero ir. (I don't want to go.)

### Answering in the Negative

- ¿Hablas japonés? — **No**, **no** hablo japonés.
  (The first "no" answers the question; the second negates the verb.)

### Negative Words

| Negative | Affirmative Counterpart |
|----------|------------------------|
| **nada** (nothing) | algo (something) |
| **nadie** (nobody) | alguien (someone) |
| **nunca / jamás** (never) | siempre (always) |
| **tampoco** (neither) | también (also) |
| **ninguno/a** (none) | alguno/a (some) |
| **ni... ni** (neither... nor) | o... o (either... or) |

## 4. Double Negatives

Unlike English, Spanish **requires** double (or even triple) negatives.
This is grammatically correct and standard:

| Spanish | English |
|---------|---------|
| **No** tengo **nada**. | I don't have anything. (lit: I don't have nothing.) |
| **No** viene **nadie**. | Nobody is coming. |
| **No** voy **nunca**. | I never go. |
| **No** quiero **ni** café **ni** té. | I want neither coffee nor tea. |
| **No** tengo **ningún** problema. | I don't have any problem. |

> When the negative word comes **before** the verb, omit *no*:
> - **Nadie** viene. (Nobody is coming.)
> - **Nunca** como carne. (I never eat meat.)
> - **Nada** es perfecto. (Nothing is perfect.)

## 5. Expressing Opinions

| Expression | Example |
|------------|---------|
| **Creo que...** | Creo que es buena idea. (I think it's a good idea.) |
| **Pienso que...** | Pienso que tienes razón. (I think you're right.) |
| **Me parece que...** | Me parece que va a llover. (It seems to me it's going to rain.) |
| **En mi opinión...** | En mi opinión, es muy caro. (In my opinion, it's very expensive.) |
| **Para mí...** | Para mí, la familia es importante. (For me, family is important.) |

### Agreeing and Disagreeing

| Agreement | Disagreement |
|-----------|-------------|
| Estoy de acuerdo. (I agree.) | No estoy de acuerdo. (I disagree.) |
| Tienes razón. (You're right.) | No tienes razón. (You're wrong.) |
| Exacto. / Exactamente. | No es así. (It's not like that.) |
| Por supuesto. (Of course.) | Para nada. (Not at all.) |
| Claro que sí. (Of course.) | Claro que no. (Of course not.) |

## 6. Phone Expressions

| Spanish | English |
|---------|---------|
| ¿Aló? / ¿Diga? / ¿Bueno? | Hello? (answering phone) |
| ¿Puedo hablar con...? | Can I speak with...? |
| ¿De parte de quién? | Who's calling? |
| Un momento, por favor. | One moment, please. |
| No está disponible. | He/She is not available. |
| ¿Quiere dejar un mensaje? | Would you like to leave a message? |
| Le devuelvo la llamada. | I'll return the call. |
| Se cortó la llamada. | The call was cut off. |
| No tengo señal. | I don't have signal. |

> Phone greetings vary by country: **¿Diga?** (Spain), **¿Bueno?** (Mexico),
> **¿Aló?** (South America), **¿Hola?** (Argentina).

## 7. Restaurant Scenarios

### Useful Phrases

| Spanish | English |
|---------|---------|
| Una mesa para dos, por favor. | A table for two, please. |
| ¿Tienen menú del día? | Do you have a set menu? |
| ¿Qué me recomienda? | What do you recommend? |
| Para mí, la sopa. | For me, the soup. |
| ¿Me trae la cuenta, por favor? | Can you bring me the check? |
| ¿Está incluida la propina? | Is the tip included? |
| ¿Puedo pagar con tarjeta? | Can I pay by card? |

### Sample Dialogue

> **Camarero**: ¡Buenas tardes! ¿Mesa para cuántos?
> **Cliente**: Para dos, por favor.
> **Camarero**: Aquí tienen el menú. ¿Qué desean tomar?
> **Cliente**: Yo quiero una limonada y ella quiere un agua con gas.
> **Camarero**: ¿Ya saben qué van a pedir?
> **Cliente**: Sí, yo quiero la paella y ella quiere el pescado a la plancha.
> **Camarero**: Muy bien. ¿Algo más?
> **Cliente**: No, gracias. Eso es todo.
> ...
> **Cliente**: La cuenta, por favor.
> **Camarero**: Aquí tiene. Son treinta y cinco euros.
> **Cliente**: ¿Puedo pagar con tarjeta?
> **Camarero**: Por supuesto.

## 9. Asking for Directions

| Spanish | English |
|---------|---------|
| Perdone, ¿dónde está...? | Excuse me, where is...? |
| ¿Cómo llego a...? | How do I get to...? |
| ¿Está lejos/cerca de aquí? | Is it far/near from here? |
| Siga todo recto / derecho. | Go straight ahead. |
| Gire a la derecha/izquierda. | Turn right/left. |
| Está en la esquina. | It's on the corner. |
| Está al lado de... | It's next to... |
| Está enfrente de... | It's across from... |
| Cruce la calle. | Cross the street. |
| A dos cuadras / manzanas. | Two blocks away. |

## 10. Practice Exercises

### Exercise 1: Form Questions

Write the question for each answer using the appropriate question word.

1. Answer: Me llamo Ana. → ¿________?
2. Answer: Vivo en Madrid. → ¿________?
3. Answer: Tengo 25 años. → ¿________?
4. Answer: Estoy bien, gracias. → ¿________?
5. Answer: Porque me gusta. → ¿________?
6. Answer: Cuesta 10 euros. → ¿________?

<details>
<summary>Answers</summary>

1. ¿**Cómo** te llamas?
2. ¿**Dónde** vives?
3. ¿**Cuántos** años tienes?
4. ¿**Cómo** estás?
5. ¿**Por qué** lo haces? / ¿**Por qué** te gusta?
6. ¿**Cuánto** cuesta?

</details>

### Exercise 2: Negate the Sentences

1. Siempre como carne. → ________
2. Alguien llamó. → ________
3. Quiero algo. → ________
4. También me gusta. → ________
5. Tengo algún problema. → ________

<details>
<summary>Answers</summary>

1. **Nunca** como carne. / **No** como carne **nunca**.
2. **Nadie** llamó. / **No** llamó **nadie**.
3. **No** quiero **nada**.
4. **Tampoco** me gusta. / A mí **tampoco**.
5. **No** tengo **ningún** problema.

</details>

Fill in the blanks in this restaurant conversation.

> Camarero: Les ________ la paella, es nuestra especialidad.
> Cliente: Perfecto, dos paellas entonces.
> ...
> Cliente: La ________, por favor. ¿________ pagar con tarjeta?
