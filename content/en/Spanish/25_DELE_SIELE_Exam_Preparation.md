## 1. DELE Overview

### What is DELE?

```
+----------------------------------------------------------+
|  DELE — Diplomas de Español como Lengua Extranjera       |
+----------------------------------------------------------+
|                                                          |
|  Issuing body: Instituto Cervantes                       |
|  Recognition: International, lifelong validity           |
|  Levels: A1, A2, B1, B2, C1, C2                         |
|  Exam sessions: Typically April, May, July, October,     |
|                 November (varies by location)            |
|                                                          |
|  SECTIONS (4 sections):                                  |
|  1. Comprensión de lectura (Reading comprehension)       |
|  2. Comprensión auditiva (Listening comprehension)       |
|  3. Expresión e interacción escritas (Written expression) |
|  4. Expresión e interacción orales (Oral expression)     |
|                                                          |
|  PASSING: Each section grouped into two pairs.            |
|  You must score at least 60% in EACH group.              |
|  Group 1: Reading + Writing                              |
|  Group 2: Listening + Speaking                           |
+----------------------------------------------------------+
```

### DELE Exam Duration by Level

| Level | Reading | Writing | Listening | Speaking |
|-------|---------|---------|-----------|---------|
| A1 | 45 min | 25 min | 20 min | 15 min |
| A2 | 60 min | 45 min | 35 min | 15 min |
| B1 | 70 min | 40 min | 40 min | 15 min |
| B2 | 70 min | 80 min | 40 min | 20 min |
| C1 | 90 min | 80 min | 50 min | 20 min |
| C2 | 105 min | 150 min | 55 min | 20 min |

## 2. SIELE Overview

### What is SIELE?

```
+----------------------------------------------------------+
|  SIELE — Servicio Internacional de Evaluación de la      |
|          Lengua Española                                  |
+----------------------------------------------------------+
|                                                          |
|  Issuing body: Instituto Cervantes + UNAM +              |
|                U. de Salamanca + U. de Buenos Aires       |
|  Recognition: International                              |
|  Validity: 5 years (unlike DELE's lifetime)              |
|  Format: 100% computer-based, online                     |
|  Scoring: 0-1000 points (mapped to CEFR levels)          |
|                                                          |
|  MODALITIES:                                             |
|  • SIELE Global: All 4 sections                          |
|  • SIELE S1: Reading + Listening                         |
|  • SIELE S2: Reading + Writing                           |
|  • SIELE S3: Listening + Writing                         |
|  • SIELE S4: Speaking only                               |
|  • SIELE S5: Speaking + Listening                        |
|                                                          |
|  Advantage: More flexible dates, faster results,         |
|  includes Latin American Spanish varieties               |
+----------------------------------------------------------+
```

### DELE vs. SIELE Comparison

| Feature | DELE | SIELE |
|---------|------|-------|
| Validity | Lifetime | 5 years |
| Format | Paper-based | Computer-based |
| Result | Pass/Fail (by level) | Score 0-1000 |
| Dates | Fixed sessions | Flexible |
| Varieties | Primarily Peninsular | All varieties |
| Preparation time | You choose the level | Adaptive |

## 3. CEFR Level Descriptions

### Can-Do Statements

| Level | Description |
|-------|-------------|
| **A1** (Beginner) | Can understand and use basic everyday expressions. Can introduce themselves and answer simple personal questions. |
| **A2** (Elementary) | Can communicate in routine tasks on familiar topics. Can describe aspects of their background and immediate needs. |
| **B1** (Intermediate) | Can deal with most situations while traveling. Can write simple connected text on familiar topics. Can describe experiences, events, and plans. |
| **B2** (Upper Intermediate) | Can interact fluently with native speakers without strain. Can produce detailed text on a wide range of subjects. Can explain a viewpoint on current issues. |
| **C1** (Advanced) | Can use language flexibly for social, academic, and professional purposes. Can produce well-structured, detailed text on complex subjects. |
| **C2** (Mastery) | Can understand virtually everything heard or read. Can summarize from different sources, reconstructing arguments coherently. Near-native fluency. |

### What Each Level Requires

```
+----------------------------------------------------------+
|          GRAMMAR REQUIREMENTS BY LEVEL                    |
+----------------------------------------------------------+
|                                                          |
|  A1: Present tense, ser/estar, basic vocabulary,         |
|      numbers, greetings, simple questions                |
|                                                          |
|  A2: Past tenses (preterite/imperfect), future           |
|      (ir a + inf), comparatives, object pronouns         |
|                                                          |
|  B1: All indicative tenses, basic subjunctive,           |
|      conditional, compound tenses, connectors            |
|                                                          |
|  B2: Full subjunctive, si-clauses, passive voice,        |
|      complex sentences, discourse markers                |
|                                                          |
|  C1: Nuanced subjunctive use, complex conditionals,      |
|      register awareness, idiomatic expressions           |
|                                                          |
|  C2: Full mastery of all structures, stylistic range,    |
|      subtle meaning distinctions, cultural references    |
+----------------------------------------------------------+
```

### Vocabulary Targets (Approximate)

| Level | Active Vocabulary |
|-------|------------------|
| A1 | ~500-700 words |
| A2 | ~1,000-1,500 words |
| B1 | ~2,500-3,000 words |
| B2 | ~4,000-5,000 words |
| C1 | ~6,000-8,000 words |
| C2 | ~10,000+ words |

## 4. Reading Section Strategies

### General Tips

```
+----------------------------------------------------------+
|          READING EXAM STRATEGIES                          |
+----------------------------------------------------------+
|                                                          |
|  1. Read the questions FIRST, then the text.              |
|     This helps you know what to look for.                |
|                                                          |
|  2. Don't panic over unknown words.                      |
|     Use context clues to guess meaning.                  |
|                                                          |
|  3. Watch for distractors.                               |
|     Wrong answers often use words from the text           |
|     but distort the meaning.                             |
|                                                          |
|  4. Pay attention to negatives and qualifiers.            |
|     "not always" ≠ "never" ≠ "sometimes"                |
|                                                          |
|  5. Manage your time.                                    |
|     Don't spend too long on one question.                |
|     Mark difficult ones and return to them.              |
+----------------------------------------------------------+
```

### Common Task Types

| Task Type | Strategy |
|-----------|----------|
| Multiple choice | Eliminate obviously wrong answers first |
| True/False | Look for the exact information in the text |
| Gap fill (cloze) | Read the whole text first for overall meaning |
| Matching (text to summary) | Identify the main idea of each text/paragraph |
| Sentence completion | Check that your answer is grammatically correct |

## 5. Writing Section Strategies

### By Level

| Level | Task Types | Tips |
|-------|-----------|------|
| **A1-A2** | Fill in forms, short messages, postcards | Use simple sentences. Focus on accuracy over complexity. |
| **B1** | Informal letter/email, short essay, blog post | Use connectors (también, pero, porque). Organize into paragraphs. |
| **B2** | Formal letter, article, essay with argumentation | Use a clear structure (intro-body-conclusion). Include counterarguments. |
| **C1** | Formal text, summary, complex essay | Vary sentence structure. Use subjunctive and conditional appropriately. |
| **C2** | Academic text, detailed analysis, creative writing | Demonstrate mastery of register, style, and complex grammar. |

### Writing Checklist

```
+----------------------------------------------------------+
|          WRITING CHECKLIST                                 |
+----------------------------------------------------------+
|                                                          |
|  □ Did I address ALL parts of the prompt?                 |
|  □ Is my text the required length?                       |
|  □ Is my text organized with clear paragraphs?           |
|  □ Did I use appropriate connectors/discourse markers?    |
|  □ Did I check gender and number agreement?              |
|  □ Did I check verb tense consistency?                   |
|  □ Did I use the correct register (formal/informal)?     |
|  □ Did I check accent marks?                             |
|  □ Did I use ¿...? and ¡...! correctly?                  |
+----------------------------------------------------------+
```

## 6. Listening Section Strategies

### General Tips

```
+----------------------------------------------------------+
|          LISTENING EXAM STRATEGIES                         |
+----------------------------------------------------------+
|                                                          |
|  BEFORE listening:                                       |
|  • Read the questions carefully                          |
|  • Predict what you might hear                           |
|  • Note key words in each question                       |
|                                                          |
|  DURING listening:                                       |
|  • Don't try to understand every word                    |
|  • Listen for key information (names, numbers, dates)    |
|  • Pay attention to tone and context                     |
|  • Take quick notes                                      |
|                                                          |
|  AFTER first listen:                                     |
|  • Answer what you can                                   |
|  • Mark uncertain answers for the second listen          |
|  • On second listen, focus on missed information         |
|                                                          |
|  NOTE: You typically hear each audio TWICE.              |
+----------------------------------------------------------+
```

### Common Audio Types

| Level | Audio Types |
|-------|-------------|
| A1-A2 | Short announcements, simple conversations, messages |
| B1 | News excerpts, everyday conversations, phone calls |
| B2 | Interviews, debates, lectures, complex conversations |
| C1-C2 | Academic lectures, nuanced discussions, multiple speakers |

## 7. Speaking Section Strategies

### DELE Speaking Format

| Level | Tasks |
|-------|-------|
| A1-A2 | Self-introduction, describe a photo, role-play a simple situation |
| B1 | Describe an experience, discuss a topic with the examiner, role-play |
| B2 | Give a short presentation on a topic, debate with the examiner |
| C1-C2 | Extended presentation, complex discussion, defend your arguments |

### Tips for the Speaking Exam

```
+----------------------------------------------------------+
|          SPEAKING EXAM TIPS                                |
+----------------------------------------------------------+
|                                                          |
|  1. Don't memorize scripts — examiners notice.            |
|     Prepare TOPICS and STRUCTURES, not exact texts.      |
|                                                          |
|  2. If you don't know a word:                            |
|     - Describe it: "Es una cosa que sirve para..."       |
|     - Use a synonym or simpler word                      |
|     - DON'T switch to English                            |
|                                                          |
|  3. Use fillers naturally:                               |
|     Bueno... / Pues... / A ver... / Es que...            |
|                                                          |
|  4. Self-correct when you notice errors:                 |
|     "Fui al... bueno, fui a la tienda."                  |
|     This shows awareness and is valued.                  |
|                                                          |
|  5. Ask for clarification if needed:                     |
|     "¿Podría repetir la pregunta?"                       |
|     "¿Se refiere a...?"                                  |
|                                                          |
|  6. Use a variety of structures to show your range.      |
+----------------------------------------------------------+
```

### Useful Speaking Phrases

| Function | Phrases |
|----------|---------|
| Starting | Me gustaría hablar sobre... / En mi opinión... |
| Adding points | Además... / También... / Otro aspecto importante es... |
| Giving examples | Por ejemplo... / Como... / Un caso sería... |
| Contrasting | Sin embargo... / Por otro lado... / Aunque... |
| Concluding | En resumen... / Para terminar... / En definitiva... |
| Buying time | Bueno, déjeme pensar... / Es una pregunta interesante... |

## 8. Sample Exercises by Level

### A1-A2 Level: Reading

Read the note and answer.

> "Hola, mamá. Hoy no puedo ir a cenar a tu casa porque tengo mucho trabajo. ¿Podemos quedar el sábado? Un beso. Luis"

1. ¿Quién escribe la nota?
2. ¿Por qué no puede ir a cenar?
3. ¿Qué día propone?

<details>
<summary>Answers</summary>

1. **Luis** writes the note (to his mother).
2. Because he has a lot of work (**tiene mucho trabajo**).
3. He proposes **Saturday** (el sábado).

</details>

### B1 Level: Writing Prompt

Write an email (120-150 words) to a friend describing your last vacation. Include:
- Where you went
- What you did
- What you liked most
- Whether you would recommend it

<details>
<summary>Sample Answer</summary>

> ¡Hola, Ana!
>
> ¿Qué tal estás? Te escribo para contarte sobre mis vacaciones. El mes pasado fui a Barcelona con mi familia. Estuvimos una semana y fue una experiencia increíble.
>
> Visitamos la Sagrada Familia, que es impresionante, y paseamos por Las Ramblas. También fuimos a la playa y comimos paella en un restaurante junto al mar. Lo que más me gustó fue el barrio Gótico, con sus calles estrechas y edificios antiguos.
>
> Sin embargo, hacía mucho calor y había muchos turistas, lo cual a veces resultaba un poco agobiante.
>
> Te lo recomiendo totalmente, pero es mejor ir en primavera cuando no hace tanto calor. ¿Tú has estado alguna vez en Barcelona?
>
> ¡Escríbeme pronto!
> Un abrazo,
> Marta

</details>

### B2 Level: Reading

Read the text and answer.

> La inteligencia artificial está transformando el mercado laboral de maneras que hace solo una década habrían parecido ciencia ficción. Si bien es cierto que la automatización eliminará algunos puestos de trabajo, especialmente aquellos que implican tareas repetitivas, también es verdad que creará nuevas oportunidades en campos que aún no existen. El desafío para los trabajadores actuales no es resistir al cambio tecnológico, sino adaptarse a él mediante la formación continua y el desarrollo de habilidades que las máquinas no pueden replicar fácilmente, como la creatividad, la empatía y el pensamiento crítico.

1. According to the text, what kind of jobs are most at risk?
2. What should workers do, according to the author?
3. What skills does the author consider "machine-proof"?

<details>
<summary>Answers</summary>

1. Jobs involving **repetitive tasks** (tareas repetitivas).
2. **Adapt** through continuous training (formación continua) and develop skills machines can't replicate.
3. **Creativity** (creatividad), **empathy** (empatía), and **critical thinking** (pensamiento crítico).

</details>

## 9. Study Plan

### General Preparation Timeline

```
+----------------------------------------------------------+
|          EXAM PREPARATION PLAN                            |
+----------------------------------------------------------+
|                                                          |
|  3-6 MONTHS BEFORE:                                     |
|  • Assess your current level (take a practice test)      |
|  • Identify weak areas                                   |
|  • Set up a study schedule                               |
|  • Start building vocabulary systematically              |
|                                                          |
|  2-3 MONTHS BEFORE:                                     |
|  • Work through grammar points for your level            |
|  • Practice reading and listening daily                  |
|  • Write practice texts weekly                           |
|  • Start practicing speaking (with a partner or tutor)   |
|                                                          |
|  1 MONTH BEFORE:                                        |
|  • Take full practice exams under timed conditions       |
|  • Focus on your weakest section(s)                      |
|  • Review common errors                                  |
|  • Practice time management                              |
|                                                          |
|  1 WEEK BEFORE:                                         |
|  • Review, don't cram                                    |
|  • Do one final practice test                            |
|  • Prepare logistics (ID, location, materials)           |
|  • Rest well                                             |
+----------------------------------------------------------+
```

### Daily Practice Routine

| Activity | Time | Purpose |
|----------|------|---------|
| Read a Spanish article | 15 min | Reading + vocabulary |
| Listen to a Spanish podcast | 15 min | Listening comprehension |
| Write a short text | 15 min | Writing practice |
| Review vocabulary (flashcards) | 10 min | Vocabulary retention |
| Grammar exercises | 15 min | Accuracy |
| Speak out loud (describe your day, opinions) | 10 min | Fluency |

### Recommended Resources

| Resource | Type | Best For |
|----------|------|----------|

## 10. Practice Exercises

### Exercise 1: Level Self-Assessment

Read each statement and check what you can do.

| | A1 | A2 | B1 | B2 |
|---|---|---|---|---|
| Introduce myself | □ | | | |
| Order food in a restaurant | | □ | | |
| Tell a story in the past | | | □ | |
| Discuss current events with opinions | | | | □ |
| Understand a newspaper article | | | | □ |
| Write a formal complaint letter | | | | □ |

### Exercise 2: Identify the Error Type

What type of error would lose marks? Classify as G (Grammar), V (Vocabulary), O (Orthography/Spelling), or R (Register).

1. "Yo soy cansado." → ________
2. "Querido señor, ¿qué onda?" → ________
3. "El problema es muy dificil." → ________
4. "Yo realicé que era verdad." (meaning: I realized) → ________

<details>
<summary>Answers</summary>

1. **G** — Grammar (should be "Estoy cansado" — temporary state)
2. **R** — Register (mixing formal greeting with informal slang)
3. **O** — Orthography (missing accent: "difícil")
4. **V** — Vocabulary (false friend: "realizar" ≠ "to realize"; should be "Me di cuenta de que era verdad")

</details>

### Exercise 3: Timed Writing Practice

Set a timer for 20 minutes. Write 100-120 words on this topic:

**"¿Qué ventajas tiene aprender un segundo idioma?"** (What advantages does learning a second language have?)

Requirements:
- Introduction with your thesis
- At least 2 supporting arguments
- A brief conclusion
- Use at least 3 discourse markers

<details>
<summary>Sample Answer</summary>

"**Hoy en día**, aprender un segundo idioma se ha convertido en una necesidad más que en un lujo. **En mi opinión**, hablar otra lengua ofrece numerosas ventajas tanto personales como profesionales.

**En primer lugar**, conocer otro idioma abre puertas en el mercado laboral. Las empresas internacionales valoran a los candidatos que pueden comunicarse con clientes y socios de diferentes países. **Además**, aprender una lengua nos permite conocer otras culturas desde dentro, entendiendo su manera de pensar y de ver el mundo.

**Por otro lado**, estudios científicos han demostrado que el bilingüismo mejora la memoria y la capacidad de concentración.

**En definitiva**, invertir tiempo en aprender un idioma es una decisión que nos beneficia en todos los aspectos de la vida."

</details>

### Exercise 4: Speaking Practice Topics

Practice speaking for 2-3 minutes on each topic.

1. **A2**: Describe your daily routine.
2. **B1**: Talk about a trip you took last year.
3. **B2**: Give your opinion: Is technology making us more or less social?
