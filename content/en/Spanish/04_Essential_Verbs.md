## 1. Why These Four Verbs?

```
┌─────────────────────────────────────────────────────────────────┐
│   These four verbs appear in almost every Spanish sentence.     │
│   Master them and you can express most basic ideas.             │
│                                                                 │
│   SER    → identity, origin, traits       (permanent "to be")  │
│   ESTAR  → location, condition, feelings  (temporary "to be")  │
│   TENER  → possession, age, many idioms   ("to have")          │
│   HABER  → existence (hay), perfect tenses ("to have" aux.)    │
│                                                                 │
│   Challenge for English speakers:                               │
│   English "to be" → SER or ESTAR (you must choose!)            │
│   English "to be" → sometimes TENER (tener hambre = "to be     │
│                     hungry", not "to have hunger")              │
└─────────────────────────────────────────────────────────────────┘
```

## 2. SER — Permanent/Essential "To Be"

### 2.1 Present Tense Conjugation

| Subject | Conjugation | Example |
|---------|-------------|---------|
| yo | **soy** | Soy estudiante. (I'm a student.) |
| tú | **eres** | Eres inteligente. (You're smart.) |
| él/ella/usted | **es** | Es profesor. (He's a teacher.) |
| nosotros/as | **somos** | Somos coreanos. (We're Korean.) |
| vosotros/as | **sois** | Sois amigos. (You all are friends.) |
| ellos/ellas/ustedes | **son** | Son las tres. (It's three o'clock.) |

### 2.2 Uses of SER — DOCTOR Mnemonic

```
┌─────────────────────────────────────────────────────────────────┐
│   D - Description (physical/personality traits)                 │
│       Ella es alta y simpática.                                 │
│       (She is tall and friendly.)                               │
│                                                                 │
│   O - Occupation (profession)                                   │
│       Soy ingeniero. (I'm an engineer.)                         │
│       Es profesora. (She's a teacher.)                          │
│                                                                 │
│   C - Characteristic (inherent quality)                         │
│       El hielo es frío. (Ice is cold.)                          │
│       La nieve es blanca. (Snow is white.)                      │
│                                                                 │
│   T - Time (clock time, dates)                                  │
│       Son las dos. (It's two o'clock.)                          │
│       Hoy es lunes. (Today is Monday.)                          │
│       Es el 15 de marzo. (It's March 15th.)                    │
│                                                                 │
│   O - Origin (where someone/something is from)                  │
│       Soy de Corea. (I'm from Korea.)                           │
│       El café es de Colombia. (The coffee is from Colombia.)    │
│                                                                 │
│   R - Relationship (possession, material)                       │
│       Es mi libro. (It's my book.)                              │
│       La mesa es de madera. (The table is made of wood.)        │
│       La fiesta es en mi casa. (The party is at my house.)      │
└─────────────────────────────────────────────────────────────────┘
```

### 2.3 More SER Examples

```
Identity:      Soy María. (I'm María.)
Nationality:   Somos coreanos. (We're Korean.)
Religion:      Es católico. (He's Catholic.)
Personality:   Eres muy amable. (You're very kind.)
Profession:    Son doctores. (They're doctors.)
Time:          Es la una. (It's one o'clock.)
Origin:        El vino es de España. (The wine is from Spain.)
Material:      El anillo es de oro. (The ring is made of gold.)
Events:        La reunión es a las tres. (The meeting is at three.)
```

## 3. ESTAR — Temporary/Situational "To Be"

### 3.1 Present Tense Conjugation

| Subject | Conjugation | Example |
|---------|-------------|---------|
| yo | **estoy** | Estoy en casa. (I'm at home.) |
| tú | **estás** | ¿Cómo estás? (How are you?) |
| él/ella/usted | **está** | Está contento. (He's happy.) |
| nosotros/as | **estamos** | Estamos cansados. (We're tired.) |
| vosotros/as | **estáis** | ¿Estáis listos? (Are you all ready?) |
| ellos/ellas/ustedes | **están** | Están en Madrid. (They're in Madrid.) |

### 3.2 Uses of ESTAR — PLACE Mnemonic

```
┌─────────────────────────────────────────────────────────────────┐
│   P - Position/Place (location)                                 │
│       Estoy en la oficina. (I'm at the office.)                 │
│       Madrid está en España. (Madrid is in Spain.)              │
│                                                                 │
│   L - Looks/Appears (how something looks right now)             │
│       Estás muy guapa hoy. (You look very pretty today.)        │
│       La comida está deliciosa. (The food is/tastes delicious.) │
│                                                                 │
│   A - Action in progress (-ando/-iendo)                         │
│       Estoy estudiando. (I'm studying.)                         │
│       Están comiendo. (They're eating.)                         │
│                                                                 │
│   C - Condition (temporary state, result of change)             │
│       Estoy cansado. (I'm tired.)                               │
│       La puerta está abierta. (The door is open.)               │
│       El café está frío. (The coffee is cold.)                  │
│                                                                 │
│   E - Emotion (how someone feels right now)                     │
│       Estoy feliz. (I'm happy right now.)                       │
│       Está triste. (He/She is sad.)                             │
│       Estamos nerviosos. (We're nervous.)                       │
└─────────────────────────────────────────────────────────────────┘
```

## 4. SER vs ESTAR Comparison

### 4.1 The Key Distinction

```
SER = what something IS (essence, identity, definition)
ESTAR = how something IS (state, condition, location)

Think of it this way:
  SER: "This is its nature."
  ESTAR: "This is its current state."
```

### 4.2 Same Adjective, Different Meaning

| Adjective | with SER (inherent) | with ESTAR (current state) |
|-----------|--------------------|-----------------------------|
| feliz | Es feliz. (He's a happy person.) | Está feliz. (He's happy right now.) |
| aburrido | Es aburrido. (He's boring.) | Está aburrido. (He's bored.) |
| listo | Es listo. (He's clever.) | Está listo. (He's ready.) |
| malo | Es malo. (He's a bad person.) | Está malo. (He's sick.) |
| bueno | Es bueno. (He's a good person.) | Está bueno. (He looks good / It tastes good.) |
| rico | Es rico. (He's rich.) | Está rico. (It tastes delicious.) |
| verde | Es verde. (It's green in color.) | Está verde. (It's unripe.) |
| vivo | Es vivo. (He's sharp/clever.) | Está vivo. (He's alive.) |
| seguro | Es seguro. (It's safe/secure.) | Está seguro. (He's sure/certain.) |
| orgulloso | Es orgulloso. (He's a proud person.) | Está orgulloso. (He's feeling proud.) |
| guapo | Es guapo. (He's handsome.) | Está guapo. (He looks handsome today.) |
| nuevo | Es nuevo. (It's new/brand new.) | Está nuevo. (It looks new/like new.) |

### 4.3 Quick Decision Chart

```
Is it about...?
  Identity / Origin / Profession  → SER
  Location                        → ESTAR
  Time / Date                     → SER
  Feeling / Emotion right now     → ESTAR
  Physical trait (always true)    → SER
  Condition (can change)          → ESTAR
  Progressive (-ando/-iendo)      → ESTAR
  Event location                  → SER
  Material / Composition          → SER
  Result of an action             → ESTAR
```

## 5. TENER — "To Have" and Idiomatic Uses

### 5.1 Present Tense Conjugation

| Subject | Conjugation | Example |
|---------|-------------|---------|
| yo | **tengo** | Tengo un gato. (I have a cat.) |
| tú | **tienes** | ¿Tienes tiempo? (Do you have time?) |
| él/ella/usted | **tiene** | Tiene 30 años. (He's 30 years old.) |
| nosotros/as | **tenemos** | Tenemos clase. (We have class.) |
| vosotros/as | **tenéis** | ¿Tenéis hambre? (Are you hungry?) |
| ellos/ellas/ustedes | **tienen** | Tienen dos hijos. (They have two children.) |

### 5.2 Literal Uses

```
Possession:    Tengo un coche. (I have a car.)
Relationships: Tiene tres hermanos. (He has three siblings.)
Age:           Tengo veinticinco años. (I'm 25 years old.)
               ※ Spanish says "I have 25 years", NOT "I am 25"
```

### 5.3 TENER Idioms (Where English uses "to be")

```
┌─────────────────────────────────────────────────────────────────┐
│   Physical sensations:                                          │
│   tener hambre       → to be hungry     (lit: have hunger)      │
│   tener sed          → to be thirsty    (lit: have thirst)      │
│   tener frío         → to be cold       (lit: have cold)        │
│   tener calor        → to be hot        (lit: have heat)        │
│   tener sueño        → to be sleepy     (lit: have sleep)       │
│                                                                 │
│   Emotional states:                                             │
│   tener miedo (de)   → to be afraid (of) (lit: have fear)      │
│   tener vergüenza    → to be embarrassed (lit: have shame)     │
│   tener cuidado      → to be careful    (lit: have care)        │
│   tener paciencia    → to be patient    (lit: have patience)    │
│                                                                 │
│   Other common idioms:                                          │
│   tener razón        → to be right      (lit: have reason)      │
│   tener prisa        → to be in a hurry (lit: have hurry)      │
│   tener suerte       → to be lucky      (lit: have luck)        │
│   tener éxito        → to be successful (lit: have success)     │
│   tener sentido      → to make sense    (lit: have sense)       │
│   tener lugar        → to take place    (lit: have place)       │
│   tener ganas de     → to feel like     (lit: have desire of)   │
└─────────────────────────────────────────────────────────────────┘
```

### 5.4 TENER QUE + Infinitive (Obligation)

```
tener que + infinitive = "to have to" / "must"

Tengo que estudiar.        → I have to study.
Tienes que comer.          → You have to eat.
Tenemos que ir.            → We have to go.
¿Tienes que trabajar hoy? → Do you have to work today?
```

## 6. HABER — "There Is" and Auxiliary

### 6.1 Impersonal HAY (There is / There are)

```
┌─────────────────────────────────────────────────────────────────┐
│   hay = there is / there are (present tense, impersonal)        │
│                                                                 │
│   Hay un libro en la mesa.        → There is a book on the table.│
│   Hay muchos estudiantes.         → There are many students.    │
│   ¿Hay un banco cerca?            → Is there a bank nearby?     │
│   No hay problema.                → There's no problem.         │
│   Hay que estudiar.               → One must study. / You have  │
│                                      to study. (impersonal)     │
│                                                                 │
│   Past:   había (there was/were)                                │
│   Future: habrá (there will be)                                 │
│                                                                 │
│   ※ HAY never changes for plural!                              │
│   Hay un gato.   Hay tres gatos.  (NOT "han tres gatos")       │
└─────────────────────────────────────────────────────────────────┘
```

### 6.2 HAY vs ESTAR for Location

```
HAY → introduces existence (there is/are), used with indefinite articles
ESTAR → specifies location of known things, used with definite articles

  Hay un banco en la calle.        → There's a bank on the street.
  El banco está en la calle Main.  → The bank is on Main Street.

  Hay libros en la mesa.           → There are books on the table.
  Los libros están en la mesa.     → The books are on the table.
```

### 6.3 HABER as Auxiliary (Compound Tenses Preview)

```
HABER (conjugated) + past participle = compound tenses

Present perfect:
  he hablado     → I have spoken
  has comido     → you have eaten
  ha vivido      → he/she has lived
  hemos visto    → we have seen

※ This will be covered in detail in Lesson 12.
```

## 7. Common Mistakes and Tips

### 7.1 Typical Errors

```
✗ Soy cansado.           ✓ Estoy cansado. (tired = temporary state)
✗ Estoy profesor.         ✓ Soy profesor. (profession = identity)
✗ Soy en Madrid.          ✓ Estoy en Madrid. (location = ESTAR)
✗ Tengo 25.               ✓ Tengo 25 años. (need "años" for age)
✗ Soy hambre.             ✓ Tengo hambre. (hungry = TENER idiom)
✗ Han muchos libros.      ✓ Hay muchos libros. (HAY is invariable)
✗ La fiesta está en mi casa. ✓ La fiesta es en mi casa. (event = SER)
```

### 7.2 Memory Tips

```
SER = DOCTOR     (Description, Occupation, Characteristic,
                   Time, Origin, Relationship)

ESTAR = PLACE    (Position, Looks, Action, Condition, Emotion)

TENER for physical sensations:
  "In Spanish, you don't ARE hungry/cold/sleepy —
   you HAVE hunger/cold/sleepiness"
```

│                                                                 │
