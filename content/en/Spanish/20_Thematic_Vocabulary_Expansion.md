## 6. Diminutives

Diminutive suffixes express small size, affection, or softening.

### -ito / -ita (Most Common)

| Original | Diminutive | Nuance |
|----------|-----------|--------|
| casa | cas**ita** | small/cozy house |
| perro | perr**ito** | puppy, cute dog |
| momento | moment**ito** | just a moment (softening) |
| ahora | ahor**ita** | right now (Mexico) |
| café | cafec**ito** | a cup of coffee (friendliness) |
| abuelo/a | abuel**ito/a** | grandpa/grandma (affection) |
| poco | poqu**ito** | a tiny bit |
| regalo | regal**ito** | small gift (modesty) |
| cerca | cerqu**ita** | very close (emphasis) |

### Formation Rules

```
+----------------------------------------------------------+
|          DIMINUTIVE FORMATION (-ito/-ita)                 |
+----------------------------------------------------------+
|                                                          |
|  1. Ending in vowel: drop vowel + -ito/-ita              |
|     casa → casita, perro → perrito                       |
|                                                          |
|  2. Ending in consonant: + -ito/-ita or -cito/-cita      |
|     amor → amorcito, pan → panecito                      |
|                                                          |
|  3. Ending in -co/-ca: c → qu + -ito/-ita                |
|     poco → poquito, chico → chiquito                     |
|                                                          |
|  4. Ending in -go/-ga: g → gu + -ito/-ita                |
|     amigo → amiguito, largo → larguito                   |
|                                                          |
|  5. One-syllable words: -ecito/-ecita                    |
|     flor → florecita, pan → panecito, pez → pececito    |
+----------------------------------------------------------+
```

### Other Diminutive Suffixes

| Suffix | Region/Tone | Example |
|--------|------------|---------|
| **-illo/-illa** | Less affectionate, sometimes lexicalized | ventanilla (counter), cigarrillo (cigarette), bolsillo (pocket) |
| **-ico/-ica** | Caribbean, Costa Rica, Aragon | momentico, ratico, poquitico |
| **-ín/-ina** | Asturias (Spain) | pequeñín, chiquitín |

## 7. Augmentatives

Augmentative suffixes express large size, intensity, or sometimes negative qualities.

### -ote / -ota (Large, Sometimes Coarse)

| Original | Augmentative | Meaning |
|----------|-------------|---------|
| libro | libr**ote** | thick/big book |
| palabra | palabr**ota** | swear word |
| grande | grand**ote** | very big |
| amigo | amig**ote** | close friend |
| perro | perr**ote** | big dog |

### -azo / -aza (Large, Impressive, or a Blow)

| Original | Augmentative | Meaning |
|----------|-------------|---------|
| coche | coch**azo** | awesome car |
| gol | gol**azo** | great/amazing goal |
| fiesta | fiest**azo** | great party |

**-azo indicating a blow/hit:**

| Noun | -azo Form | Meaning |
|------|-----------|---------|
| puño | puñ**etazo** | punch |
| codo | cod**azo** | elbow hit |
| cabeza | cabez**azo** | headbutt |
| rodilla | rodill**azo** | knee hit |
| zapato | zapat**azo** | shoe hit |

### -ón / -ona (Large, Exaggerated)

| Original | Augmentative | Meaning |
|----------|-------------|---------|
| silla | sill**ón** | armchair |
| casa | cas**ona** | mansion |
| cuchara | cuchar**ón** | ladle |
| llorar → llor**ón/a** | crybaby |
| comer → comil**ón/a** | big eater |
| dormir → dormil**ón/a** | sleepyhead |
| mandar → mand**ón/a** | bossy person |

### Nuance Summary

| Use | Example | Effect |
|-----|---------|--------|
| Affection | mi hijito | Loving, warm |
| Small size | una casita | Physically small |
| Softening | ¿Me das un favorcito? | Reduces the burden |
| Emphasis | Está cerquita | Very close |
| Friendliness | ¿Tomamos un cafecito? | Inviting, warm |
| Large/impressive | ¡Qué golazo! | Admiration |
| Negative/contempt | Es un mandón | Bossy (pejorative) |

## 8. Word Formation Patterns

### Common Prefixes

| Prefix | Meaning | Examples |
|--------|---------|---------|
| des- | un-, reverse | deshacer (undo), descargar (download) |
| re- | again, very | rehacer (redo), releer (reread) |
| in-/im- | not | imposible, incómodo, increíble |
| pre- | before | predecir (predict), prevenir (prevent) |
| sobre- | over, excess | sobrevivir (survive), sobrepasar (surpass) |
| sub- | under | subrayar (underline), subterráneo |
| auto- | self | autocuidado (self-care), autoservicio |

### Common Suffixes for Nouns

| Suffix | Creates | Examples |
|--------|---------|---------|
| -ción/-sión | action/result | contaminación, depresión, producción |
| -miento | action/result | entrenamiento, agotamiento, calentamiento |
| -dad/-tad | quality | productividad, libertad, biodiversidad |
| -ero/-era | profession/container | camarero (waiter), zapatero (shoemaker), papelera (trash can) |
| -ista | person/ideology | periodista (journalist), ecologista |

### Common Suffixes for Adjectives

| Suffix | Creates | Examples |
|--------|---------|---------|
| -able/-ible | can be done | sostenible, biodegradable, reciclable |
| -oso/-osa | full of | peligroso, famoso, nervioso |
| -al | relating to | global, social, digital, mental |
| -ivo/-iva | tending to | productivo, creativo, sostenible |

## 9. Practice Exercises

### Exercise 1: Technology Vocabulary

Match the Spanish term with its English equivalent.

1. la contraseña        a) cloud
2. la nube              b) to download
3. descargar            c) password
4. la pantalla          d) screen
5. actualizar           e) to update

<details>
<summary>Answers</summary>

1-c, 2-a, 3-b, 4-d, 5-e

</details>

### Exercise 2: Form Diminutives

Create the diminutive (-ito/-ita) form.

1. casa → ________
2. poco → ________
3. amigo → ________
4. flor → ________
5. momento → ________

<details>
<summary>Answers</summary>

1. **casita** 2. **poquito** (c→qu) 3. **amiguito** (g→gu) 4. **florecita** (one syllable → -ecita) 5. **momentito**

</details>

### Exercise 3: Identify Augmentative Meaning

What does each augmentative mean?

1. golazo
2. palabrota
3. sillón
4. cochazo
5. dormilón

<details>
<summary>Answers</summary>

1. **Amazing/great goal** (admiration)
2. **Swear word / bad word** (coarse word)
3. **Armchair** (big chair — lexicalized)
4. **Awesome/big car** (impressive)
5. **Sleepyhead** (someone who sleeps a lot)

</details>

### Exercise 4: Environment Mini-Essay

Fill in the blanks with vocabulary from this lesson.

"El ________ ________ (climate change) es un problema global. Todos debemos ________ (reduce) nuestra ________ de ________ (carbon footprint). Podemos usar ________ ________ (renewable energy), ________ (recycle) y usar ________ ________ (reusable bags)."

<details>
<summary>Answers</summary>

"El **cambio climático** es un problema global. Todos debemos **reducir** nuestra **huella** de **carbono**. Podemos usar **energía renovable**, **reciclar** y usar **bolsas reutilizables**."

</details>

### Exercise 5: Word Formation

Use prefixes or suffixes to form new words.

1. hacer → ________ (to undo) — prefix: des-
2. posible → ________ (impossible) — prefix: im-
3. producir → ________ (productivity) — suffix: -dad
4. reciclar → ________ (recyclable) — suffix: -able
5. zapato → ________ (shoemaker) — suffix: -ero

<details>
<summary>Answers</summary>

1. **deshacer** 2. **imposible** 3. **productividad** 4. **reciclable** 5. **zapatero**

</details>
