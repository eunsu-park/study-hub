## 1. Imperative: Tú Commands

### Affirmative Tú Commands

For most verbs, the affirmative tú command = **third person singular present indicative**:

| Infinitive | Command | English |
|-----------|---------|---------|
| hablar | ¡**Habla**! | Speak! |
| comer | ¡**Come**! | Eat! |
| escribir | ¡**Escribe**! | Write! |
| leer | ¡**Lee**! | Read! |

### Irregular Tú Commands (8 verbs)

| Infinitive | Command | Mnemonic |
|-----------|---------|---------|
| decir | **di** | |
| hacer | **haz** | |
| ir | **ve** | |
| poner | **pon** | |
| salir | **sal** | |
| ser | **sé** | |
| tener | **ten** | |
| venir | **ven** | |

> Mnemonic: **ven di sal haz ten ve pon sé** — "Ven, di: sal, haz ten, ve, pon sé"

## 2. Imperative: Usted/Ustedes Commands

Formal commands use the **present subjunctive** form:

| Infinitive | Usted | Ustedes |
|-----------|-------|---------|
| hablar | ¡**Hable**! | ¡**Hablen**! |
| comer | ¡**Coma**! | ¡**Coman**! |
| escribir | ¡**Escriba**! | ¡**Escriban**! |
| ir | ¡**Vaya**! | ¡**Vayan**! |
| ser | ¡**Sea**! | ¡**Sean**! |

> **Pattern**: -AR verbs use -e/-en endings, -ER/-IR verbs use -a/-an endings.
> This is the "opposite vowel" rule — the same as subjunctive formation.

---

## 3. Imperative: Nosotros and Vosotros

### Nosotros Commands ("Let's...")

Use the **nosotros subjunctive** form:
- ¡**Hablemos**! (Let's speak!)
- ¡**Comamos**! (Let's eat!)
- ¡**Vamos**! (Let's go!) — irregular (not *vayamos* in affirmative)

### Vosotros Commands (Spain only)

Change the infinitive **-r** to **-d**:
- hablar → ¡**Hablad**! (Speak!)
- comer → ¡**Comed**! (Eat!)
- escribir → ¡**Escribid**! (Write!)

---

## 4. Negative Commands

**All** negative commands use the subjunctive:

| Person | hablar | comer |
|--------|--------|-------|
| tú | ¡No **hables**! | ¡No **comas**! |
| usted | ¡No **hable**! | ¡No **coma**! |
| nosotros | ¡No **hablemos**! | ¡No **comamos**! |
| vosotros | ¡No **habléis**! | ¡No **comáis**! |
| ustedes | ¡No **hablen**! | ¡No **coman**! |

## 5. Pronoun Placement with Commands

```
+----------------------------------------------------------+
|  AFFIRMATIVE commands: pronoun ATTACHED to the end        |
|  ¡Dime!        (Tell me!)                                |
|  ¡Levántate!   (Get up!)                                 |
|  ¡Dámelo!      (Give it to me!)                          |
|                                                          |
|  NEGATIVE commands: pronoun BEFORE the verb               |
|  ¡No me digas! (Don't tell me!)                          |
|  ¡No te levantes! (Don't get up!)                        |
|  ¡No me lo des!   (Don't give it to me!)                 |
+----------------------------------------------------------+
```

> When attaching pronouns, add an **accent mark** to maintain the original stress:
> di + me = dime (no accent needed), levanta + te = levántate (accent added).

## 6. What Is the Subjunctive?

The subjunctive is a **mood** (not a tense). While the indicative states facts,
the subjunctive expresses subjectivity:

```
+----------------------------------------------------------+
|  INDICATIVE (reality)         SUBJUNCTIVE (subjectivity)  |
|  ────────────────────         ──────────────────────────  |
|  States facts                 Expresses:                  |
|  Reports events               - Wishes and desires        |
|  Describes certainty           - Emotions and feelings     |
|                                - Doubt and uncertainty     |
|                                - Influence and commands    |
|                                - Hypothetical situations   |
|                                                          |
|  Sé que habla español.       Espero que hable español.   |
|  (I know he speaks...)       (I hope he speaks...)       |
+----------------------------------------------------------+
```

> **Key**: The subjunctive almost always appears in a **subordinate clause**
> (after "que") when the main clause expresses subjectivity.

## 7. Present Subjunctive Formation

### The "Opposite Vowel" Rule

1. Start with the **yo** form of the present indicative
2. Drop the **-o**
3. Add the "opposite" vowel endings

| | -AR (hablar) | -ER (comer) | -IR (vivir) |
|---|---|---|---|
| yo | habl**e** | com**a** | viv**a** |
| tú | habl**es** | com**as** | viv**as** |
| él/ella | habl**e** | com**a** | viv**a** |
| nosotros | habl**emos** | com**amos** | viv**amos** |
| vosotros | habl**éis** | com**áis** | viv**áis** |
| ellos | habl**en** | com**an** | viv**an** |

> -AR verbs use -e endings; -ER/-IR verbs use -a endings.

### Irregular Subjunctives (Yo-form Based)

Since formation starts from the **yo indicative**, verbs irregular in yo carry
that irregularity into the subjunctive:

| Infinitive | Yo indicative | Subjunctive (yo) |
|-----------|--------------|------------------|
| tener | teng**o** | teng**a** |
| venir | veng**o** | veng**a** |
| decir | dig**o** | dig**a** |
| hacer | hag**o** | hag**a** |
| poner | pong**o** | pong**a** |
| salir | salg**o** | salg**a** |
| conocer | conozc**o** | conozc**a** |

### Fully Irregular Subjunctives

| Infinitive | Subjunctive |
|-----------|-------------|
| ser | sea, seas, sea, seamos, seáis, sean |
| estar | esté, estés, esté, estemos, estéis, estén |
| ir | vaya, vayas, vaya, vayamos, vayáis, vayan |
| haber | haya, hayas, haya, hayamos, hayáis, hayan |
| saber | sepa, sepas, sepa, sepamos, sepáis, sepan |
| dar | dé, des, dé, demos, deis, den |

## 8. WEIRDO: When to Use Subjunctive

**WEIRDO** is a mnemonic for the main triggers:

| Letter | Category | Examples |
|--------|----------|---------|
| **W** | Wishes | Quiero que **vengas**. (I want you to come.) |
| **E** | Emotions | Me alegra que **estés** bien. (I'm glad you're well.) |
| **I** | Impersonal expressions | Es importante que **estudies**. (It's important that you study.) |
| **R** | Recommendations | Te recomiendo que **descanses**. (I recommend you rest.) |
| **D** | Doubt/Denial | Dudo que **sea** verdad. (I doubt it's true.) |
| **O** | Ojalá | Ojalá que **venga**. (I hope he/she comes.) |

### Key Impersonal Expressions (always + subjunctive)

| Expression | Example |
|------------|---------|
| Es necesario que... | Es necesario que **estudies**. |
| Es posible que... | Es posible que **llueva**. |
| Es importante que... | Es importante que **hables**. |
| Es mejor que... | Es mejor que **vayas**. |
| Es probable que... | Es probable que **venga**. |

---

## 9. Subjunctive with Conjunctions

Certain conjunctions **always** require the subjunctive:

| Conjunction | Meaning | Example |
|-------------|---------|---------|
| para que | so that | Hablo despacio **para que** me **entiendas**. |
| antes de que | before | Sal **antes de que** **llueva**. |
| a menos que | unless | Iré **a menos que** **llueva**. |
| sin que | without | Salió **sin que** nadie lo **supiera**. |
| con tal de que | provided that | Te ayudo **con tal de que** me **escuches**. |

Others take subjunctive only when referring to **future/uncertain** events:
- **cuando** + subjunctive (future): Cuando **llegues**, llámame. (When you arrive, call me.)
- **cuando** + indicative (habitual): Cuando **llego**, siempre como. (When I arrive, I always eat.)

## 10. Practice Exercises

### Exercise 1: Form Commands

Give the command form indicated.

1. (hablar, tú affirmative) → ________
2. (comer, usted affirmative) → ________
3. (ir, tú affirmative) → ________
4. (hacer, tú negative) → ________
5. (sentarse, tú affirmative) → ________
6. (decir + me, tú affirmative) → ________

<details>
<summary>Answers</summary>

1. ¡**Habla**! 2. ¡**Coma**! 3. ¡**Ve**! 4. ¡No **hagas**! 5. ¡**Siéntate**! 6. ¡**Dime**!

</details>

### Exercise 2: Indicative or Subjunctive?

Choose the correct form.

1. Sé que ella (habla / hable) español.
2. Quiero que tú (vienes / vengas) a la fiesta.
3. Es posible que (llueve / llueva) mañana.
4. Creo que él (tiene / tenga) razón.
5. No creo que ella (puede / pueda) venir.

<details>
<summary>Answers</summary>

1. **habla** (indicative — certainty with "sé que")
2. **vengas** (subjunctive — wish)
3. **llueva** (subjunctive — impersonal expression of possibility)
4. **tiene** (indicative — belief = certainty in affirmative)
5. **pueda** (subjunctive — negated belief = doubt)

</details>

### Exercise 3: Complete with Subjunctive

1. Espero que tú ________ (estar) bien.
2. Es necesario que nosotros ________ (ir) al médico.
3. Dudo que ellos ________ (saber) la respuesta.
4. Ojalá que ________ (hacer) buen tiempo mañana.

<details>
<summary>Answers</summary>

1. **estés** 2. **vayamos** 3. **sepan** 4. **haga**

</details>
