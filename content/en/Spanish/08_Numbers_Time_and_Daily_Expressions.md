| 55 | cincuenta **y** cinco |
| 67 | sesenta **y** siete |
| 78 | setenta **y** ocho |
| 83 | ochenta **y** tres |
| 99 | noventa **y** nueve |

### Tens

| 30 | treinta | 70 | setenta |
|----|---------|----|---------|
| 40 | cuarenta | 80 | ochenta |
| 50 | cincuenta | 90 | noventa |
| 60 | sesenta | | |

### Hundreds

| Number | Masculine | Feminine |
|--------|-----------|----------|
| 100 | cien / ciento | cien / ciento |
| 200 | doscientos | doscientas |
| 300 | trescientos | trescientas |
| 400 | cuatrocientos | cuatrocientas |
| 500 | quinientos | quinientas |
| 600 | seiscientos | seiscientas |
| 700 | setecientos | setecientas |
| 800 | ochocientos | ochocientas |
| 900 | novecientos | novecientas |

> **cien** vs. **ciento**: Use *cien* before nouns and before *mil/millones*.
> Use *ciento* when followed by other numbers: ciento dos (102).

### Large Numbers

| Number | Spanish |
|--------|---------|
| 1,000 | mil |
| 2,000 | dos mil |
| 10,000 | diez mil |
| 100,000 | cien mil |
| 1,000,000 | un millón (de + noun) |
| 2,000,000 | dos millones (de + noun) |

> **Millón** is a noun, so it requires **de** before another noun:
> - un millón **de** personas (one million people)
> - tres millones **de** euros

### Gender Agreement

**Uno** and hundreds (200-900) agree with the noun:
- **veintiún** libros / **veintiuna** páginas
- **doscientos** euros / **doscientas** personas

## 3. Ordinal Numbers

| Number | Masculine | Feminine |
|--------|-----------|----------|
| 1st | primero (primer) | primera |
| 2nd | segundo | segunda |
| 3rd | tercero (tercer) | tercera |
| 4th | cuarto | cuarta |
| 5th | quinto | quinta |
| 6th | sexto | sexta |
| 7th | séptimo | séptima |
| 8th | octavo | octava |
| 9th | noveno | novena |
| 10th | décimo | décima |

> **Primero** and **tercero** shorten to **primer** and **tercer** before masculine
> singular nouns: el **primer** día, el **tercer** piso.

> Beyond 10th, Spanish typically uses cardinal numbers after the noun:
> el piso **once** (the 11th floor), el siglo **veintiuno** (the 21st century).

## 4. Telling Time

### Basic Structure

```
+----------------------------------------------------------+
|              TELLING TIME IN SPANISH                      |
+----------------------------------------------------------+
|                                                          |
|  1:00  Es la una.           (singular: "es la")          |
|  2:00  Son las dos.         (plural: "son las")          |
|  3:15  Son las tres y cuarto.                            |
|  4:30  Son las cuatro y media.                           |
|  5:45  Son las seis menos cuarto.                        |
|        (= Faltan quince para las seis.)                  |
|  6:10  Son las seis y diez.                              |
|  7:50  Son las ocho menos diez.                          |
|                                                          |
+----------------------------------------------------------+
```

### Time of Day

| Spanish | English |
|---------|---------|
| de la mañana | in the morning (AM) |
| de la tarde | in the afternoon (PM, ~1-7) |
| de la noche | in the evening/night (PM, ~8-12) |
| del mediodía | at noon |
| de la medianoche | at midnight |

**Examples:**
- Son las ocho **de la mañana**. (It's 8 AM.)
- Son las tres **de la tarde**. (It's 3 PM.)
- Es la una **de la tarde**. (It's 1 PM.)

### Asking and Answering

- **¿Qué hora es?** (What time is it?)
- **¿A qué hora...?** (At what time...?)
  - ¿A qué hora empieza la clase? — **A las nueve.** (At nine.)

## 7. Expressing Dates

### Format

```
el + number + de + month + de + year

el 15 de marzo de 2026
(March 15, 2026)
```

- **Exception**: The first of the month uses *primero*:
  - el **primero** de enero (January 1st)
  - But: el **dos** de enero (January 2nd)

### Asking the Date

- **¿Qué fecha es hoy?** (What's today's date?)
- **¿A cuántos estamos?** (What's the date?)
- **Hoy es el 24 de febrero de 2026.** (Today is February 24, 2026.)

### Years

- **2026**: dos mil veintiséis
- **1999**: mil novecientos noventa y nueve
- **2000**: dos mil (el año dos mil)

## 9. Expressing Frequency

| Spanish | English |
|---------|---------|
| siempre | always |
| casi siempre | almost always |
| todos los días | every day |
| a menudo / frecuentemente | often / frequently |
| a veces | sometimes |
| de vez en cuando | once in a while |
| pocas veces / raramente | rarely |
| casi nunca | almost never |
| nunca / jamás | never |

**Frequency with Time:**
- tres veces **al** día (three times a day)
- una vez **a la** semana (once a week)
- dos veces **al** mes (twice a month)
- cada año (every year)

Write the time in Spanish.

1. 3:00 PM → ________
2. 1:30 PM → ________
3. 8:15 AM → ________
4. 11:45 PM → ________
5. 12:00 (noon) → ________

<details>

Write in Spanish.

1. March 15, 2026 → ________

3. December 25 → ________

3. el veinticinco de diciembre

</details>
