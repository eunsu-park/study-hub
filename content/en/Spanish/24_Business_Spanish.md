## 1. Formal Register Overview

Business Spanish uses **usted/ustedes** and specific formal structures:

```
+----------------------------------------------------------+
|          FORMALITY SPECTRUM                                |
+----------------------------------------------------------+
|                                                          |
|  INFORMAL              NEUTRAL              FORMAL       |
|  ─────────────────────────────────────────────────────   |
|  tú                    tú/usted             usted        |
|  ¡Hola!                Buenos días.          Estimado/a   |
|  vale                  de acuerdo            conforme     |
|  ¡Genial!              Muy bien.             Excelente.   |
|  ¡Chao!                Hasta luego.          Atentamente  |
|                                                          |
|  WHEN TO USE FORMAL REGISTER:                            |
|  • First contacts and initial meetings                   |
|  • Emails to superiors or external contacts              |
|  • Presentations and official events                     |
|  • Legal and contractual contexts                        |
+----------------------------------------------------------+
```

### Key Formal Expressions

| Informal | Formal |
|----------|--------|
| ¿Puedes...? | ¿Podría usted...? / ¿Sería tan amable de...? |
| Quiero... | Quisiera... / Me gustaría... |
| ¿Qué piensas? | ¿Cuál es su opinión al respecto? |
| Gracias. | Le agradezco sinceramente. |
| Lo siento. | Le pido disculpas. / Lamento la inconveniencia. |
| OK. | De acuerdo. / Conforme. |

## 2. Business Email Writing

### Email Structure

| Section | Spanish | Example |
|---------|---------|---------|
| **Subject** | Asunto: | Asunto: Propuesta de colaboración |
| **Greeting** | Estimado/a + name/title: | Estimada Sra. López: |
| **Opening** | Me dirijo a usted para... | Me dirijo a usted en relación con... |
| **Body** | Clear, structured paragraphs | Le informo de que... / Adjunto... |
| **Closing** | Quedo a la espera de... | Quedo a su disposición para... |
| **Sign-off** | Atentamente, | Un cordial saludo, |
| **Signature** | Full name + title | Carlos Martínez, Director de Marketing |

### Opening Phrases

| Purpose | Phrase |
|---------|--------|
| First contact | Me dirijo a usted por primera vez para... |
| Follow-up | En relación con nuestra conversación del [fecha]... |
| Response | Acuso recibo de su correo del [fecha]... |
| Request | Le escribo para solicitar información sobre... |
| Complaint | Me pongo en contacto con usted para expresar mi disconformidad con... |
| Thank you | Le agradezco el envío de... |

### Closing Phrases

| Purpose | Phrase |
|---------|--------|
| Awaiting reply | Quedo a la espera de su respuesta. |
| Offering help | No dude en contactarme si necesita más información. |
| Availability | Quedo a su entera disposición. |
| Thanks | Le agradezco de antemano su atención. |
| Urgent | Le ruego nos responda a la mayor brevedad posible. |

### Attachments and References

| Spanish | English |
|---------|---------|
| Adjunto (encontrará)... | Attached (you will find)... |
| Le remito... | I am forwarding... |
| Según lo acordado... | As agreed... |
| Con referencia a... | With reference to... |
| Tal y como le comenté... | As I mentioned to you... |

## 3. Meetings and Presentations

### Meeting Vocabulary

| Spanish | English |
|---------|---------|
| la reunión | meeting |
| el orden del día | agenda |
| el acta (de la reunión) | minutes (of the meeting) |
| la sala de reuniones / juntas | meeting room |
| convocar una reunión | to call a meeting |
| asistir a una reunión | to attend a meeting |
| aplazar / posponer | to postpone |
| cancelar | to cancel |
| el punto (del orden del día) | item (on the agenda) |
| llegar a un acuerdo | to reach an agreement |
| tomar una decisión | to make a decision |
| votar | to vote |

### Conducting a Meeting

| Phase | Expressions |
|-------|------------|
| **Opening** | Empecemos / Comencemos la reunión. (Let's begin.) |
| | El objetivo de esta reunión es... (The goal of this meeting is...) |
| | ¿Podemos comenzar? (Shall we begin?) |
| **Presenting a topic** | El primer punto del orden del día es... (The first agenda item is...) |
| | Me gustaría hablar sobre... (I'd like to talk about...) |
| **Giving opinions** | Desde mi punto de vista... (From my point of view...) |
| | Propongo que... (I propose that...) |
| | En mi opinión... (In my opinion...) |
| **Asking for input** | ¿Cuál es su opinión? (What is your opinion?) |
| | ¿Qué opina(n) al respecto? (What do you think about this?) |
| | ¿Alguien tiene algo que añadir? (Does anyone have anything to add?) |
| **Agreeing** | Estoy completamente de acuerdo. (I completely agree.) |
| | Comparto su punto de vista. (I share your viewpoint.) |
| **Disagreeing** | Con todo respeto, discrepo. (With all due respect, I disagree.) |
| | Tengo algunas reservas al respecto. (I have some reservations.) |
| | Entiendo su postura, pero... (I understand your position, but...) |
| **Closing** | Para resumir... / En resumen... (To summarize...) |
| | ¿Hay alguna pregunta? (Are there any questions?) |
| | Gracias por su participación. (Thank you for your participation.) |

### Presentation Vocabulary

| Spanish | English |
|---------|---------|
| la presentación | presentation |
| la diapositiva | slide |
| el gráfico / la gráfica | chart, graph |
| la cifra / el dato | figure, data point |
| el porcentaje | percentage |
| el aumento / el incremento | increase |
| la disminución / la reducción | decrease |
| el resultado | result |
| la tendencia | trend |
| la conclusión | conclusion |

### Describing Trends

| Direction | Expressions |
|-----------|------------|
| **Increase** | aumentar, crecer, subir, incrementarse |
| | Las ventas **han aumentado** un 15%. (Sales have increased 15%.) |
| **Decrease** | disminuir, bajar, reducirse, descender |
| | Los costes **se han reducido** significativamente. |
| **Stable** | mantenerse estable, permanecer constante |
| **Fluctuate** | fluctuar, variar |

## 4. Telephone Calls

### Business Phone Protocol

| Phase | Expression |
|-------|-----------|
| **Answering** | Buenos días, [empresa], ¿en qué puedo ayudarle? |
| **Identifying yourself** | Soy [nombre] de [empresa]. / Le llamo de parte de [empresa]. |
| **Requesting someone** | ¿Podría comunicarme con el Sr./la Sra. [nombre]? |
| | Quisiera hablar con el departamento de... |
| **Connecting** | Le paso con... / Un momento, le comunico. |
| **Unavailable** | Lo siento, no se encuentra en este momento. |
| | ¿Desea dejar un mensaje? |
| | Le devolverá la llamada en cuanto sea posible. |
| **Message** | ¿Podría decirle que me llame al [número]? |
| | ¿Sería tan amable de transmitirle que...? |
| **Problems** | Lo siento, no le oigo bien. ¿Podría repetir? |
| | La línea está ocupada. / Está comunicando. |
| **Closing** | Le agradezco su tiempo. / Ha sido un placer. |

## 5. Negotiations

### Negotiation Vocabulary

| Spanish | English |
|---------|---------|
| negociar | to negotiate |
| la propuesta | proposal |
| la oferta | offer |
| la contraoferta | counteroffer |
| las condiciones | terms |
| el acuerdo | agreement |
| el contrato | contract |
| la cláusula | clause |
| el compromiso | commitment, compromise |
| la concesión | concession |
| el plazo | deadline, term |
| el presupuesto | budget |
| el margen | margin |
| rentable | profitable |

### Negotiation Phrases

| Function | Expression |
|----------|-----------|
| **Proposing** | Nos gustaría proponer... (We would like to propose...) |
| | ¿Qué les parecería si...? (How would it be if...?) |
| **Conditions** | Estaríamos dispuestos a... siempre y cuando... (We would be willing to... as long as...) |
| | Con la condición de que... (On the condition that...) |
| **Asking for concession** | ¿Podrían considerar...? (Could you consider...?) |
| | ¿Sería posible...? (Would it be possible...?) |
| **Accepting** | Nos parece aceptable. (That seems acceptable.) |
| | Estamos de acuerdo con esas condiciones. |
| **Rejecting politely** | Lamentablemente, no nos es posible aceptar esa propuesta. |
| | Tendríamos que rechazar esa oferta. |
| **Compromising** | ¿Podríamos encontrar un punto intermedio? (Could we find a middle ground?) |
| **Closing a deal** | Entonces, ¿tenemos un acuerdo? (So, do we have a deal?) |
| | Procederemos a redactar el contrato. (We'll proceed to draft the contract.) |

## 8. Numbers in Business

### Financial Figures

| Spanish | English |
|---------|---------|
| mil (1.000) | one thousand |
| diez mil (10.000) | ten thousand |
| cien mil (100.000) | one hundred thousand |
| un millón (1.000.000) | one million |
| mil millones (1.000.000.000) | one billion |
| un billón (1.000.000.000.000) | one trillion |

> **Warning**: Spanish **billón** = English **trillion** (10^12).
> English "billion" = Spanish **mil millones** (10^9).

### Percentages and Trends

| Spanish | English |
|---------|---------|
| un aumento del 10% | a 10% increase |
| una caída del 5% | a 5% drop |
| se duplicó | it doubled |
| se triplicó | it tripled |
| representar el 30% | to represent 30% |
| crecer un 20% respecto al año anterior | to grow 20% compared to the previous year |

## 9. Practice Exercises

### Exercise 1: Formal Email

Write a formal email requesting information about a training course. Include:
- Proper greeting and opening
- Your request
- Formal closing

<details>
<summary>Sample Answer</summary>

> Estimados señores:
>
> Me dirijo a ustedes para solicitar información sobre los cursos de formación en gestión de proyectos que ofrecen.
>
> Me gustaría conocer las fechas disponibles, la duración del curso, el precio y si ofrecen algún tipo de descuento para empresas.
>
> Quedo a la espera de su respuesta. No duden en contactarme en este correo electrónico o en el teléfono 612 345 678.
>
> Atentamente,
> María García López
> Departamento de Recursos Humanos

</details>

### Exercise 2: Meeting Expressions

What would you say in each situation?

1. You want to start the meeting.
2. You disagree politely with a colleague.
3. You want to ask for someone's opinion.
4. You want to summarize and close the meeting.

<details>
<summary>Answers</summary>

1. **Empecemos la reunión. El objetivo de hoy es...**
2. **Con todo respeto, tengo una opinión diferente...** or **Entiendo su postura, pero...**
3. **¿Cuál es su opinión al respecto?** or **¿Qué opina usted?**
4. **Para resumir los puntos principales... Gracias por su participación.**

</details>

### Exercise 3: Translate Business Terms

1. revenue → ________
2. budget → ________
3. deadline → ________
4. market share → ________
5. job interview → ________

<details>
<summary>Answers</summary>

1. **los ingresos** 2. **el presupuesto** 3. **el plazo / la fecha límite** 4. **la cuota de mercado** 5. **la entrevista de trabajo**

</details>

### Exercise 4: Billion Trap

Translate into Spanish numbers:

1. $2.5 million → ________
2. $1 billion → ________
3. A 15% increase → ________

<details>
<summary>Answers</summary>

1. 2,5 millones de dólares (note: comma for decimal)
2. Mil millones de dólares (NOT "un billón" — that's a trillion!)
3. Un aumento del 15% / Un incremento del quince por ciento

</details>

### Exercise 5: Interview Response

Write a brief response to: "Hábleme de usted." (Tell me about yourself.)

<details>
<summary>Sample Answer</summary>

"Buenos días. Soy licenciado/a en Administración de Empresas y cuento con cinco años de experiencia en el sector de marketing digital. En mi puesto anterior, fui responsable de gestionar las campañas de publicidad en redes sociales, logrando un aumento del 30% en el engagement de la marca. Me considero una persona organizada, creativa y con capacidad de trabajo en equipo. Me interesa este puesto porque me gustaría desarrollar mi carrera profesional en una empresa innovadora como la suya."

</details>
