## 1. Gender Agreement

Spanish adjectives must match the noun they modify in **gender** (masculine/feminine)
and **number** (singular/plural). This is called *concordancia*.

### Pattern 1: -o / -a Adjectives (Most Common)

| Masculine | Feminine | English |
|-----------|----------|---------|
| alto | alta | tall |
| bajo | baja | short |
| bonito | bonita | pretty |
| pequeño | pequeña | small |
| guapo | guapa | handsome/pretty |
| delgado | delgada | thin |
| gordo | gorda | fat |
| rico | rica | rich |

> **Rule**: Change the final **-o** to **-a** for feminine.

### Pattern 2: Invariable Adjectives (-e or consonant)

These adjectives have the **same form** for masculine and feminine:

| Adjective | English | Example |
|-----------|---------|---------|
| grande | big | un hombre **grande** / una casa **grande** |
| inteligente | intelligent | un chico **inteligente** / una chica **inteligente** |
| triste | sad | el niño **triste** / la niña **triste** |
| alegre | happy | un día **alegre** / una tarde **alegre** |
| fácil | easy | un examen **fácil** / una tarea **fácil** |
| difícil | difficult | un problema **difícil** / una pregunta **difícil** |
| joven | young | un hombre **joven** / una mujer **joven** |

### Pattern 3: Nationality Adjectives (-consonant)

Nationality adjectives ending in a consonant **add -a** for feminine:

| Masculine | Feminine | English |
|-----------|----------|---------|
| español | español**a** | Spanish |
| francés | frances**a** | French |
| alemán | aleman**a** | German |
| inglés | ingles**a** | English |
| japonés | japones**a** | Japanese |

> **Note**: The accent mark is dropped in the feminine (francés → francesa).

## 2. Number Agreement

### Vowel Ending → Add -s

| Singular | Plural | English |
|----------|--------|---------|
| alto | alto**s** | tall |
| alta | alta**s** | tall (f.) |
| grande | grande**s** | big |
| inteligente | inteligente**s** | intelligent |

### Consonant Ending → Add -es

| Singular | Plural | English |
|----------|--------|---------|
| fácil | fácil**es** | easy |
| difícil | difícil**es** | difficult |
| español | español**es** | Spanish |
| joven | jóven**es** | young |

### -z Ending → Change to -ces

| Singular | Plural | English |
|----------|--------|---------|
| feliz | feli**ces** | happy |
| capaz | capa**ces** | capable |
| feroz | fero**ces** | fierce |

## 3. Complete Agreement Pattern

```
+----------------------------------------------------------+
|              ADJECTIVE AGREEMENT SUMMARY                  |
+----------------------------------------------------------+
|                                                          |
|  Adjective Type     M.sg   F.sg    M.pl     F.pl        |
|  ────────────────   ─────  ─────   ──────   ──────      |
|  -o / -a            -o     -a      -os      -as         |
|   (alto)            alto   alta    altos    altas       |
|                                                          |
|  -e (invariable)    -e     -e      -es      -es         |
|   (grande)          grande grande  grandes  grandes     |
|                                                          |
|  consonant          -∅     -a      -es      -as         |
|   (español)         español española españoles españolas|
|                                                          |
|  -z                 -z     -z      -ces     -ces        |
|   (feliz)           feliz  feliz   felices  felices     |
|                                                          |
+----------------------------------------------------------+
```

### Mixed-Gender Groups

When an adjective modifies a group of **mixed gender**, use the **masculine plural**:

- Los chicos y las chicas son **altos**. (The boys and girls are tall.)
- Los profesores y las profesoras son **buenos**. (The male and female teachers are good.)

## 4. Adjective Placement

### Default: After the Noun

Most adjectives come **after** the noun they modify. This is the opposite of English:

| Spanish | English |
|---------|---------|
| un coche **rojo** | a red car |
| una casa **grande** | a big house |
| un libro **interesante** | an interesting book |
| una persona **amable** | a kind person |

### Before the Noun (Special Cases)

Some adjectives precede the noun for emphasis, stylistic effect, or because
they express an inherent quality:

| Category | Examples |
|----------|---------|
| Quantity | mucho, poco, otro, varios, cada |
| Order | primero, segundo, último, próximo |
| Inherent quality | la blanca nieve (the white snow) |
| Emphasis/affection | ¡Qué bonita casa! |

### Shortened Forms (Apocopation)

Some adjectives lose their final vowel when placed **before a masculine singular noun**:

| Full Form | Shortened | Example |
|-----------|-----------|---------|
| bueno | **buen** | un **buen** amigo (a good friend) |
| malo | **mal** | un **mal** día (a bad day) |
| primero | **primer** | el **primer** piso (the first floor) |
| tercero | **tercer** | el **tercer** libro (the third book) |
| grande | **gran** | un **gran** hombre (a great man) |

> **Note**: *grande* shortens to *gran* before **both** masculine and feminine singular nouns:
> un **gran** actor / una **gran** actriz.

## 5. Meaning Changes by Position

Some adjectives change meaning depending on whether they come before or after the noun:

| Adjective | Before Noun | After Noun |
|-----------|-------------|------------|
| **grande** | great, important: *un **gran** hombre* (a great man) | big, large: *un hombre **grande*** (a big man) |
| **pobre** | unfortunate: *el **pobre** niño* (the poor [pitiful] child) | not wealthy: *el niño **pobre*** (the poor [broke] child) |
| **viejo** | longstanding: *un **viejo** amigo* (an old [longtime] friend) | elderly: *un amigo **viejo*** (an old [aged] friend) |
| **nuevo** | another, different: *un **nuevo** coche* (a new [different] car) | brand-new: *un coche **nuevo*** (a new [just made] car) |
| **único** | only: *el **único** problema* (the only problem) | unique: *un problema **único*** (a unique problem) |
| **mismo** | same: *el **mismo** libro* (the same book) | -self (emphasis): *el libro **mismo*** (the book itself) |
| **cierto** | certain (some): *una **cierta** sensación* (a certain feeling) | true, sure: *una noticia **cierta*** (a true piece of news) |
| **antiguo** | former: *mi **antiguo** profesor* (my former teacher) | ancient: *un edificio **antiguo*** (an ancient building) |

## 6. Comparatives

### More than (más... que)

- María es **más alta que** Juan. (María is taller than Juan.)
- Este libro es **más interesante que** ese. (This book is more interesting than that one.)

### Less than (menos... que)

- Juan es **menos alto que** María. (Juan is less tall than María.)
- Ese restaurante es **menos caro que** este. (That restaurant is less expensive than this one.)

### As... as (tan... como)

- Pedro es **tan inteligente como** Ana. (Pedro is as intelligent as Ana.)
- Mi casa es **tan grande como** la tuya. (My house is as big as yours.)

### As much/many as (tanto/a/os/as... como)

*Tanto* agrees with the noun it modifies:

- Tengo **tantos** libros **como** tú. (I have as many books as you.)
- No tengo **tanta** paciencia **como** ella. (I don't have as much patience as her.)

## 7. Superlatives

### Relative Superlative (the most / the least)

```
el/la/los/las + más/menos + adjective + de + group
```

- Es **la más alta de** la clase. (She is the tallest in the class.)
- Es **el menos caro de** todos. (It is the least expensive of all.)
- Son **los más importantes del** mundo. (They are the most important in the world.)

### Absolute Superlative (-ísimo/a)

Expresses an extreme degree without comparison ("very, very"):

| Adjective | Superlative | Meaning |
|-----------|-------------|---------|
| alto | alt**ísimo** | extremely tall |
| guapo | guap**ísimo** | extremely handsome |
| rico | riqu**ísimo** | extremely rich/delicious |
| largo | largu**ísimo** | extremely long |
| fácil | facil**ísimo** | extremely easy |
| difícil | dificil**ísimo** | extremely difficult |

> **Spelling changes**: rico → riqu**ísimo** (c→qu), largo → largu**ísimo** (g→gu),
> feliz → felic**ísimo** (z→c).

## 8. Irregular Comparatives and Superlatives

| Adjective | Comparative | Relative Superlative |
|-----------|-------------|---------------------|
| bueno (good) | **mejor** (better) | **el/la mejor** (the best) |
| malo (bad) | **peor** (worse) | **el/la peor** (the worst) |
| grande (big) | **mayor** (older/greater) | **el/la mayor** (the oldest/greatest) |
| pequeño (small) | **menor** (younger/lesser) | **el/la menor** (the youngest/least) |

### Usage Notes

- **Mayor/menor** refer to **age** (not physical size):
  - Mi hermano **mayor** (my older brother) — NOT my bigger brother
  - Mi hermana **menor** (my younger sister)
- For physical size, use **más grande / más pequeño**:
  - Esta casa es **más grande** que la otra. (This house is bigger.)

## 10. Practice Exercises

### Exercise 1: Gender and Number Agreement

Make the adjective agree with the noun.

1. Las casas ________ (blanco)
2. Un hombre ________ (español)
3. Una mujer ________ (francés)
4. Los problemas ________ (difícil)
5. Las chicas ________ (inteligente)
6. Un día ________ (feliz)
7. Unas flores ________ (bonito)

<details>
<summary>Answers</summary>

1. Las casas **blancas**
2. Un hombre **español**
3. Una mujer **francesa**
4. Los problemas **difíciles**
5. Las chicas **inteligentes**
6. Un día **feliz**
7. Unas flores **bonitas**

</details>

### Exercise 2: Placement and Shortened Forms

Fill in the adjective in the correct position (before or after the noun), using
shortened forms where appropriate.

1. Es un ________ amigo ________. (bueno)
2. Es el ________ piso ________. (primero)
3. Tiene un coche ________. (rojo)
4. Es una persona ________. (amable)
5. Es un ________ hombre ________. (grande — meaning "great")

<details>
<summary>Answers</summary>

1. Es un **buen** amigo. (before, shortened)
2. Es el **primer** piso. (before, shortened)
3. Tiene un coche **rojo**. (after — colors go after)
4. Es una persona **amable**. (after — descriptive)
5. Es un **gran** hombre. (before, shortened — means "great")

</details>

### Exercise 3: Comparatives and Superlatives

Translate to Spanish.

1. She is taller than her brother.
2. This movie is less interesting than the book.
3. He is as intelligent as his father.
4. It is the best restaurant in the city.
5. This coffee is extremely hot. (Use -ísimo)

<details>
<summary>Answers</summary>

1. Ella es **más alta que** su hermano.
2. Esta película es **menos interesante que** el libro.
3. Él es **tan inteligente como** su padre.
4. Es **el mejor** restaurante **de** la ciudad.
5. Este café está **calentísimo**.

</details>

### Exercise 4: Meaning Changes by Position

Translate each pair, paying attention to adjective position.

1. a) A great woman → ________
   b) A big woman → ________
2. a) The poor (pitiful) boy → ________
   b) The poor (not rich) boy → ________
3. a) My old (longtime) friend → ________
   b) My old (elderly) friend → ________

<details>
<summary>Answers</summary>

1. a) Una **gran** mujer (great) / b) Una mujer **grande** (big)
2. a) El **pobre** niño (pitiful) / b) El niño **pobre** (not rich)
3. a) Mi **viejo** amigo (longtime) / b) Mi amigo **viejo** (elderly)

</details>
