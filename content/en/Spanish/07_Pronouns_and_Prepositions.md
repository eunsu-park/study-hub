## 1. Subject Pronouns

| Person | Singular | Plural |
|--------|----------|--------|
| 1st | **yo** (I) | **nosotros/nosotras** (we) |
| 2nd informal | **tú** (you) | **vosotros/vosotras** (you all — Spain) |
| 2nd formal | **usted** (you) | **ustedes** (you all) |
| 3rd | **él/ella** (he/she) | **ellos/ellas** (they) |

> **Pro-drop**: Spanish is a pro-drop language — subject pronouns are usually
> **omitted** because the verb conjugation already indicates the subject.
> - *Hablo español.* (I speak Spanish.) — "yo" omitted
> - Use subject pronouns for **emphasis, contrast, or clarity**:
>   - **Yo** hablo español, pero **él** habla francés.

---

## 2. Direct Object Pronouns

Direct object pronouns replace the **thing or person** receiving the action directly.

| Person | Singular | Plural |
|--------|----------|--------|
| 1st | **me** (me) | **nos** (us) |
| 2nd informal | **te** (you) | **os** (you all — Spain) |
| 2nd formal | **lo/la** (you) | **los/las** (you all) |
| 3rd masc. | **lo** (him/it) | **los** (them) |
| 3rd fem. | **la** (her/it) | **las** (them) |

**Examples:**
- ¿Ves **el libro**? → ¿**Lo** ves? (Do you see it?)
- Compré **las flores**. → **Las** compré. (I bought them.)
- Conozco **a María**. → **La** conozco. (I know her.)

---

## 3. Indirect Object Pronouns

Indirect object pronouns indicate **to/for whom** the action is done.

| Person | Singular | Plural |
|--------|----------|--------|
| 1st | **me** (to me) | **nos** (to us) |
| 2nd informal | **te** (to you) | **os** (to you all) |
| 3rd | **le** (to him/her/you) | **les** (to them/you all) |

**Examples:**
- **Me** dio un regalo. (He gave me a gift.)
- **Le** escribí una carta. (I wrote him/her a letter.)
- **Nos** explicó la lección. (He explained the lesson to us.)

> **Clarification with A**: Since *le/les* are ambiguous, add *a + person* for clarity:
> - **Le** di el libro **a María**. (I gave the book to María.)

## 4. Pronoun Placement Rules

### Before Conjugated Verbs

Pronouns go **before** conjugated verbs:
- **Lo** veo. (I see him.)
- **Me** llama. (He calls me.)
- No **te** conozco. (I don't know you.)

### Attached to Infinitives, Gerunds, and Affirmative Commands

Pronouns can attach to the **end** of infinitives, gerunds, and affirmative commands:

| Structure | Example |
|-----------|---------|
| Infinitive | Quiero **verlo**. = **Lo** quiero ver. (I want to see it.) |
| Gerund | Estoy **haciéndolo**. = **Lo** estoy haciendo. (I'm doing it.) |
| Affirmative command | ¡**Dímelo**! (Tell it to me!) |

### Negative Commands: Before the Verb

- ¡No **me** lo digas! (Don't tell me!)
- ¡No **lo** hagas! (Don't do it!)

## 5. Double Object Pronouns

When both indirect and direct object pronouns appear together:

```
+----------------------------------------------------------+
|         DOUBLE PRONOUN ORDER                              |
+----------------------------------------------------------+
|                                                          |
|  Indirect Object  BEFORE  Direct Object                  |
|  (me, te, le, nos, os, les)  (lo, la, los, las)         |
|                                                          |
|  LE/LES + LO/LA/LOS/LAS  →  SE + LO/LA/LOS/LAS        |
|                                                          |
|  Example:                                                |
|  ¿Me das el libro?  →  ¿Me lo das?                      |
|  Le di las llaves.  →  Se las di. (NOT *le las)         |
|                                                          |
+----------------------------------------------------------+
```

**The LE → SE Rule**: When **le** or **les** would appear before **lo/la/los/las**,
change *le/les* to **se**:

- Le doy el libro. → **Se** lo doy. (I give it to him/her.)
- Les compré las flores. → **Se** las compré. (I bought them for them.)

## 6. Reflexive Pronouns

Reflexive pronouns indicate the subject performs the action **on itself**.

| Person | Pronoun | Example |
|--------|---------|---------|
| yo | **me** | Me levanto. (I get up.) |
| tú | **te** | Te duchas. (You shower.) |
| él/ella/Ud. | **se** | Se peina. (He/she combs his/her hair.) |
| nosotros | **nos** | Nos vestimos. (We get dressed.) |
| vosotros | **os** | Os sentáis. (You all sit down.) |
| ellos/Uds. | **se** | Se acuestan. (They go to bed.) |

### Common Reflexive Verbs

| Spanish | English | Spanish | English |
|---------|---------|---------|---------|
| levantarse | to get up | peinarse | to comb one's hair |
| acostarse | to go to bed | vestirse | to get dressed |
| ducharse | to shower | sentarse | to sit down |
| lavarse | to wash oneself | irse | to leave |
| cepillarse | to brush | llamarse | to be called |
| despertarse | to wake up | sentirse | to feel |

## 7. Possessive Adjectives and Pronouns

### Possessive Adjectives (before the noun)

| Person | Singular Noun | Plural Noun |
|--------|--------------|-------------|
| yo | **mi** | **mis** |
| tú | **tu** | **tus** |
| él/ella/Ud. | **su** | **sus** |
| nosotros | **nuestro/a** | **nuestros/as** |
| vosotros | **vuestro/a** | **vuestros/as** |
| ellos/Uds. | **su** | **sus** |

> Only *nuestro* and *vuestro* agree in gender with the noun.

### Possessive Pronouns (replace noun)

| Person | Masc. sg. | Fem. sg. | Masc. pl. | Fem. pl. |
|--------|-----------|----------|-----------|----------|
| yo | **mío** | **mía** | **míos** | **mías** |
| tú | **tuyo** | **tuya** | **tuyos** | **tuyas** |
| él/ella | **suyo** | **suya** | **suyos** | **suyas** |
| nosotros | **nuestro** | **nuestra** | **nuestros** | **nuestras** |

- ¿Es tu libro? → Sí, es **mío**. (Yes, it's mine.)
- Mi casa es grande. **La tuya** es pequeña. (Mine is big. Yours is small.)

## 8. Demonstrative Pronouns

| Distance | Masculine | Feminine | Neuter |
|----------|-----------|----------|--------|
| Near (this) | **este / estos** | **esta / estas** | **esto** |
| Medium (that) | **ese / esos** | **esa / esas** | **eso** |
| Far (that over there) | **aquel / aquellos** | **aquella / aquellas** | **aquello** |

- **Este** libro es interesante. (This book is interesting.)
- **Esa** casa es bonita. (That house is pretty.)
- **Aquella** montaña es alta. (That mountain over there is tall.)

> **Neuter** forms (esto, eso, aquello) refer to abstract ideas or unidentified things:
> - ¿Qué es **esto**? (What is this?)
> - **Eso** no es verdad. (That's not true.)

## 9. Essential Prepositions

### Simple Prepositions

| Preposition | Meaning | Example |
|-------------|---------|---------|
| **a** | to, at | Voy **a** la escuela. (I go to school.) |
| **de** | of, from | Soy **de** México. (I'm from Mexico.) |
| **en** | in, on, at | Estoy **en** casa. (I'm at home.) |
| **con** | with | Voy **con** mi amigo. (I go with my friend.) |
| **sin** | without | Café **sin** azúcar. (Coffee without sugar.) |
| **por** | by, for, through | Paso **por** el parque. (I pass through the park.) |
| **para** | for, in order to | Es **para** ti. (It's for you.) |
| **entre** | between | **Entre** tú y yo. (Between you and me.) |
| **sobre** | on, about | El libro está **sobre** la mesa. (on the table) |
| **hasta** | until, up to | Trabajo **hasta** las 5. (I work until 5.) |
| **desde** | from, since | Vivo aquí **desde** 2020. (since 2020) |
| **hacia** | toward | Camino **hacia** la playa. (toward the beach) |
| **según** | according to | **Según** él... (According to him...) |

### Contractions

Spanish has only **two** contractions:
- **a + el = al**: Voy **al** cine. (I go to the movies.)
- **de + el = del**: Vengo **del** trabajo. (I come from work.)

### Compound Prepositions

| Preposition | Meaning | Example |
|-------------|---------|---------|
| delante de | in front of | delante **de** la casa |
| detrás de | behind | detrás **de** ti |
| encima de | on top of | encima **de** la mesa |
| debajo de | under | debajo **de** la cama |
| al lado de | next to | al lado **de** la escuela |
| cerca de | near | cerca **de** aquí |
| lejos de | far from | lejos **de** la ciudad |

### Personal A

When the direct object is a **specific person**, add **a** before it:
- Conozco **a** María. (I know María.)
- Veo **a** mi profesor. (I see my teacher.)
- But: Veo **∅** la película. (I see the movie. — no *a* for things)

## 10. POR vs. PARA

This is one of the trickiest distinctions in Spanish. Both can translate to "for"
in English, but they have different uses.

### PARA — Destination, Purpose, Deadline

| Use | Example |
|-----|---------|
| **Destination** | Salgo **para** Madrid. (I'm leaving for Madrid.) |
| **Purpose** (in order to) | Estudio **para** aprender. (I study to learn.) |
| **Recipient** | Es **para** ti. (It's for you.) |
| **Deadline** | Es **para** mañana. (It's for tomorrow.) |
| **Comparison** | **Para** ser joven, sabe mucho. (For being young, he knows a lot.) |
| **Opinion** | **Para** mí, es fácil. (For me, it's easy.) |
| **Employment** | Trabajo **para** Google. (I work for Google.) |

### POR — Cause, Exchange, Duration, Movement Through

| Use | Example |
|-----|---------|
| **Cause/Reason** | Lo hice **por** amor. (I did it for/because of love.) |
| **Exchange** | Pagué 10€ **por** el libro. (I paid 10€ for the book.) |
| **Duration** | Estudié **por** dos horas. (I studied for two hours.) |
| **Movement through** | Caminamos **por** el parque. (We walked through the park.) |
| **Means** | Hablamos **por** teléfono. (We talked by phone.) |
| **On behalf of** | Lo hice **por** ti. (I did it on your behalf.) |
| **Per** | Tres veces **por** semana. (Three times per week.) |

```
+----------------------------------------------------------+
|              POR vs PARA SUMMARY                          |
+----------------------------------------------------------+
|                                                          |
|  PARA (→ forward-looking)     POR (← backward-looking)  |
|  ─────────────────────        ─────────────────────      |
|  Destination (going TO)       Movement THROUGH           |
|  Purpose (IN ORDER TO)        Cause/Reason (BECAUSE OF)  |
|  Recipient (FOR someone)      On behalf of (FOR someone) |
|  Deadline (BY when)           Duration (FOR how long)    |
|  Opinion (FOR me...)          Exchange (IN EXCHANGE for)  |
|  Employer (work FOR)          Means (BY phone/email)     |
|                                                          |
+----------------------------------------------------------+
```

## 11. Practice Exercises

### Exercise 1: Replace with Pronouns

Replace the underlined words with the correct pronoun.

1. Veo **a María** todos los días. → ________ veo todos los días.
2. Compré **el libro** ayer. → ________ compré ayer.
3. Dio **el regalo** **a su madre**. → ________ ________ dio.
4. Estoy leyendo **la novela**. → Estoy ________.

<details>
<summary>Answers</summary>

1. **La** veo todos los días. (direct object: a María → la)
2. **Lo** compré ayer. (direct object: el libro → lo)
3. **Se** **lo** dio. (indirect: a su madre → le → se; direct: el regalo → lo)
4. Estoy **leyéndola**. (or: **La** estoy leyendo.)

</details>

### Exercise 2: POR or PARA

Fill in with **por** or **para**.

1. Este regalo es ________ ti.
2. Caminamos ________ la playa.
3. Estudio español ________ viajar a España.
4. Pagué 50 euros ________ los zapatos.
5. Trabajo ________ una empresa internacional.
6. Hablamos ________ teléfono.
7. La tarea es ________ mañana.
8. Lo hice ________ amor.

<details>
<summary>Answers</summary>

1. **para** (recipient)
2. **por** (movement through)
3. **para** (purpose — in order to)
4. **por** (exchange)
5. **para** (employer)
6. **por** (means)
7. **para** (deadline)
8. **por** (cause/reason)

</details>

### Exercise 3: Choose the Correct Pronoun

1. ¿Conoces a Juan? — Sí, ________ conozco.
2. ________ llamo Carlos. (My name is Carlos.)
3. ¿Me das tu número? — Sí, ________ ________ doy.
4. Ella ________ levanta a las 7. (She gets up at 7.)
5. ¿Dónde están ________ llaves? (my)

<details>
<summary>Answers</summary>

1. **lo** (direct object — Juan = masc.)
2. **Me** (reflexive — llamarse)
3. **te lo** (indirect: a ti → te; direct: mi número → lo)
4. **se** (reflexive — levantarse)
5. **mis** (possessive adjective — plural)

</details>
