## 1. Spanish Punctuation

### Inverted Marks

Spanish is the only major language that uses **inverted question and exclamation marks** at the beginning of questions and exclamations:

```
+----------------------------------------------------------+
|          SPANISH PUNCTUATION MARKS                        |
+----------------------------------------------------------+
|                                                          |
|  Questions:    ¿...?                                     |
|  ¿Cómo te llamas? (What's your name?)                   |
|  ¿Dónde vives? (Where do you live?)                      |
|                                                          |
|  Exclamations: ¡...!                                     |
|  ¡Qué bonito! (How beautiful!)                           |
|  ¡No me digas! (No way!)                                 |
|                                                          |
|  IMPORTANT: The inverted mark goes where the question     |
|  or exclamation BEGINS (not necessarily at the start      |
|  of the sentence):                                       |
|                                                          |
|  Si vienes mañana, ¿puedes traer el libro?               |
|  Después de mucho trabajo, ¡por fin terminé!             |
+----------------------------------------------------------+
```

### Other Punctuation Differences

| Symbol | Spanish Rule | Example |
|--------|-------------|---------|
| « » or " " | Quotation marks (both are used) | «Hola», dijo María. / "Hola", dijo María. |
| — | Dialogue dash (replaces quotation marks in literature) | —¿Cómo estás? —preguntó Juan. |
| ; | Semicolon (same as English) | Estudié mucho; aprobé el examen. |
| : | Colon (same, plus after salutations) | Estimado señor: |
| ... | Ellipsis (same as English) | No sé... tal vez mañana. |

### Numbers and Dates

| Convention | Spanish | English |
|-----------|---------|---------|
| Decimal point | 3**,**5 (comma) | 3**.**5 (period) |
| Thousands separator | 1**.**000 (period) | 1**,**000 (comma) |
| Date format | 15/10/2024 (dd/mm/yyyy) | 10/15/2024 (mm/dd/yyyy) |
| Currency | 29,99 € (after number) | $29.99 (before number) |

## 2. Accent Marks (Tildes)

### Stress Rules

Spanish has predictable stress patterns. Accent marks indicate **exceptions** to the default rules:

```
+----------------------------------------------------------+
|              STRESS AND ACCENT RULES                      |
+----------------------------------------------------------+
|                                                          |
|  DEFAULT STRESS (no accent needed):                      |
|  • Words ending in vowel, -n, or -s → stress on          |
|    SECOND-TO-LAST syllable (palabra llana)                |
|    Examples: ca-SA, ha-BLAN, co-MES                      |
|                                                          |
|  • Words ending in consonant (except -n, -s) → stress    |
|    on LAST syllable (palabra aguda)                      |
|    Examples: ha-BLAR, co-MER, ciu-DAD                    |
|                                                          |
|  ACCENT MARK NEEDED (exceptions):                        |
|  • Agudas stressed on last + ending in vowel/n/s:        |
|    café, canción, inglés, habló                           |
|                                                          |
|  • Llanas stressed on 2nd-to-last + NOT ending in         |
|    vowel/n/s: árbol, lápiz, fácil, difícil               |
|                                                          |
|  • Esdrújulas (stress on 3rd-to-last): ALWAYS accented   |
|    teléfono, médico, número, último, rápido              |
|                                                          |
|  • Sobresdrújulas (4th-to-last): ALWAYS accented         |
|    dígamelo, tráemelo                                    |
+----------------------------------------------------------+
```

### Accent Marks that Distinguish Meaning

| Without accent | With accent |
|---------------|-------------|
| el (the) | él (he) |
| tu (your) | tú (you) |
| mi (my) | mí (me) |
| si (if) | sí (yes) |
| se (reflexive pronoun) | sé (I know / be [imperative]) |
| de (of, from) | dé (give [subjunctive]) |
| te (you, object) | té (tea) |
| mas (but, literary) | más (more) |
| solo (alone, adj.) | sólo (only — optional in modern use) |
| como (as, like) | cómo (how?) |
| donde (where, relative) | dónde (where?) |
| que (that, relative) | qué (what?) |
| quien (who, relative) | quién (who?) |
| cuando (when, relative) | cuándo (when?) |

> **Rule**: Question words ALWAYS have an accent mark, whether in direct
> or indirect questions: "No sé **cuándo** viene." (I don't know when he's coming.)

## 3. Capitalization Rules

Spanish capitalizes LESS than English:

| Capitalize in English | NOT capitalized in Spanish |
|----------------------|---------------------------|
| Days of the week | lunes, martes, miércoles... |
| Months | enero, febrero, marzo... |
| Nationalities/languages | español, mexicano, inglés |
| Religions | católico, musulmán, budista |
| "I" (pronoun) | yo (lowercase) |
| Book/movie titles (all major words) | Only first word + proper nouns |

### What IS Capitalized in Spanish

- First word of a sentence
- Proper nouns (names of people, places)
- Country names: España, México, Argentina
- Abbreviations: Ud., Sr., Dra.
- Titles of institutions: Real Academia Española

### Examples

| English | Spanish |
|---------|---------|
| Monday, January 15 | lunes, 15 de enero |
| I speak Spanish. | Yo hablo español. |
| The Lord of the Rings | El señor de los anillos |
| She is Catholic. | Ella es católica. |

## 4. Writing an Informal Email

### Structure

```
+----------------------------------------------------------+
|          INFORMAL EMAIL STRUCTURE                         |
+----------------------------------------------------------+
|                                                          |
|  Greeting:                                               |
|  ¡Hola, [nombre]! / Querido/a [nombre]: / ¡Hey!         |
|                                                          |
|  Opening:                                                |
|  ¿Qué tal? / ¿Cómo estás? / ¿Cómo va todo?             |
|  Espero que estés bien.                                  |
|                                                          |
|  Body:                                                   |
|  Te escribo para... / Quería contarte que...             |
|                                                          |
|  Closing:                                                |
|  Bueno, te dejo. / Escríbeme pronto.                     |
|                                                          |
|  Sign-off:                                               |
|  Un abrazo, / Besos, / ¡Nos vemos! / Un beso,           |
|  [Tu nombre]                                             |
+----------------------------------------------------------+
```

### Example

> ¡Hola, Laura!
>
> ¿Qué tal estás? Espero que todo vaya bien por ahí.
>
> Te escribo porque el próximo sábado es mi cumpleaños y voy a hacer una fiesta en mi casa. ¿Te apetece venir? Va a ser una fiesta informal, con música y comida. Empieza a las 20:00.
>
> ¡Ojalá puedas venir! Avísame cuando puedas.
>
> Un abrazo,
> Ana

## 5. Writing a Formal Email or Letter

### Structure

```
+----------------------------------------------------------+
|          FORMAL EMAIL / LETTER STRUCTURE                  |
+----------------------------------------------------------+
|                                                          |
|  Greeting:                                               |
|  Estimado/a Sr./Sra. [apellido]:                         |
|  Estimados señores:                                      |
|  A quien corresponda: (To whom it may concern)           |
|                                                          |
|  Opening:                                                |
|  Me dirijo a usted(es) para...                           |
|  Le(s) escribo con respecto a...                         |
|  En relación con su [carta/email/anuncio]...             |
|                                                          |
|  Body:                                                   |
|  Clear, organized paragraphs                             |
|  Usted/ustedes forms throughout                          |
|                                                          |
|  Closing:                                                |
|  Quedo a la espera de su respuesta.                      |
|  Le(s) agradezco su atención.                            |
|  No dude(n) en contactarme si necesita(n) más            |
|  información.                                            |
|                                                          |
|  Sign-off:                                               |
|  Atentamente, / Un cordial saludo, / Reciba un           |
|  cordial saludo,                                         |
|  [Nombre completo]                                       |
|  [Cargo/departamento]                                    |
+----------------------------------------------------------+
```

### Example

> Estimada Sra. López:
>
> Me dirijo a usted en relación con el puesto de asistente administrativo publicado en su página web el pasado 5 de marzo.
>
> Soy licenciado en Administración de Empresas por la Universidad Complutense de Madrid y cuento con tres años de experiencia en puestos similares. Adjunto mi currículum vitae para su consideración.
>
> Quedo a su disposición para una entrevista en el momento que le resulte conveniente. No dude en contactarme en el número 612 345 678 o en este correo electrónico.
>
> Atentamente,
> Carlos Martínez García

### Key Formal Expressions

| Purpose | Expression |
|---------|-----------|
| Opening | Me dirijo a usted para... (I am writing to you to...) |
| Purpose | El motivo de esta carta es... (The purpose of this letter is...) |
| Requesting | Le agradecería que... (I would appreciate it if...) |
| Attaching | Adjunto... (I attach...) |
| Availability | Quedo a su disposición. (I am at your disposal.) |
| Closing | A la espera de su respuesta... (Looking forward to your reply...) |
| Thanking | Le agradezco de antemano. (I thank you in advance.) |

## 6. Writing a Short Essay

### Essay Structure

```
+----------------------------------------------------------+
|            ESSAY STRUCTURE (ENSAYO)                       |
+----------------------------------------------------------+
|                                                          |
|  INTRODUCTION (Introducción):                            |
|  • Hook: question, quote, surprising fact                 |
|  • Present the topic                                     |
|  • State your thesis                                     |
|  Markers: Hoy en día..., En la actualidad...,            |
|           Es bien sabido que...                           |
|                                                          |
|  BODY (Desarrollo): 2-3 paragraphs                       |
|  • Each paragraph = one main idea                        |
|  • Support with examples, data, arguments                |
|  Markers: En primer lugar..., Además...,                 |
|           Por otro lado..., Sin embargo...               |
|                                                          |
|  CONCLUSION (Conclusión):                                |
|  • Summarize main points                                 |
|  • Restate thesis (different words)                      |
|  • Final thought or call to action                       |
|  Markers: En conclusión..., En resumen...,               |
|           En definitiva...                                |
+----------------------------------------------------------+
```

### Example Essay Outline

**Topic**: ¿Es importante aprender idiomas? (Is it important to learn languages?)

**Introduction**:
> En la actualidad, vivimos en un mundo cada vez más globalizado. Aprender un segundo idioma ya no es un lujo, sino una necesidad. En este ensayo, argumentaré que el aprendizaje de idiomas es fundamental tanto para el desarrollo personal como profesional.

**Body Paragraph 1** (Professional benefits):
> En primer lugar, hablar otro idioma abre puertas en el mercado laboral...

**Body Paragraph 2** (Personal benefits):
> Además, aprender un idioma permite conocer otras culturas...

**Body Paragraph 3** (Counterargument + rebuttal):
> Sin embargo, algunos argumentan que la tecnología de traducción hace innecesario el estudio de idiomas. No obstante, la comunicación humana va más allá de las palabras...

**Conclusion**:
> En definitiva, aprender idiomas es una inversión en nuestro futuro...

## 7. Common Errors by English Speakers

### Grammar Errors

| Error | Wrong | Correct | Rule |
|-------|-------|---------|------|
| Gender agreement | la problema | **el** problema | -ma nouns are often masculine |
| Ser vs estar | Soy cansado | **Estoy** cansado | Temporary states use estar |
| Double negative | No tengo algo | No tengo **nada** | Spanish requires double negatives |
| Possessive with body parts | Lavo mis manos | Me lavo **las** manos | Use article, not possessive |
| Word order | rojo coche | coche **rojo** | Adjective usually after noun |
| "It" as subject | It es importante | **Es** importante | Don't translate "it" as subject |

### Spelling Errors

| Error | Wrong | Correct | Note |
|-------|-------|---------|------|
| Missing ñ | espanol | espa**ñ**ol | ñ is a separate letter |
| Missing accents | cafe | caf**é** | Accents change meaning/stress |
| b/v confusion | aver | a **ver** | Sound the same in Spanish |
| h errors | aver (to see) | a ver / haber | h is silent |
| ll vs y | — | Regional variation | No spelling distinction needed |

### False Friend Traps in Writing

| What you mean | Don't write | Write |
|--------------|-------------|-------|
| to realize (understand) | realizar | **darse cuenta (de)** |
| actually | actualmente | **en realidad** / **de hecho** |
| to attend | atender | **asistir a** |
| library | librería | **biblioteca** |
| to introduce (a person) | introducir | **presentar** |
| currently | corrientemente | **actualmente** |
| to support | soportar | **apoyar** |

## 8. Connecting Ideas in Writing

### Useful Connectors by Function

| Function | Connectors |
|----------|-----------|
| **Adding** | además (furthermore), también (also), es más (what's more), incluso (even) |
| **Contrasting** | sin embargo (however), no obstante (nevertheless), en cambio (on the other hand), aunque (although) |
| **Cause** | porque (because), ya que (since), dado que (given that), debido a (due to) |
| **Effect** | por lo tanto (therefore), por eso (that's why), así que (so), en consecuencia (as a result) |
| **Sequencing** | en primer lugar (firstly), a continuación (next), luego (then), finalmente (finally) |
| **Clarifying** | es decir (that is), o sea (in other words), por ejemplo (for example) |
| **Concluding** | en conclusión (in conclusion), en resumen (in summary), en definitiva (ultimately) |

1. Como te llamas
2. Que bonito dia
3. Si vienes manana, puedes traer el libro
4. Donde esta la biblioteca
5. Fantastico, me encanta

2. **¡Qué** bonito **día!**
3. Si vienes **mañana**, **¿**puedes traer el libro**?**
4. **¿Dónde está** la biblioteca**?**

1. Estudio Español los Lunes y Miércoles.
2. Mi amigo es Católico y Mexicano.
3. El Señor De Los Anillos es mi película favorita.

<details>
<summary>Answers</summary>

2. Mi amigo es **católico** y **mexicano**.
3. **El señor de los anillos** es mi película favorita.

</details>

### Exercise 3: Write an Informal Email

Write a short email to a friend inviting them to a movie this weekend. Include:
- Greeting
- The invitation with details (movie, time, place)
- A closing

<details>
<summary>Sample Answer</summary>

> ¡Hola, Pedro!
>
> ¿Qué tal? ¿Tienes planes para este sábado? He visto que ponen una película nueva en el cine del centro y me apetece mucho ir a verla. Es a las 19:30.
>
> ¿Te animas? Después podemos cenar algo por ahí.
>
> ¡Avísame!
> Un abrazo,
> María

</details>

### Exercise 4: Correct the Errors

Find and fix the errors in each sentence.

1. Yo soy muy cansado después del trabajo.
2. La problema es que no tengo dinero.
3. Yo lavo mis manos antes de comer.
4. No quiero algo de comer.
5. El coche roja es muy rapido.

<details>
<summary>Answers</summary>

1. Yo **estoy** muy cansado... (temporary state → estar)
2. **El** problema es que... (problema is masculine)
3. **Me** lavo **las** manos... (reflexive + article with body parts)
4. No quiero **nada** de comer. (double negative required)
5. El coche **rojo** es muy **rápido**. (gender agreement + accent)

</details>

### Exercise 5: Essay Introduction

Write an introduction paragraph for this topic: "¿Son las redes sociales buenas o malas para la sociedad?" (Are social media good or bad for society?)

Use at least two discourse markers.

<details>
<summary>Sample Answer</summary>

"**Hoy en día**, las redes sociales forman parte de la vida cotidiana de millones de personas en todo el mundo. **Sin embargo**, existe un intenso debate sobre si su impacto en la sociedad es positivo o negativo. **En mi opinión**, las redes sociales tienen tanto ventajas como desventajas, y es importante aprender a usarlas de manera responsable."

</details>
