# Probability and Statistics Study Guide

## Introduction

Welcome to the **Probability and Statistics Study Guide**! This topic provides rigorous coverage of probability theory, statistical inference, and stochastic processes — bridging the gap between Data Science's applied statistics and Math for AI's ML-focused probability.

Prerequisites: Python (for computational examples)

---

## Learning Roadmap

The 18 lessons follow a structured progression:

### Phase 1: Probability Foundations (L01--L05)
Core probability theory from counting principles to multivariate distributions.

### Phase 2: Distribution Theory (L06--L09)
Discrete and continuous distribution families, transformations, and the multivariate normal.

### Phase 3: Convergence and Limit Theorems (L10--L11)
Convergence concepts, the Law of Large Numbers, and the Central Limit Theorem.

### Phase 4: Statistical Inference (L12--L15)
Point estimation, confidence intervals, hypothesis testing, and Bayesian methods.

### Phase 5: Applied Methods (L16--L18)
Nonparametric methods, regression/ANOVA, and an introduction to stochastic processes.

## How to Use This Guide

1. **Sequential study**: Lessons build on each other -- follow the numbering
2. **Practice with code**: Each lesson has Python examples using only the standard library
3. **Cross-reference**: See Data_Science for applied methods, Math_for_AI for ML applications
4. **Exercises**: Complete the exercises in `exercises/Probability_and_Statistics/`
