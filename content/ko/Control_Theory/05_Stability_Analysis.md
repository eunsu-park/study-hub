# 레슨 5: 안정성 분석(Stability Analysis)

## 학습 목표

- LTI 시스템에 대한 BIBO 안정성(BIBO stability)과 점근적 안정성(asymptotic stability)을 정의한다
- 라우스-허위츠 판별법(Routh-Hurwitz criterion)을 적용하여 특성 다항식으로부터 안정성을 판별한다
- 라우스 배열(Routh array)을 사용하여 안정성을 위한 파라미터 범위를 구한다
- 한계 안정(marginally stable) 시스템과 불안정(unstable) 시스템을 구별한다
- 극점 위치와 안정성 사이의 연관성을 이해한다

## 1. 안정성 개념(Stability Concepts)

안정성은 모든 제어 시스템에서 **가장 근본적인 요구 사항**이다. 불안정한 시스템은 최소한 쓸모가 없고, 최악의 경우 위험하다.

### 1.1 BIBO 안정성(BIBO Stability)

**유계 입력 유계 출력(bounded-input, bounded-output, BIBO) 안정**이란 모든 유계 입력이 유계 출력을 생성하는 것을 말한다:

$$|u(t)| \leq M_u < \infty \quad \Rightarrow \quad |y(t)| \leq M_y < \infty$$

임펄스 응답 $g(t)$를 갖는 LTI 시스템에서 BIBO 안정성은 다음과 동치이다:

$$\int_0^\infty |g(t)| \, dt < \infty$$

### 1.2 점근적 안정성(Asymptotic Stability)

**점근적 안정(asymptotically stable)**한 시스템은 자유 응답(영 입력)이 0으로 감쇠한다:

$$\lim_{t \to \infty} y_{\text{free}}(t) = 0$$

### 1.3 극점 위치에 의한 안정성(Stability from Pole Locations)

LTI 시스템에서는 두 조건 모두 간단한 극점 검사로 귀결된다:

| 안정성 | 조건 |
|-----------|-----------|
| **점근적 안정** | 모든 극점이 열린 좌반평면(open left half-plane)에 위치: $\text{Re}(p_i) < 0 \; \forall i$ |
| **한계 안정(Marginally stable)** | 우반평면(RHP)에 극점 없음, 허수축(imaginary axis) 위에 단순 극점이 하나 이상 존재 |
| **불안정** | 우반평면(RHP)에 극점이 하나 이상 존재하거나, 허수축 위에 중복 극점이 존재 |

폐루프 안정성을 분석하기 위해 **특성 방정식(characteristic equation)**의 근을 분석한다:

$$1 + G(s)H(s) = 0 \quad \Leftrightarrow \quad \Delta(s) = 0$$

## 2. 라우스-허위츠 판별법(Routh-Hurwitz Criterion)

차수가 $>2$인 다항식의 근을 구하는 것은 일반적으로 어렵다. 라우스-허위츠 판별법은 **근을 직접 계산하지 않고** 안정성을 판별한다.

### 2.1 필요 조건(Necessary Condition)

**정리:** $\Delta(s) = a_n s^n + \cdots + a_1 s + a_0$의 모든 근이 음의 실부(negative real part)를 갖기 위한 필요 조건은 모든 계수 $a_i > 0$이어야 한다 ($a_n > 0$을 가정).

계수가 하나라도 0이거나 음수이면 시스템은 **불안정**하다. ($n \geq 3$에서 이는 필요 조건이지만 충분 조건은 아니다.)

### 2.2 라우스 배열 구성(Routh Array Construction)

특성 다항식이 다음과 같을 때:

$$\Delta(s) = a_n s^n + a_{n-1} s^{n-1} + a_{n-2} s^{n-2} + \cdots + a_0$$

라우스 배열을 구성한다:

| $s^n$ | $a_n$ | $a_{n-2}$ | $a_{n-4}$ | $\cdots$ |
|-------|-------|-----------|-----------|----------|
| $s^{n-1}$ | $a_{n-1}$ | $a_{n-3}$ | $a_{n-5}$ | $\cdots$ |
| $s^{n-2}$ | $b_1$ | $b_2$ | $b_3$ | $\cdots$ |
| $s^{n-3}$ | $c_1$ | $c_2$ | $c_3$ | $\cdots$ |
| $\vdots$ | | | | |
| $s^0$ | | | | |

여기서:

$$b_1 = \frac{a_{n-1}a_{n-2} - a_n a_{n-3}}{a_{n-1}}, \quad b_2 = \frac{a_{n-1}a_{n-4} - a_n a_{n-5}}{a_{n-1}}, \quad \ldots$$

$$c_1 = \frac{b_1 a_{n-3} - a_{n-1} b_2}{b_1}, \quad \ldots$$

### 2.3 라우스-허위츠 안정성 판별법(Routh-Hurwitz Stability Criterion)

**정리 (라우스-허위츠):** 라우스 배열의 첫 번째 열에서 **부호 변화 횟수**가 $\Delta(s)$ 중 양의 실부를 갖는 근의 수와 같다.

**따름 정리:** 시스템이 안정하기 위한 필요충분 조건은 **첫 번째 열의 모든 원소가 양수**인 것이다 ($a_n > 0$을 가정).

### 2.4 예제

$$\Delta(s) = s^4 + 2s^3 + 3s^2 + 4s + 5$$

라우스 배열:

| 행 | 열 1 | 열 2 | 열 3 |
|-----|-------|-------|-------|
| $s^4$ | $1$ | $3$ | $5$ |
| $s^3$ | $2$ | $4$ | $0$ |
| $s^2$ | $\frac{2 \cdot 3 - 1 \cdot 4}{2} = 1$ | $\frac{2 \cdot 5 - 1 \cdot 0}{2} = 5$ | |
| $s^1$ | $\frac{1 \cdot 4 - 2 \cdot 5}{1} = -6$ | | |
| $s^0$ | $5$ | | |

첫 번째 열: $1, 2, 1, -6, 5$

부호 변화: $1 \to -6$과 $-6 \to 5$ → **2번의 부호 변화** → 우반평면(RHP) 근 2개 → **불안정**.

## 3. 라우스 배열의 특수 경우(Special Cases in the Routh Array)

### 3.1 첫 번째 열에 0이 있는 경우(Zero in the First Column)

어떤 행의 첫 번째 원소가 0이지만 그 행 전체가 0은 아닌 경우, 0을 작은 양수 $\epsilon > 0$으로 대체하여 계속 진행한다. 배열을 완성한 후 $\epsilon \to 0^+$일 때의 부호를 검사한다.

**예제:** $\Delta(s) = s^3 + s^2 + 2s + 2$

| $s^3$ | $1$ | $2$ |
| $s^2$ | $1$ | $2$ |
| $s^1$ | $\frac{1\cdot 2 - 1\cdot 2}{1} = 0 \to \epsilon$ | |
| $s^0$ | $2$ | |

첫 번째 열: $1, 1, \epsilon, 2$ — $\epsilon \to 0^+$에서 부호 변화 없음 → 안정?

실제로, 0 행은 **허수축 위에 근이 존재함**을 나타낸다: $\Delta(s) = (s^2+2)(s+1)$이므로 극점은 $s = \pm j\sqrt{2}$ → **한계 안정(marginally stable)**, 점근적 안정은 아님.

### 3.2 전체 행이 0인 경우(Entire Row of Zeros)

전체 행이 0이 되면, 특성 다항식이 **대칭적인 근 쌍(symmetric root pairs)** — 서로 반대 부호인 근들: $\pm\sigma$, $\pm j\omega$, 또는 $\pm\sigma \pm j\omega$ — 을 가짐을 나타낸다.

**절차:**
1. 0 행 **바로 위의** 행으로부터 **보조 다항식(auxiliary polynomial)** $P(s)$를 구성한다
2. 0 행을 $\frac{dP}{ds}$의 계수로 대체한다
3. 라우스 배열을 계속 진행한다

$P(s)$의 근이 대칭적인 근 쌍이며, 허수축 위의 근을 포함한다.

## 4. 라우스-허위츠를 이용한 안정 범위(Stability Ranges Using Routh-Hurwitz)

가장 강력한 응용 중 하나: 시스템이 안정하기 위한 파라미터(보통 이득 $K$)의 범위를 구하는 것이다.

### 4.1 예제: $K$의 안정 범위 구하기

$G(s) = \frac{K}{s(s+1)(s+5)}$를 갖는 단위 귀환 시스템.

특성 방정식: $s^3 + 6s^2 + 5s + K = 0$

라우스 배열:

| $s^3$ | $1$ | $5$ |
| $s^2$ | $6$ | $K$ |
| $s^1$ | $\frac{30 - K}{6}$ | |
| $s^0$ | $K$ | |

안정성을 위해 첫 번째 열의 모든 원소가 양수여야 한다:
- $6 > 0$ ✓
- $\frac{30-K}{6} > 0 \Rightarrow K < 30$
- $K > 0$

**안정 범위:** $0 < K < 30$.

$K = 30$에서: $s^1$ 행이 0이 됨 → **지속 진동을 갖는 한계 안정**. $s^2$ 행으로부터 구성되는 보조 다항식: $6s^2 + 30 = 0 \Rightarrow s = \pm j\sqrt{5}$. 진동 주파수는 $\omega = \sqrt{5}$ rad/s.

## 5. 허위츠 행렬식(Hurwitz Determinants) — 대체 공식

허위츠(Hurwitz) 판별법은 행렬식을 통해 동일한 정보를 제공한다. $a_n > 0$인 $\Delta(s) = a_n s^n + \cdots + a_0$에서 모든 근이 음의 실부를 갖기 위한 필요충분 조건은 **허위츠 행렬식(Hurwitz determinants)**이 모두 양수인 것이다:

$$D_1 = a_{n-1} > 0$$

$$D_2 = \begin{vmatrix} a_{n-1} & a_n \\ a_{n-3} & a_{n-2} \end{vmatrix} > 0$$

$$D_3 = \begin{vmatrix} a_{n-1} & a_n & 0 \\ a_{n-3} & a_{n-2} & a_{n-1} \\ a_{n-5} & a_{n-4} & a_{n-3} \end{vmatrix} > 0$$

낮은 차수의 시스템에서는 전체 라우스 배열보다 이 방법이 더 간단할 수 있다.

### 5.1 특수 경우

**2차** $s^2 + a_1 s + a_0$: $a_1 > 0$이고 $a_0 > 0$이면 안정.

**3차** $s^3 + a_2 s^2 + a_1 s + a_0$: $a_2 > 0$, $a_0 > 0$이고 $a_2 a_1 > a_0$이면 안정.

## 6. 상대적 안정성(Relative Stability)

라우스 판별법은 극점이 좌반평면에 있는지 여부만 알려준다. **상대적 안정성** — 극점이 허수축으로부터 얼마나 멀리 있는지 — 을 알기 위해서는 이동된 변수(shifted variable)를 사용할 수 있다.

**방법:** 모든 극점이 $\text{Re}(p_i) < -\sigma_0$를 만족하는지 확인하려면, $s = \hat{s} - \sigma_0$를 $\Delta(s)$에 대입하여 $\hat{s}$에 대한 새 다항식에 라우스 판별법을 적용한다.

이동된 라우스 배열의 첫 번째 열의 모든 원소가 양수이면, 원래의 모든 극점이 $\text{Re}(p_i) < -\sigma_0$를 만족한다.
