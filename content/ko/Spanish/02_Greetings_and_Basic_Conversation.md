## 1. 시간대별 인사

```
┌─────────────────────────────────────────────────────────────────┐
│                    시간대별 인사                                   │
│                                                                 │
│   06:00 ─────── 12:00 ─────── 20:00 ─────── 06:00              │
│   │  Buenos días  │ Buenas tardes │ Buenas noches  │            │
│   │  (좋은 아침)   │(좋은 오후)     │(좋은 저녁/     │            │
│   │               │               │  안녕히 주무세요)│            │
│                                                                 │
│   ※ Buenas noches는 인사와 작별 인사 모두로 사용됨               │
│   ※ 비격식 줄임말: ¡Buenas! (아무 때나 사용 가능)                │
└─────────────────────────────────────────────────────────────────┘
```

| 시간대 | 인사 | 용법 |
|------|----------|-------|
| 오전 (~6:00-12:00) | ¡Buenos días! | 격식 및 비격식 |
| 오후 (~12:00-20:00) | ¡Buenas tardes! | 격식 및 비격식 |
| 저녁/밤 (~20:00+) | ¡Buenas noches! | 격식 및 비격식 |
| 아무 때나 | ¡Hola! | 비격식 |
| 아무 때나 | ¡Buenas! | 가벼운 줄임말 |

## 2. 비격식 인사

### 2.1 일반적인 비격식 인사

| 스페인어 | 한국어 | 비고 |
|---------|---------|-------|
| ¡Hola! | 안녕! | 보편적, 아무 때나 |
| ¿Qué tal? | 잘 지내? | 매우 흔함 |
| ¿Cómo estás? | 어떻게 지내? | 표준 비격식 |
| ¿Qué hay? | 무슨 일이야? | 가벼운 표현 |
| ¿Qué pasa? | 무슨 일이야? | 매우 가벼운 표현 |
| ¿Cómo te va? | 잘 되어 가? | 친근한 표현 |
| ¿Qué hay de nuevo? | 새로운 거 있어? | 안부를 물을 때 |

### 2.2 지역별 비격식 인사

| 표현 | 지역 | 의미 |
|-----------|--------|---------|
| ¿Qué onda? | 멕시코 | 뭐 해? |
| ¿Qué hubo? / ¿Quiubo? | 콜롬비아 | 뭐 해? |
| ¿Cómo andás? | 아르헨티나 (보세오) | 어떻게 지내? |
| ¿Qué tal, tío/tía? | 스페인 | 잘 지내, 친구? |
| ¿Cómo vas? | 일반적 | 어떻게 지내? |

## 3. 격식 vs 비격식: Tú vs Usted

### 3.1 두 가지 "당신"

```
┌─────────────────────────────────────────────────────────────────┐
│   TÚ (비격식 "너")                 USTED (격식 "당신")            │
│   ├── 친구 및 또래                  ├── 낯선 사람                  │
│   ├── 가족 구성원                   ├── 어르신                     │
│   ├── 어린이                        ├── 상사 / 윗사람              │
│   ├── 같은 나이 급우                ├── 공무원 / 경찰              │
│   └── 비격식 상황                   └── 격식 상황                  │
│                                                                 │
│   동사: 2인칭 단수                   동사: 3인칭 단수              │
│   ¿Cómo estás? (어떻게 지내?)      ¿Cómo está? (어떻게 지내세요?)  │
│   ¿Tienes tiempo? (시간 있어?)     ¿Tiene tiempo? (시간 있으세요?) │
│                                                                 │
│   ※ 확신이 없으면 USTED를 사용하세요 — 항상 예의 바릅니다.        │
│   ※ 라틴아메리카에서는 usted를 더 폭넓게 사용합니다.             │
│   ※ 스페인에서는 tú를 더 쉽게 사용합니다.                       │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 동사 형태 빠른 비교

| 맥락 | Tú | Usted |
|---------|-----|-------|
| 어떻게 지내? | ¿Cómo **estás**? | ¿Cómo **está**? |
| 어디 출신이야? | ¿De dónde **eres**? | ¿De dónde **es**? |
| 이름이 뭐야? | ¿Cómo **te** llamas? | ¿Cómo **se** llama? |
| ...를 말해? | ¿**Hablas** español? | ¿**Habla** español? |
| 도와줄 수 있어? | ¿**Puedes** ayudarme? | ¿**Puede** ayudarme? |

## 4. 자기소개

### 4.1 구성 요소

```
이름:      Me llamo [이름].        → 제 이름은 [이름]입니다.
           Mi nombre es [이름].    → 제 이름은 [이름]입니다.

출신:      Soy de [나라/도시].     → 저는 [나라/도시] 출신입니다.
           Soy [국적].            → 저는 [국적]입니다.

나이:      Tengo [숫자] años.      → 저는 [숫자]살입니다.

직업:      Trabajo como [직업].    → 저는 [직업]으로 일합니다.
           Trabajo en [회사].      → 저는 [회사]에서 일합니다.

학업:      Estudio [과목].         → 저는 [과목]을 공부합니다.
           Soy estudiante.         → 저는 학생입니다.

언어:      Hablo [언어].           → 저는 [언어]를 합니다.
           Estoy aprendiendo español. → 저는 스페인어를 배우고 있습니다.
```

### 4.2 일반적인 직업

| 스페인어 | 한국어 |
|---------|---------|
| profesor/a | 교사 |
| ingeniero/a | 엔지니어 |
| médico/a | 의사 |
| abogado/a | 변호사 |
| estudiante | 학생 |
| programador/a | 프로그래머 |
| diseñador/a | 디자이너 |
| empresario/a | 사업가 |

### 4.3 자기소개 대화 예시

```
대화 1 (비격식):
A: ¡Hola! ¿Cómo te llamas?
B: Me llamo Min-jun. ¿Y tú?
A: Soy María. ¿De dónde eres?
B: Soy de Corea del Sur. Soy estudiante.
A: ¡Qué bien! Mucho gusto, Min-jun.
B: Igualmente, María.

대화 2 (격식):
A: Buenos días. ¿Cómo se llama usted?
B: Me llamo Park Min-jun. Mucho gusto.
A: Encantada. Soy la doctora García. ¿De dónde es?
B: Soy de Seúl, Corea del Sur.
A: ¿Y qué hace aquí en Madrid?
B: Estudio español en la universidad.
```

## 5. "어떻게 지내세요?"에 대한 답변

### 5.1 답변 스펙트럼

```
매우 긍정적:
  ¡Estupendo!         → 훌륭해요!
  ¡De maravilla!      → 환상적이에요!
  ¡Genial!            → 대단해요!
  ¡Muy bien!          → 아주 잘 지내요!

긍정적:
  Bien, gracias.      → 잘 지내요, 감사합니다.
  Todo bien.          → 다 괜찮아요.
  No me quejo.        → 불평할 게 없어요.

보통:
  Así así.            → 그저 그래요.
  Regular.            → 그냥 그래요.
  Tirando.            → 그럭저럭이요. (스페인, 구어체)
  Ahí vamos.          → 버티고 있어요.

부정적:
  No muy bien.        → 별로 안 좋아요.
  Un poco cansado/a.  → 좀 피곤해요.
  Fatal.              → 최악이에요.
```

### 5.2 완전한 인사 교환

```
A: ¡Hola! ¿Qué tal?        → 안녕! 잘 지내?
B: Bien, gracias. ¿Y tú?   → 잘 지내, 고마워. 너는?
A: Muy bien, también.       → 나도 아주 잘 지내.

A: ¿Cómo está usted?       → 어떻게 지내세요? (격식)
B: Bien, gracias. ¿Y usted? → 잘 지내요, 감사합니다. 당신은요?
```

## 6. 작별 인사

```
┌─────────────────────────────────────────────────────────────────┐
│   격식 작별 인사                비격식 작별 인사                   │
│   ─────────────────             ──────────────────              │
│   Adiós.                        ¡Chao! / ¡Chau!                │
│   Hasta luego.                  ¡Nos vemos!                     │
│   Hasta pronto.                 ¡Cuídate!                       │
│   Que tenga un buen día.        ¡Que te vaya bien!              │
│   Que le vaya bien.             ¡Hasta luego!                   │
│                                                                 │
│   시간 특정:                                                      │
│   Hasta mañana.    (내일 봐요.)                                  │
│   Hasta el lunes.  (월요일에 봐요.)                               │
│   Hasta la próxima. (다음에 봐요.)                                │
│   Buenas noches.   (안녕히 주무세요. — 인사로도 사용)              │
└─────────────────────────────────────────────────────────────────┘
```

## 7. 예의 표현

### 7.1 감사

| 표현 | 의미 | 격식 수준 |
|-----------|---------|-----------|
| Gracias. | 감사합니다. | 보편적 |
| Muchas gracias. | 대단히 감사합니다. | 보편적 |
| Mil gracias. | 정말 감사합니다. | 따뜻한 표현 |
| Te/Le agradezco. | 감사드립니다. | 격식 |
| Muy amable. | 매우 친절하시네요. | 격식 |

**감사에 대한 응답:**
- De nada. — 천만에요.
- No hay de qué. — 별말씀을요.
- Es un placer. — 기꺼이요.

### 7.2 사과

```
┌─────────────────────────────────────────────────────────────────┐
│   Perdón / Perdona (tú) / Perdone (Ud.)                        │
│   → 죄송합니다 / 실례합니다 (사소한 일, 주의를 끌 때)            │
│                                                                 │
│   Lo siento (mucho).                                            │
│   → (정말) 죄송합니다. (진심 어린 사과, 공감 표현)               │
│                                                                 │
│   Disculpa (tú) / Disculpe (Ud.)                                │
│   → 실례합니다. (대화 중 끼어들기, 지나갈 때)                    │
│                                                                 │
│   Con permiso.                                                   │
│   → 실례합니다. (지나가거나 방을 나갈 때)                        │
│                                                                 │
│   Fue mi culpa. → 제 잘못이었습니다.                             │
│   No fue mi intención. → 의도한 것이 아니었습니다.                │
└─────────────────────────────────────────────────────────────────┘
```

**사과에 대한 응답:**
- No pasa nada. — 괜찮아요. / 별일 아니에요.
- No te preocupes. — 걱정하지 마세요.
- Está bien. — 괜찮습니다.

### 7.3 요청하기

| 공손함 수준 | 스페인어 | 한국어 |
|-----------------|---------|---------|
| 직접적 | ¿Puedes ayudarme? | 도와줄 수 있어? |
| 공손한 | ¿Me podrías ayudar? | 도와줄 수 있겠어? |
| 매우 공손한 | ¿Sería posible...? | ...이 가능할까요? |
| 격식 | Me gustaría... | ...하고 싶습니다. |
| 매우 격식 | Quisiera... | ...하고 싶습니다. (더 공손) |

## 8. 기본 대화 패턴

### 8.1 카페에서

```
Cliente: ¡Buenos días!
Camarero: ¡Buenos días! ¿Qué desea?
Cliente: Un café con leche, por favor.
Camarero: ¿Algo más?
Cliente: No, eso es todo. Gracias.
Camarero: Son dos euros cincuenta.
Cliente: Aquí tiene. Gracias.
Camarero: Gracias a usted.
```

### 8.2 길 묻기

```
A: Disculpe, ¿dónde está la estación de metro?
B: Siga todo recto y gire a la derecha en la segunda calle.
A: ¿Está lejos?
B: No, está a cinco minutos a pie.
A: Muchas gracias.
B: De nada.
```

### 8.3 새로운 사람 만나기

```
A: ¡Hola! ¿Eres nuevo aquí?
B: Sí, me llamo Carlos. ¡Mucho gusto!
A: Encantada, Carlos. Soy Lucía. ¿De dónde eres?
B: Soy de México. ¿Y tú?
A: De aquí, de Madrid. ¡Bienvenido!
B: ¡Gracias! Todos son muy amables.
```

## 9. 유용한 첫날 표현

### 9.1 생존 표현

```
┌─────────────────────────────────────────────────────────────────┐
│   첫날을 위한 필수 표현                                           │
│                                                                 │
│   Por favor.                    → 부탁합니다.                    │
│   Gracias.                      → 감사합니다.                    │
│   Sí. / No.                    → 네. / 아니요.                  │
│   No entiendo.                  → 이해하지 못합니다.              │
│   ¿Puede repetir, por favor?   → 다시 말씀해 주시겠어요?         │
│   ¿Cómo se dice ___ en español? → ___를 스페인어로 어떻게 말해요? │
│   ¿Qué significa ___?          → ___는 무슨 뜻이에요?            │
│   Más despacio, por favor.      → 더 천천히 말씀해 주세요.        │
│   No hablo español (muy bien).  → 스페인어를 (잘) 못합니다.      │
│   Hablo un poco de español.     → 스페인어를 조금 합니다.         │
│   ¿Habla inglés?               → 영어 하세요?                   │
│   ¿Dónde está el baño?         → 화장실이 어디에요?              │
│   ¿Cuánto cuesta?              → 얼마에요?                      │
│   La cuenta, por favor.         → 계산서 주세요.                 │
│   ¡Ayuda!                       → 도와주세요!                    │
└─────────────────────────────────────────────────────────────────┘
```

### 9.2 기차역에서

```
Viajero: Buenos días. Un billete a Barcelona, por favor.
Taquillero: ¿De ida o de ida y vuelta?
Viajero: De ida y vuelta, por favor.
Taquillero: ¿Para cuándo?
Viajero: Para hoy. ¿A qué hora sale el próximo tren?
Taquillero: A las 14:30. Andén número 5.
Viajero: Perfecto. ¿Cuánto cuesta?
Taquillero: Sesenta y cinco euros.
Viajero: Aquí tiene. Gracias.
Taquillero: Buen viaje.
```

## 10. 연습 문제

### 연습 1: 적절한 인사 고르기
각 상황에 가장 적절한 인사를 선택하세요.

<details>
<summary>정답 보기</summary>

```
1. 오전 9시에 교수님을 만날 때
   → Buenos días, profesor.

2. 오후 3시에 가게에 들어갈 때
   → Buenas tardes.

3. 아무 때나 친구에게 문자할 때
   → ¡Hola! ¿Qué tal?

4. 오후 9시에 격식 있는 저녁 식사에 도착할 때
   → Buenas noches.
```

</details>

### 연습 2: Tú인가 Usted인가?
어떤 형태를 사용하겠습니까?

<details>
<summary>정답 보기</summary>

```
1. 가장 친한 친구         → tú
2. 경찰관               → usted
3. 같은 반 친구          → tú
4. 상사 (첫 만남)        → usted
5. 어린이               → tú
6. 나이 든 모르는 분      → usted
7. 연인                 → tú
```

</details>

### 연습 3: 자기소개 완성하기
빈칸을 채우세요.

<details>
<summary>정답 보기</summary>

```
A: ¡Hola! ¿Cómo te _______?          → llamas
B: _______ llamo Soo-jin. ¿Y tú?     → Me
A: Soy Pablo. ¿De _______ eres?       → dónde
B: _______ de Corea del Sur.          → Soy
A: ¿Qué _______ en España?            → haces
B: _______ español.                    → Estudio
A: ¡Mucho _______!                     → gusto
B: _______.                            → Igualmente
```

</details>

### 연습 4: 역할극 — 대화 만들기
아래 요소들을 사용하여 대화문을 작성하세요. 이 레슨의 패턴을 활용하세요.

```
상황: 마드리드의 카페에 있습니다. 웨이터에게 인사하고,
커피를 주문하고, 계산서를 요청하고, 작별 인사를 하세요.

핵심 어휘: Buenos días, por favor, un café,
la cuenta, gracias, adiós
```
