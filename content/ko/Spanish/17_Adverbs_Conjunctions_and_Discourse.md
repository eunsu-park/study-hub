## 1. 시간 부사

| 스페인어 | 한국어 | 스페인어 | 한국어 |
|---------|---------|---------|---------|
| ahora | 지금 | todavía / aún | 아직, 여전히 |
| hoy | 오늘 | ya | 이미 |
| ayer | 어제 | luego | 나중에/그 다음에 |
| mañana | 내일 | después | ~후에, 나중에 |
| siempre | 항상 | antes | ~전에 |
| nunca / jamás | 절대 ~않다 | pronto | 곧 |
| a veces | 가끔 | temprano | 일찍 |
| a menudo | 자주 | tarde | 늦게 |
| mientras | 그 동안에 | entonces | 그때, 그러면 |
| de repente | 갑자기 | enseguida | 즉시 |

## 2. 장소 부사

| 스페인어 | 한국어 | 스페인어 | 한국어 |
|---------|---------|---------|---------|
| aquí / acá | 여기 | dentro | 안에 |
| ahí | 거기 (상대방 근처) | fuera | 밖에 |
| allí / allá | 저기 (먼 곳) | cerca | 가까이 |
| arriba | 위에, 위층에 | lejos | 멀리 |
| abajo | 아래에, 아래층에 | enfrente | 맞은편에 |
| adelante | 앞으로 | alrededor | 주위에 |
| atrás | 뒤에, 뒤로 | | |

> **Aquí/acá와 allí/allá**: *aquí/allí*는 더 구체적인 위치를 나타내고,
> *acá/allá*는 대략적인 방향을 나타냅니다. 라틴아메리카에서는 *acá/allá*를 더 많이 사용합니다.

## 3. 방법 부사

| 스페인어 | 한국어 |
|---------|---------|
| bien | 잘 |
| mal | 나쁘게, 못 |
| así | 이렇게, 그렇게 |
| despacio | 천천히 |
| rápido | 빠르게 (형용사로도 사용) |
| mejor | 더 잘 |
| peor | 더 나쁘게 |

## 4. 수량 부사

| 스페인어 | 한국어 | 스페인어 | 한국어 |
|---------|---------|---------|---------|
| muy | 매우 | más | 더 |
| mucho | 많이 | menos | 덜, 더 적게 |
| poco | 조금 | casi | 거의 |
| bastante | 꽤, 충분히 | apenas | 겨우, 간신히 |
| demasiado | 너무 많이 | nada | 전혀 (~않다) |
| tan | 그렇게, 그토록 | tanto | 그만큼 많이 |

> **Muy vs. mucho**: *Muy*는 형용사/부사를 수식하고, *mucho*는 동사를 수식합니다.
> - Es **muy** alto. (그는 매우 키가 크다.)
> - Trabaja **mucho**. (그는 많이 일한다.)

## 5. -MENTE 부사 형성

### 규칙: 형용사의 여성형 + -mente

| 남성형 형용사 | 여성형 형용사 | 부사 | 한국어 |
|---------------|--------------|--------|---------|
| rápido | rápid**a** | rápidamente | 빠르게 |
| lento | lent**a** | lentamente | 천천히 |
| tranquilo | tranquil**a** | tranquilamente | 차분하게 |
| perfecto | perfect**a** | perfectamente | 완벽하게 |

### -e 또는 자음으로 끝나는 형용사: -mente를 바로 붙임

| 형용사 | 부사 | 한국어 |
|-----------|--------|---------|
| fácil | fácil**mente** | 쉽게 |
| frecuente | frecuente**mente** | 자주 |
| feliz | feliz**mente** | 행복하게 |
| normal | normal**mente** | 보통, 정상적으로 |

### 연속 -mente 부사 두 개 사용 시

첫 번째 부사는 **형용사 형태만** 사용하고, 마지막 부사에만 -mente를 붙입니다:
- Habla **clara** y **lentamente**. (그는 명확하고 천천히 말한다.)
- Lo hizo **rápida** y **eficientemente**. (그녀는 빠르고 효율적으로 해냈다.)

## 6. 등위 접속사

| 접속사 | 의미 | 예시 |
|-------------|---------|---------|
| **y** | 그리고 | Juan **y** María |
| **e** | 그리고 (i-/hi- 앞에서) | padre **e** hijo |
| **o** | 또는 | ¿café **o** té? |
| **u** | 또는 (o-/ho- 앞에서) | siete **u** ocho |
| **pero** | 그러나 | Es caro, **pero** bueno. |
| **sino** | ~이 아니라 (부정문 뒤에서) | No es rojo, **sino** azul. |
| **ni... ni** | ~도 ~도 아닌 | **Ni** café **ni** té. |

> **Pero vs. sino**: *sino*는 부정문 뒤에서 모순되는 대안을 제시할 때만 사용합니다:
> - No quiero café **sino** té. (나는 커피가 아니라 차를 원한다.)

---

## 7. 종속 접속사

| 접속사 | 의미 | 예시 |
|-------------|---------|---------|
| **porque** | ~때문에 | Estudio **porque** quiero aprender. |
| **ya que / puesto que** | ~이므로 | **Ya que** llueve, me quedo. |
| **aunque** | ~임에도 불구하고 | **Aunque** es caro, lo compro. |
| **mientras** | ~하는 동안 | Leo **mientras** como. |
| **cuando** | ~할 때 | Llámame **cuando** llegues. |
| **si** | 만약 ~라면 | **Si** llueve, no voy. |
| **para que** | ~하기 위해 (+접속법) | Hablo despacio **para que** entiendas. |
| **antes de que** | ~하기 전에 (+접속법) | Sal **antes de que** llueva. |
| **después de que** | ~한 후에 | **Después de que** comimos, salimos. |
| **como** | ~이므로, ~처럼 | **Como** no tengo dinero, no voy. |

## 8. 담화 표지: 구성하기

### 도입 (Introduction)

| 스페인어 | 한국어 |
|---------|---------|
| En primer lugar | 우선, 첫째로 |
| Para empezar | 시작하자면 |
| Ante todo | 무엇보다도 |

### 순서 (Sequencing)

| 스페인어 | 한국어 |
|---------|---------|
| En segundo lugar | 둘째로 |
| A continuación | 다음으로 |
| Luego / Después | 그 다음에 / 그 후에 |
| Por último | 마지막으로 |
| Finalmente | 최종적으로 |

### 결론 (Conclusion)

| 스페인어 | 한국어 |
|---------|---------|
| En resumen | 요약하면 |
| En conclusión | 결론적으로 |
| Para terminar | 마무리하자면 |
| En definitiva | 결국 |
| Al fin y al cabo | 어쨌든, 결국에는 |

## 9. 담화 표지: 추가, 대조, 명확화

### 정보 추가 (Adding Information)

| 스페인어 | 한국어 |
|---------|---------|
| Además | 게다가 |
| También | 또한 |
| Es más | 더 나아가 |
| Incluso | 심지어 |
| Por cierto | 그런데, 참고로 |
| Encima | 게다가, 그 위에 |

### 대조 (Contrasting)

| 스페인어 | 한국어 |
|---------|---------|
| Sin embargo | 그러나, 하지만 |
| No obstante | 그럼에도 불구하고 |
| En cambio | 반면에 |
| Por el contrario | 반대로 |
| A pesar de (que) | ~에도 불구하고 |
| De todos modos | 어쨌든 |

### 명확화 (Clarifying)

| 스페인어 | 한국어 |
|---------|---------|
| Por ejemplo | 예를 들어 |
| Es decir | 즉 |
| O sea | 다시 말해서 |
| En otras palabras | 다른 말로 하면 |
| Concretamente | 구체적으로 |

### 원인과 결과 (Cause and Effect)

| 원인 | 결과 |
|-------|--------|
| Porque (~때문에) | Por lo tanto (그러므로) |
| Ya que (~이므로) | Por eso (그래서) |
| Dado que (~을 감안하면) | Así que (그래서) |
| Debido a (~로 인해) | En consecuencia (결과적으로) |

## 10. 대화 전략

### 시간 벌기 (Buying Time)

| 스페인어 | 한국어 |
|---------|---------|
| Pues... | 글쎄... |
| Bueno... | 음... |
| A ver... | 어디 보자... |
| Es que... | 사실은... |
| O sea... | 그러니까... |

### 확인 구하기 (Seeking Confirmation)

| 스페인어 | 한국어 |
|---------|---------|
| ¿Vale? | 알겠지? (스페인) |
| ¿Verdad? | 그렇지? |
| ¿No? | 그렇지? / 아닌가? |
| ¿Sabes? | 알지? |
| ¿Entiendes? | 이해하겠지? |

### 화제 전환 (Changing Subject)

| 스페인어 | 한국어 |
|---------|---------|
| Bueno, ... | 그래, ... |
| Cambiando de tema... | 화제를 바꾸면... |
| A todo esto... | 그런데 말이야... |
| Hablando de otra cosa... | 다른 얘기인데... |

## 11. 연습 문제

### 연습 1: -mente 부사 형성하기

1. tranquilo → ________
2. fácil → ________
3. rápido → ________
4. probable → ________
5. "명확하고 천천히" (부사 두 개) → ________

<details>
<summary>정답</summary>

1. tranquilamente 2. fácilmente 3. rápidamente 4. probablemente
5. clara y lentamente (첫 번째는 형용사 형태만 사용)

</details>

### 연습 2: 올바른 접속사 고르기

1. No quiero café ________ té. (~이 아니라)
2. Estudio ________ quiero aprobar. (~때문에)
3. Son las siete ________ ocho. (또는 → o- 앞에서)
4. Hablo despacio ________ me entiendas. (~하기 위해)

<details>
<summary>정답</summary>

1. **sino** 2. **porque** 3. **u** 4. **para que** (+ 접속법)

</details>

### 연습 3: 담화 표지 추가하기

적절한 담화 표지를 추가하여 다음 글을 개선하세요.

"Viajé a España. Visité Madrid. Fui a Barcelona. Me gustó mucho Barcelona. Fue un viaje increíble."

<details>
<summary>모범 답안</summary>

"**El verano pasado**, viajé a España. **En primer lugar**, visité Madrid.
**Después**, fui a Barcelona. **Sin embargo**, me gustó mucho **más** Barcelona.
**En definitiva**, fue un viaje increíble."

</details>
