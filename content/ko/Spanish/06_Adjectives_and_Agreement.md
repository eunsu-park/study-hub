## 1. 성 일치

스페인어 형용사는 수식하는 명사의 **성**(남성/여성)과 **수**(단수/복수)에 일치해야 한다.
이를 *concordancia*(일치)라고 한다.

### 패턴 1: -o / -a 형용사 (가장 흔한 유형)

| 남성형 | 여성형 | 한국어 |
|-----------|----------|---------|
| alto | alta | 키가 큰 |
| bajo | baja | 키가 작은 |
| bonito | bonita | 예쁜 |
| pequeño | pequeña | 작은 |
| guapo | guapa | 잘생긴/예쁜 |
| delgado | delgada | 마른 |
| gordo | gorda | 뚱뚱한 |
| rico | rica | 부유한 |

> **규칙**: 여성형은 끝의 **-o**를 **-a**로 바꾼다.

### 패턴 2: 변화 없는 형용사 (-e 또는 자음으로 끝나는 경우)

이 형용사들은 남성형과 여성형이 **같은 형태**이다:

| 형용사 | 한국어 | 예시 |
|-----------|---------|---------|
| grande | 큰 | un hombre **grande** / una casa **grande** |
| inteligente | 지적인 | un chico **inteligente** / una chica **inteligente** |
| triste | 슬픈 | el niño **triste** / la niña **triste** |
| alegre | 기쁜 | un día **alegre** / una tarde **alegre** |
| fácil | 쉬운 | un examen **fácil** / una tarea **fácil** |
| difícil | 어려운 | un problema **difícil** / una pregunta **difícil** |
| joven | 젊은 | un hombre **joven** / una mujer **joven** |

### 패턴 3: 국적 형용사 (자음으로 끝나는 경우)

자음으로 끝나는 국적 형용사는 여성형에 **-a를 추가**한다:

| 남성형 | 여성형 | 한국어 |
|-----------|----------|---------|
| español | español**a** | 스페인의 |
| francés | frances**a** | 프랑스의 |
| alemán | aleman**a** | 독일의 |
| inglés | ingles**a** | 영국의 |
| japonés | japones**a** | 일본의 |

> **참고**: 여성형에서는 악센트 부호가 사라진다 (francés → francesa).

## 2. 수 일치

### 모음으로 끝나는 경우 → -s 추가

| 단수 | 복수 | 한국어 |
|----------|--------|---------|
| alto | alto**s** | 키가 큰 |
| alta | alta**s** | 키가 큰 (여성) |
| grande | grande**s** | 큰 |
| inteligente | inteligente**s** | 지적인 |

### 자음으로 끝나는 경우 → -es 추가

| 단수 | 복수 | 한국어 |
|----------|--------|---------|
| fácil | fácil**es** | 쉬운 |
| difícil | difícil**es** | 어려운 |
| español | español**es** | 스페인의 |
| joven | jóven**es** | 젊은 |

### -z로 끝나는 경우 → -ces로 변환

| 단수 | 복수 | 한국어 |
|----------|--------|---------|
| feliz | feli**ces** | 행복한 |
| capaz | capa**ces** | 유능한 |
| feroz | fero**ces** | 사나운 |

## 3. 일치 패턴 종합

```
+----------------------------------------------------------+
|              ADJECTIVE AGREEMENT SUMMARY                  |
+----------------------------------------------------------+
|                                                          |
|  Adjective Type     M.sg   F.sg    M.pl     F.pl        |
|  ────────────────   ─────  ─────   ──────   ──────      |
|  -o / -a            -o     -a      -os      -as         |
|   (alto)            alto   alta    altos    altas       |
|                                                          |
|  -e (invariable)    -e     -e      -es      -es         |
|   (grande)          grande grande  grandes  grandes     |
|                                                          |
|  consonant          -∅     -a      -es      -as         |
|   (español)         español española españoles españolas|
|                                                          |
|  -z                 -z     -z      -ces     -ces        |
|   (feliz)           feliz  feliz   felices  felices     |
|                                                          |
+----------------------------------------------------------+
```

### 혼합 성별 그룹

형용사가 **혼합 성별** 그룹을 수식할 때는 **남성 복수형**을 사용한다:

- Los chicos y las chicas son **altos**. (소년들과 소녀들은 키가 크다.)
- Los profesores y las profesoras son **buenos**. (남녀 선생님들은 훌륭하다.)

## 4. 형용사 위치

### 기본: 명사 뒤에 위치

대부분의 형용사는 수식하는 명사 **뒤에** 온다. 이는 영어와 반대이다:

| 스페인어 | 한국어 |
|---------|---------|
| un coche **rojo** | 빨간 자동차 |
| una casa **grande** | 큰 집 |
| un libro **interesante** | 흥미로운 책 |
| una persona **amable** | 친절한 사람 |

### 명사 앞에 오는 경우 (특별한 경우)

일부 형용사는 강조, 문체적 효과, 또는 본질적 성질을 표현하기 위해
명사 앞에 올 수 있다:

| 범주 | 예시 |
|----------|---------|
| 수량 | mucho, poco, otro, varios, cada |
| 순서 | primero, segundo, último, próximo |
| 본질적 성질 | la blanca nieve (하얀 눈) |
| 강조/애정 | ¡Qué bonita casa! |

### 단축형 (어미탈락, Apocopation)

일부 형용사는 **남성 단수 명사 앞에** 올 때 끝 모음이 탈락한다:

| 완전형 | 단축형 | 예시 |
|-----------|-----------|---------|
| bueno | **buen** | un **buen** amigo (좋은 친구) |
| malo | **mal** | un **mal** día (나쁜 날) |
| primero | **primer** | el **primer** piso (1층) |
| tercero | **tercer** | el **tercer** libro (세 번째 책) |
| grande | **gran** | un **gran** hombre (위대한 사람) |

> **참고**: *grande*는 **남성과 여성** 단수 명사 앞에서 모두 *gran*으로 단축된다:
> un **gran** actor / una **gran** actriz.

---

## 5. 위치에 따른 의미 변화

일부 형용사는 명사 앞에 오느냐 뒤에 오느냐에 따라 의미가 달라진다:

| 형용사 | 명사 앞 | 명사 뒤 |
|-----------|-------------|------------|
| **grande** | 위대한, 중요한: *un **gran** hombre* (위대한 사람) | 큰: *un hombre **grande*** (큰 사람) |
| **pobre** | 불쌍한: *el **pobre** niño* (불쌍한 아이) | 가난한: *el niño **pobre*** (가난한 아이) |
| **viejo** | 오래된(관계): *un **viejo** amigo* (오랜 친구) | 나이 든: *un amigo **viejo*** (늙은 친구) |
| **nuevo** | 다른, 새로운: *un **nuevo** coche* (다른 차) | 새것인: *un coche **nuevo*** (새로 만든 차) |
| **único** | 유일한: *el **único** problema* (유일한 문제) | 독특한: *un problema **único*** (독특한 문제) |
| **mismo** | 같은: *el **mismo** libro* (같은 책) | ~자체(강조): *el libro **mismo*** (그 책 자체) |
| **cierto** | 어떤: *una **cierta** sensación* (어떤 느낌) | 확실한: *una noticia **cierta*** (확실한 소식) |
| **antiguo** | 이전의: *mi **antiguo** profesor* (예전 선생님) | 오래된: *un edificio **antiguo*** (오래된 건물) |

## 6. 비교급

### ~보다 더 (más... que)

- María es **más alta que** Juan. (마리아는 후안보다 키가 크다.)
- Este libro es **más interesante que** ese. (이 책은 저 책보다 더 흥미롭다.)

### ~보다 덜 (menos... que)

- Juan es **menos alto que** María. (후안은 마리아보다 키가 작다.)
- Ese restaurante es **menos caro que** este. (저 식당은 이 식당보다 덜 비싸다.)

### ~만큼 (tan... como)

- Pedro es **tan inteligente como** Ana. (페드로는 아나만큼 지적이다.)
- Mi casa es **tan grande como** la tuya. (우리 집은 네 집만큼 크다.)

### ~만큼 많은 (tanto/a/os/as... como)

*Tanto*는 수식하는 명사에 맞게 변화한다:

- Tengo **tantos** libros **como** tú. (나는 너만큼 많은 책을 가지고 있다.)
- No tengo **tanta** paciencia **como** ella. (나는 그녀만큼 인내심이 없다.)

## 7. 최상급

### 상대 최상급 (가장 ~한 / 가장 덜 ~한)

```
el/la/los/las + más/menos + adjective + de + group
```

- Es **la más alta de** la clase. (그녀는 반에서 가장 키가 크다.)
- Es **el menos caro de** todos. (이것은 전부 중에서 가장 덜 비싸다.)
- Son **los más importantes del** mundo. (그들은 세계에서 가장 중요하다.)

### 절대 최상급 (-ísimo/a)

비교 없이 극단적인 정도를 표현한다 ("매우 매우"):

| 형용사 | 최상급 | 의미 |
|-----------|-------------|---------|
| alto | alt**ísimo** | 극히 키가 큰 |
| guapo | guap**ísimo** | 극히 잘생긴 |
| rico | riqu**ísimo** | 극히 부유한/맛있는 |
| largo | largu**ísimo** | 극히 긴 |
| fácil | facil**ísimo** | 극히 쉬운 |
| difícil | dificil**ísimo** | 극히 어려운 |

> **철자 변화**: rico → riqu**ísimo** (c→qu), largo → largu**ísimo** (g→gu),
> feliz → felic**ísimo** (z→c).

## 8. 불규칙 비교급과 최상급

| 형용사 | 비교급 | 상대 최상급 |
|-----------|-------------|---------------------|
| bueno (좋은) | **mejor** (더 좋은) | **el/la mejor** (가장 좋은) |
| malo (나쁜) | **peor** (더 나쁜) | **el/la peor** (가장 나쁜) |
| grande (큰) | **mayor** (나이가 더 많은/더 큰) | **el/la mayor** (가장 나이 많은/가장 큰) |
| pequeño (작은) | **menor** (나이가 더 어린/더 작은) | **el/la menor** (가장 나이 어린/가장 작은) |

### 사용 시 유의사항

- **Mayor/menor**는 **나이**를 가리킨다 (물리적 크기가 아님):
  - Mi hermano **mayor** (형/오빠) — "더 큰 형제"가 아님
  - Mi hermana **menor** (여동생/누나)
- 물리적 크기에는 **más grande / más pequeño**를 사용한다:
  - Esta casa es **más grande** que la otra. (이 집이 저 집보다 크다.)

## 9. 범주별 필수 형용사

### 색상

| 스페인어 | 한국어 | 스페인어 | 한국어 |
|---------|---------|---------|---------|
| rojo/a | 빨간 | morado/a | 보라색의 |
| azul | 파란 | rosa | 분홍색의 |
| verde | 초록색의 | marrón | 갈색의 |
| amarillo/a | 노란 | gris | 회색의 |
| naranja | 주황색의 | negro/a | 검은 |
| blanco/a | 흰 | dorado/a | 금색의 |

> **참고**: *azul*, *verde*, *gris*, *marrón*, *rosa*, *naranja*는 성에 따라 변하지 않는다.

### 크기와 형태

| 스페인어 | 한국어 | 스페인어 | 한국어 |
|---------|---------|---------|---------|
| grande | 큰 | ancho/a | 넓은 |
| pequeño/a | 작은 | estrecho/a | 좁은 |
| largo/a | 긴 | grueso/a | 두꺼운 |
| corto/a | 짧은 | fino/a | 얇은, 섬세한 |
| alto/a | 키가 큰 | redondo/a | 둥근 |

### 성격

| 스페인어 | 한국어 | 스페인어 | 한국어 |
|---------|---------|---------|---------|
| amable | 친절한 | tímido/a | 수줍은 |
| simpático/a | 호감이 가는 | valiente | 용감한 |
| antipático/a | 불쾌한 | perezoso/a | 게으른 |
| generoso/a | 관대한 | trabajador(a) | 부지런한 |
| inteligente | 지적인 | honesto/a | 정직한 |

### 감정과 상태

| 스페인어 | 한국어 | 스페인어 | 한국어 |
|---------|---------|---------|---------|
| feliz | 행복한 | nervioso/a | 긴장한 |

## 10. 연습 문제

### 연습 1: 성과 수 일치

형용사를 명사에 맞게 일치시키시오.

1. Las casas ________ (blanco)
2. Un hombre ________ (español)
3. Una mujer ________ (francés)
4. Los problemas ________ (difícil)
5. Las chicas ________ (inteligente)
6. Un día ________ (feliz)
7. Unas flores ________ (bonito)

<details>
<summary>정답</summary>

1. Las casas **blancas**
2. Un hombre **español**
3. Una mujer **francesa**
4. Los problemas **difíciles**
5. Las chicas **inteligentes**
6. Un día **feliz**
7. Unas flores **bonitas**

</details>

### 연습 2: 위치와 단축형

형용사를 올바른 위치(명사 앞 또는 뒤)에 넣고,
해당되는 경우 단축형을 사용하시오.

1. Es un ________ amigo ________. (bueno)
2. Es el ________ piso ________. (primero)
3. Tiene un coche ________. (rojo)
4. Es una persona ________. (amable)
5. Es un ________ hombre ________. (grande — "위대한"의 의미)

<details>
<summary>정답</summary>

1. Es un **buen** amigo. (앞, 단축형)
2. Es el **primer** piso. (앞, 단축형)
3. Tiene un coche **rojo**. (뒤 — 색상은 뒤에 위치)
4. Es una persona **amable**. (뒤 — 묘사적 형용사)
5. Es un **gran** hombre. (앞, 단축형 — "위대한"의 의미)

</details>

### 연습 3: 비교급과 최상급

스페인어로 번역하시오.

1. 그녀는 오빠보다 키가 크다.
2. 이 영화는 그 책보다 덜 흥미롭다.
3. 그는 아버지만큼 지적이다.
4. 이곳은 도시에서 가장 좋은 식당이다.
5. 이 커피는 극히 뜨겁다. (-ísimo 사용)

<details>
<summary>정답</summary>

1. Ella es **más alta que** su hermano.
2. Esta película es **menos interesante que** el libro.
3. Él es **tan inteligente como** su padre.
4. Es **el mejor** restaurante **de** la ciudad.
5. Este café está **calentísimo**.

</details>

### 연습 4: 위치에 따른 의미 변화

각 쌍을 번역하시오. 형용사 위치에 주의할 것.

1. a) 위대한 여성 → ________
   b) 큰 여성 → ________
2. a) 불쌍한 소년 → ________
   b) 가난한 소년 → ________
3. a) 오랜 친구 → ________
   b) 나이 든 친구 → ________

<details>
<summary>정답</summary>

1. a) Una **gran** mujer (위대한) / b) Una mujer **grande** (큰)
2. a) El **pobre** niño (불쌍한) / b) El niño **pobre** (가난한)
3. a) Mi **viejo** amigo (오랜) / b) Mi amigo **viejo** (나이 든)

</details>

```
+----------------------------------------------------------+
|         ADJECTIVE AGREEMENT CHEAT SHEET                   |
+----------------------------------------------------------+
|                                                          |
|  1. Match GENDER (masc/fem) and NUMBER (sg/pl)           |
|  2. -o adjectives: -o/-a/-os/-as                         |

|  8. Superlative: el/la más/menos... de, -ísimo/a         |

|                                                          |
+----------------------------------------------------------+
```
