## 1. 스페인어 동사 기초

### 1.1 부정사(Infinitive) 형태

모든 스페인어 동사는 **-ar**, **-er**, **-ir** 중 하나로 끝나는 부정사(Infinitive) 형태를 가집니다:

```
┌─────────────────────────────────────────────────────────────────┐
│   Group 1: -AR verbs (~75% of all verbs — the largest group)   │
│   hablar (to speak), estudiar (to study), trabajar (to work)   │
│                                                                 │
│   Group 2: -ER verbs (~15%)                                     │
│   comer (to eat), beber (to drink), leer (to read)             │
│                                                                 │
│   Group 3: -IR verbs (~10%)                                     │
│   vivir (to live), escribir (to write), abrir (to open)        │
│                                                                 │
│   To conjugate: remove the ending (-ar/-er/-ir) to get the     │
│   stem, then add the appropriate person/number ending.          │
│                                                                 │
│   hablar → habl- + -o = hablo (I speak)                        │
│   comer  → com-  + -o = como (I eat)                           │
│   vivir  → viv-  + -o = vivo (I live)                          │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 주격 대명사(Subject Pronouns)

| 단수 | 복수 |
|------|------|
| yo (나) | nosotros/as (우리) |
| tú (너, 비격식) | vosotros/as (너희들, 스페인) |
| él (그) | ellos (그들, 남성) |
| ella (그녀) | ellas (그들, 여성) |
| usted (당신, 격식) | ustedes (당신들, 격식 / 중남미) |

> **주어 생략(Pro-drop)**: 스페인어에서는 동사 어미가 이미 주어를 나타내기 때문에 주격 대명사를 자주 생략합니다. *Hablo español* = "나는 스페인어를 한다" (*yo*가 필요 없음). 대명사는 강조하거나 의미를 명확히 할 때만 사용합니다.

## 2. -AR 동사 활용

### 2.1 패턴: hablar (말하다)

| 주어 | 어간 + 어미 | 활용형 | 의미 |
|------|------------|--------|------|
| yo | habl + **-o** | habl**o** | 나는 말한다 |
| tú | habl + **-as** | habl**as** | 너는 말한다 |
| él/ella/Ud. | habl + **-a** | habl**a** | 그/그녀는 말한다 |
| nosotros/as | habl + **-amos** | habl**amos** | 우리는 말한다 |
| vosotros/as | habl + **-áis** | habl**áis** | 너희들은 말한다 |
| ellos/ellas/Uds. | habl + **-an** | habl**an** | 그들은 말한다 |

### 2.2 예시: estudiar (공부하다)

```
Estudio español todos los días.   → I study Spanish every day.
¿Estudias en la universidad?      → Do you study at the university?
Ella estudia medicina.            → She studies medicine.
Estudiamos juntos.                → We study together.
Estudian mucho para el examen.    → They study a lot for the exam.
```

### 2.3 필수 -AR 동사 10개

| 동사 | 의미 | 예문 |
|------|------|------|
| hablar | 말하다 | Hablo dos idiomas. (나는 두 가지 언어를 말한다.) |
| estudiar | 공부하다 | Estudio español. (나는 스페인어를 공부한다.) |
| trabajar | 일하다 | Trabajo en una oficina. (나는 사무실에서 일한다.) |
| comprar | 사다 | Compro comida. (나는 음식을 산다.) |
| llamar | 전화하다 | Llamo a mi madre. (나는 어머니에게 전화한다.) |
| mirar | 보다 | Miro la televisión. (나는 텔레비전을 본다.) |
| escuchar | 듣다 | Escucho música. (나는 음악을 듣는다.) |
| caminar | 걷다 | Camino al parque. (나는 공원까지 걷는다.) |
| cocinar | 요리하다 | Cocino la cena. (나는 저녁을 요리한다.) |
| necesitar | 필요하다 | Necesito ayuda. (나는 도움이 필요하다.) |

## 3. -ER 동사 활용

### 3.1 패턴: comer (먹다)

| 주어 | 어간 + 어미 | 활용형 | 의미 |
|------|------------|--------|------|
| yo | com + **-o** | com**o** | 나는 먹는다 |
| tú | com + **-es** | com**es** | 너는 먹는다 |
| él/ella/Ud. | com + **-e** | com**e** | 그/그녀는 먹는다 |
| nosotros/as | com + **-emos** | com**emos** | 우리는 먹는다 |
| vosotros/as | com + **-éis** | com**éis** | 너희들은 먹는다 |
| ellos/ellas/Uds. | com + **-en** | com**en** | 그들은 먹는다 |

### 3.2 예시: beber (마시다)

```
Bebo café por la mañana.          → I drink coffee in the morning.
¿Bebes agua?                       → Do you drink water?
Él bebe jugo de naranja.          → He drinks orange juice.
Bebemos vino con la cena.         → We drink wine with dinner.
No beben alcohol.                  → They don't drink alcohol.
```

### 3.3 필수 -ER 동사 10개

| 동사 | 의미 | 예문 |
|------|------|------|
| comer | 먹다 | Como a las dos. (나는 두 시에 먹는다.) |
| beber | 마시다 | Bebo mucha agua. (나는 물을 많이 마신다.) |
| leer | 읽다 | Leo un libro. (나는 책을 읽는다.) |
| aprender | 배우다 | Aprendo español. (나는 스페인어를 배운다.) |
| comprender | 이해하다 | No comprendo. (나는 이해하지 못한다.) |
| correr | 달리다 | Corro por la mañana. (나는 아침에 달린다.) |
| creer | 믿다 | Creo que sí. (나는 그렇다고 생각한다/믿는다.) |
| vender | 팔다 | Vende coches. (그는 자동차를 판다.) |
| responder | 대답하다 | Respondo al correo. (나는 이메일에 답장한다.) |
| deber | ~해야 하다 | Debo estudiar. (나는 공부해야 한다.) |

## 4. -IR 동사 활용

### 4.1 패턴: vivir (살다)

| 주어 | 어간 + 어미 | 활용형 | 의미 |
|------|------------|--------|------|
| yo | viv + **-o** | viv**o** | 나는 산다 |
| tú | viv + **-es** | viv**es** | 너는 산다 |
| él/ella/Ud. | viv + **-e** | viv**e** | 그/그녀는 산다 |
| nosotros/as | viv + **-imos** | viv**imos** | 우리는 산다 |
| vosotros/as | viv + **-ís** | viv**ís** | 너희들은 산다 |
| ellos/ellas/Uds. | viv + **-en** | viv**en** | 그들은 산다 |

> **참고**: -ER 동사와 -IR 동사는 nosotros (-emos vs -imos)와 vosotros (-éis vs -ís)를 제외하면 같은 어미를 공유합니다.

### 4.2 예시: escribir (쓰다)

```
Escribo en mi diario.             → I write in my diary.
¿Escribes correos electrónicos?   → Do you write emails?
Ella escribe poemas.              → She writes poems.
Escribimos cartas.                → We write letters.
Escriben mensajes.                → They write messages.
```

### 4.3 필수 -IR 동사 10개

| 동사 | 의미 | 예문 |
|------|------|------|
| vivir | 살다 | Vivo en Seúl. (나는 서울에 산다.) |
| escribir | 쓰다 | Escribo un correo. (나는 이메일을 쓴다.) |
| abrir | 열다 | Abro la puerta. (나는 문을 연다.) |
| recibir | 받다 | Recibo un regalo. (나는 선물을 받는다.) |
| decidir | 결정하다 | Decido ahora. (나는 지금 결정한다.) |
| compartir | 나누다, 공유하다 | Comparto mi comida. (나는 음식을 나눈다.) |
| asistir | 참석하다 | Asisto a clase. (나는 수업에 참석한다.) |
| subir | 올라가다 | Subo las escaleras. (나는 계단을 올라간다.) |
| descubrir | 발견하다 | Descubro algo nuevo. (나는 새로운 것을 발견한다.) |
| existir | 존재하다 | Existe una solución. (해결책이 존재한다.) |

## 5. 기본 문장 구성

### 5.1 어순: 주어-동사-목적어 (SVO)

```
┌─────────────────────────────────────────────────────────────────┐
│   Basic pattern: Subject + Verb + Object/Complement             │
│                                                                 │
│   (Yo) estudio español.        → I study Spanish.               │
│   María come una manzana.      → María eats an apple.           │
│   (Nosotros) vivimos en Madrid. → We live in Madrid.            │
│                                                                 │
│   ※ Subject pronouns are usually omitted (pro-drop):           │
│   Hablo español. = I speak Spanish. (yo is implied)             │
│   Estudia medicina. = He/She studies medicine.                  │
│                                                                 │
│   ※ Spanish word order is more flexible than English:           │
│   Leo un libro. = Un libro leo. (emphasis on "a book")          │
│   Both are grammatically correct.                               │
└─────────────────────────────────────────────────────────────────┘
```

### 5.2 세부 정보 추가하기

```
Time expressions:
  Estudio español por la mañana.         → I study Spanish in the morning.
  Trabajo todos los días.                → I work every day.
  Siempre como a las dos.               → I always eat at two.

Location:
  Estudio en la biblioteca.              → I study in the library.
  Trabajo en una oficina.                → I work in an office.

Manner:
  Habla español muy bien.               → He/She speaks Spanish very well.
  Caminamos lentamente.                 → We walk slowly.

Frequency:
  siempre (always), a menudo (often), a veces (sometimes),
  casi nunca (almost never), nunca (never)
```

## 6. 의문문 만들기

### 6.1 의문사 (Interrogatives)

| 스페인어 | 한국어 | 예문 |
|----------|--------|------|
| ¿Qué? | 무엇? | ¿Qué estudias? (무엇을 공부하니?) |
| ¿Quién? / ¿Quiénes? | 누구? | ¿Quién habla? (누가 말하니?) |
| ¿Cuándo? | 언제? | ¿Cuándo comes? (언제 먹니?) |
| ¿Dónde? | 어디? | ¿Dónde vives? (어디에 사니?) |
| ¿Adónde? | 어디로? | ¿Adónde vas? (어디로 가니?) |
| ¿Por qué? | 왜? | ¿Por qué estudias? (왜 공부하니?) |
| ¿Cómo? | 어떻게? | ¿Cómo estás? (어떻게 지내니?) |
| ¿Cuánto/a? | 얼마나? | ¿Cuánto cuesta? (얼마예요?) |
| ¿Cuántos/as? | 몇 개? 얼마나 많은? | ¿Cuántos años tienes? (몇 살이니?) |
| ¿Cuál? / ¿Cuáles? | 어느 것? | ¿Cuál prefieres? (어느 것을 선호하니?) |

> **모든 의문사에는 악센트 부호(Accent Mark)가 붙습니다.** 악센트가 없으면 관계대명사가 됩니다: *que* (that/~하는), *donde* (where/~하는 곳) 등.

### 6.2 예/아니오 의문문을 만드는 세 가지 방법

```
Method 1: Intonation only (raise pitch at end)
  ¿Hablas español?           → Do you speak Spanish?
  (Same word order as statement, just with rising intonation)

Method 2: Invert subject and verb
  ¿Habla usted español?      → Do you speak Spanish?
  ¿Estudia María medicina?   → Does María study medicine?

Method 3: Add a tag question
  Hablas español, ¿no?       → You speak Spanish, right?
  Hablas español, ¿verdad?   → You speak Spanish, right?
```

> **참고**: 스페인어는 문장 시작에 역물음표(¿)를 사용합니다. 쓸 때는 반드시 ¿와 ?를 모두 포함해야 합니다.

## 7. 부정문

### 7.1 기본 부정: 동사 앞에 *no* 배치

```
Hablo español.       → I speak Spanish.
No hablo español.    → I don't speak Spanish.

Tiene tiempo.        → He/She has time.
No tiene tiempo.     → He/She doesn't have time.

Es mi libro.         → It's my book.
No es mi libro.      → It's not my book.
```

### 7.2 부정어 (Negative Words)

| 부정어 | 반의어 (긍정) | 의미 |
|--------|--------------|------|
| no | sí | 아니오 / ~아니다 |
| nada | algo | 아무것도 (~없다) / 무언가 |
| nadie | alguien | 아무도 (~없다) / 누군가 |
| nunca / jamás | siempre | 절대 (~않다) / 항상 |
| ninguno/a | alguno/a | 아무 ~도 없다 / 어떤 |
| tampoco | también | ~도 아니다 / ~도 |
| ni...ni | o...o | ~도 ~도 아니다 / ~이거나 ~이거나 |

### 7.3 이중 부정 (스페인어에서는 올바른 문법!)

```
┌─────────────────────────────────────────────────────────────────┐
│   Unlike English, double negatives are REQUIRED in Spanish.     │
│                                                                 │
│   No tengo nada.       → I don't have anything. (lit: not nothing)│
│   No veo a nadie.      → I don't see anyone.                   │
│   No como nunca carne. → I never eat meat.                      │
│   No quiero ni café ni té. → I want neither coffee nor tea.    │
│                                                                 │
│   Alternative (negative word before verb, no "no"):             │
│   Nunca como carne.     → I never eat meat.                     │
│   Nadie habla.          → Nobody speaks.                        │
│   Nada funciona.        → Nothing works.                        │
│                                                                 │
│   Both patterns are correct:                                    │
│   No viene nunca. = Nunca viene. (He/She never comes.)         │
└─────────────────────────────────────────────────────────────────┘
```

## 8. 현재 시제의 용법

스페인어의 현재 시제는 영어의 여러 시제에 해당합니다:

```
1. Habitual actions:
   Estudio español todos los días.
   → I study Spanish every day.

2. Current states:
   Vivo en Madrid.
   → I live in Madrid.

3. General truths:
   La Tierra gira alrededor del Sol.
   → The Earth revolves around the Sun.

4. Near future (with time expression):
   Mañana trabajo temprano.
   → Tomorrow I work early. (= I will work / I'm working tomorrow)

5. Ongoing action (informal, instead of progressive):
   ¿Qué lees? (= ¿Qué estás leyendo?)
   → What are you reading?
```

## 9. 요약: 세 그룹 비교

```
┌────────┬──────────┬──────────┬──────────┐
│ Person │ -AR      │ -ER      │ -IR      │
│        │ (hablar) │ (comer)  │ (vivir)  │
├────────┼──────────┼──────────┼──────────┤
│ yo     │ habl-o   │ com-o    │ viv-o    │
│ tú     │ habl-as  │ com-es   │ viv-es   │
│ él     │ habl-a   │ com-e    │ viv-e    │
│ nos.   │ habl-amos│ com-emos │ viv-imos │
│ vos.   │ habl-áis │ com-éis  │ viv-ís   │
│ ellos  │ habl-an  │ com-en   │ viv-en   │
├────────┴──────────┴──────────┴──────────┤
│ Pattern:                                 │
│ • "yo" always ends in -o                │
│ • -AR: a-themed vowels (a, as, amos)    │
│ • -ER/-IR: e-themed vowels (es, e, en)  │
│ • -ER vs -IR differ only in nos. & vos. │
└──────────────────────────────────────────┘
```
