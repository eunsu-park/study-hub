## 2. 스페인: 이베리아 반도 스페인어

### Vosotros 활용 (스페인 전용)

라틴아메리카에서는 모든 복수 "you"에 **ustedes**를 사용한다. 스페인에서는 다음과 같이 구분한다:
- **vosotros/as** → 비격식 복수
- **ustedes** → 격식 복수

| 시제 | -AR (hablar) | -ER (comer) | -IR (vivir) |
|-------|-------------|-------------|-------------|
| 현재 | habl**áis** | com**éis** | viv**ís** |
| 단순과거 | habl**asteis** | com**isteis** | viv**isteis** |
| 불완료과거 | habl**abais** | com**íais** | viv**íais** |
| 미래 | habl**aréis** | com**eréis** | viv**iréis** |
| 조건 | habl**aríais** | com**eríais** | viv**iríais** |
| 접속법 현재 | habl**éis** | com**áis** | viv**áis** |
| 명령법 | habl**ad** | com**ed** | viv**id** |

관련 형태:
- 목적어 대명사: **os** — Os quiero. (여러분을 사랑해요.)
- 소유격: **vuestro/a/os/as** — Vuestra casa es bonita.

### 발음 특징

```
+----------------------------------------------------------+
|          PENINSULAR SPANISH PRONUNCIATION                 |
+----------------------------------------------------------+
|                                                          |
|  1. DISTINCIÓN (c/z vs s):                               |
|     cena → [θéna]  (not [séna])                         |
|     zapato → [θapáto]                                    |
|     cerveza → [θerβéθa]                                  |
|                                                          |
|  2. Strong /x/ (j, ge, gi):                              |
|     jota → [xóta]  (harsh, from back of throat)         |
|     gente → [xénte]                                      |
|                                                          |
|  3. Final -d:                                            |
|     Madrid → [maðríð] or [maðrí] (weakened/dropped)     |
|     ciudad → [θjuðáð]                                    |
|                                                          |
|  4. ll pronunciation:                                    |
|     Many regions: [ʎ] (lateral palatal)                  |
|     Yeísmo zones: [j] (same as y)                        |
+----------------------------------------------------------+
```

## 3. 멕시코

### 특징

```
+----------------------------------------------------------+
|               MEXICAN SPANISH FEATURES                    |
+----------------------------------------------------------+
|                                                          |
|  Pronunciation:                                          |
|  • Seseo: c/z pronounced as [s] (like all of L.A.)      |
|  • Final -s retained clearly                             |
|  • Consonant clusters preserved: extra [éks.tra]         |
|                                                          |
|  Grammar:                                                |
|  • Ustedes used (no vosotros)                            |
|  • Preterite preferred: Hoy comí. (not He comido hoy.)  |
|  • Diminutives (-ito/-ita) used very frequently          |
|                                                          |
|  Influence:                                              |
|  • Náhuatl (Aztec) vocabulary: chocolate, tomate,        |
|    aguacate, chile, guajolote                            |
|  • US English influence (northern border regions)        |
+----------------------------------------------------------+
```

### 멕시코 표현

| 표현 | 의미 |
|------------|---------|
| ¿Qué onda? | 잘 지내? / 무슨 일이야? |
| ¡Órale! | 멋지다! / 진짜? / 가자! |
| ¡Híjole! | 와! (놀라움) |
| No manches | 말도 안 돼! / 장난이지! |
| ¡Ándale! | 어서! / 빨리! |
| Padre / Padrísimo | 최고의, 멋진 |
| Chido/Chida | 멋진, 좋은 |
| Güey | 친구, 야 (매우 비격식) |
| Neta | 진짜, 정말로 |
| Fresa | 잘난 척하는 사람, 도도한 사람 |

### 축소사(Diminutive) 문화

멕시코는 대부분의 다른 스페인어권 국가보다 축소사(diminutive)를 더 많이 사용한다:

| 축소사 | 원형 | 의미 |
|-----------|------|---------|
| ahorita | ahora | 지금 당장 / 잠깐 후에 |
| tantito | tanto | 아주 조금만 |
| momentito | momento | 잠깐만 |
| lueguito | luego | 바로 |
| cerquita | cerca | 아주 가까이 |
| poquito | poco | 아주 조금 |

> **Ahorita**는 문맥과 어조에 따라 "지금 당장," "잠깐 후에," 심지어 "나중에"를 의미할 수 있다 — 유명한 다의어이다.

## 4. 아르헨티나와 우루과이: 리오플라텐세 스페인어

### 보세오(Voseo) 체계

아르헨티나와 우루과이에서는 **tú** 대신 **vos**를 사용한다:

| 동사 | tú 형태 | vos 형태 |
|------|---------|----------|
| hablar | tú **hablas** | vos **hablás** |
| comer | tú **comes** | vos **comés** |
| vivir | tú **vives** | vos **vivís** |
| ser | tú **eres** | vos **sos** |
| tener | tú **tienes** | vos **tenés** |
| poder | tú **puedes** | vos **podés** |
| querer | tú **quieres** | vos **querés** |
| pensar | tú **piensas** | vos **pensás** |
| venir | tú **vienes** | vos **venís** |

```
+----------------------------------------------------------+
|              VOSEO RULES                                  |
+----------------------------------------------------------+
|                                                          |
|  1. Stress falls on the FINAL syllable                   |
|  2. NO stem changes (pensás, not piensás)                |
|  3. Imperative: hablá, comé, viví (stress on last vowel) |
|                                                          |
|  Examples:                                               |
|  ¿Vos tenés tiempo? (Do you have time?)                  |
|  ¿De dónde sos? (Where are you from?)                    |
|  Vení acá. (Come here.)                                  |
|  Decime la verdad. (Tell me the truth.)                  |
+----------------------------------------------------------+
```

> **vos는 어디서 사용되는가?** 주로 아르헨티나와 우루과이에서 사용된다. 또한 중앙아메리카 일부 지역(과테말라, 온두라스, 엘살바도르, 코스타리카, 니카라과)과 콜롬비아의 파이사(Paisa) 지역(메데인)에서도 사용된다.

### 발음: 셰이스모(Sheísmo)

아르헨티나/우루과이 스페인어의 가장 특징적인 현상:

| 문자 | 일반 스페인어 | 리오플라텐세 |
|--------|----------------|-------------|
| **ll** | [j] (영어 "y") | [ʃ] (영어 "sh") 또는 [ʒ] ("zh") |
| **y** | [j] | [ʃ] 또는 [ʒ] |

- pollo: 일반 [pójo] → 아르헨티나 [póʃo]
- yo: 일반 [jo] → 아르헨티나 [ʃo]
- llamar: 일반 [jamár] → 아르헨티나 [ʃamár]

아르헨티나의 억양은 이탈리아 이민의 영향을 받은 독특한 선율적 특성을 가지고 있다.

### 아르헨티나 표현 (Lunfardo)

| 표현 | 의미 |
|-----------|---------|
| Che | 이봐, 친구 (호칭) |
| Boludo/a | 친구 / 바보 (문맥에 따라 다름) |
| ¡Qué quilombo! | 정말 엉망이다! |
| Bárbaro | 훌륭한, 멋진 |
| Copado/a | 멋진, 좋은 |
| Morfar | 먹다 |
| Laburo | 일, 직장 |
| Mango | 돈, 페소 |
| Bondi | 버스 |
| Pibe/Piba | 젊은 사람, 아이 |
| Re- | 매우 (re lindo = 매우 예쁜) |

> **룬파르도(Lunfardo)**는 부에노스아이레스의 속어로, 이탈리아 이민자들의 영향을 크게 받았다. 탱고 가사에서 흔히 발견된다.

## 5. 카리브해 스페인어

### 쿠바, 도미니카 공화국, 푸에르토리코

```
+----------------------------------------------------------+
|           CARIBBEAN PRONUNCIATION FEATURES                |
+----------------------------------------------------------+
|                                                          |
|  1. Final/syllable-final -s → [h] or DROPPED            |
|     estos → [éhtoh] or [éto]                             |
|     más → [máh]                                          |
|                                                          |
|  2. Final -r → -l or DROPPED                             |
|     comer → [komél]                                      |
|     puerta → [pwélta]                                    |
|                                                          |
|  3. Rapid speech with syllable shortening                |
|                                                          |
|  4. African-influenced vocabulary and rhythm              |
+----------------------------------------------------------+
```

## 7. 칠레

### 발음

- 어말 -s 약화: [h] 또는 탈락 (카리브해와 유사)
- -ado → -ao: cansado → cansao
- 독특한 ch 발음: [tʃ]보다 [ʃ]에 가까움

### 칠레 표현

| 표현 | 의미 |
|-----------|---------|
| ¿Cachai? | 알겠어? (이해해?) |
| Po (pues에서 유래) | 문장 끝 강조: Sí po, No po |
| Fome | 지루한 |
| Pololo/Polola | 남자친구/여자친구 |

| Luca | 1,000 페소 |

## 8. 발음 변이 지도

### 세 가지 핵심 현상

| 현상 | 설명 | 사용 지역 |
|-----------|-------------|-------|
| **Distinción** | c/z = [θ], s = [s] | 스페인 대부분 |
| **Seseo** | c/z/s 모두 = [s] | 라틴아메리카 전역, 스페인 남부 |
| **Ceceo** | c/z/s 모두 = [θ] | 안달루시아 일부 |
| **Yeísmo** | ll = y = [j] | 스페인어권 대부분 |
| **Sheísmo** | ll = y = [ʃ] 또는 [ʒ] | 아르헨티나, 우루과이 |
| **S 기식음화(S-aspiration)** | 어말 -s → [h] | 카리브해, 칠레, 안달루시아, 라틴아메리카 일부 |
