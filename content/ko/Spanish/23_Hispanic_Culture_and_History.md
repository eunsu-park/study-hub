## 2. 주요 역사적 사건

### 연표

```
+----------------------------------------------------------+
|          KEY DATES IN HISPANIC HISTORY                     |
+----------------------------------------------------------+
|                                                          |
|  711     Moorish invasion of Iberian Peninsula            |
|  1492    Fall of Granada / Columbus arrives in Americas   |
|  1519-21 Hernán Cortés conquers Aztec Empire             |
|  1532-33 Francisco Pizarro conquers Inca Empire          |
|  1808    Napoleonic invasion of Spain triggers            |
|          independence movements in Latin America          |
|  1810-30 Latin American independence wars                 |
|  1898    Spanish-American War — Spain loses Cuba,         |
|          Puerto Rico, Philippines                        |
|  1936-39 Spanish Civil War                               |
|  1975    Death of Franco, transition to democracy        |
|  1994    NAFTA — Mexico enters free trade agreement      |
+----------------------------------------------------------+
```

### 무어인의 유산 (711-1492)

약 800년 동안 무어인(Moors, 북아프리카의 무슬림)이 스페인(España) 대부분을 지배했습니다. 그들의 영향은 오늘날에도 다음에서 확인할 수 있습니다:

- **건축**: La Alhambra (그라나다(Granada)), Mezquita de Córdoba (코르도바 모스크)
- **언어**: 아랍어에서 유래한 스페인어 단어 약 4,000개 (almohada/베개, alfombra/카펫, azúcar/설탕, algodón/면, ojalá/바라건대)
- **과학**: 수학, 천문학, 의학

### 대항해 시대 (1492-)

크리스토퍼 콜럼버스(Cristóbal Colón)가 1492년 아메리카 대륙에 도달하면서 스페인의 식민 지배가 시작되었습니다. 그 결과:

- 스페인어와 가톨릭 종교의 확산
- 원주민, 유럽인, 아프리카인 간의 문화적 혼합(**mestizaje**, 메스티사헤)
- 식품 교환: 감자, 토마토, 초콜릿, 옥수수 → 유럽 / 말, 밀, 소 → 아메리카

### 라틴아메리카 독립 (1810-1830)

주요 인물 (**libertadores**, 해방자들):
- **Simón Bolívar** — 베네수엘라, 콜롬비아, 에콰도르, 페루, 볼리비아 해방
- **José de San Martín** — 아르헨티나, 칠레, 페루 해방
- **Miguel Hidalgo** — 멕시코 독립 운동 시작 (El Grito, 1810년 9월 16일)

```
+----------------------------------------------------------+
|          TYPICAL MEAL SCHEDULE (SPAIN)                     |
+----------------------------------------------------------+
|                                                          |
|  Desayuno (7:00-9:00):  Light — coffee, toast, pastry   |
|  Almuerzo (14:00-15:30): Main meal of the day            |
```

### 사회적 규범

| 개념 | 설명 |
|---------|-------------|
| **La familia** | 가족은 히스패닉 문화의 핵심입니다. 확대 가족(조부모, 사촌, 삼촌/이모)이 근처에 살며 자주 모입니다. |
| **La sobremesa** | 식후 대화 시간으로, 식사 직후 바로 자리를 뜨는 것은 무례한 것으로 여겨집니다. |
| **La siesta** | 오후 휴식 전통 (현대 도시에서는 줄어들고 있지만 여전히 문화적으로 중요합니다). |
| **시간 관념(Punctuality)** | 나라와 상황에 따라 다릅니다. 사교 행사는 종종 정해진 시간보다 늦게 시작됩니다. 비즈니스 회의는 일반적으로 시간을 지킵니다. |
| **Tú vs. Usted** | 잘못된 형태를 사용하면 지나치게 친밀하거나 차갑게 느껴질 수 있습니다. 확신이 없을 때는 usted를 사용하세요. |

### 유용한 문화 어휘

| 스페인어 | 한국어 |
|---------|---------|
| la tertulia | 대화를 위한 정기적인 사교 모임 |
| el piropo | 길거리 칭찬 (논란이 있는 전통) |
| el paseo | 저녁 산책 (마을/도시에서 흔한 관습) |
| las tapas | 함께 나눠 먹는 소량의 요리 (스페인) |
| la quinceañera | 15번째 생일 축하 |
| el compadrazgo | 대부모 제도 (사회적 유대 관계) |

### 연습 2: 문화 지식

다음 질문에 답하세요.

1. Sobremesa란 무엇인가요?
2. 고인을 추모하는 멕시코의 전통 명절 이름은?
3. *Don Quijote de la Mancha*의 저자는 누구인가요?
4. Alhambra는 무엇이며 어디에 있나요?
5. "Las doce uvas"는 무엇인가요?

<details>
<summary>정답</summary>

1. 식사 후 식탁에 남아 대화를 나누는 전통입니다.
2. **Día de los Muertos** (죽은 자의 날, 11월 1-2일).
3. **Miguel de Cervantes** (1605년).
4. 스페인(España) **그라나다(Granada)**에 있는 무어 양식 궁전 및 요새입니다.
5. 자정에 종이 울릴 때마다 포도 한 알씩 총 12알을 먹는 스페인의 새해 전통입니다.

</details>
