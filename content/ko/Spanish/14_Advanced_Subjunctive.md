## 1. 복습: 현재 접속법

"반대 모음" 규칙 빠른 참조:

| | -AR → -e | -ER/-IR → -a |
|---|---|---|
| yo | hable | coma / viva |
| tú | hables | comas / vivas |
| él | hable | coma / viva |
| nosotros | hablemos | comamos / vivamos |
| ellos | hablen | coman / vivan |

주요 불규칙 동사: **sea, esté, vaya, haya, sepa, dé**

---

## 2. 불완료과거 접속법

### 형성 규칙

**ellos 단순과거(preterite)** 형태에서 **-ron**을 제거하고 어미를 추가합니다:

| Ellos 단순과거 | 어간 | -ra 형 (yo) | -se 형 (yo) |
|----------------|------|---------------|---------------|
| habla**ron** | habla- | habla**ra** | habla**se** |
| comie**ron** | comie- | comie**ra** | comie**se** |
| vivie**ron** | vivie- | vivie**ra** | vivie**se** |
| fue**ron** | fue- | fue**ra** | fue**se** |
| tuvie**ron** | tuvie- | tuvie**ra** | tuvie**se** |
| dije**ron** | dije- | dije**ra** | dije**se** |

### 전체 활용표 (-ra 형, 더 일반적)

| 인칭 | hablar | tener | ir/ser |
|--------|--------|-------|--------|
| yo | hablara | tuviera | fuera |
| tú | hablaras | tuvieras | fueras |
| él/ella | hablara | tuviera | fuera |
| nosotros | habláramos | tuviéramos | fuéramos |
| vosotros | hablarais | tuvierais | fuerais |
| ellos | hablaran | tuvieran | fueran |

> -ra 형과 -se 형 모두 올바른 형태입니다. -ra 형은 구어에서 더 일반적이고,
> -se 형은 문어에서 더 일반적입니다.

## 3. 현재완료 접속법

```
HABER (현재 접속법) + 과거분사
```

| 인칭 | 형태 |
|--------|------|
| yo | haya hablado |
| tú | hayas comido |
| él/ella | haya vivido |
| nosotros | hayamos hecho |
| ellos | hayan dicho |

**용법**: 접속법 유발 표현이 현재이지만 동작이 과거인 경우:
- Espero que **hayas dormido** bien. (잘 잤기를 바란다.)
- No creo que **haya llegado**. (그가 도착했다고 생각하지 않는다.)
- Me alegra que **hayas aprobado**. (네가 합격해서 기쁘다.)

---

## 4. 과거완료 접속법

```
HABER (불완료과거 접속법) + 과거분사
```

| 인칭 | 형태 |
|--------|------|
| yo | hubiera hablado |
| tú | hubieras comido |
| él/ella | hubiera vivido |
| nosotros | hubiéramos hecho |
| ellos | hubieran dicho |

**용법**: 사실에 반하는 과거 상황:
- Si **hubiera estudiado** más, habría aprobado.
  (더 공부했더라면, 합격했을 텐데.)
- Ojalá **hubiéramos** llegado a tiempo.
  (제시간에 도착했더라면 좋았을 텐데.)

---

## 5. 시제 일치

```
+------------------------------------------------------------+
|  주절 시제                      →  종속절 접속법              |
+------------------------------------------------------------+
|  현재 / 미래 / 명령             →  현재 접속법                |
|  Quiero que vengas.               (네가 오기를 원한다.)       |
|                                                            |
|  현재 + 과거 동작               →  현재완료 접속법            |
|  Espero que haya llegado.         (그가 도착했기를 바란다.)    |
|                                                            |
|  과거 / 조건법                  →  불완료과거 접속법           |
|  Quería que vinieras.             (네가 오기를 원했다.)       |
|                                                            |
|  과거 + 더 이전의 과거          →  과거완료 접속법             |
|  Dudaba que hubiera venido.       (그가 왔었는지 의심했다.)    |
+------------------------------------------------------------+
```

## 6. Si-절 (조건문)

### 유형 1: 실현 가능/현실적 (Si + 현재 직설법)

- Si **llueve**, no vamos. (비가 오면, 우리는 가지 않는다.)
- Si **tienes** hambre, come algo. (배가 고프면, 뭔가 먹어라.)

### 유형 2: 가능성 낮음/가정 (Si + 불완료과거 접속법)

- Si **tuviera** dinero, **viajaría** por el mundo.
  (돈이 있다면, 세계를 여행할 텐데.)
- Si **fueras** presidente, ¿qué **harías**?
  (네가 대통령이라면, 무엇을 할 것인가?)

### 유형 3: 불가능/과거 (Si + 과거완료 접속법)

- Si **hubiera** estudiado más, **habría** aprobado.
  (더 공부했더라면, 합격했을 텐데.)
- Si no **hubiéramos** salido tarde, **habríamos** llegado a tiempo.
  (늦게 출발하지 않았더라면, 제시간에 도착했을 텐데.)

```
+----------------------------------------------------------+
|  조건문 유형 요약                                          |
+----------------------------------------------------------+
|  유형 | Si 절                  | 결과절                    |
|  ─────┼────────────────────────┼──────────────────────── |
|   1   | present indicative     | present/future/command   |
|   2   | imperfect subjunctive  | conditional              |
|   3   | pluperfect subjunctive | conditional perfect      |
+----------------------------------------------------------+
|  SI 뒤에 현재 접속법을 절대 사용하지 마세요!               |
|  Si llueva... ✗   Si llueve... ✓                         |
+----------------------------------------------------------+
```

## 7. 접속법 유발 표현 확장

### 소망과 영향 (Wishes and Influence)

| 유발 표현 | 예문 |
|---------|---------|
| querer que | Quiero que **vengas**. |
| esperar que | Espero que **estés** bien. |
| desear que | Deseo que **tengas** éxito. |
| pedir que | Te pido que **hables** más despacio. |
| decir que (명령) | Le dije que **viniera**. |
| aconsejar que | Te aconsejo que **estudies**. |
| prohibir que | Prohíben que **fumemos** aquí. |

### 감정 (Emotions)

| 유발 표현 | 예문 |
|---------|---------|
| alegrarse de que | Me alegro de que **hayas** venido. |
| sentir que | Siento que **estés** enfermo. |
| temer que | Temo que no **llegue** a tiempo. |
| sorprender que | Me sorprende que **sepas** eso. |

### 의심과 부정 (Doubt and Denial)

| 유발 표현 | 예문 |
|---------|---------|
| dudar que | Dudo que **sea** verdad. |
| no creer que | No creo que **pueda** venir. |
| negar que | Niega que lo **haya** hecho. |
| no es seguro que | No es seguro que **funcione**. |

## 8. 접속법 vs. 직설법: 까다로운 경우

| 긍정문 → 직설법 | 부정문 → 접속법 |
|-------------------------|----------------------|
| Creo que **tiene** razón. | No creo que **tenga** razón. |
| Es seguro que **viene**. | No es seguro que **venga**. |
| Es verdad que **sabe**. | No es verdad que **sepa**. |

**Aunque** (~이지만/~일지라도):
- Aunque **llueve**, salgo. (비가 오지만 [사실], 나간다.) — 직설법(Indicative)
- Aunque **llueva**, saldré. (비가 오더라도 [불확실], 나갈 것이다.) — 접속법(Subjunctive)

**Tal vez / Quizás** (아마):
- Tal vez **viene**. (아마 그가 올 것이다. — 화자가 어느 정도 확신)
- Tal vez **venga**. (아마 그가 올지도 모른다. — 더 불확실)

## 9. 연습 문제

### 연습 1: 불완료과거 접속법

불완료과거 접속법(-ra 형)으로 활용하세요.

1. Si yo (tener) ________ más tiempo...
2. Quería que tú (venir) ________.
3. Ojalá (ser) ________ más fácil.
4. Nos pidió que (hacer) ________ el trabajo.
5. Si ellos (poder) ________, vendrían.

<details>
<summary>정답</summary>

1. **tuviera** 2. **vinieras** 3. **fuera** 4. **hiciéramos** 5. **pudieran**

</details>

### 연습 2: Si-절 (조건문)

올바른 시제로 완성하세요.

1. Si (llover) ________, no iremos al parque. [유형 1]
2. Si yo (ser) ________ tú, estudiaría más. [유형 2]
3. Si nosotros (saber) ________ antes, habríamos ayudado. [유형 3]
4. Si (tener, tú) ________ hambre, come algo. [유형 1]

<details>
<summary>정답</summary>

1. **llueve** (현재 직설법)
2. **fuera** (불완료과거 접속법)
3. **hubiéramos sabido** (과거완료 접속법)
4. **tienes** (현재 직설법)

</details>

### 연습 3: 올바른 서법 선택

직설법(Indicative)인가 접속법(Subjunctive)인가?

1. Es cierto que Juan (habla / hable) tres idiomas.
2. No es cierto que María (habla / hable) chino.
3. Aunque (llueve / llueva) mañana, iremos.
4. Creo que (puedes / puedas) hacerlo.
5. No creo que (puedes / puedas) terminarlo hoy.

<details>
<summary>정답</summary>

1. **habla** (직설법 — es cierto = 확실성)
2. **hable** (접속법 — no es cierto = 부정)
3. **llueva** (접속법 — 불확실한 미래: ~일지라도)
4. **puedes** (직설법 — 긍정 creer)
5. **puedas** (접속법 — 부정 creer = 의심)

</details>
