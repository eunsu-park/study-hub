## 1. 격식체 개요

비즈니스 스페인어는 **usted/ustedes**와 특정 격식 구조를 사용합니다:

```
+----------------------------------------------------------+
|          FORMALITY SPECTRUM                                |
+----------------------------------------------------------+
|                                                          |
|  INFORMAL              NEUTRAL              FORMAL       |
|  ─────────────────────────────────────────────────────   |
|  tú                    tú/usted             usted        |
|  ¡Hola!                Buenos días.          Estimado/a   |
|  vale                  de acuerdo            conforme     |
|  ¡Genial!              Muy bien.             Excelente.   |
|  ¡Chao!                Hasta luego.          Atentamente  |
|                                                          |
|  WHEN TO USE FORMAL REGISTER:                            |
|  • First contacts and initial meetings                   |
|  • Emails to superiors or external contacts              |
|  • Presentations and official events                     |
|  • Legal and contractual contexts                        |
+----------------------------------------------------------+
```

### 주요 격식 표현

| 비격식(Informal) | 격식(Formal) |
|----------|--------|
| ¿Puedes...? | ¿Podría usted...? / ¿Sería tan amable de...? |
| Quiero... | Quisiera... / Me gustaría... |
| ¿Qué piensas? | ¿Cuál es su opinión al respecto? |
| Gracias. | Le agradezco sinceramente. |
| Lo siento. | Le pido disculpas. / Lamento la inconveniencia. |
| OK. | De acuerdo. / Conforme. |

## 2. 비즈니스 이메일 작성

### 이메일 구조

| 구성 요소 | 스페인어 | 예시 |
|---------|---------|---------|
| **제목(Subject)** | Asunto: | Asunto: Propuesta de colaboración |
| **인사말(Greeting)** | Estimado/a + 이름/직함: | Estimada Sra. López: |
| **서두(Opening)** | Me dirijo a usted para... | Me dirijo a usted en relación con... |
| **본문(Body)** | 명확하고 구조화된 단락 | Le informo de que... / Adjunto... |
| **마무리(Closing)** | Quedo a la espera de... | Quedo a su disposición para... |
| **끝인사(Sign-off)** | Atentamente, | Un cordial saludo, |
| **서명(Signature)** | 전체 이름 + 직함 | Carlos Martínez, Director de Marketing |

### 서두 표현

| 용도 | 표현 |
|---------|--------|
| 첫 연락 | Me dirijo a usted por primera vez para... |
| 후속 연락 | En relación con nuestra conversación del [fecha]... |
| 회신 | Acuso recibo de su correo del [fecha]... |
| 요청 | Le escribo para solicitar información sobre... |
| 항의 | Me pongo en contacto con usted para expresar mi disconformidad con... |
| 감사 | Le agradezco el envío de... |

### 마무리 표현

| 용도 | 표현 |
|---------|--------|
| 답변 대기 | Quedo a la espera de su respuesta. |
| 도움 제안 | No dude en contactarme si necesita más información. |
| 가용성 표시 | Quedo a su entera disposición. |
| 감사 | Le agradezco de antemano su atención. |
| 긴급 | Le ruego nos responda a la mayor brevedad posible. |

### 첨부 및 참조 표현

| 스페인어 | 한국어 |
|---------|---------|
| Adjunto (encontrará)... | 첨부합니다... |
| Le remito... | 전달해 드립니다... |
| Según lo acordado... | 합의한 대로... |
| Con referencia a... | ~에 관하여... |
| Tal y como le comenté... | 말씀드린 바와 같이... |

## 3. 회의와 프레젠테이션

### 회의 어휘

| 스페인어 | 한국어 |
|---------|---------|
| la reunión | 회의 |
| el orden del día | 안건(Agenda) |
| el acta (de la reunión) | 회의록(Minutes) |
| la sala de reuniones / juntas | 회의실 |
| convocar una reunión | 회의를 소집하다 |
| asistir a una reunión | 회의에 참석하다 |
| aplazar / posponer | 연기하다 |
| cancelar | 취소하다 |
| el punto (del orden del día) | 안건 항목 |
| llegar a un acuerdo | 합의에 도달하다 |
| tomar una decisión | 결정을 내리다 |
| votar | 투표하다 |

### 회의 진행

| 단계 | 표현 |
|-------|------------|
| **개회(Opening)** | Empecemos / Comencemos la reunión. (회의를 시작합시다.) |
| | El objetivo de esta reunión es... (이번 회의의 목표는...) |
| | ¿Podemos comenzar? (시작할까요?) |
| **주제 제시(Presenting a topic)** | El primer punto del orden del día es... (첫 번째 안건은...) |
| | Me gustaría hablar sobre... (~에 대해 이야기하고 싶습니다.) |
| **의견 제시(Giving opinions)** | Desde mi punto de vista... (제 관점에서...) |
| | Propongo que... (~을 제안합니다.) |
| | En mi opinión... (제 의견으로는...) |
| **의견 요청(Asking for input)** | ¿Cuál es su opinión? (어떻게 생각하십니까?) |
| | ¿Qué opina(n) al respecto? (이에 대해 어떻게 생각하십니까?) |
| | ¿Alguien tiene algo que añadir? (추가할 사항이 있으십니까?) |
| **동의(Agreeing)** | Estoy completamente de acuerdo. (전적으로 동의합니다.) |
| | Comparto su punto de vista. (귀하의 관점에 동의합니다.) |
| **반대(Disagreeing)** | Con todo respeto, discrepo. (모든 존경을 담아, 동의하지 않습니다.) |
| | Tengo algunas reservas al respecto. (그 부분에 대해 다소 유보적입니다.) |
| | Entiendo su postura, pero... (귀하의 입장은 이해하지만...) |
| **폐회(Closing)** | Para resumir... / En resumen... (요약하자면...) |
| | ¿Hay alguna pregunta? (질문 있으십니까?) |
| | Gracias por su participación. (참여해 주셔서 감사합니다.) |

### 프레젠테이션 어휘

| 스페인어 | 한국어 |
|---------|---------|
| la presentación | 프레젠테이션(Presentation) |
| la diapositiva | 슬라이드(Slide) |
| el gráfico / la gráfica | 차트, 그래프 |
| la cifra / el dato | 수치, 데이터 포인트 |
| el porcentaje | 백분율(Percentage) |
| el aumento / el incremento | 증가 |
| la disminución / la reducción | 감소 |
| el resultado | 결과 |
| la tendencia | 추세(Trend) |
| la conclusión | 결론 |

### 추세 설명

| 방향 | 표현 |
|-----------|------------|
| **증가(Increase)** | aumentar, crecer, subir, incrementarse |
| | Las ventas **han aumentado** un 15%. (매출이 15% 증가했습니다.) |
| **감소(Decrease)** | disminuir, bajar, reducirse, descender |
| | Los costes **se han reducido** significativamente. |
| **안정(Stable)** | mantenerse estable, permanecer constante |
| **변동(Fluctuate)** | fluctuar, variar |

### 협상 표현

| 기능 | 표현 |
|----------|-----------|
| **제안하기(Proposing)** | Nos gustaría proponer... (~을 제안하고 싶습니다.) |
| | ¿Qué les parecería si...? (~하면 어떻겠습니까?) |
| **조건 제시(Conditions)** | Estaríamos dispuestos a... siempre y cuando... (~하는 한 기꺼이 ~하겠습니다.) |
| | Con la condición de que... (~라는 조건 하에...) |
| **양보 요청(Asking for concession)** | ¿Podrían considerar...? (~을 고려해 주실 수 있습니까?) |
| | ¿Sería posible...? (~이 가능하겠습니까?) |
| **수락(Accepting)** | Nos parece aceptable. (수용 가능해 보입니다.) |
| | Estamos de acuerdo con esas condiciones. |
| **정중한 거절(Rejecting politely)** | Lamentablemente, no nos es posible aceptar esa propuesta. |
| | Tendríamos que rechazar esa oferta. |
| **절충(Compromising)** | ¿Podríamos encontrar un punto intermedio? (절충점을 찾을 수 있을까요?) |
| **거래 성사(Closing a deal)** | Entonces, ¿tenemos un acuerdo? (그럼, 합의된 건가요?) |
| | Procederemos a redactar el contrato. (계약서 작성을 진행하겠습니다.) |

## 8. 비즈니스 숫자 표현

### 금융 수치

| 스페인어 | 한국어 |
|---------|---------|
| mil (1.000) | 1천(One Thousand) |
| diez mil (10.000) | 1만(Ten Thousand) |
| cien mil (100.000) | 10만(One Hundred Thousand) |
| un millón (1.000.000) | 100만(One Million) |
| mil millones (1.000.000.000) | 10억(One Billion) |
| un billón (1.000.000.000.000) | 1조(One Trillion) |

> **주의**: 스페인어의 **billón** = 영어의 **trillion** (10^12)입니다.
> 영어의 "billion" = 스페인어의 **mil millones** (10^9)입니다.

### 백분율과 추세

| 스페인어 | 한국어 |
|---------|---------|
| un aumento del 10% | 10% 증가 |
| una caída del 5% | 5% 하락 |
| se duplicó | 2배가 되었다 |
| se triplicó | 3배가 되었다 |
| representar el 30% | 30%를 차지하다 |
| crecer un 20% respecto al año anterior | 전년 대비 20% 성장하다 |

<summary>예시 답안</summary>

"Buenos días. Soy licenciado/a en Administración de Empresas y cuento con cinco años de experiencia en el sector de marketing digital. En mi puesto anterior, fui responsable de gestionar las campañas de publicidad en redes sociales, logrando un aumento del 30% en el engagement de la marca. Me considero una persona organizada, creativa y con capacidad de trabajo en equipo.
