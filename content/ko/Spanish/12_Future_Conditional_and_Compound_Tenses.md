## 1. IR A + 부정사 (가까운 미래)

일상 대화에서 미래 행동을 표현하는 가장 일반적인 방법:

```
ir (현재형으로 활용) + a + 부정사
```

| 주어 | 예문 |
|---------|---------|
| yo | **Voy a** estudiar. (공부할 거예요.) |
| tú | **Vas a** comer. (먹을 거예요.) |
| él/ella | **Va a** llover. (비가 올 거예요.) |
| nosotros | **Vamos a** viajar. (여행할 거예요.) |
| ellos | **Van a** llegar tarde. (그들은 늦게 도착할 거예요.) |

> 구어 스페인어(특히 라틴아메리카)에서는 가까운 미래 사건에 대해 단순 미래보다
> 이 형태가 훨씬 더 자주 사용됩니다.

---

## 2. 단순 미래 시제

**부정사 전체**에 어미를 붙여서 형성합니다:

| | -AR (hablar) | -ER (comer) | -IR (vivir) |
|---|---|---|---|
| yo | hablar**é** | comer**é** | vivir**é** |
| tú | hablar**ás** | comer**ás** | vivir**ás** |
| él/ella | hablar**á** | comer**á** | vivir**á** |
| nosotros | hablar**emos** | comer**emos** | vivir**emos** |
| vosotros | hablar**éis** | comer**éis** | vivir**éis** |
| ellos | hablar**án** | comer**án** | vivir**án** |

> 세 동사 그룹 모두 **같은** 미래 어미를 공유합니다.

### 불규칙 미래 어간 (12개 동사)

| 부정사 | 어간 | yo 형태 |
|-----------|------|---------|
| tener | **tendr-** | tendré |
| venir | **vendr-** | vendré |
| poner | **pondr-** | pondré |
| salir | **saldr-** | saldré |
| poder | **podr-** | podré |
| saber | **sabr-** | sabré |
| querer | **querr-** | querré |
| haber | **habr-** | habré |
| hacer | **har-** | haré |
| decir | **dir-** | diré |

### 단순 미래의 용법

| 용법 | 예문 |
|-----|---------|
| 예측 | Mañana lloverá. (내일 비가 올 거예요.) |
| 약속 | Te llamaré esta noche. (오늘 밤 전화할게요.) |
| **추측 (현재)** | ¿Qué hora será? (지금 몇 시쯤 될까?) |
| 격식체 발언 | El presidente hablará a las ocho. |

---

## 3. 조건 시제

미래와 같은 형성 패턴: **부정사 전체**에 어미를 붙이고, 불규칙 어간도 동일합니다.

| | hablar | comer | vivir |
|---|---|---|---|
| yo | hablar**ía** | comer**ía** | vivir**ía** |
| tú | hablar**ías** | comer**ías** | vivir**ías** |
| él/ella | hablar**ía** | comer**ía** | vivir**ía** |
| nosotros | hablar**íamos** | comer**íamos** | vivir**íamos** |
| vosotros | hablar**íais** | comer**íais** | vivir**íais** |
| ellos | hablar**ían** | comer**ían** | vivir**ían** |

### 조건법의 용법

| 용법 | 예문 |
|-----|---------|
| 가정 | Si tuviera dinero, **viajaría**. (돈이 있다면, 여행할 텐데.) |
| 정중한 요청 | ¿**Podrías** ayudarme? (도와주실 수 있나요?) |
| 추측 (과거) | **Serían** las diez. (아마 열 시쯤이었을 거예요.) |
| 간접 화법 | Dijo que **vendría**. (그는 올 거라고 말했어요.) |

---

## 4. 과거분사

| 동사 유형 | 규칙 | 예시 |
|-----------|------|---------|
| -AR | 어간 + **-ado** | hablar → habl**ado** |
| -ER | 어간 + **-ido** | comer → com**ido** |
| -IR | 어간 + **-ido** | vivir → viv**ido** |

### 불규칙 과거분사 (필수 암기)

| 부정사 | 분사형 | 부정사 | 분사형 |
|-----------|-----------|-----------|-----------|
| hacer | **hecho** | abrir | **abierto** |
| decir | **dicho** | poner | **puesto** |
| escribir | **escrito** | volver | **vuelto** |
| ver | **visto** | morir | **muerto** |
| romper | **roto** | cubrir | **cubierto** |

> 복합 시제에서 과거분사는 **변하지 않습니다** (항상 -o로 끝남):
> Ella **ha** comid**o**. (*comida*가 아님)

---

## 5. 현재완료

```
HABER (현재형) + 과거분사
```

| 주어 | HABER | 예문 |
|---------|-------|---------|
| yo | **he** | He comido. (먹었어요.) |
| tú | **has** | ¿Has visto esa película? (그 영화 봤어요?) |
| él/ella | **ha** | Ha terminado. (끝냈어요.) |
| nosotros | **hemos** | Hemos estudiado. (공부했어요.) |
| vosotros | **habéis** | ¿Habéis llegado? (도착했어요?) |
| ellos | **han** | Han salido. (나갔어요.) |

**주요 용법**: 인생 경험, 현재와 관련 있는 최근 과거.
- **Nunca** he viajado a Asia. (아시아에 가본 적이 없어요.)
- **Ya** hemos comido. (이미 먹었어요.)
- Hoy he estudiado mucho. (오늘 많이 공부했어요.)

> **지역 참고**: 스페인에서는 최근 사건에 현재완료를 사용합니다 (Hoy he ido al mercado).
> 라틴아메리카에서는 점과거(Preterite)를 선호합니다 (Hoy fui al mercado). 둘 다 맞습니다.

---

## 6. 대과거 (과거완료)

다른 과거 행동 **이전에** 완료된 행동을 표현합니다.

```
HABER (불완료과거형) + 과거분사
```

| 주어 | HABER | 예문 |
|---------|-------|---------|
| yo | **había** | Había terminado. (끝내 놓았었어요.) |
| tú | **habías** | ¿Habías comido? (먹었었어요?) |
| él/ella | **había** | Había salido. (나가 있었어요.) |
| nosotros | **habíamos** | No habíamos visto. (보지 못했었어요.) |
| ellos | **habían** | Habían llegado. (도착해 있었어요.) |

- Cuando llegué, ella ya **había salido**. (내가 도착했을 때, 그녀는 이미 나간 뒤였어요.)
- No **habíamos** visto esa película antes. (우리는 이전에 그 영화를 본 적이 없었어요.)

---

## 7. 미래완료와 조건완료

### 미래완료 (~했을 것이다)

```
HABER (미래형) + 과거분사
```

- Para las cinco, **habré terminado**. (다섯 시까지 끝내 놓을 거예요.)
- Cuando llegues, ya **habremos comido**. (네가 도착할 때, 우리는 이미 먹었을 거예요.)
- **추측**에도 사용: ¿**Habrá** llegado ya? (벌써 도착했을까?)

### 조건완료 (~했을 텐데)

```
HABER (조건형) + 과거분사
```

- **Habría** ido, pero estaba enfermo. (갔을 텐데, 아팠어요.)
- Sin tu ayuda, no **habríamos** terminado. (네 도움이 없었다면, 끝내지 못했을 거예요.)

## 8. 시제 비교 차트

```
+------------------------------------------------------------+
|           COMPOUND TENSE MASTER FORMULA                     |
+------------------------------------------------------------+
|                                                            |
|  Present Perfect:     he/has/ha/hemos/habéis/han + pp      |
|  Pluperfect:          había/habías/.../habían + pp          |
|  Future Perfect:      habré/habrás/.../habrán + pp          |
|  Conditional Perfect: habría/habrías/.../habrían + pp       |
|                                                            |
|  pp = past participle (invariable -o form)                 |
|  Nothing between HABER and the participle!                  |
|                                                            |
+------------------------------------------------------------+
```

### IR A vs. 미래 vs. 조건법 선택

| | IR A + inf. | 단순 미래 | 조건법 |
|---|---|---|---|
| 격식 수준 | 비격식 | 격식 | 가정적 |
| 확실성 | 높음 (계획된) | 가변적 | ~할 텐데/~할 수 있을 텐데 |
| 예문 | Voy a comer. | Comeré. | Comería. |

## 9. 연습 문제

### 연습 1: 단순 미래

미래 시제로 활용하세요.

1. Mañana nosotros ________ (salir) temprano.
2. ¿Tú ________ (poder) venir?
3. Yo ________ (hacer) la cena.
4. Ellos ________ (hablar) con el director.
5. Ella ________ (tener) que trabajar.

<details>
<summary>정답</summary>

1. **saldremos** 2. **podrás** 3. **haré** 4. **hablarán** 5. **tendrá**

</details>

### 연습 2: 조건법

1. Si tuviera tiempo, ________ (viajar) a Japón.
2. ¿Qué ________ (hacer) tú en mi situación?
3. Dijo que ________ (venir) a las ocho.
4. ¿________ (poder) usted repetir eso?
5. Me ________ (gustar) un café.

<details>
<summary>정답</summary>

1. **viajaría** 2. **harías** 3. **vendría** 4. **Podría** 5. **gustaría**

</details>

### 연습 3: 복합 시제

올바른 복합 시제를 선택하세요.

1. Todavía no ________ (comer, nosotros). [현재완료]
2. Cuando llegaste, yo ya ________ (salir). [대과거]
3. Para junio, ellos ________ (terminar) el proyecto. [미래완료]
4. Sin tu consejo, yo ________ (tomar) una mala decisión. [조건완료]

<details>
<summary>정답</summary>

1. **hemos comido** 2. **había salido** 3. **habrán terminado** 4. **habría tomado**

</details>
