## 1. 신체 관련 관용구

| 관용구 | 직역 | 실제 의미 |
|-------|------|----------|
| Costar un ojo de la cara | 얼굴에서 눈 하나 값 | 매우 비싸다 |
| Estar hasta las narices | 코까지 차다 | 진절머리가 나다 |
| No tener pelos en la lengua | 혀에 털이 없다 | 거침없이 말하다 |
| Meter la pata | 발을 들이밀다 | 실수하다 |
| Tomar el pelo a alguien | 누군가의 머리카락을 잡다 | 놀리다, 장난치다 |
| Dar en el clavo | 못을 맞히다 | 핵심을 찌르다 |
| Estar con el agua al cuello | 목까지 물이 차다 | 큰 곤경에 빠지다 |
| Hacer la vista gorda | 시야를 뚱뚱하게 만들다 | 못 본 척하다 |
| Quedarse con la boca abierta | 입이 벌어진 채로 있다 | 깜짝 놀라다 |
| Ser uña y carne | 손톱과 살이 되다 | 떼려야 뗄 수 없는 사이다 |

### 예문

- Ese restaurante **cuesta un ojo de la cara**. (그 레스토랑은 매우 비싸다.)
- ¡**Metí la pata** en la reunión! (회의에서 실수해 버렸어!)
- No me **tomes el pelo**, dime la verdad. (나를 놀리지 마, 진실을 말해.)
- María y yo somos **uña y carne**. (마리아와 나는 떼려야 뗄 수 없는 사이야.)

## 2. 동물 관련 관용구

| 관용구 | 직역 | 실제 의미 |
|-------|------|----------|
| Ser un gallina | 닭이 되다 | 겁쟁이다 |
| Estar como una cabra | 염소 같다 | 미쳤다 |
| Llevarse como el perro y el gato | 개와 고양이처럼 지내다 | 사이가 나쁘다 |
| Ser un burro | 당나귀가 되다 | 멍청하다/고집이 세다 |
| A paso de tortuga | 거북이 걸음으로 | 매우 느리게 |
| Tener memoria de elefante | 코끼리 기억력을 가지다 | 기억력이 매우 좋다 |
| Ser un pez gordo | 뚱뚱한 물고기가 되다 | 거물이다 |
| Dormir como un tronco | 통나무처럼 자다 | 깊이 자다 |
| Matar dos pájaros de un tiro | 한 발로 새 두 마리를 잡다 | 일석이조 |
| Buscar tres pies al gato | 고양이에게서 세 번째 발을 찾다 | 일을 복잡하게 만들다 |

## 3. 음식 관련 관용구

| 관용구 | 직역 | 실제 의미 |
|-------|------|----------|
| Pan comido | 먹은 빵 | 식은 죽 먹기 (쉬운 일) |
| Importar un pepino | 오이 하나만큼 중요하다 | 전혀 신경 쓰지 않다 |
| Ponerse como un tomate | 토마토처럼 되다 | 얼굴이 빨개지다 |
| Dar calabazas | 호박을 주다 | (연애에서) 거절하다 |
| Ser la media naranja | 반쪽 오렌지가 되다 | 운명의 짝이다 |
| Ir al grano | 곡식 알갱이로 가다 | 본론으로 들어가다 |
| Estar en su salsa | 자기 소스 안에 있다 | 물 만난 고기처럼 신나다 |
| Tener mala leche | 나쁜 우유를 가지다 | 성질이 나쁘다 |
| Comerse el coco | 코코넛을 먹다 | 너무 많이 생각하다 |
| Ponerse las pilas | 배터리를 넣다 | 정신 차리다 |

### 예문

- Este examen es **pan comido**. (이 시험은 식은 죽 먹기야.)
- Me importa un **pepino** lo que digan. (남들이 뭐라 하든 전혀 신경 안 써.)
- Se puso como un **tomate** cuando le hablé. (내가 말을 걸자 얼굴이 빨개졌어.)
- ¡Deja de **comerte el coco** y actúa! (그만 고민하고 행동해!)

## 4. 색깔 관련 관용구

| 관용구 | 직역 | 실제 의미 |
|-------|------|----------|
| Ponerse rojo/a | 빨갛게 되다 | 얼굴이 빨개지다 |
| Estar verde | 초록이다 | 경험이 부족하다 |
| Quedarse en blanco | 하얗게/빈 상태가 되다 | 머리가 하얗게 되다 (기억이 안 나다) |
| Ver todo negro | 모든 것이 검게 보이다 | 비관적이다 |
| Pasar la noche en blanco | 하얀 밤을 보내다 | 밤을 새우다 |
| Príncipe azul | 파란 왕자 | 백마 탄 왕자 |
| Verlo todo de color de rosa | 모든 것이 분홍색으로 보이다 | 지나치게 낙관적이다 |
| Poner verde a alguien | 누군가를 초록으로 만들다 | 뒷담화를 하다 |

## 5. 날씨와 자연 관용구

| 관용구 | 의미 |
|-------|------|
| Llueve a cántaros | 억수같이 쏟아지다 (항아리로 비가 온다) |
| Hacer un frío que pela | 살을 에는 추위 (껍질을 벗기는 추위) |
| Después de la tormenta viene la calma | 폭풍 뒤에 고요가 온다 |
| Llover sobre mojado | 젖은 땅 위에 비가 오다 (설상가상) |
| Estar en las nubes | 구름 위에 있다 (멍 때리다) |
| Ser un rayo de sol | 한 줄기 햇살이다 (밝은 사람) |
| Quedarse helado | 얼어붙다 (충격을 받다) |

## 6. 돈과 직업 관용구

| 관용구 | 의미 |
|-------|------|
| Ganarse la vida | 생계를 꾸리다 |
| Estar sin blanca | 무일푼이다 (한 푼도 없다) |
| Tirar la casa por la ventana | 아낌없이 쓰다 (집을 창문으로 던지다) |
| Costar un riñón | 신장 하나 값이다 (매우 비싸다) |
| Trabajar como un burro | 당나귀처럼 일하다 (매우 열심히 일하다) |
| Vivir del cuento | 빈둥거리며 살다 (이야기로 먹고 살다) |
| Apretarse el cinturón | 허리띠를 졸라매다 |
| Ganarse el pan | 빵을 벌다 (생계를 유지하다) |
| Estar forrado | 돈이 많다 (부자이다) |
| Hacer el agosto | 대박을 치다 (8월을 만들다) |

## 8. 감정과 관계 관용구

### 감정

| 관용구 | 의미 |
|-------|------|
| Morirse de risa | 웃겨 죽겠다 |
| Ponerse de los nervios | 매우 초조해지다 |
| Estar hecho polvo | 녹초가 되다 (먼지가 되다) |
| Quedarse de piedra | 돌이 되다 (경악하다) |
| Llorar a moco tendido | 펑펑 울다 |
| Hervir la sangre | 피가 끓다 (분노하다) |
| Perder los estribos | 등자를 잃다 (이성을 잃다) |
| Subirse por las paredes | 벽을 타고 올라가다 (극도로 화나다) |

### 관계

| 관용구 | 의미 |
|-------|------|
| Llevarse bien/mal | 사이가 좋다/나쁘다 |
| Caer bien/mal | 첫인상이 좋다/나쁘다 |
| Romper el hielo | 얼음을 깨다 (어색함을 풀다) |
| Dar la espalda | 등을 돌리다 |
| Echar de menos | 그리워하다 |
| Hacer buenas migas | 잘 어울리다 (좋은 빵 부스러기를 만들다) |
| Tirar la toalla | 수건을 던지다 (포기하다) |
| Sentar cabeza | 정착하다, 안정되다 |

## 9. 일반 속담

### 삶의 지혜

| 속담 | 한국어 대응 표현 |
|------|----------------|
| Más vale tarde que nunca. | 늦더라도 안 하는 것보다 낫다. |
| No hay mal que por bien no venga. | 전화위복 (모든 구름에는 은빛 테두리가 있다). |
| Dime con quién andas y te diré quién eres. | 네 친구를 보면 네가 누군지 안다. |
| El que mucho abarca, poco aprieta. | 많이 움켜쥐면 적게 쥔다. (이것저것 다 하면 하나도 못한다.) |
| A quien madruga, Dios le ayuda. | 일찍 일어나는 새가 벌레를 잡는다. |
| Más vale prevenir que curar. | 예방이 치료보다 낫다. |
| No es oro todo lo que reluce. | 빛나는 것이 다 금은 아니다. |
| En boca cerrada no entran moscas. | 닫힌 입에는 파리가 들어가지 않는다. (침묵은 금이다.) |
| Quien no arriesga, no gana. | 호랑이 굴에 들어가야 호랑이를 잡는다. |
| A caballo regalado no le mires el diente. | 선물 받은 말의 이빨을 보지 마라. (받은 선물에 흠잡지 마라.) |

### 관계와 사람

| 속담 | 한국어 대응 표현 |
|------|----------------|
| Más vale solo que mal acompañado. | 나쁜 동행보다 혼자가 낫다. |
| Ojos que no ven, corazón que no siente. | 보지 않으면 마음도 아프지 않다. (눈에서 멀어지면 마음에서도 멀어진다.) |
| De tal palo, tal astilla. | 그 나무에 그 조각. (부전자전.) |
| Perro que ladra no muerde. | 짖는 개는 물지 않는다. |
| Cada maestrillo tiene su librillo. | 각 선생에게는 자기만의 교본이 있다. (사람마다 자기만의 방법이 있다.) |

### 노력과 인내

| 속담 | 한국어 대응 표현 |
|------|----------------|
| La práctica hace al maestro. | 연습이 달인을 만든다. |
| Quien la sigue la consigue. | 끈기 있게 하면 이룬다. |
| No dejes para mañana lo que puedas hacer hoy. | 오늘 할 수 있는 일을 내일로 미루지 마라. |
| Roma no se construyó en un día. | 로마는 하루아침에 이루어지지 않았다. |

## 10. 구어체로 감정 표현하기

### 기쁨과 흥분

| 표현 | 상황/의미 |
|------|----------|
| ¡Qué alegría! | 정말 기쁘다! |
| ¡Genial! / ¡Estupendo! / ¡Fenomenal! | 대단해! / 훌륭해! / 환상적이야! |
| ¡Qué ilusión! | 정말 설레! (스페인) |
| ¡No me lo puedo creer! | 믿을 수가 없어! |
| ¡Me hace muy feliz! | 정말 행복해! |

### 슬픔과 실망

| 표현 | 상황/의미 |
|------|----------|
| ¡Qué pena! | 정말 안타깝다! |
| ¡Qué decepción! | 정말 실망이야! |
| Me siento fatal. | 기분이 최악이야. |
| No puedo más. | 더 이상 못 참겠어. |

### 분노와 짜증

| 표현 | 상황/의미 |
|------|----------|
| ¡Qué rabia! | 정말 짜증나! |
| ¡Es el colmo! | 이건 도를 넘었어! |
| ¡Ya está bien! | 이제 그만 해! |
| ¡Basta! | 그만! |
| ¡No aguanto más! | 더 이상 못 참겠어! |
| Me tiene harto/a. | 정말 질렸어. |

### 놀라움과 불신

| 표현 | 상황/의미 |
|------|----------|
| ¡No me digas! | 설마! / 정말? |
| ¡No puede ser! | 그럴 리가! |
| ¿En serio? / ¿De verdad? | 정말? / 진짜? |
| ¡Anda! | 어머! (스페인) |
| ¡Qué fuerte! | 대박! / 세상에! (스페인) |

## 11. 연습 문제

### 연습 1: 관용구 짝짓기

각 관용구와 그 의미를 연결하세요.

1. Costar un ojo de la cara        a) 실수하다
2. Meter la pata                   b) 매우 비싸다
3. Pan comido                      c) 본론으로 들어가다
4. Ir al grano                     d) 식은 죽 먹기
5. Quedarse en blanco              e) 머리가 하얗게 되다

<details>
<summary>정답</summary>

1-b, 2-a, 3-d, 4-c, 5-e

</details>

### 연습 2: 신체 부위 채우기

빠진 신체 부위를 채우세요.

1. Costar un ojo de la ________ (매우 비싸다)
2. Estar hasta las ________ (진절머리가 나다)
3. No tener pelos en la ________ (거침없이 말하다)
4. Quedarse con la ________ abierta (깜짝 놀라다)
5. Ser ________ y carne (떼려야 뗄 수 없는 사이다)

<details>
<summary>정답</summary>

1. **cara** (얼굴) 2. **narices** (코) 3. **lengua** (혀)
4. **boca** (입) 5. **uña** (손톱)

</details>

### 연습 3: 속담 완성하기

기억을 더듬어 속담을 완성하세요.

1. Más vale tarde que ________
2. No es oro todo lo que ________
3. Dime con quién andas y te diré quién ________
4. A quien madruga, Dios le ________
5. La práctica hace al ________

<details>
<summary>정답</summary>

1. **nunca** (늦더라도 안 하는 것보다 낫다)
2. **reluce** (빛나는 것이 다 금은 아니다)
3. **eres** (네 친구를 보면 네가 누군지 안다...)
4. **ayuda** (일찍 일어나는 새가 벌레를 잡는다)
5. **maestro** (연습이 달인을 만든다)

</details>

### 연습 4: 상황에 맞는 표현 찾기

각 상황에서 어떤 표현을 사용하겠습니까?

1. 친구가 끔찍한 머리를 하고 마음에 드냐고 물었을 때.
2. 매우 어려운 시험을 쉽게 통과했을 때.
3. 무언가를 설명하는데 상대방이 계속 주제에서 벗어날 때.
4. 12시간 근무 후 녹초가 되었을 때.
5. 친구와 떼려야 뗄 수 없는 사이일 때.

<details>
<summary>정답</summary>

1. **Hacer la vista gorda** (못 본 척하다) 또는 적당히 넘기기
2. ¡Fue **pan comido**! (식은 죽 먹기였어!)
3. ¡Ve **al grano**! (본론으로 들어가!)
4. Estoy **hecho/a polvo**. (녹초가 됐어.)
5. Somos **uña y carne**. (우리는 떼려야 뗄 수 없는 사이야.)

</details>
