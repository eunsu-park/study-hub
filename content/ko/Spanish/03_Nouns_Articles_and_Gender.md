## 1. 문법적 성이란?

모든 스페인어 명사에는 문법적 성(Grammatical Gender)이 있습니다: **남성(masculino)** 또는 **여성(femenino)**. 이것은 생물학적 성별과는 관계가 없습니다 — "탁자"(la mesa, 여성)나 "책"(el libro, 남성)과 같은 사물에도 문법적 성이 있습니다.

```
┌─────────────────────────────────────────────────────────────────┐
│   English: "the book", "the table", "the house"                │
│   (명사에 관계없이 동일한 관사 사용)                                │
│                                                                 │
│   Spanish: "el libro", "la mesa", "la casa"                    │
│   (명사의 성에 맞게 관사가 바뀜)                                   │
│                                                                 │
│   왜 중요한가:                                                    │
│   • 관사가 일치해야 함: el libro / la mesa                        │
│   • 형용사가 일치해야 함: libro nuevo / mesa nueva                 │
│   • 대명사가 일치해야 함: lo compré / la compré                    │
└─────────────────────────────────────────────────────────────────┘
```

> **한국어 화자를 위한 참고**: 한국어 명사에는 문법적 성이 없습니다 (책, 탁자는 성 중립입니다). 스페인어에서는 모든 명사의 성을 관사와 함께 외워야 합니다. 단순히 "libro"가 아니라 **el libro**로 학습하세요.

## 2. 성 판별 규칙

### 2.1 남성 명사 표지

```
일반적으로 남성인 경우:
  -o 어미:    el libro (책), el gato (고양이), el año (해)
  -or 어미:   el doctor, el amor (사랑), el color
  -aje 어미:  el viaje (여행), el paisaje (풍경)
  -mente 어미: el accidente, el continente
  요일:       el lunes, el martes, el miércoles...
  달:         enero, febrero (보통 관사 없이 쓰지만 남성)
  언어:       el español, el inglés, el coreano
```

### 2.2 여성 명사 표지

```
일반적으로 여성인 경우:
  -a 어미:      la casa (집), la mesa (탁자), la persona (사람)
  -ción 어미:   la nación (국가), la canción (노래)
  -sión 어미:   la televisión, la decisión
  -dad 어미:    la ciudad (도시), la universidad (대학교)
  -tad 어미:    la libertad (자유), la amistad (우정)
  -ez 어미:     la vez (번, 회), la vejez (노년)
  -eza 어미:    la belleza (아름다움), la naturaleza (자연)
  -tud 어미:    la actitud, la juventud (청춘)
  글자:         la a, la be, la ce...
```

### 2.3 중요한 예외

```
┌─────────────────────────────────────────────────────────────────┐
│   -a로 끝나지만 남성인 단어 (외워야 합니다!):                        │
│   el día (날)          el mapa (지도)      el problema (문제)    │
│   el programa          el sistema          el clima (기후)       │
│   el tema (주제)       el idioma (언어)     el drama              │
│   el planeta           el poema            el diploma            │
│                                                                 │
│   ※ 그리스어에서 온 -ma 단어 다수가 남성입니다                       │
│                                                                 │
│   -o로 끝나지만 여성인 단어 (외워야 합니다!):                        │
│   la mano (손)         la foto (사진)      la moto (오토바이)     │
│   la radio (라디오)                                              │
│                                                                 │
│   성 중립 (형태는 같고 관사만 변경):                                 │
│   el/la estudiante    el/la artista       el/la turista         │
│   el/la periodista    el/la pianista      el/la dentista        │
└─────────────────────────────────────────────────────────────────┘
```

## 3. 정관사

### 3.1 "the"의 네 가지 형태

| | 남성 | 여성 |
|---|-----------|----------|
| **단수** | **el** | **la** |
| **복수** | **los** | **las** |

```
el libro   → the book (그 책)         la casa   → the house (그 집)
los libros → the books (그 책들)      las casas → the houses (그 집들)
```

### 3.2 특수 규칙: 강세 a-/ha-로 시작하는 여성 명사 + el

강세가 있는 **a-** 또는 **ha-**로 시작하는 여성 단수 명사는 발음상의 이유로 **el**을 사용하지만, 여전히 여성입니다:

```
el agua    → the water (물)      (여성! → las aguas, el agua fría)
el águila  → the eagle (독수리)   (여성! → las águilas)
el alma    → the soul (영혼)     (여성! → las almas)
el hambre  → the hunger (배고픔)  (여성! → las hambres)
el aula    → the classroom (교실) (여성! → las aulas)

※ a-/ha-에 강세가 있을 때만 해당:
la amiga (여자 친구) → "el amiga"가 아님 (강세가 -mi-에 있지 a-에 없음)
```

### 3.3 축약형

스페인어에는 관사와 함께 쓰이는 필수 축약형이 정확히 두 가지 있습니다:

```
a + el = al       →  Voy al parque. (나는 공원에 간다.)
de + el = del     →  El libro del profesor. (교수님의 책.)

※ la/los/las에서는 축약하지 않음:
  Voy a la escuela.  ("al escuela"가 아님)
  El libro de la profesora.
```

### 3.4 정관사 사용 시점 (영어와 다른 경우)

```
영어에서 쓰지 않지만 스페인어에서 "the"를 쓰는 경우:
  Me gusta el chocolate.       → I like chocolate. (일반적 개념)
  La vida es bella.            → Life is beautiful.
  Los perros son leales.       → Dogs are loyal.
  El español es útil.          → Spanish is useful.
  Tengo clase los lunes.       → I have class on Mondays.

영어에서 쓰지만 스페인어에서 "the"를 생략하는 경우:
  Soy estudiante.              → I'm a student. (직업, 관사 없음)
  Necesito café.               → I need coffee. (일부, 관사 없음)
```

## 4. 부정관사

### 4.1 "a/an/some"의 네 가지 형태

| | 남성 | 여성 |
|---|-----------|----------|
| **단수** | **un** | **una** |
| **복수** | **unos** | **unas** |

```
un libro   → a book (책 한 권)       una casa   → a house (집 한 채)
unos libros → some books (책 몇 권)  unas casas → some houses (집 몇 채)
```

### 4.2 부정관사를 생략하는 경우

```
ser + 직업 (형용사 없이):
  Soy profesor.          → I'm a teacher. (나는 교사다.)
  Es médica.             → She's a doctor. (그녀는 의사다.)
  BUT: Es un buen médico. → He's a good doctor. (형용사 → 관사 사용)

sin (없이) 뒤에서:
  Salgo sin paraguas.    → I go out without an umbrella. (우산 없이 나간다.)

tener 부정문에서:
  No tengo coche.        → I don't have a car. (차가 없다.)
```

## 5. 복수형 만들기

### 5.1 세 가지 규칙

```
┌─────────────────────────────────────────────────────────────────┐
│   규칙 1: 모음으로 끝나면 → -s 추가                                │
│   libro → libros        casa → casas       estudiante → estudiantes │
│                                                                 │
│   규칙 2: 자음으로 끝나면 → -es 추가                               │
│   papel → papeles       ciudad → ciudades   profesor → profesores │
│                                                                 │
│   규칙 3: -z로 끝나면 → z를 c로 바꾸고 -es 추가                    │
│   lápiz → lápices       vez → veces         pez → peces         │
│                                                                 │
│   특수한 경우:                                                    │
│   비강세 -s로 끝나는 단어 → 변화 없음                               │
│   el lunes → los lunes     la crisis → las crisis               │
│   el paraguas → los paraguas                                    │
└─────────────────────────────────────────────────────────────────┘
```

| el hijo / la hija | 아들 / 딸 | m / f |
| el hermano / la hermana | 형제 / 자매 | m / f |
| el abuelo / la abuela | 할아버지 / 할머니 | m / f |
| el tío / la tía | 삼촌 / 이모(고모) | m / f |
| el primo / la prima | 사촌 (남) / 사촌 (여) | m / f |
| el esposo / la esposa | 남편 / 아내 | m / f |
| el nieto / la nieta | 손자 / 손녀 | m / f |
| el sobrino / la sobrina | 조카 (남) / 조카 (여) | m / f |
| el suegro / la suegra | 시아버지(장인) / 시어머니(장모) | m / f |
| el cuñado / la cuñada | 처남(시숙) / 처형(시누이) | m / f |
| el bebé | 아기 | m |
| los padres | 부모 | m pl |
| los hijos | 자녀 (아들과 딸) | m pl |

**예문:**
- Tengo dos hermanos. — 나는 형제자매가 둘 있다.
- Mi madre es profesora. — 우리 어머니는 교사이시다.
- Mis abuelos viven en Busan. — 우리 조부모님은 부산에 사신다.

> **참고**: 남성 복수형은 양성 모두를 포함할 수 있습니다: *los hermanos* = 형제들, 또는 형제자매 전체.

## 10. 연습 문제

### 연습 1: 성 판별
*el*인가 *la*인가?

<details>
<summary>정답 보기</summary>

```
1. ___ libro     → el libro (남성, -o 어미)
2. ___ mesa      → la mesa (여성, -a 어미)
3. ___ ciudad    → la ciudad (여성, -dad 어미)
4. ___ problema  → el problema (남성, 그리스어 -ma 예외)
5. ___ mano      → la mano (여성, 예외)
6. ___ día       → el día (남성, 예외)
7. ___ canción   → la canción (여성, -ción 어미)
8. ___ agua      → el agua (여성 명사이지만 강세 a- 때문에 el 사용)
9. ___ foto      → la foto (여성, 예외)
10. ___ idioma   → el idioma (남성, 그리스어 -ma)
```

</details>

### 연습 2: 관사 맞추기
올바른 관사를 넣으세요 (el, la, los, las, un, una).

<details>
<summary>정답 보기</summary>

```
1. Tengo ___ hermana y ___ hermano. → una, un
2. ___ libros están en ___ mesa. → Los, la
3. Es ___ buen estudiante. → un
4. ___ manos de ___ niña. → Las, la
5. Voy a ___ escuela. → la
6. Vengo ___ mercado. → del (de + el)
```

### 연습 4: 문장 만들기

## 빠른 참조표

```
┌─────────────────────────────────────────────────────────────────┐
│   관사(ARTICLES)     남성(Masculine)  여성(Feminine)              │
│   정관사 (the)       el / los         la / las                   │
│   부정관사 (a)       un / unos        una / unas                 │
│                                                                 │
│   성 판별 단서 (GENDER CLUES)                                     │
│   남성: -o, -or, -aje, 그리스어 -ma                               │
│   여성: -a, -ción, -sión, -dad, -tad, -ez, -eza                 │
│   예외: el día, el mapa, la mano, la foto                       │
│                                                                 │
│   복수형 (PLURALS)                                                │
│   모음 → +s    자음 → +es    -z → -ces                           │
│                                                                 │
│   축약형 (CONTRACTIONS)                                           │
│   a + el = al          de + el = del                            │
└─────────────────────────────────────────────────────────────────┘
```
