## 1. 주격 대명사

| 인칭 | 단수 | 복수 |
|--------|----------|--------|
| 1인칭 | **yo** (나) | **nosotros/nosotras** (우리) |
| 2인칭 비격식 | **tú** (너) | **vosotros/vosotras** (너희 — 스페인) |
| 2인칭 격식 | **usted** (당신) | **ustedes** (당신들) |
| 3인칭 | **él/ella** (그/그녀) | **ellos/ellas** (그들) |

> **주어 생략(Pro-drop)**: 스페인어는 주어 생략 언어(Pro-drop Language)입니다 — 동사 활용형이
> 이미 주어를 나타내므로 주격 대명사는 보통 **생략**됩니다.
> - *Hablo español.* (나는 스페인어를 말한다.) — "yo" 생략
> - **강조, 대조, 또는 명확화**를 위해 주격 대명사를 사용합니다:
>   - **Yo** hablo español, pero **él** habla francés.

## 2. 직접 목적어 대명사

직접 목적어 대명사(Direct Object Pronoun)는 동작을 직접 받는 **사물이나 사람**을 대체합니다.

| 인칭 | 단수 | 복수 |
|--------|----------|--------|
| 1인칭 | **me** (나를) | **nos** (우리를) |
| 2인칭 비격식 | **te** (너를) | **os** (너희를 — 스페인) |
| 2인칭 격식 | **lo/la** (당신을) | **los/las** (당신들을) |
| 3인칭 남성 | **lo** (그를/그것을) | **los** (그들을) |
| 3인칭 여성 | **la** (그녀를/그것을) | **las** (그들을) |

**예시:**
- ¿Ves **el libro**? → ¿**Lo** ves? (그것이 보이니?)
- Compré **las flores**. → **Las** compré. (나는 그것들을 샀다.)
- Conozco **a María**. → **La** conozco. (나는 그녀를 안다.)

---

## 3. 간접 목적어 대명사

간접 목적어 대명사(Indirect Object Pronoun)는 동작이 **누구에게/누구를 위해** 행해지는지를 나타냅니다.

| 인칭 | 단수 | 복수 |
|--------|----------|--------|
| 1인칭 | **me** (나에게) | **nos** (우리에게) |
| 2인칭 비격식 | **te** (너에게) | **os** (너희에게) |
| 3인칭 | **le** (그/그녀/당신에게) | **les** (그들/당신들에게) |

**예시:**
- **Me** dio un regalo. (그가 나에게 선물을 주었다.)
- **Le** escribí una carta. (나는 그/그녀에게 편지를 썼다.)
- **Nos** explicó la lección. (그가 우리에게 수업을 설명해 주었다.)

> **A를 이용한 명확화**: *le/les*는 의미가 모호할 수 있으므로, 명확히 하려면 *a + 사람*을 추가합니다:
> - **Le** di el libro **a María**. (나는 María에게 그 책을 주었다.)

---

## 4. 대명사 위치 규칙

### 활용된 동사 앞에 위치

대명사는 활용된 동사(Conjugated Verb) **앞**에 놓입니다:
- **Lo** veo. (나는 그를 본다.)
- **Me** llama. (그가 나에게 전화한다.)
- No **te** conozco. (나는 너를 모른다.)

### 부정사, 현재분사, 긍정 명령문에 붙이기

대명사는 부정사(Infinitive), 현재분사(Gerund), 긍정 명령문(Affirmative Command)의 **끝**에 붙일 수 있습니다:

| 구조 | 예시 |
|-----------|---------|
| 부정사 | Quiero **verlo**. = **Lo** quiero ver. (나는 그것을 보고 싶다.) |
| 현재분사 | Estoy **haciéndolo**. = **Lo** estoy haciendo. (나는 그것을 하고 있다.) |
| 긍정 명령 | ¡**Dímelo**! (나에게 그것을 말해!) |

### 부정 명령문: 동사 앞에 위치

- ¡No **me** lo digas! (나에게 말하지 마!)
- ¡No **lo** hagas! (그것을 하지 마!)

## 5. 이중 목적어 대명사

간접 목적어 대명사와 직접 목적어 대명사가 함께 나타날 때:

```
+----------------------------------------------------------+
|         DOUBLE PRONOUN ORDER                              |
+----------------------------------------------------------+
|                                                          |
|  Indirect Object  BEFORE  Direct Object                  |
|  (me, te, le, nos, os, les)  (lo, la, los, las)         |
|                                                          |
|  LE/LES + LO/LA/LOS/LAS  →  SE + LO/LA/LOS/LAS        |
|                                                          |
|  Example:                                                |
|  ¿Me das el libro?  →  ¿Me lo das?                      |
|  Le di las llaves.  →  Se las di. (NOT *le las)         |
|                                                          |
+----------------------------------------------------------+
```

**LE → SE 규칙**: **le** 또는 **les**가 **lo/la/los/las** 앞에 오게 될 경우,
*le/les*를 **se**로 바꿉니다:

- Le doy el libro. → **Se** lo doy. (나는 그것을 그/그녀에게 준다.)
- Les compré las flores. → **Se** las compré. (나는 그것들을 그들에게 사 주었다.)

## 6. 재귀 대명사

재귀 대명사(Reflexive Pronoun)는 주어가 **자기 자신에게** 동작을 수행함을 나타냅니다.

| 인칭 | 대명사 | 예시 |
|--------|---------|---------|
| yo | **me** | Me levanto. (나는 일어난다.) |
| tú | **te** | Te duchas. (너는 샤워한다.) |
| él/ella/Ud. | **se** | Se peina. (그/그녀는 머리를 빗는다.) |
| nosotros | **nos** | Nos vestimos. (우리는 옷을 입는다.) |
| vosotros | **os** | Os sentáis. (너희는 앉는다.) |
| ellos/Uds. | **se** | Se acuestan. (그들은 잠자리에 든다.) |

### 주요 재귀 동사

| 스페인어 | 한국어 | 스페인어 | 한국어 |
|---------|---------|---------|---------|
| levantarse | 일어나다 | peinarse | 머리를 빗다 |
| acostarse | 잠자리에 들다 | vestirse | 옷을 입다 |
| ducharse | 샤워하다 | sentarse | 앉다 |
| lavarse | 씻다 | irse | 떠나다 |
| cepillarse | 양치하다/빗다 | llamarse | ~라고 불리다 |
| despertarse | 잠에서 깨다 | sentirse | 느끼다 |

## 7. 소유 형용사와 소유 대명사

### 소유 형용사 (명사 앞에 위치)

| 인칭 | 단수 명사 | 복수 명사 |
|--------|--------------|-------------|
| yo | **mi** | **mis** |
| tú | **tu** | **tus** |
| él/ella/Ud. | **su** | **sus** |
| nosotros | **nuestro/a** | **nuestros/as** |
| vosotros | **vuestro/a** | **vuestros/as** |
| ellos/Uds. | **su** | **sus** |

> *nuestro*와 *vuestro*만 명사의 성(Gender)에 따라 변합니다.

### 소유 대명사 (명사를 대체)

| 인칭 | 남성 단수 | 여성 단수 | 남성 복수 | 여성 복수 |
|--------|-----------|----------|-----------|----------|
| yo | **mío** | **mía** | **míos** | **mías** |
| tú | **tuyo** | **tuya** | **tuyos** | **tuyas** |
| él/ella | **suyo** | **suya** | **suyos** | **suyas** |
| nosotros | **nuestro** | **nuestra** | **nuestros** | **nuestras** |

- ¿Es tu libro? → Sí, es **mío**. (네, 그것은 내 것이다.)
- Mi casa es grande. **La tuya** es pequeña. (내 집은 크다. 네 것은 작다.)

## 8. 지시 대명사

| 거리 | 남성형 | 여성형 | 중성형 |
|----------|-----------|----------|--------|
| 가까움 (이것) | **este / estos** | **esta / estas** | **esto** |
| 중간 (그것) | **ese / esos** | **esa / esas** | **eso** |
| 먼 거리 (저것) | **aquel / aquellos** | **aquella / aquellas** | **aquello** |

- **Este** libro es interesante. (이 책은 재미있다.)
- **Esa** casa es bonita. (그 집은 예쁘다.)
- **Aquella** montaña es alta. (저 산은 높다.)

> **중성형** (esto, eso, aquello)은 추상적인 개념이나 정체가 불분명한 것을 가리킵니다:
> - ¿Qué es **esto**? (이것은 뭐야?)
> - **Eso** no es verdad. (그것은 사실이 아니다.)

## 9. 필수 전치사

### 단순 전치사

| 전치사 | 의미 | 예시 |
|-------------|---------|---------|
| **a** | ~에, ~로 | Voy **a** la escuela. (나는 학교에 간다.) |
| **de** | ~의, ~에서 | Soy **de** México. (나는 멕시코 출신이다.) |
| **en** | ~안에, ~위에, ~에서 | Estoy **en** casa. (나는 집에 있다.) |
| **con** | ~와 함께 | Voy **con** mi amigo. (나는 친구와 함께 간다.) |
| **sin** | ~없이 | Café **sin** azúcar. (설탕 없는 커피.) |
| **por** | ~에 의해, ~을 위해, ~을 통해 | Paso **por** el parque. (나는 공원을 지나간다.) |
| **para** | ~을 위해, ~하기 위해 | Es **para** ti. (이것은 너를 위한 것이다.) |
| **entre** | ~사이에 | **Entre** tú y yo. (너와 나 사이에.) |
| **sobre** | ~위에, ~에 대해 | El libro está **sobre** la mesa. (책이 테이블 위에 있다.) |
| **hasta** | ~까지 | Trabajo **hasta** las 5. (나는 5시까지 일한다.) |
| **desde** | ~부터, ~이래로 | Vivo aquí **desde** 2020. (2020년부터 여기 산다.) |
| **hacia** | ~을 향해 | Camino **hacia** la playa. (해변을 향해 걷는다.) |
| **según** | ~에 따르면 | **Según** él... (그에 따르면...) |

### 축약형

스페인어에는 **두 가지** 축약형(Contraction)만 있습니다:
- **a + el = al**: Voy **al** cine. (나는 영화관에 간다.)
- **de + el = del**: Vengo **del** trabajo. (나는 직장에서 온다.)

### 복합 전치사

| 전치사 | 의미 | 예시 |
|-------------|---------|---------|
| delante de | ~앞에 | delante **de** la casa |
| detrás de | ~뒤에 | detrás **de** ti |
| encima de | ~위에 | encima **de** la mesa |
| debajo de | ~아래에 | debajo **de** la cama |
| al lado de | ~옆에 | al lado **de** la escuela |
| cerca de | ~근처에 | cerca **de** aquí |
| lejos de | ~에서 먼 | lejos **de** la ciudad |

### 인칭 전치사 A (Personal A)

직접 목적어가 **특정 사람**인 경우, 그 앞에 **a**를 붙입니다:
- Conozco **a** María. (나는 María를 안다.)
- Veo **a** mi profesor. (나는 선생님을 본다.)
- 하지만: Veo **∅** la película. (나는 영화를 본다. — 사물에는 *a*를 쓰지 않음)

## 10. POR vs. PARA

이것은 스페인어에서 가장 까다로운 구별 중 하나입니다. 둘 다 한국어로 "~을 위해"로
번역될 수 있지만, 용법이 다릅니다.

### PARA — 목적지, 목적, 기한

| 용법 | 예시 |
|-----|---------|
| **목적지** | Salgo **para** Madrid. (나는 마드리드로 떠난다.) |
| **목적** (~하기 위해) | Estudio **para** aprender. (나는 배우기 위해 공부한다.) |
| **수신자** | Es **para** ti. (이것은 너를 위한 것이다.) |
| **기한** | Es **para** mañana. (이것은 내일까지이다.) |
| **비교** | **Para** ser joven, sabe mucho. (젊은 것치고 많이 안다.) |
| **의견** | **Para** mí, es fácil. (나에게는 쉽다.) |
| **고용** | Trabajo **para** Google. (나는 구글에서 일한다.) |

### POR — 원인, 교환, 기간, 통과

| 용법 | 예시 |
|-----|---------|
| **원인/이유** | Lo hice **por** amor. (나는 사랑 때문에 그것을 했다.) |
| **교환** | Pagué 10€ **por** el libro. (나는 그 책에 10유로를 냈다.) |
| **기간** | Estudié **por** dos horas. (나는 두 시간 동안 공부했다.) |
| **통과** | Caminamos **por** el parque. (우리는 공원을 걸었다.) |
| **수단** | Hablamos **por** teléfono. (우리는 전화로 이야기했다.) |
| **대리** | Lo hice **por** ti. (나는 너를 대신해서 그것을 했다.) |
| **~당** | Tres veces **por** semana. (일주일에 세 번.) |

```
+----------------------------------------------------------+
|              POR vs PARA SUMMARY                          |
+----------------------------------------------------------+
|                                                          |
|  PARA (→ forward-looking)     POR (← backward-looking)  |
|  ─────────────────────        ─────────────────────      |
|  Destination (going TO)       Movement THROUGH           |
|  Purpose (IN ORDER TO)        Cause/Reason (BECAUSE OF)  |
|  Recipient (FOR someone)      On behalf of (FOR someone) |
|  Deadline (BY when)           Duration (FOR how long)    |
|  Opinion (FOR me...)          Exchange (IN EXCHANGE for)  |
|  Employer (work FOR)          Means (BY phone/email)     |
|                                                          |
+----------------------------------------------------------+
```

## 11. 연습 문제

### 연습 1: 대명사로 바꾸기

밑줄 친 단어를 올바른 대명사로 바꾸세요.

1. Veo **a María** todos los días. → ________ veo todos los días.
2. Compré **el libro** ayer. → ________ compré ayer.
3. Dio **el regalo** **a su madre**. → ________ ________ dio.
4. Estoy leyendo **la novela**. → Estoy ________.

<details>
<summary>정답</summary>

1. **La** veo todos los días. (직접 목적어: a María → la)
2. **Lo** compré ayer. (직접 목적어: el libro → lo)
3. **Se** **lo** dio. (간접: a su madre → le → se; 직접: el regalo → lo)
4. Estoy **leyéndola**. (또는: **La** estoy leyendo.)

</details>

### 연습 2: POR 또는 PARA

**por** 또는 **para**를 넣으세요.

1. Este regalo es ________ ti.
2. Caminamos ________ la playa.
3. Estudio español ________ viajar a España.
4. Pagué 50 euros ________ los zapatos.
5. Trabajo ________ una empresa internacional.
6. Hablamos ________ teléfono.
7. La tarea es ________ mañana.
8. Lo hice ________ amor.

<details>
<summary>정답</summary>

1. **para** (수신자)
2. **por** (통과)
3. **para** (목적 — ~하기 위해)
4. **por** (교환)
5. **para** (고용)
6. **por** (수단)
7. **para** (기한)
8. **por** (원인/이유)

</details>

### 연습 3: 올바른 대명사 선택

1. ¿Conoces a Juan? — Sí, ________ conozco.
2. ________ llamo Carlos. (내 이름은 Carlos이다.)
3. ¿Me das tu número? — Sí, ________ ________ doy.
4. Ella ________ levanta a las 7. (그녀는 7시에 일어난다.)
5. ¿Dónde están ________ llaves? (나의)

<details>
<summary>정답</summary>

1. **lo** (직접 목적어 — Juan = 남성)
2. **Me** (재귀 — llamarse)
3. **te lo** (간접: a ti → te; 직접: mi número → lo)
4. **se** (재귀 — levantarse)
5. **mis** (소유 형용사 — 복수)

</details>
