## 1. 동사 분류 개요

```
+----------------------------------------------------------+
|           SPANISH VERB CLASSIFICATION                     |
+----------------------------------------------------------+
|                                                          |
|  By ending:   -AR (most common)                          |
|               -ER                                        |
|               -IR                                        |
|                                                          |
|  By regularity:  Regular                                 |
|                   Stem-changing (boot verbs)              |
|                   Irregular (unique patterns)             |
|                   Reflexive (-se ending)                  |
|                                                          |
|  By mood:  Indicative (facts)                            |
|            Subjunctive (subjectivity)                     |
|            Imperative (commands)                          |
|                                                          |
+----------------------------------------------------------+
```

## 2. 어간 변화 동사: E→IE

어간 모음 **e**가 nosotros와 vosotros를 **제외한** 모든 인칭에서 **ie**로 변한다.

### PENSAR (생각하다) — 직설법 현재

| 인칭 | 활용형 |
|------|--------|
| yo | p**ie**nso |
| tú | p**ie**nsas |
| él/ella | p**ie**nsa |
| nosotros | p**e**nsamos |
| vosotros | p**e**nsáis |
| ellos | p**ie**nsan |

### 주요 E→IE 동사

| -AR | -ER | -IR |
|-----|-----|-----|
| cerrar (닫다) | entender (이해하다) | preferir (선호하다) |
| empezar (시작하다) | perder (잃다) | sentir (느끼다) |
| pensar (생각하다) | querer (원하다) | mentir (거짓말하다) |
| despertar (깨우다) | tener (가지다)* | divertirse (즐기다) |
| sentar (앉히다) | | venir (오다)* |

> *tener와 venir은 추가적인 불규칙 변화가 있다 (tengo, vengo).

## 3. 어간 변화 동사: O→UE

### PODER (~할 수 있다) — 직설법 현재

| 인칭 | 활용형 |
|------|--------|
| yo | p**ue**do |
| tú | p**ue**des |
| él/ella | p**ue**de |
| nosotros | p**o**demos |
| vosotros | p**o**déis |
| ellos | p**ue**den |

### 주요 O→UE 동사

| -AR | -ER | -IR |
|-----|-----|-----|
| contar (세다/이야기하다) | poder (~할 수 있다) | dormir (자다) |
| encontrar (찾다) | volver (돌아오다) | morir (죽다) |
| recordar (기억하다) | mover (움직이다) | |
| costar (비용이 들다) | llover (비가 오다) | |
| almorzar (점심을 먹다) | | |

> 특별: **jugar** (놀다)는 u→ue로 변한다: juego, juegas, juega...

## 4. 어간 변화 동사: E→I

-IR 동사에서만 나타난다. 현재 시제와 점과거(Preterite) 3인칭에서 변화가 일어난다.

### PEDIR (요청하다) — 직설법 현재

| 인칭 | 활용형 |
|------|--------|
| yo | p**i**do |
| tú | p**i**des |
| él/ella | p**i**de |
| nosotros | p**e**dimos |
| vosotros | p**e**dís |
| ellos | p**i**den |

### 주요 E→I 동사

| 동사 | 의미 |
|------|------|
| pedir | 요청하다 |
| servir | 봉사하다, 제공하다 |
| repetir | 반복하다 |
| seguir | 따르다 |
| vestirse | 옷을 입다 |
| reír | 웃다 |
| sonreír | 미소 짓다 |
| elegir | 선택하다 |

---

## 5. 부츠 패턴

어간 변화는 활용표에서 "부츠" 모양을 따른다 — nosotros와 vosotros를 **제외한** 모든 인칭에서 변화가 일어난다:

```
+------------------+
|  yo    ✓  |  nosotros ✗  |
|  tú    ✓  |  vosotros ✗  |
|  él    ✓  |  ellos    ✓  |
+------------------+
     \_________/
      The "boot"
```

> **점과거(Preterite)**에서는 E→I 및 O→UE -IR 동사가 3인칭에서만 변화한다:
> - dormir → durmió, durmieron
> - pedir → pidió, pidieron

## 6. 재귀 동사 복습

### 재귀 대명사 위치

| 활용된 동사와 함께 | 부정사/동명사와 함께 |
|-------------------|-------------------|
| **Me** levanto. | Voy a levant**arme**. |
| **Te** duchas. | Estoy duchándo**me**. |
| **Se** viste. | Quiere vestir**se**. |

### 범주별 주요 재귀 동사

| 일상 루틴 | 감정 | 기타 |
|----------|------|------|
| levantarse (일어나다) | alegrarse (기뻐하다) | irse (떠나다) |
| acostarse (잠자리에 들다) | enojarse (화나다) | llamarse (~라고 불리다) |
| ducharse (샤워하다) | preocuparse (걱정하다) | sentarse (앉다) |
| vestirse (옷을 입다) | sentirse (느끼다) | quedarse (머무르다) |
| lavarse (씻다) | aburrirse (지루해하다) | mudarse (이사하다) |
| peinarse (머리를 빗다) | divertirse (즐기다) | darse cuenta (깨닫다) |

## 7. 진행 시제

### 형성 규칙

```
ESTAR (conjugated) + gerund (-ando / -iendo)
```

| 동사 유형 | 동명사(Gerund) | 예시 |
|----------|---------------|------|
| -AR | -**ando** | hablando (말하는 중) |
| -ER | -**iendo** | comiendo (먹는 중) |
| -IR | -**iendo** | viviendo (사는 중) |

### 불규칙 동명사(Gerund)

| 부정사 | 동명사 | 이유 |
|--------|--------|------|
| leer | **leyendo** | 모음 + -iendo → -yendo |
| oír | **oyendo** | |
| ir | **yendo** | |
| dormir | **durmiendo** | o→u 어간 변화 |
| pedir | **pidiendo** | e→i 어간 변화 |
| decir | **diciendo** | e→i |
| seguir | **siguiendo** | e→i |

### 시제 조합

| 시제 | 예문 | 번역 |
|------|------|------|
| 현재 진행형(Present progressive) | Estoy hablando. | 나는 말하고 있다. |
| 과거 진행형(Past progressive) | Estaba hablando. | 나는 말하고 있었다. |
| 미래 진행형(Future progressive) | Estaré hablando. | 나는 말하고 있을 것이다. |

## 8. 종합 동사 변화표

**HABLAR**의 주요 시제별 전체 활용:

| 시제 | yo | tú | él/ella | nosotros | ellos |
|------|----|----|---------|----------|-------|
| **현재(Present)** | hablo | hablas | habla | hablamos | hablan |
| **점과거(Preterite)** | hablé | hablaste | habló | hablamos | hablaron |
| **불완료과거(Imperfect)** | hablaba | hablabas | hablaba | hablábamos | hablaban |
| **미래(Future)** | hablaré | hablarás | hablará | hablaremos | hablarán |
| **조건법(Conditional)** | hablaría | hablarías | hablaría | hablaríamos | hablarían |
| **현재 접속법(Pres. Subj.)** | hable | hables | hable | hablemos | hablen |
| **불완료 접속법(Imp. Subj.)** | hablara | hablaras | hablara | habláramos | hablaran |

| 복합 시제(Compound Tense) | yo 형태 |
|--------------------------|---------|
| 현재완료(Present Perfect) | he hablado |
| 대과거(Pluperfect) | había hablado |
| 미래완료(Future Perfect) | habré hablado |
| 조건완료(Conditional Perfect) | habría hablado |
| 현재완료 접속법(Present Perfect Subj.) | haya hablado |
| 대과거 접속법(Pluperfect Subj.) | hubiera hablado |

## 9. 연습 문제

### 연습 1: 어간 변화 동사

현재 시제로 활용하시오.

1. Yo ________ (querer) ir al cine.
2. Ellos ________ (dormir) ocho horas.
3. Tú ________ (pedir) la cuenta.
4. Nosotros ________ (poder) ayudarte.
5. Ella ________ (volver) a las cinco.
6. Yo ________ (seguir) estudiando.

<details>
<summary>정답</summary>

1. **quiero** (e→ie) 2. **duermen** (o→ue) 3. **pides** (e→i)
4. **podemos** (변화 없음 — nosotros) 5. **vuelve** (o→ue) 6. **sigo** (e→i + 불규칙 yo)

</details>

### 연습 2: 진행 시제

현재 진행형을 만드시오.

1. (yo / estudiar) → ________
2. (ella / dormir) → ________
3. (nosotros / leer) → ________
4. (ellos / pedir) → ________

<details>
<summary>정답</summary>

1. Estoy **estudiando**. 2. Está **durmiendo**. 3. Estamos **leyendo**. 4. Están **pidiendo**.

</details>

### 연습 3: 시제 식별

각 동사 형태가 어떤 시제인지 답하시오.

1. habíamos comido → ________
2. habrán llegado → ________
3. comiera → ________
4. haya dicho → ________
5. estaría hablando → ________

<details>
<summary>정답</summary>

1. 대과거(Pluperfect) (직설법)
2. 미래완료(Future Perfect)
3. 불완료 접속법(Imperfect Subjunctive)
4. 현재완료 접속법(Present Perfect Subjunctive)
5. 조건 진행형(Conditional Progressive)

</details>
