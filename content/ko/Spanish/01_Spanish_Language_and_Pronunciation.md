## 1. 스페인어 소개

### 1.1 세계 속의 스페인어

```
┌─────────────────────────────────────────────────────────────────┐
│                Spanish: A Global Language                        │
│                                                                 │
│   Native speakers: ~500 million (4th worldwide)                 │
│   Total speakers:  ~600 million (including L2)                  │
│   Official in:     21 countries                                 │
│   UN official language: Yes (1 of 6)                            │
│                                                                 │
│   Language family: Indo-European → Romance → Ibero-Romance      │
│   Closest relatives: Portuguese, Catalan, Italian               │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 스페인어 사용 국가

```
Americas (18 countries):
  Mexico, Guatemala, Honduras, El Salvador, Nicaragua,
  Costa Rica, Panama, Cuba, Dominican Republic,
  Puerto Rico (US territory), Colombia, Venezuela,
  Ecuador, Peru, Bolivia, Paraguay, Chile, Argentina, Uruguay

Europe: Spain
Africa: Equatorial Guinea
```

### 1.3 스페인어 발음이 학습자에게 친화적인 이유

스페인어는 가장 음성학적으로 일관된 언어 중 하나입니다:
- **보이는 대로 읽는다**: 거의 완벽한 철자-발음 대응 관계
- **5개의 순수 모음**: 영어처럼 이중모음화되는 모음이 없음
- **예측 가능한 강세**: 두 가지 간단한 규칙으로 대부분의 단어를 커버
- **묵음 없음**: `h`를 제외하면 모든 글자를 발음

## 3. 모음 체계

### 3.1 다섯 개의 순수 모음

스페인어는 주요 세계 언어 중 가장 단순한 모음 체계를 가지고 있습니다.

```
┌─────────────────────────────────────────────────────────────────┐
│                    Spanish Vowel Triangle                        │
│                                                                 │
│           Front         Central         Back                    │
│   High     i /i/                        u /u/                   │
│   Mid      e /e/                        o /o/                   │
│   Low                   a /a/                                   │
│                                                                 │
│   All vowels are:                                               │
│   • Pure (no glide/diphthong)                                   │
│   • Short (consistent length)                                   │
│   • Clear (never reduced to schwa)                              │
└─────────────────────────────────────────────────────────────────┘
```

| 모음 | IPA | 영어 유사 발음 | 한국어 유사 발음 | 예시 |
|-------|-----|-------------------|-------------------|---------|
| a | /a/ | "father"의 "ah" | 아 | c**a**sa (집) |
| e | /e/ | "bet"의 "e" (활음 없이) | 에 | m**e**sa (탁자) |
| i | /i/ | "see"의 "ee" (더 짧게) | 이 | s**i** (네) |
| o | /o/ | "go"의 "o" (활음 없이) | 오 | n**o** (아니오) |
| u | /u/ | "food"의 "oo" (더 짧게) | 우 | **u**no (하나) |

> **영어와의 핵심 차이**: 스페인어 모음은 비강세 음절에서도 절대 "uh" (슈와 /ə/)로 약화되지 않습니다. 강세와 관계없이 모든 모음이 완전하고 명확한 소리를 유지합니다.

### 3.2 이중모음(Diphthong)

이중모음(Diphthong)은 강모음(a, e, o)과 약모음(i, u)이 결합하거나, 두 약모음이 하나의 음절 안에서 결합할 때 발생합니다.

```
Strong + Weak (falling diphthong):
  ai / ay → aire (air), hay (there is)
  ei / ey → seis (six), rey (king)
  oi / oy → oigo (I hear), hoy (today)
  au      → auto (car), causa (cause)
  eu      → Europa (Europe), deuda (debt)
  ou      → (rare) bou

Weak + Strong (rising diphthong):
  ia      → familia (family), dia → día (day, with accent = no diphthong)
  ie      → tiempo (time), fiesta (party)
  io      → radio (radio), acción (action)
  ua      → agua (water), cuatro (four)
  ue      → bueno (good), puerta (door)
  uo      → antiguo (old), cuota (quota)

Weak + Weak:
  iu      → ciudad (city), viuda (widow)
  ui / uy → cuidar (to care), muy (very)
```

### 3.3 모음 충돌(Hiatus) — 두 모음이 별도 음절에 위치하는 경우

두 강모음이 만나거나, 약모음에 악센트 부호가 있으면 모음 충돌(Hiatus)이 발생하여 별도의 음절이 됩니다:

```
Two strong vowels → hiatus:
  le-er (to read)     po-e-ta (poet)     ca-os (chaos)

Accented weak vowel → breaks diphthong:
  dí-a (day)          pa-ís (country)    ba-úl (trunk)
  rí-o (river)        ma-íz (corn)       re-ír (to laugh)
```

## 4. 자음 체계

### 4.1 B와 V

스페인어에서 **b**와 **v**는 동일하게 발음됩니다. /v/ 소리는 존재하지 않습니다.

```
After a pause or nasal (n, m): [b] (hard, like English "b")
  - vamos [ˈba.mos]        buenos [ˈbwe.nos]
  - un beso [um ˈbe.so]    enviar [em.biˈar]

Between vowels or after other consonants: [β] (soft, lips don't fully close)
  - saber [sa.ˈβeɾ]        la vida [la ˈβi.ða]
  - hablar [a.ˈβlaɾ]       uva [ˈu.βa]
```

### 4.2 C, Z, S — *Distinción*(구별 발음) vs *Seseo*(세세오)

```
┌─────────────────────────────────────────────────────────────────┐
│   Spain (most regions): Distinción                              │
│   c + e/i = /θ/ (like English "th" in "think")                  │
│   z       = /θ/                                                 │
│   s       = /s/                                                 │
│   cena = [ˈθe.na]    zapato = [θa.ˈpa.to]    sol = [sol]       │
│                                                                 │
│   Latin America & southern Spain: Seseo                         │
│   c + e/i = /s/                                                 │
│   z       = /s/                                                 │
│   s       = /s/                                                 │
│   cena = [ˈse.na]    zapato = [sa.ˈpa.to]    sol = [sol]       │
│                                                                 │
│   ※ Both pronunciations are correct and standard.              │
│   ※ Seseo is used by far more speakers worldwide.              │
└─────────────────────────────────────────────────────────────────┘
```

기타 C 발음:
- c + a/o/u = /k/: casa, comer, cubo
- cc = /ks/: acción, lección

### 4.3 D — 연음 D

```
After a pause or nasal/lateral: [d] (hard, like English "d" but dental)
  - donde [ˈdon.de]        aldea [al.ˈde.a]

Between vowels: [ð] (soft, like "th" in English "this")
  - vida [ˈbi.ða]          nada [ˈna.ða]
  - todo [ˈto.ðo]          cansado [kan.ˈsa.ðo]

Word-final -d: very soft [ð] or silent in casual speech
  - ciudad [θju.ˈðað] or [θju.ˈða]
  - Madrid [ma.ˈðɾið] or [ma.ˈðɾi]
```

### 4.4 G와 J — 후두음(Guttural Sound)

```
G + a/o/u = /g/ or /ɣ/ (like English "g" in "go")
  - gato [ˈga.to]          agua [ˈa.ɣwa]

G + e/i = /x/ (harsh guttural, like "ch" in Scottish "loch")
  - gente [ˈxen.te]        girar [xi.ˈɾaɾ]

J (always) = /x/
  - jardín [xaɾ.ˈðin]      ojo [ˈo.xo]      joven [ˈxo.βen]

GU + e/i = /g/ (u is silent, just keeps g hard)
  - guerra [ˈge.ra]        guitarra [gi.ˈta.ra]

GÜ + e/i = /gw/ (u is pronounced, shown by diaeresis ¨)
  - pingüino [piŋ.ˈgwi.no]    vergüenza [beɾ.ˈgwen.θa]
```

### 4.5 H — 항상 묵음

```
The letter H is always silent in Spanish:
  - hola [ˈo.la]           hacer [a.ˈθeɾ]
  - hora [ˈo.ɾa]           huevo [ˈwe.βo]
  - hotel [o.ˈtel]         hay [ai]

※ Exception: "ch" digraph is pronounced /tʃ/
  - chocolate [tʃo.ko.ˈla.te]
```

### 4.6 LL과 Y — *Yeísmo*(예이스모)

```
Historically:
  ll = /ʎ/ (palatal lateral, like Italian "gli")
  y  = /ʝ/ (palatal fricative)

Modern Spanish (yeísmo — used by vast majority):
  ll = y = /ʝ/ (both sound the same)
  - llamar = yamar [ʝa.ˈmaɾ]
  - calle = caye [ˈka.ʝe]
  - yo [ˈʝo]

Argentina/Uruguay (sheísmo/zheísmo):
  ll = y = /ʃ/ or /ʒ/ (like "sh" or "zh")
  - yo [ˈʃo]    calle [ˈka.ʃe]
```

### 4.7 Ñ — 스페인어 고유 문자

```
ñ = /ɲ/ (like "ny" in "canyon" or "ni" in "onion")
  - año [ˈa.ɲo] (year)
  - mañana [ma.ˈɲa.na] (morning/tomorrow)
  - niño [ˈni.ɲo] (child)
  - España [es.ˈpa.ɲa] (Spain)
  - señor [se.ˈɲoɾ] (sir)

Similar to Korean ㄴ + ㅣ combination (냐, 녀, 뇨, 뉴)
```

### 4.8 R과 RR — 탄설음(Tap) vs 전동음(Trill)

```
┌─────────────────────────────────────────────────────────────────┐
│   Single R /ɾ/ (tap — tongue briefly touches ridge once):      │
│   Between vowels: pero [ˈpe.ɾo] (but)                          │
│   After consonant: tres [tɾes] (three)                          │
│   End of word: hablar [a.ˈβlaɾ] (to speak)                     │
│                                                                 │
│   RR /r/ (trill — tongue vibrates multiple times):              │
│   Written rr: perro [ˈpe.ro] (dog)                              │
│                carro [ˈka.ro] (car)                              │
│                correr [ko.ˈrer] (to run)                        │
│                                                                 │
│   Single R pronounced as trill /r/:                             │
│   Word-initial: rosa [ˈro.sa]    rey [ˈrej]                    │
│   After n: Enrique [en.ˈri.ke]                                  │
│   After l: alrededor [al.re.ðe.ˈðoɾ]                           │
│   After s: Israel [is.ra.ˈel]                                   │
│                                                                 │
│   ※ Minimal pairs:                                             │
│   pero (but)  vs  perro (dog)                                   │
│   caro (expensive) vs carro (car)                               │
│   coral (coral) vs corral (pen/yard)                            │
│└─────────────────────────────────────────────────────────────────┘
```

## 5. 음절 구조

### 5.1 기본 패턴

```
Most common syllable patterns:
  CV    → ca-sa (house), me-sa (table)         Most common
  CVC   → par-te (part), can-tar (to sing)
  V     → a-gua (water), u-no (one)
  VC    → es-tar (to be), al-to (tall)
  CCV   → tre-s (three), pla-ya (beach)
  CCVC  → tren (train), plan (plan)
```

### 5.2 음절 분리 규칙

```
1. Single consonant between vowels → goes with second vowel:
   ca-sa, me-sa, li-bro → li-bro? No: li-bro

2. Two consonants between vowels → usually split:
   par-te, can-to, al-to
   Exception: consonant + l/r stays together (inseparable):
   li-bro, ha-blar, o-tro, a-brir

3. Three consonants → first goes with preceding vowel,
   last two go with next (if they form valid cluster):
   com-prar, en-trar, ins-truir → ins-tru-ir

4. Two vowels:
   Diphthong (strong+weak or weak+weak) = one syllable:
   bue-no, ciu-dad, vie-jo
   Hiatus (strong+strong or accented weak) = separate:
   le-er, pa-ís, dí-a
```

## 6. 강세와 악센트 규칙

### 6.1 두 가지 기본 강세 규칙

```
┌─────────────────────────────────────────────────────────────────┐
│   Rule 1: Words ending in vowel, -n, or -s                     │
│   → Stress falls on the SECOND-TO-LAST syllable                │
│                                                                 │
│   ca-SA          (house)         ha-BLAN       (they speak)    │
│   me-SA          (table)         li-BROS       (books)         │
│   es-TU-dian-TE  (student)       co-MEN        (they eat)      │
│                                                                 │
│   Rule 2: Words ending in any other consonant                   │
│   → Stress falls on the LAST syllable                           │
│                                                                 │
│   ha-BLAR   (to speak)      ciu-DAD   (city)                   │
│   co-MER    (to eat)        es-pa-ÑOL (Spanish)                │
│   vi-VIR    (to live)       pa-PEL    (paper)                  │
└─────────────────────────────────────────────────────────────────┘
```

### 6.2 악센트 부호 (Tilde)

단어의 강세가 기본 규칙을 따르지 않을 때, 악센트 부호(´)가 필수입니다:

```
Violating Rule 1 (stress on last syllable despite ending in vowel/n/s):
  café [ka.ˈfe]       (coffee)
  canción [kan.ˈθjon] (song)
  japonés [xa.po.ˈnes] (Japanese)
  mamá [ma.ˈma]       (mom)

Violating Rule 2 (stress on second-to-last despite ending in consonant):
  árbol [ˈaɾ.βol]     (tree)
  fácil [ˈfa.θil]     (easy)
  lápiz [ˈla.piθ]     (pencil)

Stress on third-to-last syllable (esdrújulas — always need accent):
  música [ˈmu.si.ka]    (music)
  teléfono [te.ˈle.fo.no] (telephone)
  médico [ˈme.ði.ko]    (doctor)
  rápido [ˈra.pi.ðo]    (fast)
```

### 6.3 구별 악센트(Distinguishing Accent)

철자는 같지만 악센트 유무에 따라 의미가 달라지는 단어들이 있습니다:

| 악센트 없음 | 악센트 있음 |
|---------------|-------------|
| el (그, 정관사) | él (그, 대명사) |
| tu (너의) | tú (너) |
| mi (나의) | mí (나를/나에게) |
| si (만약) | sí (네) |
| se (자기 자신) | sé (나는 안다 / ~해라!) |
| te (너를) | té (차) |
| de (~의, ~에서) | dé (주다, 접속법) |
| mas (그러나) | más (더) |
| solo (혼자의) | sólo (오직, 선택적) |
| que (~하는 것) | qué (무엇?) |
| como (~처럼) | cómo (어떻게?) |
| donde (~하는 곳) | dónde (어디?) |
| cuando (~할 때) | cuándo (언제?) |

## 7. 억양 패턴

### 7.1 서술문(Declarative Sentence)

```
Pitch pattern: rises slightly, then falls at end

  Tengo un libro.  (I have a book.)
  ─────╱──────╲
       ↑       ↓

  Estudio español en la universidad.
  ───╱────────────────────────╲
```

### 7.2 예/아니오 의문문(Yes/No Question)

```
Pitch pattern: rises at the end

  ¿Tienes un libro?  (Do you have a book?)
  ──────────────╱
               ↑

  ¿Hablas español?  (Do you speak Spanish?)
  ────────────╱
```

### 7.3 정보 의문문 (의문사 의문문)

```
Pitch pattern: starts high on question word, falls at end

  ¿Dónde vives?  (Where do you live?)
  ╱─────╲
  ↑      ↓

  ¿Cómo te llamas?  (What's your name?)
  ╱──────────╲
```

### 7.4 감탄문(Exclamation)

```
Wide pitch range:

  ¡Qué bonito!  (How beautiful!)
  ╱╲
  ↑ ↓  (sharp rise and fall)
```

> **참고**: 스페인어에서는 의문문과 감탄문의 시작 부분에 역전된 구두점(¿ ¡)을 사용합니다. 이것은 스페인어 표기법의 고유한 특징입니다.

## 8. 흔한 발음 실수

### 8.1 영어 화자의 경우

| 실수 | 올바른 발음 | 설명 |
|---------|---------|-------------|
| p, t, k에 기식음 넣기 | 기식음 없이 발음 | 스페인어 폐쇄음에는 공기 분출이 없음 |
| 비강세 모음을 "uh"로 약화 | 모든 모음을 명확하게 유지 | *hospital* ≠ "hah-SPIT-uhl" |
| 영어 "v"를 스페인어 v에 적용 | /b/ 또는 /β/ 사용 | b와 v는 동일한 소리 |
| z에 영어 "z" 유성음 적용 | /θ/ 또는 /s/ 사용 | 유성 /z/는 절대 없음 |
| h를 발음하기 | 항상 묵음 | *hola* = [ˈo.la], "ho-la"가 아님 |
| 영어식 r 사용 | 탄설음 /ɾ/ 또는 전동음 /r/ | 혀끝 탄설음 연습 필요 |
| 모음을 이중모음화 | 모음을 순수하게 유지 | *no* ≠ "noh-oo" |

### 8.2 한국어 화자의 경우

| 실수 | 올바른 발음 | 설명 |
|---------|---------|-------------|
| r/l 혼동 | r = 탄설음/전동음, l = 설측음 | 스페인어 r은 한국어 ㄹ이 아님 |
| 자음 뒤에 모음 추가 | 어말 자음을 깔끔하게 유지 | *sol* = [sol], "so-le"가 아님 |
| b/p 유성/무성 구별 안 함 | b는 유성, p는 무성 | 빠 ≠ 바 구별이 도움됨 |
| ñ를 n+y로 따로 발음 | 하나의 구개음화된 소리 /ɲ/ | 한국어 녀와 유사하지만 하나의 소리 |
| 평탄한 억양 | 상승/하강 패턴 사용 | 스페인어는 음고 변화가 더 큼 |
| /x/ (j, ge, gi) 발음 어려움 | 목구멍 뒷부분의 마찰음 | 목을 가다듬는 소리와 유사, 한국어 ㅎ과 비슷하지만 더 강함 |

## 9. 연습 문제

### 연습 1: 모음 연습
각 단어를 소리 내어 읽으며, 모음을 순수하고 명확하게 유지하세요.

```
a: casa, papa, banana, mañana, agua
e: mesa, leche, este, verde, mente
i: sí, mil, isla, igual, vivir
o: ojo, otro, como, solo, todo
u: uno, mucho, punto, nunca, fruta
```

### 연습 2: 최소 대립쌍(Minimal Pair)
다음 쌍의 차이를 구별하며 연습하세요:

```
1. pero (but)     vs  perro (dog)      — tap r vs trill rr
2. caro (expensive) vs carro (car)     — tap r vs trill rr
3. casa (house)   vs  caza (hunt)      — s vs θ (Spain)
4. vino (wine)    vs  fino (fine)      — b/β vs f
5. peso (weight)  vs  beso (kiss)      — p vs b
```

### 연습 3: 강세 위치 찾기
강세가 있는 음절을 표시하세요. 어떤 단어에 악센트 부호가 필요한가요?

<details>
<summary>정답 보기</summary>

```
1. hablar     → ha-BLAR (Rule 2: consonant ending, last syllable ✓) No accent
2. mesa       → ME-sa (Rule 1: vowel ending, second-to-last ✓) No accent
3. telefono   → te-LE-fo-no → teléfono (stress on third-to-last, needs accent)
4. ciudad     → ciu-DAD (Rule 2 ✓) No accent
5. cafe       → ca-FE → café (Rule 1 violated: vowel ending, last syllable stressed)
6. cancion    → can-CION → canción (Rule 1 violated: -n ending, last syllable stressed)
7. arbol      → AR-bol → árbol (Rule 2 violated: consonant ending, second-to-last stressed)
8. musica     → MU-si-ca → música (esdrújula: always needs accent)
```

</details>
