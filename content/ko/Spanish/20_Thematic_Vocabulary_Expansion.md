## 6. 축소사 (Diminutivos)

축소사 접미사는 작은 크기, 애정, 또는 어조를 부드럽게 하는 효과를 나타냅니다.

### -ito / -ita (가장 일반적)

| 원형 | 축소사 | 뉘앙스 |
|----------|-----------|--------|
| casa | cas**ita** | 작은/아늑한 집 |
| perro | perr**ito** | 강아지, 귀여운 개 |
| momento | moment**ito** | 잠깐만 (어조 완화) |
| ahora | ahor**ita** | 바로 지금 (멕시코) |
| café | cafec**ito** | 커피 한 잔 (친근감) |
| abuelo/a | abuel**ito/a** | 할아버지/할머니 (애정) |
| poco | poqu**ito** | 아주 조금 |
| regalo | regal**ito** | 작은 선물 (겸손) |
| cerca | cerqu**ita** | 아주 가까이 (강조) |

### 형성 규칙

```
+----------------------------------------------------------+
|          DIMINUTIVE FORMATION (-ito/-ita)                 |
+----------------------------------------------------------+
|                                                          |
|  1. Ending in vowel: drop vowel + -ito/-ita              |
|     casa → casita, perro → perrito                       |
|                                                          |
|  2. Ending in consonant: + -ito/-ita or -cito/-cita      |
|     amor → amorcito, pan → panecito                      |
|                                                          |
|  3. Ending in -co/-ca: c → qu + -ito/-ita                |
|     poco → poquito, chico → chiquito                     |
|                                                          |
|  4. Ending in -go/-ga: g → gu + -ito/-ita                |
|     amigo → amiguito, largo → larguito                   |
|                                                          |
|  5. One-syllable words: -ecito/-ecita                    |
|     flor → florecita, pan → panecito, pez → pececito    |
+----------------------------------------------------------+
```

### 기타 축소사 접미사

| 접미사 | 지역/어조 | 예시 |
|--------|------------|---------|
| **-illo/-illa** | 덜 애정적, 때로 어휘화됨 | ventanilla (창구), cigarrillo (담배), bolsillo (주머니) |
| **-ico/-ica** | 카리브해, 코스타리카, 아라곤 | momentico, ratico, poquitico |
| **-ín/-ina** | 아스투리아스 (스페인) | pequeñín, chiquitín |

## 7. 확대사 (Aumentativos)

확대사 접미사는 큰 크기, 강도, 또는 때로는 부정적인 뉘앙스를 나타냅니다.

### -ote / -ota (큰, 때로 거친)

| 원형 | 확대사 | 의미 |
|----------|-------------|---------|
| libro | libr**ote** | 두꺼운/큰 책 |
| palabra | palabr**ota** | 욕설 |
| grande | grand**ote** | 아주 큰 |
| amigo | amig**ote** | 친한 친구 |
| perro | perr**ote** | 큰 개 |

### -azo / -aza (큰, 인상적인, 또는 타격)

| 원형 | 확대사 | 의미 |
|----------|-------------|---------|
| coche | coch**azo** | 멋진 차 |
| gol | gol**azo** | 환상적인 골 |
| fiesta | fiest**azo** | 대단한 파티 |

**-azo: 타격/때림을 나타내는 경우:**

| 명사 | -azo 형태 | 의미 |
|------|-----------|---------|
| puño | puñ**etazo** | 주먹질 |
| codo | cod**azo** | 팔꿈치 치기 |
| cabeza | cabez**azo** | 박치기 |
| rodilla | rodill**azo** | 무릎 치기 |
| zapato | zapat**azo** | 신발로 때리기 |

### -ón / -ona (큰, 과장된)

| 원형 | 확대사 | 의미 |
|----------|-------------|---------|
| silla | sill**ón** | 안락의자 |
| casa | cas**ona** | 대저택 |
| cuchara | cuchar**ón** | 국자 |
| llorar → llor**ón/a** | 울보 |
| comer → comil**ón/a** | 대식가 |
| dormir → dormil**ón/a** | 잠꾸러기 |
| mandar → mand**ón/a** | 우두머리 행세하는 사람 |

### 뉘앙스 요약

| 용법 | 예시 | 효과 |
|-----|---------|--------|
| 애정 | mi hijito | 따뜻하고 사랑스러운 |
| 작은 크기 | una casita | 물리적으로 작은 |
| 어조 완화 | ¿Me das un favorcito? | 부담을 줄임 |
| 강조 | Está cerquita | 아주 가까운 |
| 친근감 | ¿Tomamos un cafecito? | 다정하고 따뜻한 |
| 큰/인상적인 | ¡Qué golazo! | 감탄 |
| 부정적/경멸 | Es un mandón | 우두머리 행세하는 (비하) |

## 8. 단어 형성 패턴 (Patrones de Formación de Palabras)

### 주요 접두사 (Prefijos)

| 접두사 | 의미 | 예시 |
|--------|---------|---------|
| des- | 반대, 되돌림 | deshacer (되돌리다), descargar (다운로드하다) |
| re- | 다시, 매우 | rehacer (다시 하다), releer (다시 읽다) |
| in-/im- | ~이 아닌 | imposible, incómodo, increíble |
| pre- | ~전에 | predecir (예측하다), prevenir (예방하다) |
| sobre- | 과도한, 위의 | sobrevivir (생존하다), sobrepasar (초과하다) |
| sub- | 아래의 | subrayar (밑줄 치다), subterráneo |
| auto- | 자기 | autocuidado (자기 돌봄), autoservicio |

### 명사를 만드는 주요 접미사 (Sufijos)

| 접미사 | 생성 | 예시 |
|--------|---------|---------|
| -ción/-sión | 행위/결과 | contaminación, depresión, producción |
| -miento | 행위/결과 | entrenamiento, agotamiento, calentamiento |
| -dad/-tad | 성질 | productividad, libertad, biodiversidad |
| -ero/-era | 직업/용기 | camarero (웨이터), zapatero (구두장이), papelera (휴지통) |
| -ista | 사람/이념 | periodista (기자), ecologista |

### 형용사를 만드는 주요 접미사 (Sufijos)

| 접미사 | 생성 | 예시 |
|--------|---------|---------|
| -able/-ible | ~할 수 있는 | sostenible, biodegradable, reciclable |
| -oso/-osa | ~이 가득한 | peligroso, famoso, nervioso |
| -al | ~에 관련된 | global, social, digital, mental |
| -ivo/-iva | ~하는 경향의 | productivo, creativo, sostenible |

## 9. 연습문제

### 연습 1: 기술 어휘

스페인어 용어와 한국어 뜻을 연결하세요.

1. la contraseña        a) 클라우드
2. la nube              b) 다운로드하다
3. descargar            c) 비밀번호
4. la pantalla          d) 화면
5. actualizar           e) 업데이트하다

<details>
<summary>정답</summary>

1-c, 2-a, 3-b, 4-d, 5-e

</details>

### 연습 2: 축소사 만들기

축소사(-ito/-ita) 형태를 만들어 보세요.

1. casa → ________
2. poco → ________
3. amigo → ________
4. flor → ________
5. momento → ________

<details>
<summary>정답</summary>

1. **casita** 2. **poquito** (c→qu) 3. **amiguito** (g→gu) 4. **florecita** (단음절 → -ecita) 5. **momentito**

</details>

### 연습 3: 확대사 의미 파악

각 확대사의 의미는 무엇인가요?

1. golazo
2. palabrota
3. sillón
4. cochazo
5. dormilón

<details>
<summary>정답</summary>

1. **환상적인/대단한 골** (감탄)
2. **욕설 / 거친 말** (거친 단어)
3. **안락의자** (큰 의자 — 어휘화됨)
4. **멋진/큰 차** (인상적인)
5. **잠꾸러기** (잠을 많이 자는 사람)

</details>

### 연습 4: 환경 미니 에세이

이 레슨의 어휘를 사용하여 빈칸을 채우세요.

"El ________ ________ (기후 변화) es un problema global. Todos debemos ________ (줄이다) nuestra ________ de ________ (탄소 발자국). Podemos usar ________ ________ (재생 에너지), ________ (재활용하다) y usar ________ ________ (재사용 가방)."

<details>
<summary>정답</summary>

"El **cambio climático** es un problema global. Todos debemos **reducir** nuestra **huella** de **carbono**. Podemos usar **energía renovable**, **reciclar** y usar **bolsas reutilizables**."

</details>

### 연습 5: 단어 형성

접두사 또는 접미사를 사용하여 새로운 단어를 만드세요.

1. hacer → ________ (되돌리다) — 접두사: des-
2. posible → ________ (불가능한) — 접두사: im-
3. producir → ________ (생산성) — 접미사: -dad
4. reciclar → ________ (재활용 가능한) — 접미사: -able
5. zapato → ________ (구두장이) — 접미사: -ero

<details>
<summary>정답</summary>

1. **deshacer** 2. **imposible** 3. **productividad** 4. **reciclable** 5. **zapatero**

</details>
