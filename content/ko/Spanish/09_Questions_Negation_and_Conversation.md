## 1. 의문사 (Interrogatives)

스페인어의 모든 의문사는 의문문에서 사용될 때 **악센트 부호(Accent Mark)**를 붙입니다.

| 의문사 | 의미 | 예문 |
|--------|------|------|
| **¿Qué?** | 무엇? | ¿Qué quieres? (무엇을 원하니?) |
| **¿Quién? / ¿Quiénes?** | 누구? | ¿Quién es? (누구예요?) |
| **¿Cuándo?** | 언제? | ¿Cuándo vienes? (언제 와?) |
| **¿Dónde?** | 어디? | ¿Dónde vives? (어디에 살아?) |
| **¿Adónde?** | 어디로? | ¿Adónde vas? (어디로 가?) |
| **¿Por qué?** | 왜? | ¿Por qué estudias? (왜 공부해?) |
| **¿Cómo?** | 어떻게? | ¿Cómo estás? (어떻게 지내?) |
| **¿Cuánto/a?** | 얼마나? | ¿Cuánto cuesta? (얼마예요?) |
| **¿Cuántos/as?** | 몇 개? | ¿Cuántos años tienes? (몇 살이야?) |
| **¿Cuál? / ¿Cuáles?** | 어떤 것? | ¿Cuál prefieres? (어떤 것을 선호해?) |

### ¿Qué? vs. ¿Cuál?

```
+----------------------------------------------------------+
|  ¿QUÉ? = 정의나 정체를 묻는 경우                            |
|  ¿Qué es esto?  (이것이 무엇인가? — 정체를 묻는 질문)       |
|  ¿Qué hora es?  (지금 몇 시야?)                           |
|                                                          |
|  ¿CUÁL? = 선택을 묻는 경우                                  |
|  ¿Cuál es tu nombre?  (이름이 뭐야? — 선택형 질문)          |
|  ¿Cuál prefieres?     (어떤 것을 선호해?)                   |
|                                                          |
|  명사(NOUN) 앞에서는 → 항상 ¿Qué? 사용                      |
|  ¿Qué libro quieres?  (*¿Cuál libro?는 불가)              |
+----------------------------------------------------------+
```

## 2. 의문문 구성 방법

### 방법 1: 도치(Inversion) (주어-동사 → 동사-주어)

- Juan habla español. → **¿Habla** Juan español?
- Ella trabaja aquí. → **¿Trabaja** ella aquí?

### 방법 2: 억양(Intonation)만 사용 (올라가는 어조)

- **¿**Tú hablas español**?** (어순은 그대로, 올라가는 어조)

### 방법 3: 부가 의문문(Tag Questions)

- Hablas español, **¿verdad?** (스페인어 할 줄 알지, 그렇지?)
- Hablas español, **¿no?** (스페인어 할 줄 알지, 아닌가?)
- Vienes mañana, **¿cierto?** (내일 오지, 맞지?)

> 스페인어는 문장 앞뒤에 **거꾸로 된 물음표** ¿...?를 사용합니다.
> 느낌표도 마찬가지입니다: ¡...!

## 3. 부정문 기초

### 기본 부정: 동사 앞에 NO

- **No** hablo francés. (나는 프랑스어를 못해요.)
- **No** tengo dinero. (나는 돈이 없어요.)
- **No** quiero ir. (나는 가고 싶지 않아요.)

### 부정으로 대답하기

- ¿Hablas japonés? — **No**, **no** hablo japonés.
  (첫 번째 "no"는 질문에 대한 대답이고, 두 번째 "no"는 동사를 부정합니다.)

### 부정어

| 부정어 | 긍정 대응어 |
|--------|------------|
| **nada** (아무것도 ~않다) | algo (무언가) |
| **nadie** (아무도 ~않다) | alguien (누군가) |
| **nunca / jamás** (절대 ~않다) | siempre (항상) |
| **tampoco** (~도 아니다) | también (~도) |
| **ninguno/a** (하나도 ~않다) | alguno/a (어떤) |
| **ni... ni** (~도 ~도 아닌) | o... o (~이거나 ~이거나) |

## 4. 이중 부정

영어와 달리, 스페인어에서는 이중 부정(또는 삼중 부정)이 **필수적**입니다.
이것은 문법적으로 올바르며 표준적인 용법입니다:

| 스페인어 | 한국어 |
|----------|--------|
| **No** tengo **nada**. | 나는 아무것도 없다. (직역: 나는 아무것도 가지고 있지 않다.) |
| **No** viene **nadie**. | 아무도 오지 않는다. |
| **No** voy **nunca**. | 나는 절대 가지 않는다. |
| **No** quiero **ni** café **ni** té. | 나는 커피도 차도 원하지 않는다. |
| **No** tengo **ningún** problema. | 나는 아무 문제도 없다. |

> 부정어가 동사 **앞에** 올 경우, *no*는 생략합니다:
> - **Nadie** viene. (아무도 오지 않는다.)
> - **Nunca** como carne. (나는 절대 고기를 먹지 않는다.)
> - **Nada** es perfecto. (완벽한 것은 아무것도 없다.)

## 5. 의견 표현하기

| 표현 | 예문 |
|------|------|
| **Creo que...** | Creo que es buena idea. (좋은 생각인 것 같아요.) |
| **Pienso que...** | Pienso que tienes razón. (당신 말이 맞는 것 같아요.) |
| **Me parece que...** | Me parece que va a llover. (비가 올 것 같아요.) |
| **En mi opinión...** | En mi opinión, es muy caro. (제 생각에는 너무 비싸요.) |
| **Para mí...** | Para mí, la familia es importante. (저에게 가족은 중요합니다.) |

### 동의와 반대

| 동의 | 반대 |
|------|------|
| Estoy de acuerdo. (동의해요.) | No estoy de acuerdo. (동의하지 않아요.) |
| Tienes razón. (당신 말이 맞아요.) | No tienes razón. (당신 말이 틀려요.) |
| Exacto. / Exactamente. | No es así. (그렇지 않아요.) |
| Por supuesto. (물론이죠.) | Para nada. (전혀 아니에요.) |
| Claro que sí. (물론이죠.) | Claro que no. (물론 아니죠.) |

## 6. 전화 표현

| 스페인어 | 한국어 |
|----------|--------|
| ¿Aló? / ¿Diga? / ¿Bueno? | 여보세요? (전화 받을 때) |
| ¿Puedo hablar con...? | ...와 통화할 수 있을까요? |
| ¿De parte de quién? | 누구시죠? (전화한 사람 확인) |
| Un momento, por favor. | 잠시만요. |
| No está disponible. | 지금 통화할 수 없습니다. |
| ¿Quiere dejar un mensaje? | 메시지를 남기시겠어요? |
| Le devuelvo la llamada. | 다시 전화 드리겠습니다. |
| Se cortó la llamada. | 전화가 끊겼어요. |
| No tengo señal. | 신호가 없어요. |

> 전화 인사는 나라마다 다릅니다: **¿Diga?** (스페인), **¿Bueno?** (멕시코),
> **¿Aló?** (남미), **¿Hola?** (아르헨티나).

## 7. 레스토랑 시나리오

### 유용한 표현

| 스페인어 | 한국어 |
|----------|--------|
| Una mesa para dos, por favor. | 2인 테이블 부탁합니다. |
| ¿Tienen menú del día? | 오늘의 메뉴가 있나요? |
| ¿Qué me recomienda? | 뭘 추천하시나요? |
| Para mí, la sopa. | 저는 수프로 할게요. |
| ¿Me trae la cuenta, por favor? | 계산서 좀 가져다주시겠어요? |
| ¿Está incluida la propina? | 팁이 포함되어 있나요? |
| ¿Puedo pagar con tarjeta? | 카드로 결제할 수 있나요? |

### 대화 예시

> **Camarero**: ¡Buenas tardes! ¿Mesa para cuántos?
> **Cliente**: Para dos, por favor.
> **Camarero**: Aquí tienen el menú. ¿Qué desean tomar?
> **Cliente**: Yo quiero una limonada y ella quiere un agua con gas.
> **Camarero**: ¿Ya saben qué van a pedir?
> **Cliente**: Sí, yo quiero la paella y ella quiere el pescado a la plancha.
> **Camarero**: Muy bien. ¿Algo más?
> **Cliente**: No, gracias. Eso es todo.
> ...
> **Cliente**: La cuenta, por favor.
> **Camarero**: Aquí tiene. Son treinta y cinco euros.
> **Cliente**: ¿Puedo pagar con tarjeta?
> **Camarero**: Por supuesto.

## 8. 쇼핑 시나리오

| 스페인어 | 한국어 |
|----------|--------|
| ¿Cuánto cuesta esto? | 이거 얼마예요? |
| ¿Tiene una talla más grande/pequeña? | 더 큰/작은 사이즈 있나요? |
| ¿Lo puedo probar? | 입어 봐도 되나요? |
| ¿Dónde está el probador? | 탈의실이 어디예요? |
| Me lo llevo. | 이걸로 살게요. |
| ¿Tienen descuento? | 할인이 있나요? |
| Está en oferta / en rebajas. | 세일 중이에요. |
| ¿Aceptan tarjeta de crédito? | 신용카드 되나요? |
| Solo estoy mirando. | 그냥 구경하는 거예요. |

## 9. 길 묻기

| 스페인어 | 한국어 |
|----------|--------|
| Perdone, ¿dónde está...? | 실례합니다, ...이/가 어디에 있나요? |
| ¿Cómo llego a...? | ...에 어떻게 가나요? |
| ¿Está lejos/cerca de aquí? | 여기서 멀어요/가까워요? |
| Siga todo recto / derecho. | 쭉 직진하세요. |
| Gire a la derecha/izquierda. | 오른쪽/왼쪽으로 도세요. |
| Está en la esquina. | 모퉁이에 있어요. |
| Está al lado de... | ... 옆에 있어요. |
| Está enfrente de... | ... 맞은편에 있어요. |
| Cruce la calle. | 길을 건너세요. |
| A dos cuadras / manzanas. | 두 블록 거리에 있어요. |

## 10. 연습 문제

### 연습 1: 의문문 만들기

각 대답에 맞는 의문사를 사용하여 질문을 만드세요.

1. 대답: Me llamo Ana. → ¿________?
2. 대답: Vivo en Madrid. → ¿________?
3. 대답: Tengo 25 años. → ¿________?
4. 대답: Estoy bien, gracias. → ¿________?
5. 대답: Porque me gusta. → ¿________?
6. 대답: Cuesta 10 euros. → ¿________?

<details>
<summary>정답</summary>

1. ¿**Cómo** te llamas?
2. ¿**Dónde** vives?
3. ¿**Cuántos** años tienes?
4. ¿**Cómo** estás?
5. ¿**Por qué** lo haces? / ¿**Por qué** te gusta?
6. ¿**Cuánto** cuesta?

</details>

### 연습 2: 문장을 부정문으로 바꾸기

1. Siempre como carne. → ________
2. Alguien llamó. → ________
3. Quiero algo. → ________
4. También me gusta. → ________
5. Tengo algún problema. → ________

<details>
<summary>정답</summary>

1. **Nunca** como carne. / **No** como carne **nunca**.
2. **Nadie** llamó. / **No** llamó **nadie**.
3. **No** quiero **nada**.
4. **Tampoco** me gusta. / A mí **tampoco**.
5. **No** tengo **ningún** problema.

</details>

### 연습 3: 대화 완성하기

레스토랑 대화의 빈칸을 채우세요.

> Camarero: ¡Buenas noches! ¿________ para cuántos?
> Cliente: ________ tres, por favor.
> Camarero: ¿Qué ________ tomar?
> Cliente: Para mí, un vino tinto. Y ________, ¿qué quieres?
> Amigo: Una cerveza, por favor.
> Camarero: ¿Ya saben ________ van a pedir?
> Cliente: Sí, ¿qué nos ________?
> Camarero: Les ________ la paella, es nuestra especialidad.
> Cliente: Perfecto, dos paellas entonces.
> ...
> Cliente: La ________, por favor. ¿________ pagar con tarjeta?

<details>
<summary>정답</summary>

Mesa, Para, desean, tú, qué, recomienda, recomiendo, cuenta, Puedo

</details>
