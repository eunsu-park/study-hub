## 1. 읽기 전략 개요

```
+----------------------------------------------------------+
|            READING STRATEGIES FOR SPANISH                 |
+----------------------------------------------------------+
|                                                          |
|  BEFORE reading:                                         |
|  1. Look at the title, headings, images                  |
|  2. Predict what the text is about                       |
|  3. Activate background knowledge                        |
|                                                          |
|  DURING reading:                                         |
|  4. Don't stop at every unknown word                     |
|  5. Use context clues to guess meaning                   |
|  6. Look for cognates (words similar to English)         |
|  7. Pay attention to connecting words                    |
|                                                          |
|  AFTER reading:                                          |
|  8. Summarize the main idea                              |
|  9. Review unknown vocabulary                            |
|  10. Re-read difficult sections                          |
+----------------------------------------------------------+
```

> **핵심 원칙**: 텍스트를 이해하기 위해 모든 단어를 알 필요는 없습니다.
> 먼저 주요 아이디어에 집중한 후, 세부 사항을 채워 나가세요.

## 2. 훑어읽기와 찾아읽기

### 훑어읽기 (Lectura rápida)

목적: 텍스트의 **대략적인 내용**을 빠르게 파악합니다.

훑어읽기 방법:
- 제목과 소제목을 읽는다
- 각 문단의 첫 번째 문장을 읽는다
- 굵은 글씨나 기울임꼴 단어를 확인한다
- 결론을 읽는다

### 찾아읽기 (Lectura selectiva)

목적: **특정 정보**를 빠르게 찾습니다.

찾아읽기 방법:
- 읽기 시작 전에 무엇을 찾을지 파악한다
- 텍스트 위로 시선을 빠르게 이동한다
- 핵심 단어, 숫자, 이름, 날짜를 찾는다
- 목표 정보를 찾았을 때만 멈춘다

### 정독 (Lectura intensiva)

목적: 텍스트를 **상세하게** 이해합니다.

사용 시기:
- 학습 자료, 지시사항, 계약서
- 훑어읽기 후 중요한 부분을 파악했을 때
- 시험 준비 시

## 3. 문맥 단서

모르는 단어를 만났을 때, 다음과 같은 단서를 찾아보세요:

### 유형 1: 정의 단서

텍스트가 단어를 직접 정의합니다:
> "La **biodiversidad**, es decir, la variedad de especies animales y vegetales..."
> (생물다양성(biodiversidad), 즉 동물과 식물 종의 다양성...)

단서 표현: **es decir** (즉), **o sea** (다시 말해), **esto es** (이것은)

### 유형 2: 동의어 단서

근처에 동의어가 나타납니다:
> "Estaba **exhausto**, completamente **agotado** después del viaje."
> (그는 지쳐 있었다(exhausto), 여행 후 완전히 기진맥진한(agotado) 상태였다.)

### 유형 3: 반의어 단서

반대 단어가 대조를 제공합니다:
> "No era **tímido** sino muy **extrovertido**."
> (그는 소심한(tímido) 것이 아니라 매우 외향적(extrovertido)이었다.)

단서 표현: **sino** (~이 아니라), **en cambio** (반면에), **a diferencia de** (~와 달리)

### 유형 4: 예시 단서

예시가 의미를 명확히 합니다:
> "Los **cítricos**, como las naranjas, los limones y las mandarinas..."
> (감귤류(cítricos), 예를 들어 오렌지, 레몬, 귤...)

단서 표현: **como** (~처럼), **por ejemplo** (예를 들어), **tales como** (~와 같은)

### 유형 5: 논리와 일반 상식

세상에 대한 지식을 활용합니다:
> "Después de tres días sin comer, tenía mucha **hambre**."
> (3일 동안 먹지 않은 후, 그는 매우 ___했다.) → 배고픈(hambre)

## 4. 동족어와 거짓 동족어

### 동족어 (Cognados)

스페인어와 영어에서 비슷하게 생기고 같은 뜻을 가진 단어들:

| 스페인어 | 영어 |
|---------|---------|
| importante | important |
| información | information |
| problema | problem |
| natural | natural |
| diferente | different |
| familia | family |
| músico | musician |
| teléfono | telephone |
| universidad | university |
| hospital | hospital |

### 거짓 동족어 (Falsos amigos)

비슷하게 생겼지만 의미가 다른 단어들:

| 스페인어 | 영어처럼 보이는 뜻 | 실제 의미 |
|---------|--------------|------------------|
| **actual** | actual (실제의) | 현재의, 오늘날의 |
| **embarazada** | embarrassed (당황한) | 임신한 |
| **éxito** | exit (출구) | 성공 |
| **librería** | library (도서관) | 서점 |
| **constipado** | constipated (변비에 걸린) | 감기에 걸린 |
| **sensible** | sensible (분별 있는) | 민감한 |
| **realizar** | realize (깨닫다) | 수행하다, 달성하다 |
| **asistir** | assist (돕다) | 참석하다 |
| **carpeta** | carpet (카펫) | 폴더 |
| **fábrica** | fabric (직물) | 공장 |
| **recordar** | record (기록하다) | 기억하다 |
| **soportar** | support (지지하다) | 견디다, 참다 |
| **pretender** | pretend (~인 척하다) | 시도하다, 의도하다 |
| **bizarro** | bizarre (기이한) | 용감한, 관대한 |

> "너무 쉬워 보이는" 단어는 항상 다시 확인하세요 — 거짓 동족어(Falsos amigos)일 수 있습니다!

## 9. 주석 기법

### 능동적 읽기 표시

텍스트를 읽으면서 다음 표시를 사용하여 적극적으로 참여하세요:

| 표시 | 의미 | 사용 시기 |
|------|---------|------------|
| ✓ | 이해했음 | 명확한 구절 |
| ? | 이해하지 못했음 | 혼동되는 단어/문장 |
| ! | 중요함 / 놀라움 | 핵심 정보 |
| ★ | 핵심 어휘 | 학습할 단어 |
| → | 아는 것과 연결됨 | 연결점 만들기 |

### 주석 달기 단계

```
+----------------------------------------------------------+
|          ANNOTATION WORKFLOW                               |
+----------------------------------------------------------+
|                                                          |
|  1. First read: DON'T stop. Get the general idea.        |
|                                                          |
|  2. Second read: Mark unknown words with ?                |
|     Try to guess their meaning from context first.        |
|                                                          |
|  3. Third read: Look up remaining ? words.                |
|     Write the meaning in the margin.                     |
|                                                          |
|  4. Summarize: Write 1-2 sentences about the main idea.  |
|                                                          |
|  5. Review: Add ★ words to your vocabulary notebook.      |
+----------------------------------------------------------+
```

## 10. 연습 문제

### 연습 1: 거짓 동족어 확인

올바른 의미를 선택하세요.

1. "El éxito del proyecto fue increíble." — éxito의 의미는:
   a) 출구  b) 성공  c) 흥분

2. "Asistí a la conferencia ayer." — asistir의 의미는:
   a) 돕다  b) 참석하다  c) 고집하다

3. "Es una persona muy sensible." — sensible의 의미는:
   a) 분별 있는  b) 민감한  c) 단순한

4. "La fábrica produce coches." — fábrica의 의미는:
   a) 직물  b) 공장  c) 조작

5. "Estoy constipado." — constipado의 의미는:
   a) 변비에 걸린  b) 감기에 걸린  c) 일정한

<details>
<summary>정답</summary>

1. **b) 성공** 2. **b) 참석하다** 3. **b) 민감한** 4. **b) 공장** 5. **b) 감기에 걸린**

</details>

### 연습 2: 문맥 단서 연습

문맥 단서를 사용하여 밑줄 친 단어의 의미를 추측하세요.

1. "No podíamos ver nada porque había mucha **niebla**; todo estaba blanco y gris."
2. "El agua estaba tan **turbia** que no podíamos ver el fondo del río."
3. "Después del terremoto, muchos edificios quedaron **derrumbados**, completamente destruidos."

<details>
<summary>정답</summary>

1. **niebla** = 안개 (단서: 아무것도 볼 수 없었고, 모든 것이 하얗고 회색이었다)
2. **turbia** = 탁한/흐린 (단서: 강 바닥을 볼 수 없었다)
3. **derrumbados** = 무너진 (단서: 동의어 제공 — "completamente destruidos")

</details>

### 연습 3: 텍스트 유형 식별

각 발췌문은 어떤 유형의 텍스트에서 나온 것인가요?

1. "Estimado Sr. López: Le escribo en relación con su pedido número 4521..."
2. "¡2x1 en todos los productos! Solo este fin de semana."
3. "Érase una vez, en un pueblo pequeño, vivía una anciana muy sabia..."
4. "Según el informe del Instituto Nacional de Estadística, el desempleo bajó un 2%..."

<details>
<summary>정답</summary>

1. **공식 이메일 / 비즈니스 서신**
2. **광고 / 상업 홍보**
3. **이야기 / 동화** ("Érase una vez" = 옛날 옛적에)
4. **뉴스 기사 / 보고서**

</details>
