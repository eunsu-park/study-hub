## 1. 명령법: Tú 명령형

### 긍정 Tú 명령형

대부분의 동사에서 긍정 tú 명령형은 **직설법 현재 3인칭 단수형**과 같습니다:

| 원형 | 명령형 | 한국어 |
|-----------|---------|---------|
| hablar | ¡**Habla**! | 말해! |
| comer | ¡**Come**! | 먹어! |
| escribir | ¡**Escribe**! | 써! |
| leer | ¡**Lee**! | 읽어! |

### 불규칙 Tú 명령형 (8개 동사)

| 원형 | 명령형 | 암기법 |
|-----------|---------|---------|
| decir | **di** | |
| hacer | **haz** | |
| ir | **ve** | |
| poner | **pon** | |
| salir | **sal** | |
| ser | **sé** | |
| tener | **ten** | |
| venir | **ven** | |

> 암기법: **ven di sal haz ten ve pon sé** — "Ven, di: sal, haz ten, ve, pon sé"

---

## 2. 명령법: Usted/Ustedes 명령형

격식체 명령형은 **현재 접속법(Present Subjunctive)** 형태를 사용합니다:

| 원형 | Usted | Ustedes |
|-----------|-------|---------|
| hablar | ¡**Hable**! | ¡**Hablen**! |
| comer | ¡**Coma**! | ¡**Coman**! |
| escribir | ¡**Escriba**! | ¡**Escriban**! |
| ir | ¡**Vaya**! | ¡**Vayan**! |
| ser | ¡**Sea**! | ¡**Sean**! |

> **패턴**: -AR 동사는 -e/-en 어미를, -ER/-IR 동사는 -a/-an 어미를 사용합니다.
> 이것이 "반대 모음(Opposite Vowel)" 규칙이며, 접속법 형성과 동일합니다.

---

## 3. 명령법: Nosotros와 Vosotros

### Nosotros 명령형 ("~하자...")

**nosotros 접속법** 형태를 사용합니다:
- ¡**Hablemos**! (말하자!)
- ¡**Comamos**! (먹자!)
- ¡**Vamos**! (가자!) — 불규칙 (긍정형에서 *vayamos*가 아님)

### Vosotros 명령형 (스페인 전용)

원형의 **-r**을 **-d**로 바꿉니다:
- hablar → ¡**Hablad**! (말해!)
- comer → ¡**Comed**! (먹어!)
- escribir → ¡**Escribid**! (써!)

---

## 4. 부정 명령형

**모든** 부정 명령형은 접속법을 사용합니다:

| 인칭 | hablar | comer |
|--------|--------|-------|
| tú | ¡No **hables**! | ¡No **comas**! |
| usted | ¡No **hable**! | ¡No **coma**! |
| nosotros | ¡No **hablemos**! | ¡No **comamos**! |
| vosotros | ¡No **habléis**! | ¡No **comáis**! |
| ustedes | ¡No **hablen**! | ¡No **coman**! |

---

## 5. 명령형에서의 대명사 위치

```
+----------------------------------------------------------+
|  AFFIRMATIVE commands: pronoun ATTACHED to the end        |
|  ¡Dime!        (Tell me!)                                |
|  ¡Levántate!   (Get up!)                                 |
|  ¡Dámelo!      (Give it to me!)                          |
|                                                          |
|  NEGATIVE commands: pronoun BEFORE the verb               |
|  ¡No me digas! (Don't tell me!)                          |
|  ¡No te levantes! (Don't get up!)                        |
|  ¡No me lo des!   (Don't give it to me!)                 |
+----------------------------------------------------------+
```

> 대명사를 붙일 때, 원래 강세를 유지하기 위해 **악센트 부호(Accent Mark)**를 추가합니다:
> di + me = dime (악센트 불필요), levanta + te = levántate (악센트 추가).

---

## 6. 접속법이란?

접속법(Subjunctive)은 **서법(Mood)**입니다 (시제가 아님). 직설법(Indicative)이 사실을 진술하는 반면,
접속법은 주관성을 표현합니다:

```
+----------------------------------------------------------+
|  INDICATIVE (reality)         SUBJUNCTIVE (subjectivity)  |
|  ────────────────────         ──────────────────────────  |
|  States facts                 Expresses:                  |
|  Reports events               - Wishes and desires        |
|  Describes certainty           - Emotions and feelings     |
|                                - Doubt and uncertainty     |
|                                - Influence and commands    |
|                                - Hypothetical situations   |
|                                                          |
|  Sé que habla español.       Espero que hable español.   |
|  (I know he speaks...)       (I hope he speaks...)       |
+----------------------------------------------------------+
```

> **핵심**: 접속법은 거의 항상 **종속절(Subordinate Clause)**에서 나타나며
> (주로 "que" 뒤), 주절이 주관성을 표현할 때 사용됩니다.

## 7. 현재 접속법 형성

### "반대 모음(Opposite Vowel)" 규칙

1. 직설법 현재의 **yo** 형태에서 시작
2. **-o**를 제거
3. "반대" 모음 어미를 추가

| | -AR (hablar) | -ER (comer) | -IR (vivir) |
|---|---|---|---|
| yo | habl**e** | com**a** | viv**a** |
| tú | habl**es** | com**as** | viv**as** |
| él/ella | habl**e** | com**a** | viv**a** |
| nosotros | habl**emos** | com**amos** | viv**amos** |
| vosotros | habl**éis** | com**áis** | viv**áis** |
| ellos | habl**en** | com**an** | viv**an** |

> -AR 동사는 -e 어미를, -ER/-IR 동사는 -a 어미를 사용합니다.

### 불규칙 접속법 (Yo 형태 기반)

형성이 **yo 직설법**에서 시작되므로, yo에서 불규칙인 동사는
그 불규칙성이 접속법으로 이어집니다:

| 원형 | Yo 직설법 | 접속법 (yo) |
|-----------|--------------|------------------|
| tener | teng**o** | teng**a** |
| venir | veng**o** | veng**a** |
| decir | dig**o** | dig**a** |
| hacer | hag**o** | hag**a** |
| poner | pong**o** | pong**a** |
| salir | salg**o** | salg**a** |
| conocer | conozc**o** | conozc**a** |

### 완전 불규칙 접속법

| 원형 | 접속법 |
|-----------|-------------|
| ser | sea, seas, sea, seamos, seáis, sean |
| estar | esté, estés, esté, estemos, estéis, estén |
| ir | vaya, vayas, vaya, vayamos, vayáis, vayan |
| haber | haya, hayas, haya, hayamos, hayáis, hayan |
| saber | sepa, sepas, sepa, sepamos, sepáis, sepan |
| dar | dé, des, dé, demos, deis, den |

## 8. WEIRDO: 접속법을 사용하는 경우

**WEIRDO**는 주요 트리거(Trigger)에 대한 암기법입니다:

| 문자 | 범주 | 예문 |
|--------|----------|---------|
| **W** | 소망(Wishes) | Quiero que **vengas**. (네가 오길 원해.) |
| **E** | 감정(Emotions) | Me alegra que **estés** bien. (네가 잘 지내서 기뻐.) |
| **I** | 비인칭 표현(Impersonal expressions) | Es importante que **estudies**. (네가 공부하는 것이 중요해.) |
| **R** | 권고(Recommendations) | Te recomiendo que **descanses**. (쉬는 걸 추천해.) |
| **D** | 의심/부정(Doubt/Denial) | Dudo que **sea** verdad. (그것이 사실인지 의심스러워.) |
| **O** | Ojalá | Ojalá que **venga**. (그/그녀가 오길 바란다.) |

### 주요 비인칭 표현 (항상 + 접속법)

| 표현 | 예문 |
|------------|---------|
| Es necesario que... | Es necesario que **estudies**. |
| Es posible que... | Es posible que **llueva**. |
| Es importante que... | Es importante que **hables**. |
| Es mejor que... | Es mejor que **vayas**. |
| Es probable que... | Es probable que **venga**. |

## 9. 접속사와 접속법

특정 접속사는 **항상** 접속법을 요구합니다:

| 접속사 | 의미 | 예문 |
|-------------|---------|---------|
| para que | ~하기 위해 | Hablo despacio **para que** me **entiendas**. |
| antes de que | ~하기 전에 | Sal **antes de que** **llueva**. |
| a menos que | ~하지 않는 한 | Iré **a menos que** **llueva**. |
| sin que | ~없이 | Salió **sin que** nadie lo **supiera**. |
| con tal de que | ~하는 조건으로 | Te ayudo **con tal de que** me **escuches**. |

나머지 접속사는 **미래/불확실한** 사건을 가리킬 때만 접속법을 취합니다:
- **cuando** + 접속법 (미래): Cuando **llegues**, llámame. (네가 도착하면, 전화해.)
- **cuando** + 직설법 (습관): Cuando **llego**, siempre como. (내가 도착하면, 항상 먹는다.)

## 10. 연습 문제

### 연습 1: 명령형 만들기

지시된 명령형을 만드세요.

1. (hablar, tú 긍정) → ________
2. (comer, usted 긍정) → ________
3. (ir, tú 긍정) → ________
4. (hacer, tú 부정) → ________
5. (sentarse, tú 긍정) → ________
6. (decir + me, tú 긍정) → ________

<details>
<summary>정답</summary>

1. ¡**Habla**! 2. ¡**Coma**! 3. ¡**Ve**! 4. ¡No **hagas**! 5. ¡**Siéntate**! 6. ¡**Dime**!

</details>

### 연습 2: 직설법 또는 접속법?

올바른 형태를 선택하세요.

1. Sé que ella (habla / hable) español.
2. Quiero que tú (vienes / vengas) a la fiesta.
3. Es posible que (llueve / llueva) mañana.
4. Creo que él (tiene / tenga) razón.
5. No creo que ella (puede / pueda) venir.

<details>
<summary>정답</summary>

1. **habla** (직설법 — "sé que"로 확실성 표현)
2. **vengas** (접속법 — 소망)
3. **llueva** (접속법 — 가능성을 나타내는 비인칭 표현)
4. **tiene** (직설법 — 긍정문에서의 믿음 = 확실성)
5. **pueda** (접속법 — 부정된 믿음 = 의심)

</details>

### 연습 3: 접속법으로 완성하기

1. Espero que tú ________ (estar) bien.
2. Es necesario que nosotros ________ (ir) al médico.
3. Dudo que ellos ________ (saber) la respuesta.
4. Ojalá que ________ (hacer) buen tiempo mañana.

<details>
<summary>정답</summary>

1. **estés** 2. **vayamos** 3. **sepan** 4. **haga**

</details>
