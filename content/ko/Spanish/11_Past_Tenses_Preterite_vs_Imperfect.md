## 1. 점과거: 규칙 활용

점과거(Preterite)는 과거에 **완료된** 행위를 표현합니다.

| | -AR (hablar) | -ER (comer) | -IR (vivir) |
|---|---|---|---|
| yo | habl**é** | com**í** | viv**í** |
| tú | habl**aste** | com**iste** | viv**iste** |
| él/ella/Ud. | habl**ó** | com**ió** | viv**ió** |
| nosotros | habl**amos** | com**imos** | viv**imos** |
| vosotros | habl**asteis** | com**isteis** | viv**isteis** |
| ellos/Uds. | habl**aron** | com**ieron** | viv**ieron** |

> **참고**: -ER 동사와 -IR 동사는 점과거에서 동일한 어미를 공유합니다.
> -AR 동사와 -IR 동사의 **nosotros** 형태는 현재 시제와 동일합니다.
> 문맥으로 구분합니다.

---

## 2. 점과거: 주요 불규칙 동사

### SER와 IR (점과거에서 동일한 형태!)

| 인칭 | ser/ir |
|--------|--------|
| yo | **fui** |
| tú | **fuiste** |
| él/ella | **fue** |
| nosotros | **fuimos** |
| vosotros | **fuisteis** |
| ellos | **fueron** |

> 문맥으로 어떤 동사인지 판단합니다:
> - **Fui** al cine. (나는 영화관에 **갔다**.) — ir
> - **Fue** un buen día. (좋은 하루**였다**.) — ser

### 어간 변화 불규칙 동사 (Stem-Changing Irregulars)

이 동사들은 공통된 **비강세** 어미를 공유합니다: -e, -iste, -o, -imos, -isteis, -ieron.

| 동사 | 어간 | yo | él/ella |
|------|------|----|---------|
| estar | estuv- | estuve | estuvo |
| tener | tuv- | tuve | tuvo |
| poder | pud- | pude | pudo |
| poner | pus- | puse | puso |
| saber | sup- | supe | supo |
| hacer | hic-/hiz- | hice | hizo |
| querer | quis- | quise | quiso |
| venir | vin- | vine | vino |
| decir | dij- | dije | dijo |
| traer | traj- | traje | trajo |

> **j** 뒤에서는 3인칭 복수 어미가 **-eron**이 됩니다 (-ieron이 아님):
> dijeron, trajeron.

### DAR, VER (단음절 불규칙 동사)

| 인칭 | dar | ver |
|--------|-----|-----|
| yo | di | vi |
| tú | diste | viste |
| él/ella | dio | vio |
| nosotros | dimos | vimos |
| ellos | dieron | vieron |

---

## 3. 점과거: 철자 변화

이 변화들은 발음 규칙을 유지하기 위한 것으로, 진정한 불규칙이 아닙니다:

| 변화 | 동사 | 예시 (yo) |
|--------|-------|-------------|
| c → qu (-é 앞) | buscar, tocar, sacar | busqué |
| g → gu (-é 앞) | llegar, pagar, jugar | llegué |
| z → c (-é 앞) | empezar, almorzar | empecé |

---

## 4. 불완료 과거: 규칙 활용

불완료 과거(Imperfect)는 과거의 **진행 중인, 습관적인, 또는 묘사적인** 행위를 표현합니다.

| | -AR (hablar) | -ER (comer) | -IR (vivir) |
|---|---|---|---|
| yo | habl**aba** | com**ía** | viv**ía** |
| tú | habl**abas** | com**ías** | viv**ías** |
| él/ella/Ud. | habl**aba** | com**ía** | viv**ía** |
| nosotros | habl**ábamos** | com**íamos** | viv**íamos** |
| vosotros | habl**abais** | com**íais** | viv**íais** |
| ellos/Uds. | habl**aban** | com**ían** | viv**ían** |

> **-ER 동사와 -IR 동사는 불완료 과거에서도 동일한 어미를 공유합니다** (점과거와 마찬가지).
> **yo**와 **él/ella** 형태가 동일하므로 — 문맥으로 구분합니다.

---

## 5. 불완료 과거: 단 세 개의 불규칙 동사

| 인칭 | ser | ir | ver |
|--------|-----|-----|-----|
| yo | **era** | **iba** | **veía** |
| tú | **eras** | **ibas** | **veías** |
| él/ella | **era** | **iba** | **veía** |
| nosotros | **éramos** | **íbamos** | **veíamos** |
| vosotros | **erais** | **ibais** | **veíais** |
| ellos | **eran** | **iban** | **veían** |

> 스페인어의 다른 모든 동사는 불완료 과거에서 규칙 활용합니다.

---

## 6. 각 시제를 사용하는 경우

```
+-------------------------------+-------------------------------+
|         PRETERITE             |         IMPERFECT             |
+-------------------------------+-------------------------------+
|  COMPLETED action             |  ONGOING / HABITUAL action    |
|  (has a clear end point)      |  (no focus on start or end)   |
|                               |                               |
|  "What happened?"             |  "What was happening?"        |
|  "What did you do?"           |  "What used to happen?"       |
+-------------------------------+-------------------------------+
```

### 점과거(Preterite)를 사용하는 경우:

1. **완료된 행위**: Estudié tres horas. (나는 세 시간 동안 공부했다.)
2. **특정 횟수**: Fui al cine dos veces. (나는 두 번 영화관에 갔다.)
3. **연속적인 행위**: Me levanté, desayuné y salí. (나는 일어나서 아침을 먹고 나갔다.)
4. **갑작스러운 사건**: De repente, sonó el teléfono. (갑자기 전화가 울렸다.)

### 불완료 과거(Imperfect)를 사용하는 경우:

1. **습관적인 행위**: Siempre caminaba al trabajo. (나는 항상 직장까지 걸어갔다.)
2. **장면 묘사/배경 설정**: Llovía y hacía frío. (비가 오고 있었고 추웠다.)
3. **신체적/감정적 상태**: Estaba cansado. (나는 피곤했다.)
4. **나이, 시간, 날씨**: Tenía diez años. (나는 열 살이었다.) / Eran las ocho. (여덟 시였다.)
5. **동시 진행 행위**: Mientras cocinaba, él leía. (내가 요리하는 동안, 그는 책을 읽고 있었다.)

## 7. 시제 신호 단어 (Trigger Words)

| 점과거 신호 단어 | 불완료 과거 신호 단어 |
|---|---|
| ayer (어제) | siempre (항상) |
| anoche (어젯밤) | todos los días (매일) |
| una vez (한 번) | cada semana (매주) |
| el año pasado (작년) | a menudo (자주) |
| de repente (갑자기) | mientras (~하는 동안) |
| finalmente (마침내) | de niño/a (어릴 때) |
| primero... luego... (먼저... 그다음...) | normalmente (보통) |
| en + 특정 날짜 | en aquella época (그 시절에) |

## 8. 시제에 따라 의미가 달라지는 동사

| 동사 | 점과거(Preterite) | 불완료 과거(Imperfect) |
|------|-----------|-----------|
| saber | **supe** = 알게 되었다 | **sabía** = 알고 있었다 |
| conocer | **conocí** = (처음) 만났다 | **conocía** = 알고 지내던 사이였다 |
| querer | **quise** = ~하려고 했다 | **quería** = ~하고 싶었다 |
| no querer | **no quise** = 거부했다 | **no quería** = ~하고 싶지 않았다 |
| poder | **pude** = 해냈다 | **podía** = (일반적으로) ~할 수 있었다 |
| tener | **tuve** = 받았다/얻었다 | **tenía** = 가지고 있었다 |

---

## 9. 서사에서의 활용: 두 시제의 협력

```
+----------------------------------------------------------+
|              STORYTELLING FORMULA                          |
+----------------------------------------------------------+
|  IMPERFECT = The stage (background)                       |
|  • Scene-setting, descriptions, ongoing actions           |
|                                                          |
|  PRETERITE = The action (plot events)                     |
|  • Events that advance the story, interruptions           |
|                                                          |
|  Pattern: IMPERFECT + cuando + PRETERITE                  |
|  "was doing X when Y happened"                           |
+----------------------------------------------------------+
```

### 서사 예시

> **Era** (불완료) una noche fría. **Nevaba** (불완료) suavemente y las calles
> **estaban** (불완료) vacías. Yo **caminaba** (불완료) hacia casa cuando de repente
> **escuché** (점과거) un ruido. Me **di** (점과거) la vuelta, pero no **vi** (점과거)
> a nadie. **Seguí** (점과거) caminando más rápido. Cuando **llegué** (점과거)
> a casa, **descubrí** (점과거) que el ruido **venía** (불완료) de un gato.

패턴: 불완료 과거가 장면을 설정하고 → 점과거가 줄거리를 이끌어 갑니다.

## 10. 연습 문제

### 연습 1: 점과거 활용

점과거(Preterite)로 활용하세요.

1. Yo ________ (estudiar) toda la noche.
2. Ellos ________ (ir) al cine anoche.
3. ¿Qué ________ (hacer) tú ayer?
4. Nosotros no ________ (poder) dormir.
5. Ella me ________ (decir) la verdad.

<details>
<summary>정답</summary>

1. **estudié** 2. **fueron** 3. **hiciste** 4. **pudimos** 5. **dijo**

</details>

### 연습 2: 불완료 과거 활용

1. De niña, yo ________ (jugar) en el parque todos los días.
2. Mi abuelo siempre ________ (contar) historias.
3. ________ (ser) las diez de la noche.
4. Nosotros ________ (vivir) cerca de la playa.
5. Ella ________ (ir) a la biblioteca cada semana.

<details>
<summary>정답</summary>

1. **jugaba** 2. **contaba** 3. **Eran** 4. **vivíamos** 5. **iba**

</details>

### 연습 3: 점과거 또는 불완료 과거?

올바른 시제를 선택하세요.

1. Cuando yo (tener) ________ diez años, mi familia (mudarse) ________ a otra ciudad.
2. Mientras ella (estudiar) ________, su hermano (escuchar) ________ música.
3. De repente, la luz (apagarse) ________ y todos (gritar) ________.
4. Mi abuela siempre (preparar) ________ chocolate caliente.

<details>
<summary>정답</summary>

1. **tenía** (불완료 — 나이) / **se mudó** (점과거 — 완료된 사건)
2. **estudiaba** (불완료 — 진행 중) / **escuchaba** (불완료 — 동시 진행)
3. **se apagó** (점과거 — 갑작스러운 사건) / **gritaron** (점과거 — 반응)
4. **preparaba** (불완료 — 습관적)

</details>

### 연습 4: 의미 변화

시제에 주의하여 스페인어로 번역하세요.

1. 어제 진실을 알게 되었다. (saber, 점과거)
2. 진실을 알고 있었다. (saber, 불완료 과거)
3. 파티에서 María를 만났다. (conocer, 점과거)
4. 어릴 때부터 María를 알고 있었다. (conocer, 불완료 과거)

<details>
<summary>정답</summary>

1. **Supe** la verdad ayer.
2. **Sabía** la verdad.
3. **Conocí** a María en la fiesta.
4. **Conocía** a María desde la infancia.

</details>
