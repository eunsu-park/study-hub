## 2. 기수 31-1,000,000

### 31-99: 십의 자리 + y + 일의 자리 (세 단어)

| 숫자 | 스페인어 |
|------|----------|
| 31 | treinta **y** uno |
| 42 | cuarenta **y** dos |
| 55 | cincuenta **y** cinco |
| 67 | sesenta **y** siete |
| 78 | setenta **y** ocho |
| 83 | ochenta **y** tres |
| 99 | noventa **y** nueve |

### 십의 자리

| 30 | treinta | 70 | setenta |
|----|---------|----|---------|
| 40 | cuarenta | 80 | ochenta |
| 50 | cincuenta | 90 | noventa |
| 60 | sesenta | | |

### 백의 자리

| 숫자 | 남성형 | 여성형 |
|------|--------|--------|
| 100 | cien / ciento | cien / ciento |
| 200 | doscientos | doscientas |
| 300 | trescientos | trescientas |
| 400 | cuatrocientos | cuatrocientas |
| 500 | quinientos | quinientas |
| 600 | seiscientos | seiscientas |
| 700 | setecientos | setecientas |
| 800 | ochocientos | ochocientas |
| 900 | novecientos | novecientas |

> **cien** vs. **ciento**: 명사 앞이나 *mil/millones* 앞에서는 *cien*을 사용합니다.
> 다른 숫자가 뒤따를 때는 *ciento*를 사용합니다: ciento dos (102).

### 큰 숫자

| 숫자 | 스페인어 |
|------|----------|
| 1,000 | mil |
| 2,000 | dos mil |
| 10,000 | diez mil |
| 100,000 | cien mil |
| 1,000,000 | un millón (de + 명사) |
| 2,000,000 | dos millones (de + 명사) |

> **Millón**은 명사이므로 다른 명사 앞에 **de**가 필요합니다:
> - un millón **de** personas (백만 명의 사람들)
> - tres millones **de** euros

### 성 일치 (Gender Agreement)

**Uno**와 백의 자리(200-900)는 명사의 성에 일치시킵니다:
- **veintiún** libros / **veintiuna** páginas
- **doscientos** euros / **doscientas** personas

## 3. 서수

| 숫자 | 남성형 | 여성형 |
|------|--------|--------|
| 1번째 | primero (primer) | primera |
| 2번째 | segundo | segunda |
| 3번째 | tercero (tercer) | tercera |
| 4번째 | cuarto | cuarta |
| 5번째 | quinto | quinta |
| 6번째 | sexto | sexta |
| 7번째 | séptimo | séptima |
| 8번째 | octavo | octava |
| 9번째 | noveno | novena |
| 10번째 | décimo | décima |

> **Primero**와 **tercero**는 남성 단수 명사 앞에서 **primer**과 **tercer**로
> 축약됩니다: el **primer** día, el **tercer** piso.

> 10번째를 넘어서면, 스페인어에서는 보통 명사 뒤에 기수를 사용합니다:
> el piso **once** (11층), el siglo **veintiuno** (21세기).

## 4. 시간 말하기

### 기본 구조

```
+----------------------------------------------------------+
|              TELLING TIME IN SPANISH                      |
+----------------------------------------------------------+
|                                                          |
|  1:00  Es la una.           (singular: "es la")          |
|  2:00  Son las dos.         (plural: "son las")          |
|  3:15  Son las tres y cuarto.                            |
|  4:30  Son las cuatro y media.                           |
|  5:45  Son las seis menos cuarto.                        |
|        (= Faltan quince para las seis.)                  |
|  6:10  Son las seis y diez.                              |
|  7:50  Son las ocho menos diez.                          |
|                                                          |
+----------------------------------------------------------+
```

### 시간대

| 스페인어 | 의미 |
|----------|------|
| de la mañana | 아침에 (AM) |
| de la tarde | 오후에 (PM, ~1-7시) |
| de la noche | 저녁/밤에 (PM, ~8-12시) |
| del mediodía | 정오에 |
| de la medianoche | 자정에 |

**예시:**
- Son las ocho **de la mañana**. (오전 8시입니다.)
- Son las tres **de la tarde**. (오후 3시입니다.)
- Es la una **de la tarde**. (오후 1시입니다.)

### 시간 묻고 답하기

- **¿Qué hora es?** (지금 몇 시입니까?)
- **¿A qué hora...?** (몇 시에...?)
  - ¿A qué hora empieza la clase? — **A las nueve.** (9시에.)

## 7. 날짜 표현하기

### 형식

```
el + 숫자 + de + 월 + de + 연도

el 15 de marzo de 2026
(2026년 3월 15일)
```

- **예외**: 매월 1일에는 *primero*를 사용합니다:
  - el **primero** de enero (1월 1일)
  - 하지만: el **dos** de enero (1월 2일)

### 날짜 묻기

- **¿Qué fecha es hoy?** (오늘 날짜가 어떻게 됩니까?)
- **¿A cuántos estamos?** (오늘 며칠입니까?)
- **Hoy es el 24 de febrero de 2026.** (오늘은 2026년 2월 24일입니다.)

### 연도

- **2026**: dos mil veintiséis
- **1999**: mil novecientos noventa y nueve
- **2000**: dos mil (el año dos mil)

## 9. 빈도 표현하기

| 스페인어 | 한국어 |
|----------|--------|
| siempre | 항상 |
| casi siempre | 거의 항상 |
| todos los días | 매일 |
| a menudo / frecuentemente | 자주 |
| a veces | 가끔 |
| de vez en cuando | 때때로 |
| pocas veces / raramente | 드물게 |
| casi nunca | 거의 ~않다 |
| nunca / jamás | 절대 ~않다 |

**빈도와 시간 표현:**
- tres veces **al** día (하루에 세 번)
- una vez **a la** semana (일주일에 한 번)
- dos veces **al** mes (한 달에 두 번)
- cada año (매년)
