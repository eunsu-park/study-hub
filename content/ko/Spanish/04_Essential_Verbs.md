## 1. 왜 이 네 가지 동사인가?

```
┌─────────────────────────────────────────────────────────────────┐
│   이 네 가지 동사는 거의 모든 스페인어 문장에 등장합니다.        │
│   이것들을 마스터하면 대부분의 기본 아이디어를 표현할 수 있습니다.│
│                                                                 │
│   SER    → 정체성, 출신, 특성       (영구적 "~이다")            │
│   ESTAR  → 위치, 상태, 감정         (일시적 "~이다")            │
│   TENER  → 소유, 나이, 많은 관용표현 ("가지다")                 │
│   HABER  → 존재 (hay), 완료 시제    ("가지다" 조동사)           │
│                                                                 │
│   영어 화자를 위한 도전:                                        │
│   영어 "to be" → SER 또는 ESTAR (반드시 선택해야 합니다!)       │
│   영어 "to be" → 때때로 TENER (tener hambre = "배고프다",       │
│                  "배고픔을 가지다"가 아님)                       │
└─────────────────────────────────────────────────────────────────┘
```

## 2. SER — 영구적/본질적 "~이다"

### 2.1 현재 시제 활용(Conjugation)

| 주어 | 활용형 | 예문 |
|------|--------|------|
| yo | **soy** | Soy estudiante. (나는 학생이다.) |
| tú | **eres** | Eres inteligente. (너는 똑똑하다.) |
| él/ella/usted | **es** | Es profesor. (그는 교수이다.) |
| nosotros/as | **somos** | Somos coreanos. (우리는 한국인이다.) |
| vosotros/as | **sois** | Sois amigos. (너희는 친구이다.) |
| ellos/ellas/ustedes | **son** | Son las tres. (3시이다.) |

### 2.2 SER의 용법 — DOCTOR 기억법(Mnemonic)

```
┌─────────────────────────────────────────────────────────────────┐
│   D - Description 묘사 (외모/성격적 특성)                       │
│       Ella es alta y simpática.                                 │
│       (그녀는 키가 크고 친절하다.)                               │
│                                                                 │
│   O - Occupation 직업                                           │
│       Soy ingeniero. (나는 엔지니어이다.)                       │
│       Es profesora. (그녀는 교사이다.)                          │
│                                                                 │
│   C - Characteristic 특성 (본질적 성질)                         │
│       El hielo es frío. (얼음은 차갑다.)                        │
│       La nieve es blanca. (눈은 하얗다.)                        │
│                                                                 │
│   T - Time 시간 (시각, 날짜)                                    │
│       Son las dos. (2시이다.)                                   │
│       Hoy es lunes. (오늘은 월요일이다.)                        │
│       Es el 15 de marzo. (3월 15일이다.)                        │
│                                                                 │
│   O - Origin 출신 (어디에서 왔는지)                             │
│       Soy de Corea. (나는 한국 출신이다.)                       │
│       El café es de Colombia. (이 커피는 콜롬비아산이다.)       │
│                                                                 │
│   R - Relationship 관계 (소유, 재질)                            │
│       Es mi libro. (그것은 내 책이다.)                          │
│       La mesa es de madera. (그 탁자는 나무로 만들어졌다.)      │
│       La fiesta es en mi casa. (파티는 내 집에서 열린다.)       │
└─────────────────────────────────────────────────────────────────┘
```

### 2.3 SER 추가 예문

```
Identity:      Soy María. (나는 마리아이다.)
Nationality:   Somos coreanos. (우리는 한국인이다.)
Religion:      Es católico. (그는 가톨릭 신자이다.)
Personality:   Eres muy amable. (너는 매우 친절하다.)
Profession:    Son doctores. (그들은 의사이다.)
Time:          Es la una. (1시이다.)
Origin:        El vino es de España. (이 와인은 스페인산이다.)
Material:      El anillo es de oro. (이 반지는 금으로 만들어졌다.)
Events:        La reunión es a las tres. (회의는 3시에 열린다.)
```

## 3. ESTAR — 일시적/상황적 "~이다"

### 3.1 현재 시제 활용(Conjugation)

| 주어 | 활용형 | 예문 |
|------|--------|------|
| yo | **estoy** | Estoy en casa. (나는 집에 있다.) |
| tú | **estás** | ¿Cómo estás? (어떻게 지내?) |
| él/ella/usted | **está** | Está contento. (그는 기쁘다.) |
| nosotros/as | **estamos** | Estamos cansados. (우리는 피곤하다.) |
| vosotros/as | **estáis** | ¿Estáis listos? (너희 준비됐어?) |
| ellos/ellas/ustedes | **están** | Están en Madrid. (그들은 마드리드에 있다.) |

### 3.2 ESTAR의 용법 — PLACE 기억법(Mnemonic)

```
┌─────────────────────────────────────────────────────────────────┐
│   P - Position/Place 위치/장소                                  │
│       Estoy en la oficina. (나는 사무실에 있다.)                │
│       Madrid está en España. (마드리드는 스페인에 있다.)        │
│                                                                 │
│   L - Looks/Appears 외관 (지금 어떻게 보이는지)                 │
│       Estás muy guapa hoy. (오늘 정말 예쁘다.)                  │
│       La comida está deliciosa. (음식이 맛있다.)                │
│                                                                 │
│   A - Action 진행 중인 동작 (-ando/-iendo)                      │
│       Estoy estudiando. (나는 공부하고 있다.)                   │
│       Están comiendo. (그들은 먹고 있다.)                       │
│                                                                 │
│   C - Condition 상태 (일시적 상태, 변화의 결과)                 │
│       Estoy cansado. (나는 피곤하다.)                           │
│       La puerta está abierta. (문이 열려 있다.)                 │
│       El café está frío. (커피가 식었다.)                       │
│                                                                 │
│   E - Emotion 감정 (지금 느끼는 감정)                           │
│       Estoy feliz. (나는 지금 행복하다.)                        │
│       Está triste. (그/그녀는 슬프다.)                          │
│       Estamos nerviosos. (우리는 긴장된다.)                     │
└─────────────────────────────────────────────────────────────────┘
```

## 4. SER vs ESTAR 비교

### 4.1 핵심 구분

```
SER = 무엇인가가 무엇인지 (본질, 정체성, 정의)
ESTAR = 무엇인가가 어떤 상태인지 (상태, 조건, 위치)

이렇게 생각하세요:
  SER: "이것이 그것의 본질이다."
  ESTAR: "이것이 그것의 현재 상태이다."
```

### 4.2 같은 형용사, 다른 의미

| 형용사 | SER 사용 (본질적) | ESTAR 사용 (현재 상태) |
|--------|-------------------|------------------------|
| feliz | Es feliz. (그는 행복한 사람이다.) | Está feliz. (그는 지금 행복하다.) |
| aburrido | Es aburrido. (그는 지루한 사람이다.) | Está aburrido. (그는 지루해하고 있다.) |
| listo | Es listo. (그는 영리하다.) | Está listo. (그는 준비되었다.) |
| malo | Es malo. (그는 나쁜 사람이다.) | Está malo. (그는 아프다.) |
| bueno | Es bueno. (그는 좋은 사람이다.) | Está bueno. (그가 멋져 보인다. / 맛있다.) |
| rico | Es rico. (그는 부자이다.) | Está rico. (맛있다.) |
| verde | Es verde. (그것은 초록색이다.) | Está verde. (그것은 덜 익었다.) |
| vivo | Es vivo. (그는 영리하다/재빠르다.) | Está vivo. (그는 살아 있다.) |
| seguro | Es seguro. (그것은 안전하다.) | Está seguro. (그는 확신한다.) |
| orgulloso | Es orgulloso. (그는 자존심이 강한 사람이다.) | Está orgulloso. (그는 자랑스러워하고 있다.) |
| guapo | Es guapo. (그는 잘생겼다.) | Está guapo. (그가 오늘 멋져 보인다.) |
| nuevo | Es nuevo. (그것은 새것이다.) | Está nuevo. (그것이 새것처럼 보인다.) |

### 4.3 빠른 판별 차트

```
무엇에 관한 것인가?
  정체성 / 출신 / 직업          → SER
  위치                          → ESTAR
  시간 / 날짜                   → SER
  지금의 감정 / 기분            → ESTAR
  신체적 특성 (항상 그러한)     → SER
  상태 (변할 수 있는)           → ESTAR
  진행형 (-ando/-iendo)         → ESTAR
  행사 장소                     → SER
  재질 / 구성                   → SER
  행동의 결과                   → ESTAR
```

## 5. TENER — "가지다"와 관용적 표현

### 5.1 현재 시제 활용(Conjugation)

| 주어 | 활용형 | 예문 |
|------|--------|------|
| yo | **tengo** | Tengo un gato. (나는 고양이 한 마리를 키운다.) |
| tú | **tienes** | ¿Tienes tiempo? (시간 있어?) |
| él/ella/usted | **tiene** | Tiene 30 años. (그는 30살이다.) |
| nosotros/as | **tenemos** | Tenemos clase. (우리는 수업이 있다.) |
| vosotros/as | **tenéis** | ¿Tenéis hambre? (너희 배고파?) |
| ellos/ellas/ustedes | **tienen** | Tienen dos hijos. (그들은 두 자녀가 있다.) |

### 5.2 직역적 용법(Literal Uses)

```
Possession:    Tengo un coche. (나는 차가 있다.)
Relationships: Tiene tres hermanos. (그는 형제자매가 셋 있다.)
Age:           Tengo veinticinco años. (나는 25살이다.)
               ※ 스페인어에서는 "나는 25년을 가지고 있다"로 표현하며,
                  "나는 25이다"라고 하지 않음
```

### 5.3 TENER 관용 표현 (영어에서 "to be"를 사용하는 경우)

```
┌─────────────────────────────────────────────────────────────────┐
│   신체적 감각:                                                  │
│   tener hambre       → 배고프다       (직역: 배고픔을 가지다)   │
│   tener sed          → 목마르다       (직역: 갈증을 가지다)     │
│   tener frío         → 춥다           (직역: 추위를 가지다)     │
│   tener calor        → 덥다           (직역: 더위를 가지다)     │
│   tener sueño        → 졸리다         (직역: 졸음을 가지다)     │
│                                                                 │
│   감정 상태:                                                    │
│   tener miedo (de)   → 두렵다         (직역: 두려움을 가지다)   │
│   tener vergüenza    → 부끄럽다       (직역: 수치를 가지다)     │
│   tener cuidado      → 조심하다       (직역: 주의를 가지다)     │
│   tener paciencia    → 인내심 있다    (직역: 인내를 가지다)     │
│                                                                 │
│   기타 주요 관용 표현:                                          │
│   tener razón        → 맞다           (직역: 이유를 가지다)     │
│   tener prisa        → 서두르다       (직역: 서두름을 가지다)   │
│   tener suerte       → 운이 좋다      (직역: 행운을 가지다)     │
│   tener éxito        → 성공하다       (직역: 성공을 가지다)     │
│   tener sentido      → 말이 되다      (직역: 의미를 가지다)     │
│   tener lugar        → 개최되다       (직역: 장소를 가지다)     │
│   tener ganas de     → ~하고 싶다     (직역: ~의 욕구를 가지다) │
└─────────────────────────────────────────────────────────────────┘
```

### 5.4 TENER QUE + 부정사(Infinitive) (의무 표현)

```
tener que + 부정사 = "~해야 한다" / "~해야만 한다"

Tengo que estudiar.        → 나는 공부해야 한다.
Tienes que comer.          → 너는 먹어야 한다.
Tenemos que ir.            → 우리는 가야 한다.
¿Tienes que trabajar hoy? → 오늘 일해야 해?
```

## 6. HABER — "~이 있다"와 조동사(Auxiliary)

### 6.1 비인칭(Impersonal) HAY (~이 있다)

```
┌─────────────────────────────────────────────────────────────────┐
│   hay = ~이 있다 (현재 시제, 비인칭)                            │
│                                                                 │
│   Hay un libro en la mesa.        → 탁자 위에 책이 한 권 있다. │
│   Hay muchos estudiantes.         → 학생이 많이 있다.           │
│   ¿Hay un banco cerca?            → 근처에 은행이 있나요?       │
│   No hay problema.                → 문제없다.                   │
│   Hay que estudiar.               → 공부해야 한다. (비인칭)     │
│                                                                 │
│   과거:   había (~이 있었다)                                    │
│   미래:   habrá (~이 있을 것이다)                               │
│                                                                 │
│   ※ HAY는 복수형에서도 변하지 않습니다!                        │
│   Hay un gato.   Hay tres gatos.  ("han tres gatos"가 아님)    │
└─────────────────────────────────────────────────────────────────┘
```

### 6.2 HAY vs ESTAR (위치 표현)

```
HAY → 존재를 소개 (~이 있다), 부정관사(Indefinite Article)와 함께 사용
ESTAR → 알려진 것의 위치 지정, 정관사(Definite Article)와 함께 사용

  Hay un banco en la calle.        → 거리에 은행이 하나 있다.
  El banco está en la calle Main.  → 그 은행은 메인 거리에 있다.

  Hay libros en la mesa.           → 탁자 위에 책들이 있다.
  Los libros están en la mesa.     → 그 책들은 탁자 위에 있다.
```

### 6.3 조동사(Auxiliary)로서의 HABER (복합 시제 미리보기)

```
HABER (활용형) + 과거분사 = 복합 시제(Compound Tenses)

현재완료(Present Perfect):
  he hablado     → 나는 말했다
  has comido     → 너는 먹었다
  ha vivido      → 그/그녀는 살았다
  hemos visto    → 우리는 보았다

※ 이 내용은 12과에서 자세히 다룹니다.
```

## 7. 흔한 실수와 팁

### 7.1 자주 하는 실수

```
✗ Soy cansado.           ✓ Estoy cansado. (피곤함 = 일시적 상태)
✗ Estoy profesor.         ✓ Soy profesor. (직업 = 정체성)
✗ Soy en Madrid.          ✓ Estoy en Madrid. (위치 = ESTAR)
✗ Tengo 25.               ✓ Tengo 25 años. (나이에는 "años" 필요)
✗ Soy hambre.             ✓ Tengo hambre. (배고프다 = TENER 관용표현)
✗ Han muchos libros.      ✓ Hay muchos libros. (HAY는 불변)
✗ La fiesta está en mi casa. ✓ La fiesta es en mi casa. (행사 = SER)
```

### 7.2 기억 팁

```
SER = DOCTOR     (Description 묘사, Occupation 직업, Characteristic 특성,
                   Time 시간, Origin 출신, Relationship 관계)

ESTAR = PLACE    (Position 위치, Looks 외관, Action 동작, Condition 상태, Emotion 감정)

신체적 감각에는 TENER 사용:
  "스페인어에서는 배고프다/춥다/졸리다가 아니라
   배고픔/추위/졸음을 '가진다(TENER)'고 표현합니다"
```

```
6. No _______ problema.              → hay (존재)
```

</details>

### 연습 4: 문장 번역

```
1. 나는 한국인이다. 나는 서울 출신이다.
   → Soy coreano/a. Soy de Seúl.

2. 그녀는 피곤하지만 행복하다.
   → Está cansada pero está feliz.

3. 탁자 위에 책이 세 권 있다.
   → Hay tres libros en la mesa.

4. 나는 25살이고 학생이다.
   → Tengo veinticinco años y soy estudiante.

5. 배고프니? 부엌에 음식이 있어.
   → ¿Tienes hambre? Hay comida en la cocina.
```
