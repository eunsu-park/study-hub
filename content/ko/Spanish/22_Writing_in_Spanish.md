## 1. 스페인어 구두점

### 역부호 (Inverted Marks)

스페인어는 주요 언어 중 유일하게 질문과 감탄의 시작 부분에 **역물음표와 역느낌표**를 사용합니다:

```
+----------------------------------------------------------+
|          SPANISH PUNCTUATION MARKS                        |
+----------------------------------------------------------+
|                                                          |
|  Questions:    ¿...?                                     |
|  ¿Cómo te llamas? (What's your name?)                   |
|  ¿Dónde vives? (Where do you live?)                      |
|                                                          |
|  Exclamations: ¡...!                                     |
|  ¡Qué bonito! (How beautiful!)                           |
|  ¡No me digas! (No way!)                                 |
|                                                          |
|  IMPORTANT: The inverted mark goes where the question     |
|  or exclamation BEGINS (not necessarily at the start      |
|  of the sentence):                                       |
|                                                          |
|  Si vienes mañana, ¿puedes traer el libro?               |
|  Después de mucho trabajo, ¡por fin terminé!             |
+----------------------------------------------------------+
```

### 기타 구두점 차이

| 기호 | 스페인어 규칙 | 예시 |
|--------|-------------|---------|
| « » 또는 " " | 인용 부호 (둘 다 사용 가능) | «Hola», dijo María. / "Hola", dijo María. |
| — | 대화 대시 (문학에서 인용 부호 대신 사용) | —¿Cómo estás? —preguntó Juan. |
| ; | 세미콜론 (영어와 동일) | Estudié mucho; aprobé el examen. |
| : | 콜론 (영어와 동일, 추가로 인사말 뒤에 사용) | Estimado señor: |
| ... | 줄임표 (영어와 동일) | No sé... tal vez mañana. |

### 숫자와 날짜

| 규칙 | 스페인어 | 영어 |
|-----------|---------|---------|
| 소수점 | 3**,**5 (쉼표) | 3**.**5 (마침표) |
| 천 단위 구분 | 1**.**000 (마침표) | 1**,**000 (쉼표) |
| 날짜 형식 | 15/10/2024 (일/월/년) | 10/15/2024 (월/일/년) |
| 통화 | 29,99 € (숫자 뒤) | $29.99 (숫자 앞) |

## 2. 악센트 부호 (Tildes)

### 강세 규칙 (Stress Rules)

스페인어는 예측 가능한 강세 패턴을 가지고 있습니다. 악센트 부호는 기본 규칙에 대한 **예외**를 나타냅니다:

```
+----------------------------------------------------------+
|              STRESS AND ACCENT RULES                      |
+----------------------------------------------------------+
|                                                          |
|  DEFAULT STRESS (no accent needed):                      |
|  • Words ending in vowel, -n, or -s → stress on          |
|    SECOND-TO-LAST syllable (palabra llana)                |
|    Examples: ca-SA, ha-BLAN, co-MES                      |
|                                                          |
|  • Words ending in consonant (except -n, -s) → stress    |
|    on LAST syllable (palabra aguda)                      |
|    Examples: ha-BLAR, co-MER, ciu-DAD                    |
|                                                          |
|  ACCENT MARK NEEDED (exceptions):                        |
|  • Agudas stressed on last + ending in vowel/n/s:        |
|    café, canción, inglés, habló                           |
|                                                          |
|  • Llanas stressed on 2nd-to-last + NOT ending in         |
|    vowel/n/s: árbol, lápiz, fácil, difícil               |
|                                                          |
|  • Esdrújulas (stress on 3rd-to-last): ALWAYS accented   |
|    teléfono, médico, número, último, rápido              |
|                                                          |
|  • Sobresdrújulas (4th-to-last): ALWAYS accented         |
|    dígamelo, tráemelo                                    |
+----------------------------------------------------------+
```

### 의미를 구별하는 악센트 부호

| 악센트 없음 | 악센트 있음 |
|---------------|-------------|
| el (그, 정관사) | él (그, 대명사) |
| tu (너의) | tú (너) |
| mi (나의) | mí (나를/나에게) |
| si (만약) | sí (네, 예) |
| se (재귀 대명사) | sé (나는 안다 / ~해라 [명령형]) |
| de (~의, ~에서) | dé (주다 [접속법]) |
| te (너를, 목적격) | té (차) |
| mas (그러나, 문어체) | más (더) |
| solo (홀로, 형용사) | sólo (오직 — 현대 용법에서는 선택적) |
| como (~처럼, ~로서) | cómo (어떻게?) |
| donde (~하는 곳, 관계사) | dónde (어디?) |
| que (~하는, 관계사) | qué (무엇?) |
| quien (~하는 사람, 관계사) | quién (누구?) |
| cuando (~할 때, 관계사) | cuándo (언제?) |

> **규칙**: 의문사(Question Words)는 직접 의문문이든 간접 의문문이든 **항상** 악센트 부호를 가집니다: "No sé **cuándo** viene." (그가 언제 오는지 모르겠다.)

## 3. 대문자 사용 규칙

스페인어는 영어보다 대문자를 **적게** 사용합니다:

| 영어에서 대문자 사용 | 스페인어에서 소문자 사용 |
|----------------------|---------------------------|
| 요일 | lunes, martes, miércoles... |
| 월 | enero, febrero, marzo... |
| 국적/언어 | español, mexicano, inglés |
| 종교 | católico, musulmán, budista |
| "I" (대명사) | yo (소문자) |
| 책/영화 제목 (주요 단어 전체) | 첫 단어 + 고유명사만 대문자 |

### 스페인어에서 대문자를 사용하는 경우

- 문장의 첫 단어
- 고유명사 (사람, 장소 이름)
- 국가명: España, México, Argentina
- 약어: Ud., Sr., Dra.
- 기관명: Real Academia Española

### 예시

| 영어 | 스페인어 |
|---------|---------|
| Monday, January 15 | lunes, 15 de enero |
| I speak Spanish. | Yo hablo español. |
| The Lord of the Rings | El señor de los anillos |
| She is Catholic. | Ella es católica. |

## 4. 비공식 이메일 작성

### 구조

```
+----------------------------------------------------------+
|          INFORMAL EMAIL STRUCTURE                         |
+----------------------------------------------------------+
|                                                          |
|  Greeting:                                               |
|  ¡Hola, [nombre]! / Querido/a [nombre]: / ¡Hey!         |
|                                                          |
|  Opening:                                                |
|  ¿Qué tal? / ¿Cómo estás? / ¿Cómo va todo?             |
|  Espero que estés bien.                                  |
|                                                          |
|  Body:                                                   |
|  Te escribo para... / Quería contarte que...             |
|                                                          |
```

## 5. 공식 이메일 또는 편지 작성

### 구조

```
+----------------------------------------------------------+
|          FORMAL EMAIL / LETTER STRUCTURE                  |
+----------------------------------------------------------+
|                                                          |
|  Greeting:                                               |
|  Estimado/a Sr./Sra. [apellido]:                         |
|  Estimados señores:                                      |
|  A quien corresponda: (To whom it may concern)           |
|                                                          |
|  Opening:                                                |
|  Me dirijo a usted(es) para...                           |
|  Le(s) escribo con respecto a...                         |
|  En relación con su [carta/email/anuncio]...             |
|                                                          |
|  Body:                                                   |
|  Clear, organized paragraphs                             |
|  Usted/ustedes forms throughout                          |
|                                                          |
|  Closing:                                                |
|  Quedo a la espera de su respuesta.                      |
|  Le(s) agradezco su atención.                            |
|  No dude(n) en contactarme si necesita(n) más            |
|  información.                                            |
|                                                          |
|  Sign-off:                                               |
|  Atentamente, / Un cordial saludo, / Reciba un           |
|  cordial saludo,                                         |
|  [Nombre completo]                                       |
|  [Cargo/departamento]                                    |
+----------------------------------------------------------+
```

### 예시

> Estimada Sra. López:
>
> Me dirijo a usted en relación con el puesto de asistente administrativo publicado en su página web el pasado 5 de marzo.
>
> Soy licenciado en Administración de Empresas por la Universidad Complutense de Madrid y cuento con tres años de experiencia en puestos similares. Adjunto mi currículum vitae para su consideración.
>
> Quedo a su disposición para una entrevista en el momento que le resulte conveniente. No dude en contactarme en el número 612 345 678 o en este correo electrónico.
>
> Atentamente,
> Carlos Martínez García

### 주요 공식 표현

| 용도 | 표현 |
|---------|-----------|
| 서두 | Me dirijo a usted para... (귀하에게 ~을 위해 글을 씁니다) |
| 목적 | El motivo de esta carta es... (이 편지의 목적은 ~입니다) |
| 요청 | Le agradecería que... (~해 주시면 감사하겠습니다) |
| 첨부 | Adjunto... (~을 첨부합니다) |
| 가용성 | Quedo a su disposición. (언제든지 연락 주십시오) |
| 마무리 | A la espera de su respuesta... (답변을 기다리겠습니다) |
| 감사 | Le agradezco de antemano. (미리 감사드립니다) |

## 6. 짧은 에세이 작성

### 에세이 구조 (Essay Structure)

```
+----------------------------------------------------------+
|            ESSAY STRUCTURE (ENSAYO)                       |
+----------------------------------------------------------+
|                                                          |
|  INTRODUCTION (Introducción):                            |
|  • Hook: question, quote, surprising fact                 |
|  • Present the topic                                     |
|  • State your thesis                                     |
|  Markers: Hoy en día..., En la actualidad...,            |
|           Es bien sabido que...                           |
|                                                          |
|  BODY (Desarrollo): 2-3 paragraphs                       |
|  • Each paragraph = one main idea                        |
|  • Support with examples, data, arguments                |
|  Markers: En primer lugar..., Además...,                 |
|           Por otro lado..., Sin embargo...               |
|                                                          |
|  CONCLUSION (Conclusión):                                |
|  • Summarize main points                                 |
|  • Restate thesis (different words)                      |
|  • Final thought or call to action                       |
|  Markers: En conclusión..., En resumen...,               |
|           En definitiva...                                |
+----------------------------------------------------------+
```

### 에세이 개요 예시

**주제**: ¿Es importante aprender idiomas? (언어를 배우는 것이 중요한가?)

**서론**:
> En la actualidad, vivimos en un mundo cada vez más globalizado. Aprender un segundo idioma ya no es un lujo, sino una necesidad. En este ensayo, argumentaré que el aprendizaje de idiomas es fundamental tanto para el desarrollo personal como profesional.

**본문 1단락** (직업적 이점):
> En primer lugar, hablar otro idioma abre puertas en el mercado laboral...

**본문 2단락** (개인적 이점):
> Además, aprender un idioma permite conocer otras culturas...

**본문 3단락** (반론 + 재반박):
> Sin embargo, algunos argumentan que la tecnología de traducción hace innecesario el estudio de idiomas. No obstante, la comunicación humana va más allá de las palabras...

**결론**:
> En definitiva, aprender idiomas es una inversión en nuestro futuro...

## 7. 영어 화자가 범하는 흔한 오류

### 문법 오류

| 오류 유형 | 잘못된 표현 | 올바른 표현 | 규칙 |
|-------|-------|---------|------|
| 성 일치 | la problema | **el** problema | -ma로 끝나는 명사는 대개 남성형 |
| Ser vs estar | Soy cansado | **Estoy** cansado | 일시적 상태에는 estar 사용 |
| 이중 부정 | No tengo algo | No tengo **nada** | 스페인어는 이중 부정이 필수 |
| 신체 부위의 소유격 | Lavo mis manos | Me lavo **las** manos | 소유격 대신 정관사 사용 |
| 어순 | rojo coche | coche **rojo** | 형용사는 보통 명사 뒤에 위치 |
| 주어로서의 "It" | It es importante | **Es** importante | "it"을 주어로 번역하지 않음 |

### 철자 오류

| 오류 유형 | 잘못된 표현 | 올바른 표현 | 참고 |
|-------|-------|---------|------|
| ñ 누락 | espanol | espa**ñ**ol | ñ는 별개의 문자 |
| 악센트 누락 | cafe | caf**é** | 악센트는 의미/강세를 변경 |
| b/v 혼동 | aver | a **ver** | 스페인어에서 발음이 동일 |
| h 오류 | aver (보다) | a ver / haber | h는 묵음 |
| ll vs y | — | 지역별 변이 | 철자 구별 불필요 |

### 작문에서의 거짓 동족어(False Friends) 함정

| 의도하는 의미 | 잘못된 표현 | 올바른 표현 |
|--------------|-------------|-------|
| ~을 깨닫다 (이해하다) | realizar | **darse cuenta (de)** |
| 사실은, 실제로 | actualmente | **en realidad** / **de hecho** |
| 참석하다 | atender | **asistir a** |
| 도서관 | librería | **biblioteca** |
| (사람을) 소개하다 | introducir | **presentar** |
| 현재, 지금 | corrientemente | **actualmente** |
| 지지하다, 지원하다 | soportar | **apoyar** |

## 8. 글에서 아이디어 연결하기

### 기능별 유용한 접속사

| 기능 | 접속사 |
|----------|-----------|
| **추가** | además (게다가), también (또한), es más (더 나아가), incluso (심지어) |
| **대조** | sin embargo (그러나), no obstante (그럼에도 불구하고), en cambio (반면에), aunque (비록 ~이지만) |
| **원인** | porque (~이기 때문에), ya que (~이므로), dado que (~인 점을 감안하면), debido a (~로 인해) |
| **결과** | por lo tanto (따라서), por eso (그래서), así que (그러므로), en consecuencia (결과적으로) |
| **순서** | en primer lugar (첫째로), a continuación (다음으로), luego (그 다음에), finalmente (마지막으로) |
| **명확화** | es decir (즉), o sea (다시 말해서), por ejemplo (예를 들어) |
| **결론** | en conclusión (결론적으로), en resumen (요약하면), en definitiva (궁극적으로) |

## 9. 연습 문제

### 연습 1: 구두점 추가

빠진 구두점(¿ ¡ 악센트)을 추가하세요.

1. Como te llamas
2. Que bonito dia
3. Si vienes manana, puedes traer el libro
4. Donde esta la biblioteca
5. Fantastico, me encanta

<details>
<summary>정답</summary>

1. **¿Cómo** te llamas**?**
2. **¡Qué** bonito **día!**
3. Si vienes **mañana**, **¿**puedes traer el libro**?**
4. **¿Dónde está** la biblioteca**?**
5. **¡Fantástico**, me encanta**!**

</details>

### 연습 2: 대문자 수정

대문자 오류를 수정하세요 (일부 단어는 소문자여야 합니다).

1. Estudio Español los Lunes y Miércoles.
2. Mi amigo es Católico y Mexicano.
3. El Señor De Los Anillos es mi película favorita.

<details>
<summary>정답</summary>

1. Estudio **español** los **lunes** y **miércoles**.
2. Mi amigo es **católico** y **mexicano**.
3. **El señor de los anillos** es mi película favorita.

</details>

### 연습 3: 비공식 이메일 작성

이번 주말에 영화를 보러 가자고 친구를 초대하는 짧은 이메일을 작성하세요. 다음을 포함하세요:
- 인사말
- 세부사항이 포함된 초대 (영화, 시간, 장소)
- 마무리 인사

<details>
<summary>모범 답안</summary>

> ¡Hola, Pedro!
>
> ¿Qué tal? ¿Tienes planes para este sábado? He visto que ponen una película nueva en el cine del centro y me apetece mucho ir a verla. Es a las 19:30.
>
> ¿Te animas? Después podemos cenar algo por ahí.
>
> ¡Avísame!
> Un abrazo,
> María

</details>

### 연습 4: 오류 수정

각 문장에서 오류를 찾아 수정하세요.

1. Yo soy muy cansado después del trabajo.
2. La problema es que no tengo dinero.
3. Yo lavo mis manos antes de comer.
4. No quiero algo de comer.
5. El coche roja es muy rapido.

<details>
<summary>정답</summary>

1. Yo **estoy** muy cansado... (일시적 상태 → estar)
2. **El** problema es que... (problema는 남성형)
3. **Me** lavo **las** manos... (신체 부위에는 재귀동사 + 정관사)
4. No quiero **nada** de comer. (이중 부정 필수)
5. El coche **rojo** es muy **rápido**. (성 일치 + 악센트)

</details>

### 연습 5: 에세이 서론

다음 주제에 대한 서론 단락을 작성하세요: "¿Son las redes sociales buenas o malas para la sociedad?" (소셜 미디어는 사회에 좋은가 나쁜가?)

담화 표지를 최소 2개 이상 사용하세요.

<details>
<summary>모범 답안</summary>

"**Hoy en día**, las redes sociales forman parte de la vida cotidiana de millones de personas en todo el mundo. **Sin embargo**, existe un intenso debate sobre si su impacto en la sociedad es positivo o negativo. **En mi opinión**, las redes sociales tienen tanto ventajas como desventajas, y es importante aprender a usarlas de manera responsable."

</details>
