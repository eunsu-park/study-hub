Exercise 2: Fluid Typography with clamp()

Problem: Set body font size that smoothly changes between 16px and 24px.

Fluid Typography Demo

This text scales smoothly between 16px and 24px using clamp(1rem, 2vw, 1.5rem). No media queries are needed — the font size adjusts continuously as the viewport width changes. The clamp() function takes three arguments: a minimum value, a preferred value (usually viewport-relative), and a maximum value.

This smaller text uses clamp(0.875rem, 1.5vw, 1.125rem) for proportional scaling.

CSS Used:

body {
    /* Smoothly scales between 16px and 24px */
    font-size: clamp(1rem, 2vw, 1.5rem);

    /* Alternative with calc for finer control: */
    /* font-size: clamp(16px, 1.5vw + 12px, 24px); */
}
