\section*{Exercise 3: Personalized Document}

% The solution is the very structure of this file.  Below is a standalone
% example one could compile separately.

\begin{verbatim}
\documentclass[12pt, a4paper]{article}
\usepackage[utf8]{inputenc}
\usepackage[T1]{fontenc}

\title{A Journey Through \LaTeX{}}
\author{Jane Doe}
\date{\today}

\begin{document}
\maketitle

\section{What I Will Learn}
In this course I plan to master:
\begin{itemize}
    \item Mathematical typesetting
    \item Professional document layout
    \item Figure and table management
\end{itemize}

\section{Why \LaTeX{} Matters}
\LaTeX{} produces beautiful, publication-quality documents and
is the standard tool in science and engineering.

\end{document}
\end{verbatim}

% === Exercise 5: Error Recovery ===
% Problem: Remove \end{document}, observe the error, then fix it.

\section*{Exercise 5: Error Recovery}

Removing \verb|\end{document}| produces the error:

\begin{verbatim}
! Emergency stop.
<*> document.tex
*** (job aborted, no legal \end found)
\end{verbatim}

The fix is simply to restore the missing \verb|\end{document}| at the end of
the file.  \LaTeX{} requires every \verb|\begin{...}| to have a matching
\verb|\end{...}|.

\end{document}
