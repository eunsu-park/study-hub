Component Model Exercises

Exercise 1: Counter Component

Create a reusable counter component with:

  • Props: initial value, step size, min/max bounds
  • State: current count
  • Events: increment, decrement, reset
  • Display: current value, whether at min/max

Exercise 2: Data Flow

Build a temperature converter with two inputs (Celsius and Fahrenheit).

  • Changing one input should update the other
  • State should be lifted to a parent component
  • Each input is a child component receiving value via props

Exercise 3: Component Composition

Build a Card component that accepts:

  • title (string prop)
  • children (slot/content area)
  • footer (optional slot)
  • variant: "default" | "highlighted" | "danger"

Use the Card to build a user profile card and a notification card.
