SEO Optimization Example

Written on January 15, 2024 | Modified on January 20, 2024 | Author: Hong Gildong

1. Meta Tag Basics

Search engines understand page content through meta tags. Here are the most important meta tags:

Essential Meta Tags

<!-- Page title (50-60 characters recommended) -->
<title>Page Title | Site Name</title>

<!-- Page description (150-160 characters recommended) -->
<meta name="description" content="A concise and compelling description of the page...">

<!-- Viewport (mobile optimization) -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">

<!-- Canonical URL -->
<link rel="canonical" href="https://example.com/page">
Tip: The title tag is the most prominent element in search results. Place key keywords at the beginning and append the brand name at the end.

2. Open Graph & Twitter Card

Controls the information displayed when shared on social media.

Open Graph (Facebook, LinkedIn)

<meta property="og:type" content="article">
<meta property="og:title" content="Page Title">
<meta property="og:description" content="Page Description">
<meta property="og:image" content="https://example.com/image.jpg">
<meta property="og:url" content="https://example.com/page">

Twitter Card

<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:title" content="Page Title">
<meta name="twitter:description" content="Page Description">
<meta name="twitter:image" content="https://example.com/image.jpg">
Note: og:image should be at least 1200x630 pixels. Images that are too small may be cropped or not displayed on social media.

3. Structured Data (JSON-LD)

Using structured data can result in rich snippets appearing in search results.

Article Schema

<script type="application/ld+json">
{
    "@context": "https://schema.org",
    "@type": "Article",
    "headline": "Article Title",
    "author": {
        "@type": "Person",
        "name": "Author Name"
    },
    "datePublished": "2024-01-15T09:00:00+09:00",
    "image": "https://example.com/image.jpg"
}
</script>

Key Schema Types

Type Use Case Rich Snippet
Article News, blog posts Title, image, date
Product Product pages Price, rating, availability
Recipe Cooking recipes Cook time, calories, image
FAQPage Frequently asked questions Q&A list
HowTo How-to guides Step-by-step instructions
LocalBusiness Local businesses Address, hours, phone

4. Semantic HTML and SEO

Proper HTML structure helps search engines understand the content.

<!-- Good example -->
<article>
    <header>
        <h1>Article Title</h1>
        <time datetime="2024-01-15">January 15, 2024</time>
    </header>

    <section>
        <h2>Section Title</h2>
        <p>Content...</p>
    </section>

    <footer>
        <p>Author: Hong Gildong</p>
    </footer>
</article>

<!-- Bad example -->
<div class="article">
    <div class="title">Article Title</div>
    <div>Content...</div>
</div>

5. Core Web Vitals

Google's Core Web Vitals directly impact user experience and SEO.

Metric Meaning Good Needs Improvement
LCP Largest Contentful Paint ≤ 2.5s > 4.0s
INP Interaction to Next Paint ≤ 200ms > 500ms
CLS Cumulative Layout Shift ≤ 0.1 > 0.25

Optimization Tips

<!-- LCP optimization: Preload critical resources -->
<link rel="preload" href="hero-image.jpg" as="image">

<!-- CLS prevention: Specify image dimensions -->
<img src="image.jpg" width="800" height="600" alt="description">

<!-- Font optimization -->
<link rel="preload" href="font.woff2" as="font" type="font/woff2" crossorigin>

6. SEO Checklist

  • Set unique title and description for every page
  • Use only one h1 tag per page
  • Add alt attributes to images
  • Mobile-friendly responsive design
  • Use HTTPS
  • Generate sitemap (sitemap.xml)
  • Configure robots.txt
  • Add structured data
  • Optimize page loading speed
  • Optimize internal link structure

Frequently Asked Questions

What is SEO?

SEO (Search Engine Optimization) is the process of optimizing a website to rank higher in search engine results.

Why are meta tags important?

Meta tags inform search engines about the page content and provide the title and description displayed in search results. Proper meta tags can increase click-through rate (CTR).
