# Networking Examples

Python implementations of networking concepts and protocols.

## Files

| File | Lesson | Description |
|------|--------|-------------|
| `02_osi_packet_builder.py` | L02 | Build Ethernet/IP/TCP headers as bytes |
| `06_subnet_calculator.py` | L06 | CIDR calculator, VLSM planner |
| `07_subnetting_practice.py` | L07 | Interactive subnetting quiz |
| `08_routing_table_sim.py` | L08 | Longest prefix match simulator |
| `09_routing_protocol_sim.py` | L09 | Distance vector + link-state simulator |
| `10_tcp_state_machine.py` | L10 | TCP state machine (handshake/teardown) |
| `12_dns_resolver.py` | L12 | Recursive DNS resolver simulation |
| `13_http_client.py` | L13 | Raw HTTP/1.1 client |
| `15_firewall_sim.py` | L15 | Packet-filter firewall rules simulator |
| `17_network_tools.py` | L17 | Ping/traceroute/port scanner simulation |
| `18_ipv6_demo.py` | L18 | IPv6 manipulation, SLAAC generation |

## Running

All examples use Python standard library only:

```bash
python 02_osi_packet_builder.py
python 06_subnet_calculator.py
python 10_tcp_state_machine.py
python 12_dns_resolver.py
```
