# Probability and Statistics Examples

This directory contains example files demonstrating key probability and statistics concepts. Python examples use only the standard library (no external dependencies).

## Files Overview

| # | File | Topic | Key Demos |
|---|------|-------|-----------|
| 01 | `01_combinatorics.py` | Combinatorics and Counting | Permutations, combinations, inclusion-exclusion, derangements |
| 02 | `02_probability_axioms.py` | Probability Axioms and Rules | Sample spaces, conditional probability, Bayes' theorem, independence |
| 03 | `03_random_variables.py` | Random Variables and Distributions | PMF/PDF/CDF simulation, quantile function, distribution comparison |
| 04 | `04_expectation_moments.py` | Expectation and Moments | E[X], Var(X), MGF, Chebyshev inequality, skewness/kurtosis |
| 05 | `05_joint_distributions.py` | Joint Distributions | Joint PMF table, marginals, covariance, iterated expectation |
| 06 | `06_discrete_distributions.py` | Discrete Distribution Families | Binomial, Poisson, Geometric, Negative Binomial, Hypergeometric |
| 07 | `07_continuous_distributions.py` | Continuous Distribution Families | Normal, Exponential, Gamma, Beta, Chi-squared, t, F |
| 08 | `08_transformations.py` | Transformations of Random Variables | CDF technique, Jacobian, Box-Muller, order statistics |
| 09 | `09_multivariate_normal.py` | Multivariate Normal Distribution | Cholesky generation, conditional distributions, Mahalanobis distance |
| 10 | `10_convergence.py` | Convergence Concepts | Convergence in probability, a.s., in distribution, Slutsky's theorem |
| 11 | `11_lln_clt.py` | LLN and CLT | LLN simulation, CLT histogram, normal approximation |
| 12 | `12_point_estimation.py` | Point Estimation | MoM, MLE, Fisher information, CRLB, bias analysis |
| 13 | `13_interval_estimation.py` | Interval Estimation | z/t-intervals, proportion CI, sample size, bootstrap CI |
| 14 | `14_hypothesis_testing.py` | Hypothesis Testing | z-test, t-test, chi-squared, power analysis, multiple testing |
| 15 | `15_bayesian_inference.py` | Bayesian Inference | Beta-Binomial, Normal-Normal, MAP, posterior predictive |
| 16 | `16_nonparametric.py` | Nonparametric Methods | Sign test, Wilcoxon, Mann-Whitney, permutation test, KDE |
| 17 | `17_regression_anova.py` | Regression and ANOVA | OLS regression, R², residual analysis, one-way ANOVA |
| 18 | `18_stochastic_processes.py` | Stochastic Processes | Markov chains, stationary distribution, Poisson process, random walk |

## Running Examples

All examples run with Python 3.9+ and require no external packages:

```bash
python 01_combinatorics.py
```
